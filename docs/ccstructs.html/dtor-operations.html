4.2 destructor interface: operations

Function: ccstructs_dtor_I ccname_new(ccstructs_dtor_I) (ccstructs_core_t const * S, ccname_iface_table_type(ccstructs_dtor_I) const * M)

Build and return a new destructor interface instance as implemented for the struct type of S.

Function: ccstructs_core_t const * ccstructs_dtor_self (ccstructs_dtor_I I)

Return a pointer to the subject struct to which the interface belongs.

Function: void ccstructs_dtor_delete (ccstructs_dtor_I I)

Apply the final() method to the subject struct, then apply the release() method to the subject struct.
