6 The serialiser interface

The serialiser interface, represented by the type ccstructs_serialiser_I, allows us to serialise a struct into a destination storage. The use of this interface is suggested to serialise “small” data structures that can be written to storage with a single operation (that is: it may be not good if the serialisation is an XML document gigabytes in size).

For implementation examples we should look in the source distribution of CCStructs.
