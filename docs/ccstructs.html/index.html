CCStructs

This document describes version 0.2.0-devel.4 of CCStructs, a C11 language library implementing common programming interfaces for C data structures.

This package supports POSIX platforms. This package depends on the external packages: CCExceptions, at least version 0.8.0; CCMemory, at least version 0.2.1. To run the test suite: this package depends upon the external package CCTests, at least version 0.3.0.
