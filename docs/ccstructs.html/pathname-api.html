Function: ccstructs_pathname_I ccname_new(ccstructs_pathname_I) (ccstructs_core_t const * S, ccname_iface_table_type(ccstructs_pathname_I) const * M)

Interface constructor. Return a new interface struct given a pointer to the source struct and a pointer to the interface methods.

Function: ccstructs_core_t const * ccstructs_pathname_self (ccstructs_pathname_I I)

Return a pointer to the struct to which the interface belongs.

Function: ccmem_asciiz_t ccstructs_pathname_asciiz (cce_destination_t L, ccstructs_pathname_I I)

Apply the method asciiz() to the interface instance. Return an instance of ccmem_asciiz_t holding an ASCIIZ representation of the pathname. If an error occurs generating the representation: raise an exception by performing a non–local exit to L.
