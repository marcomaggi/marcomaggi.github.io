6.2 serialiser interface: operations

Function: ccstructs_serialiser_I ccname_new(ccstructs_serialiser_I) (ccstructs_core_t const * S, ccname_iface_table_type(ccstructs_serialiser_I) const * M)

Interface constructor. Return a new interface struct given a pointer to the source struct and a pointer to the interface methods.

Function: ccstructs_core_t const * ccstructs_serialiser_self (ccstructs_serialiser_I I)

Return a pointer to the struct to which the interface belongs.

Function: size_t ccstructs_serialiser_required_size (ccstructs_serialiser_I I)

Apply the method required_size() to the given argument.

Function: ccmem_block_t ccstructs_serialiser_write (cce_destination_t L, ccstructs_serialiser_I I, ccmem_block_t B)

Apply the method write() to the given arguments. If an error occurs: raise an exception by performing a non–local exit to L.
