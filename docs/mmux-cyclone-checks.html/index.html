Checks for Cyclone

This document describes version 0.1.0-devel.1 of MMUX Cyclone Checks, a distribution of libraries for Cyclone Scheme, a Scheme–to–C compiler supporting the language features as defined in the “Revised^7 Report on Scheme”; the library implements features to organise test suites and it is derived from SRFI-78 “Lightweight testing”.

The package supports POSIX platforms. This package should work with Cyclone version 0.17.

The latest release can be downloaded from:

https://bitbucket.org/marcomaggi/mmux-cyclone-checks/downloads
