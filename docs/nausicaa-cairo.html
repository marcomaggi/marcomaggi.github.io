1 Overview of the package

Nausicaa/Cairo is a binding between the Cairo 2D graphics library and the R6RS Scheme implementations supported by Nausicaa. It is built upon the Nausicaa/Scheme package, making use of its foreign functions interface; as such, it currently supports only GNU+Linux systems.

The installed libraries are:

(foreign graphics cairo sizeof)
Exports bindings for constant values and data structure accessors and mutators.
(foreign graphics cairo shared-library)
Exports the single binding cairo-shared-library, which references the Cairo shared library.
(foreign graphics cairo platform)
Implements a one–to–one map between Cairo's foreign functions and Scheme functions. All the Cairo functions are available as Scheme functions with the same name; for example cairo_create() is exported as cairo_create.
(foreign graphics cairo primitives)
Implements a middle level interface with marshaling of function arguments and return values. All the Cairo functions are available as Scheme functions with the same name but underscores turned into dashes; for example cairo_create() is exported as cairo-create.
(foreign graphics cairo)
Exports the high level API. Re–exports everything from (foreign graphics cairo primitive) and (foreign graphics cairo sizeof). Additionally, when possible, it exports an interface for constructors and destructors using the (compensations) library. Compensation stacks

The original Cairo documentation can be used as guide for the (foreign graphics cairo platform) library. A brief Cairo tutorial is available here (last verified Thu Oct 22, 2009):

http://cairographics.org/tutorial/

another long tutorial is available here (last verified Thu Oct 22, 2009):

http://zetcode.com/tutorials/cairographicstutorial/

the home page of Cairo has additional documentation (last verified Thu Oct 22, 2009):

5.1 Marshaling C strings

When the type is a pointer to a C string, the string is interpreted as zero–terminated and UTF8–encoded; it is converted from/to a Scheme string using string->cstring/c and cstring->string from the (foreign cstrings) library. C language strings handling

For function arguments: Memory is allocated on the spot and released after the foreign function has returned. For return values: It is assumed that the string is owned by the Cairo library, so it is converted but the memory block itself is left alone.

As a special case, the marshaling function cairo-scaled-font-text-to-glyphs collapses the two arguments of cairo_scaled_font_text_to_glyphs() representing the pointer to the string and the string length, into a single Scheme string argument. So the C language call: 

     cairo_scaled_font_text_to_glyphs(scaled_font, x, y,
                                      utf8, utf8_len,
                                      glyphs, num_glyphs,
                                      clusters, num_clusters,
                                      cluster_flags);

becomes the Scheme call: 

     (cairo-scaled-font-text-to-glyphs scaled_font x y
                                       text
                                       glyphs num_glyphs
                                       clusters num_clusters
                                       cluster_flags)

5.2 Miscellaneous functions

— Function: cairo-get-current-point cr

Given a pointer to a Cairo context, return two values representing the coordinates of the current point in the current path: the first is the X coordinate, the second is the Y coordinate.

— Function: cairo-set-dash cr dashes
— Function: cairo-set-dash cr dashes offset

Given a pointer to a Cairo context, configure the dash pattern for drawing lines. dashes must be a Scheme vector of positive inexact rational numbers (according to rational?), whose sum must be different from zero. offset must be an inexact rational number; when not given, it defaults to zero.

— Function: cairo-get-dash cr

Given a pointer to a Cairo context, return two values representing the current line dash configuration: the first is a vector of rational numbers (according to rational?) representing the dashes, the second is a number representing the offset.

— Function: cairo-surface-get-device-offset surface

Given a pointer to a surface, return two values representing the previous device offset set by cairo-surface-set-device-offset: the first is the offset in the X direction in device units, the second is the offset in the Y direction in device units.

— Function: cairo-surface-get-fallback-resolution surface

Given a pointer to a surface, return two values representing the previous fallback resolution set by cairo-surface-set-fallback-resolution: the first is the number of horizontal pixels per inch, the second is the number of vertical pixels per inch.

— Function: cairo-pattern-get-rgba pattern

Given a pointer to a pattern structure, return four values describing the color of a solid color pattern in the order: red, green, blue, alpha component. Raise an assertion violation exception if the pattern is not a solid color.

— Function: cairo-pattern-get-color-stop-rgba pattern index

Given a pointer to a pattern structure and a color stop index, return five values describing the color stop in this order: offset, red, green, blue, alpha component. Raise an assertion violation exception if index is invalid or the pattern is not a gradient pattern.

— Function: cairo-pattern-get-color-stop-count pattern

Given a pointer to a pattern structure return the number of color stops in it. Raise an assertion violation exception if the pattern is not a gradient pattern.

7.1 Writing text

The following example draws a 390 x 60 image with text in it and saves it to a file in PNG format. Notice that the functions required to save a PNG file are available only if ‘CAIRO_HAS_PNG_FUNCTIONS’ is bound to #t. 

     (import (rnrs)
       (foreign cairo)
       (foreign cstrings)
       (compensations))
     
     (with-compensations
       (letrec* ((surface (cairo-image-surface-create/c
                            CAIRO_FORMAT_ARGB32 390 60))
                 (cr      (cairo-create/c surface)))
     
         ;; draw in black ink
         (cairo-set-source-rgb cr 0. 0. 0.)
     
         ;; select the font
         (cairo-select-font-face cr "Sans"
                                 CAIRO_FONT_SLANT_NORMAL
                                 CAIRO_FONT_WEIGHT_NORMAL)
         (cairo_set_font_size cr 40.0)
     
         ;; move to the beginning of the text
         (cairo_move_to cr 10.0 50.0)
     
         ;; draw text
         (cairo_show_text cr "Disziplin ist Macht.")
     
         ;; draw the image on the output
         (cairo-surface-write-to-png surface "image.png")))

The following example does the same but outputs a PDF file. Notice that the functions required to create a PDF surface are available only if ‘CAIRO_HAS_PDF_SURFACE’ is bound to #t. 

     (import (rnrs)
       (foreign cairo)
       (foreign cstrings)

         (cairo-move-to cr 10.0 50.0)
         (cairo-show-text cr "Disziplin ist Macht.")
         (cairo-show-page cr)))

7.2 Drawing lines

The following example draws a 300 x 300 image with a red square in it and saves it to a file in PNG format. Notice that the functions required to save a PNG file are available only if ‘CAIRO_HAS_PNG_FUNCTIONS’ is bound to #t. 

     (import (rnrs)
       (foreign cairo)
       (compensations))
     
     (with-compensations
       (letrec* ((surface (cairo-image-surface-create/c
                            CAIRO_FORMAT_ARGB32 300 300))
                 (cr      (cairo-create/c surface)))
     
         (cairo-set-source-rgb cr 100. 0. 0.)
         (cairo-set-line-width cr 1.)
     
         (cairo-move-to cr 10. 10.)
         (cairo-line-to cr 290. 10.)
         (cairo-line-to cr 290. 290.)
         (cairo-line-to cr 10. 290.)
         (cairo-line-to cr 10. 10.)
         (cairo-stroke cr)
     
         (cairo-surface-write-to-png surface "test-line.png")))

To fill the square we just change: 

     (cairo-stroke cr)

to: 

     (cairo-fill cr)

To draw a circle we do: 

     (define pi  (acos -1))
     (define pi2 (* 2 pi))
     
     (cairo-arc cr 150. 150. 90. 0. pi2)
     (cairo-stroke cr)
