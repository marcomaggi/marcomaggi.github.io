4.6 Comparison functions

Generic Function: mpq-cmp (OP1 OP2)
Method: mpq-cmp ((OP1 mpq) (OP2 mpq))

Compare OP1 and OP2. 

(mpq-cmp (mpq 1 2) (mpq 3 2))   ⇒ -1
(mpq-cmp (mpq 1 2) (mpq 1 2))   ⇒  0
(mpq-cmp (mpq 3 2) (mpq 1 2))   ⇒ +1
Generic Function: mpq-cmp-z (OP1 OP2)
Method: mpq-cmp-z ((OP1 mpq) (OP2 mpz))

Compare OP1 and OP2. 

(mpq-cmp-z (mpq 1 2) (mpz 1))   ⇒ -1
(mpq-cmp-z (mpq 1 1) (mpz 1))   ⇒  0
(mpq-cmp-z (mpq 3 2) (mpz 1))   ⇒ +1
Generic Function: mpq-cmp-ui (OP1 num2 den2)
Method: mpq-cmp-ui ((OP1 mpq) (NUM2 integer) (DEN2 integer))

Compare OP1 and NUM2/DEN2. 

(mpq-cmp-ui (mpq 1 2) 3 2)      ⇒ -1
(mpq-cmp-ui (mpq 1 2) 1 2)      ⇒  0
(mpq-cmp-ui (mpq 3 2) 1 2)      ⇒ +1
Generic Function: mpq-cmp-si (OP1 num2 den2)
Method: mpq-cmp-si ((OP1 mpq) (NUM2 integer) (DEN2 integer))

Compare OP1 and NUM2/DEN2. 

(mpq-cmp-si (mpq -1 2) -3 2)    ⇒ +1
(mpq-cmp-si (mpq -1 2) -1 2)    ⇒  0
(mpq-cmp-si (mpq -3 2) -1 2)    ⇒ -1
Generic Function: mpq-sgn (OP)
Method: mpq-sgn ((OP mpq))

Return +1 if OP > 0, 0 if OP = 0, and -1 if OP < 0. 

(mpq-sgn (mpq +1 2))            ⇒ +1
(mpq-sgn (mpq  0 2))            ⇒  0
(mpq-sgn (mpq -1 2))            ⇒ -1
Generic Function: mpq-equal (OP1 OP2)
Method: mpq-equal ((OP1 mpq) (OP2 mpq))

Return non-zero if OP1 and OP2 are equal, zero if they are non-equal. 

(mpq-equal (mpq 1 2) (mpq 1 2)) ⇒ t
(mpq-equal (mpq 1 2) (mpq 3 2)) ⇒ nil
Generic Function: mpq-zero-p (OP)
Method: mpq-zero-p ((OP mpq))

Return true if OP is zero; otherwise return false. 

(mpq-zero-p (mpq -1 2))         ⇒ nil
(mpq-zero-p (mpq  0 2))         ⇒ t
(mpq-zero-p (mpq +1 2))         ⇒ nil
Generic Function: mpq-non-zero-p (OP)
Method: mpq-non-zero-p ((OP mpq))

Return true if OP is non-zero; otherwise return false. 

(mpq-non-zero-p (mpq -1 2))     ⇒ t
(mpq-non-zero-p (mpq  0 2))     ⇒ nil
(mpq-non-zero-p (mpq +1 2))     ⇒ t
Generic Function: mpq-positive-p (OP)
Method: mpq-positive-p ((OP mpq))

(mpq-positive-p (mpq -1 2))     ⇒ nil

(mpq-positive-p (mpq +1 2))     ⇒ t
Generic Function: mpq-negative-p (OP)
Method: mpq-negative-p ((OP mpq))

(mpq-non-positive-p (mpq  0 2)) ⇒ t
(mpq-non-positive-p (mpq +1 2)) ⇒ nil
Generic Function: mpq-non-negative-p (OP)

Generic Function: mpq< (OP &rest OPS)
Method: mpq< ((OP1 mpq) (OP2 mpq))

Return true if each argument is strictly less than the following argument; otherwise return false. 

(mpq< (mpq 1 2) (mpq 2 2))      ⇒ t
(mpq< (mpq 1 2) (mpq 1 2))      ⇒ nil
(mpq< (mpq 2 2) (mpq 1 2))      ⇒ nil
Generic Function: mpq> (OP &rest OPS)
Method: mpq> ((OP1 mpq) (OP2 mpq))

Return true if each argument is strictly greater than the following argument; otherwise return false. 

(mpq> (mpq 1 2) (mpq 2 2))      ⇒ nil
(mpq> (mpq 1 2) (mpq 1 2))      ⇒ nil
(mpq> (mpq 2 2) (mpq 1 2))      ⇒ t
Generic Function: mpq<= (OP &rest OPS)
Method: mpq<= ((OP1 mpq) (OP2 mpq))

Return true if each argument is strictly less than, or equal to, the following argument; otherwise return false. 

(mpq<= (mpq 1 2) (mpq 2 2))     ⇒ t
(mpq<= (mpq 1 2) (mpq 1 2))     ⇒ t
(mpq<= (mpq 2 2) (mpq 1 2))     ⇒ nil
Generic Function: mpq>= (OP &rest OPS)
Method: mpq>= ((OP1 mpq) (OP2 mpq))

Return true if each argument is greater than, or equal to, the following argument; otherwise return false. 

(mpq>= (mpq 1 2) (mpq 2 2))     ⇒ nil
(mpq>= (mpq 1 2) (mpq 1 2))     ⇒ t
(mpq>= (mpq 2 2) (mpq 1 2))     ⇒ t
Generic Function: mpq= (OP &rest OPS)
Method: mpq= ((OP1 mpq) (OP2 mpq))

Return true if each argument is equal to the following argument; otherwise return false. 

(mpq= (mpq 1 2) (mpq 2 2))      ⇒ nil
(mpq= (mpq 1 2) (mpq 1 2))      ⇒ t
(mpq= (mpq 2 2) (mpq 1 2))      ⇒ nil
