3.11 Logical and bit manipulation functions

Generic Function: mpz-and (ROP OP1 OP2)
Method: mpz-and ((ROP mpz) (OP1 mpz) (OP2 mpz))

Set ROP to OP1 bitwise–and OP2. 

(let ((rop    (mpz))
      (op1    (mpz #b1010101))
      (op2    (mpz #b1101011)))
  (mpz-and rop op1 op2)
  (mpz-get-si rop))
⇒ #b1000001
Generic Function: mpz-ior (ROP OP1 OP2)
Method: mpz-ior ((ROP mpz) (OP1 mpz) (OP2 mpz))

Set ROP to OP1 bitwise inclusive–or OP2. 

(let ((rop    (mpz))
      (op1    (mpz #b1010101))
      (op2    (mpz #b1110101)))
  (mpz-ior rop op1 op2)
  (mpz-get-si rop))
⇒ #b1110101
Generic Function: mpz-xor (ROP OP1 OP2)
Method: mpz-xor ((ROP mpz) (OP1 mpz) (OP2 mpz))

Set ROP to OP1 bitwise exclusive–or OP2. 

(let ((rop    (mpz))
      (op1    (mpz #b1110101))
      (op2    (mpz #b1011101)))
  (mpz-xor rop op1 op2)
  (mpz-get-si rop))
⇒ #b0101000
Generic Function: mpz-com (ROP OP)
Method: mpz-com ((ROP mpz) (OP mpz))

Set ROP to the one’s complement of OP. 

(let ((rop    (mpz))
      (op     (mpz #b10)))
  (mpz-com rop op)
  (mpz-get-si rop))
⇒ -3
Generic Function: mpz-popcount (op)
Method: mpz-popcount ((OP mpz))

Return the population count of OP. 

(mpz-popcount (mpz #b1010101))
⇒ 4
Generic Function: mpz-hamdist (OP1 OP2)
Method: mpz-hamdist ((OP1 mpz) (OP2 mpz))

Return the hamming distance between the two operands. 

(let ((op1    (mpz 63))
      (op2    (mpz 70)))
  (mpz-hamdist op1 op2))
⇒ 5
Generic Function: mpz-scan0 (op STARTING-BIT)
Method: mpz-scan0 ((OP mpz) (STARTING-BIT integer))

Scan OP for the first 0 bit. 

(let ((op             (mpz #b1000110))
      (starting-bit   1))
  (mpz-scan0 op starting-bit))
⇒ 3

(let ((op             (mpz #b1011111))
      (starting-bit   2))
  (mpz-scan0 op starting-bit))
⇒ 5

Generic Function: mpz-setbit (ROP BIT-INDEX)
Method: mpz-setbit ((ROP mpz) (BIT-INDEX integer))

Set bit BIT_INDEX in ROP. 

(let ((rop            (mpz))
      (bit-index      3))
  (mpz-setbit rop bit-index)
  (mpz-get-si rop))
⇒ 8

⇒ 0
Generic Function: mpz-combit (ROP BIT-INDEX)
Method: mpz-combit ((ROP mpz) (BIT-INDEX integer))

(let ((rop            (mpz #b00100))

Generic Function: mpz-tstbit (op BIT-INDEX)
Method: mpz-tstbit ((OP mpz) (BIT-INDEX integer))

Test bit BIT_INDEX in OP and return 0 or 1 accordingly. 

(let ((op             (mpz #b01000))
      (bit-index      3))
  (mpz-tstbit op bit-index))
⇒ t

(let ((op             (mpz #b10111))
      (bit-index      3))
  (mpz-tstbit op bit-index))
⇒ nil
