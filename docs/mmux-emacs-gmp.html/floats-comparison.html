5.7 Comparison functions

Generic Function: mpf-cmp (OP1 OP2)
Method: mpf-cmp ((OP1 mpf) (OP2 mpf))

Compare OP1 and OP2. 

(mpf-cmp (mpf 1.2) (mpf 3.4))           ⇒ -1
(mpf-cmp (mpf 1.2) (mpf 1.2))           ⇒  0
(mpf-cmp (mpf 3.4) (mpf 1.2)))          ⇒ +1
Generic Function: mpf-cmp-z (OP1 OP2)
Method: mpf-cmp-z ((OP1 mpf) (OP2 mpz))

Compare OP1 and OP2. 

(mpf-cmp-z (mpf 1.2) (mpz 2))           ⇒ -1
(mpf-cmp-z (mpf 1.0) (mpz 1))           ⇒  0
(mpf-cmp-z (mpf 1.2) (mpz 1))           ⇒ +1
Generic Function: mpf-cmp-d (OP1 OP2)
Method: mpf-cmp-d ((OP1 mpf) (OP2 float))

Compare OP1 and OP2. 

(mpf-cmp-d (mpf 1.2) 3.4)               ⇒ -1
(mpf-cmp-d (mpf 1.2) 1.2)               ⇒  0
(mpf-cmp-d (mpf 3.4) 1.2)               ⇒ +1
Generic Function: mpf-cmp-ui (op1 op2)
Method: mpf-cmp-ui ((OP1 mpf) (OP2 integer))

Compare OP1 and OP2. 

(mpf-cmp-ui (mpf 1.2) 2)                ⇒ -1
(mpf-cmp-ui (mpf 1.0) 1)                ⇒  0
(mpf-cmp-ui (mpf 1.2) 1)                ⇒ +1
Generic Function: mpf-cmp-si (OP1 OP2)
Method: mpf-cmp-si ((OP1 mpf) (OP2 integer))

Compare OP1 and OP2. 

(mpf-cmp-si (mpf -2.1) -2)              ⇒ -1
(mpf-cmp-si (mpf -1.0) -1)              ⇒  0
(mpf-cmp-si (mpf -1.0) -2)              ⇒ +1
Generic Function: mpf-sgn (OP)
Method: mpf-sgn ((OP mpf))

Return +1 if OP > 0, 0 if OP = 0, and -1 if OP < 0. 

(mpf-sgn (mpf -1.0))            ⇒ -1
(mpf-sgn (mpf  0.0))            ⇒  0
(mpf-sgn (mpf +1.0))            ⇒ +1
Generic Function: mpf-reldiff (ROP OP1 OP2)
Method: mpf-reldiff ((ROP mpf) (OP1 mpf) (OP2 mpf))

Compute the relative difference between OP1 and OP2 and store the result in ROP. This is abs(OP1-OP2)/OP1. 

(let ((rop    (mpf))
      (op1    (mpf 1.2))
      (op2    (mpf 3.4)))
  (mpf-reldiff rop op1 op2)
  (let ((X (/ (abs (- 1.2 3.4)) 1.2)))
    (< (- X 0.00001)
       (mpf-get-d rop)
       (+ X 0.00001))))
⇒ t
Generic Function: mpf-equal (OP1 OP2)
Method: mpf-equal ((OP1 mpf) (OP2 mpf))

Return true the operands are equal; otherwise return false. 

(mpf-equal (mpf 1.2) (mpf 1.2)))                ⇒ t
(mpf-equal (mpf 1.2) (mpf 3.4)))))              ⇒ nil
Generic Function: mpf-zero-p (OP)
Method: mpf-zero-p ((OP mpf))

Return true if OP is zero; otherwise return false. 

(mpf-zero-p (mpf 0.0)))         ⇒ t
(mpf-zero-p (mpf 1.0)))))       ⇒ nil
Generic Function: mpf-non-zero-p (OP)
Method: mpf-non-zero-p ((OP mpf))

Return true if OP is non-zero; otherwise return false. 

(mpf-non-zero-p (mpf 1.0)))     ⇒ t
(mpf-non-zero-p (mpf 0.0)))     ⇒ nil
Generic Function: mpf-positive-p (OP)
Method: mpf-positive-p ((OP mpf))

Return true if OP is strictly positive; otherwise return false. 

(mpf-positive-p (mpf +1.0)))    ⇒ t
(mpf-positive-p (mpf  0.0)))    ⇒ nil
(mpf-positive-p (mpf -1.0)))    ⇒ nil
Generic Function: mpf-negative-p (OP)
Method: mpf-negative-p ((OP mpf))

Return true if OP is strictly negative; otherwise return false. 

(mpf-negative-p (mpf +1.0))     ⇒ nil
(mpf-negative-p (mpf  0.0))     ⇒ nil
(mpf-negative-p (mpf -1.0))     ⇒ t
Generic Function: mpf-non-positive-p (OP)
Method: mpf-non-positive-p ((OP mpf))

Return true if OP is non-positive; otherwise return false. 

(mpf-non-positive-p (mpf +1.0)) ⇒ nil
(mpf-non-positive-p (mpf  0.0)) ⇒ t
(mpf-non-positive-p (mpf -1.0)) ⇒ t
Generic Function: mpf-non-negative-p (OP)
Method: mpf-non-negative-p ((OP mpf))

Return true if OP is non-negative; otherwise return false. 

(mpf-non-negative-p (mpf +1.0)) ⇒ t
(mpf-non-negative-p (mpf  0.0)) ⇒ t
(mpf-non-negative-p (mpf -1.0)) ⇒ nil
Generic Function: mpf< (OP &rest OPS)
Method: mpf< ((OP1 mpf) (OP2 mpf))

Return true if each argument is strictly less than the following argument; otherwise return false. 

(mpf< (mpf 1.0) (mpf 2.0))      ⇒ t
(mpf< (mpf 1.0) (mpf 1.0))      ⇒ nil
(mpf< (mpf 2.0) (mpf 1.0))      ⇒ nil
Generic Function: mpf> (OP &rest OPS)
Method: mpf> ((OP1 mpf) (OP2 mpf))

Return true if each argument is strictly greater than the following argument; otherwise return false. 

(mpf> (mpf 1.0) (mpf 2.0))      ⇒ nil
(mpf> (mpf 1.0) (mpf 1.0))      ⇒ nil
(mpf> (mpf 2.0) (mpf 1.0))      ⇒ t
Generic Function: mpf<= (OP &rest OPS)
Method: mpf<= ((OP1 mpf) (OP2 mpf))

Return true if each argument is strictly less than, or equal to, the following argument; otherwise return false. 

(mpf<= (mpf 1.0) (mpf 2.0)))    ⇒ t
(mpf<= (mpf 1.0) (mpf 1.0)))    ⇒ t
(mpf<= (mpf 2.0) (mpf 1.0)))    ⇒ nil
Generic Function: mpf>= (OP &rest OPS)
Method: mpf>= ((OP1 mpf) (OP2 mpf))

Return true if each argument is greater than, or equal to, the following argument; otherwise return false. 

(mpf>= (mpf 1.0) (mpf 2.0))     ⇒ nil
(mpf>= (mpf 1.0) (mpf 1.0))     ⇒ t
(mpf>= (mpf 2.0) (mpf 1.0))     ⇒ t
Generic Function: mpf= (OP &rest OPS)
Method: mpf= ((OP1 mpf) (OP2 mpf))

Return true if each argument is equal to the following argument; otherwise return false. 

(mpf= (mpf 1.0) (mpf 2.0))      ⇒ nil
(mpf= (mpf 1.0) (mpf 1.0))      ⇒ t
(mpf= (mpf 2.0) (mpf 1.0))      ⇒ nil
