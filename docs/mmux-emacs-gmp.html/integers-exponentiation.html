3.7 Exponentiation operations

Generic Function: mpz-powm (ROP BASE EXP MOD)
Method: mpz-powm ((ROP mpz) (BASE mpz) (EXP mpz) (MOD mpz))

Set ROP to (BASE raised to EXP) modulo MOD. 

(let ((rop    (mpz))
      (base   (mpz 5))
      (exp    (mpz 3))
      (mod    (mpz 2)))
  (mpz-powm rop base exp mod)
  (equal (mod (expt 5 3) 2)
         (mpz-get-ui rop)))
⇒ t
