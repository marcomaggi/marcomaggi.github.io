3.9 Commparison functions

Generic Function: mpz-cmp (OP1 OP2)
Method: mpz-cmp ((OP1 mpz) (OP2 mpz))

Compare OP1 and OP2. 

(mpz-cmp (mpz 1) (mpz 2))       ⇒ -1
(mpz-cmp (mpz 0) (mpz 0))       ⇒ 0
(mpz-cmp (mpz 2) (mpz 1))       ⇒ +1
Generic Function: mpz-cmp-d (OP1 OP2)
Method: mpz-cmp-d ((OP1 mpz) (OP2 float))

Compare OP1 and OP2. 

(mpz-cmp-d (mpz 1) 2.0))        ⇒ -1
(mpz-cmp-d (mpz 0) 0.0))        ⇒ 0
(mpz-cmp-d (mpz 2) 1.0))        ⇒ +1
Generic Function: mpz-cmp-si (OP1 OP2)
Method: mpz-cmp-si ((OP1 mpz) (OP2 integer))

Compare OP1 and OP2. 

(mpz-cmp-si (mpz -2) -1))       ⇒ -1
(mpz-cmp-si (mpz  0)  0))       ⇒ 0
(mpz-cmp-si (mpz -1) -2))       ⇒ +1
Generic Function: mpz-cmp-ui (OP1 OP2)
Method: mpz-cmp-ui ((OP1 mpz) (OP2 integer))

Compare OP1 and OP2. 

(mpz-cmp-ui (mpz 1) 2))         ⇒ -1
(mpz-cmp-ui (mpz 0) 0))         ⇒ 0
(mpz-cmp-ui (mpz 2) 1))         ⇒ +1
Generic Function: mpz-cmpabs (OP1 OP2)
Method: mpz-cmpabs ((OP1 mpz) (OP2 mpz))

Compare the absolute values of OP1 and OP2. 

(mpz-cmpabs (mpz 1) (mpz -2))   ⇒ -1
(mpz-cmpabs (mpz 0) (mpz  0))   ⇒ 0
(mpz-cmpabs (mpz 2) (mpz  1))   ⇒ +1
Generic Function: mpz-cmpabs-d (OP1 OP2)
Method: mpz-cmpabs-d ((OP1 mpz) (OP2 float))

Compare the absolute values of OP1 and OP2. 

(mpz-cmpabs-d (mpz 1) -2.0)     ⇒ -1
(mpz-cmpabs-d (mpz 0)  0.0)     ⇒ 0
(mpz-cmpabs-d (mpz 2)  1.0)     ⇒ +1
Generic Function: mpz-cmpabs-ui (OP1 OP2)
Method: mpz-cmpabs-ui ((OP1 mpz) (OP2 integer))

Compare the absolute values of OP1 and OP2. 

(mpz-cmpabs-ui (mpz  1) 2)      ⇒ -1
(mpz-cmpabs-ui (mpz  0) 0)      ⇒ 0
(mpz-cmpabs-ui (mpz -2) 1)      ⇒ +1
Generic Function: mpz-sgn (OP)
Method: mpz-sgn ((OP mpz))

Return an integer representing the sign of the operand. 

(mpz-sgn (mpz +1))      ⇒ +1
(mpz-sgn (mpz  0))      ⇒ 0
(mpz-sgn (mpz -1))      ⇒ -1

Generic Function: mpz< (OP &rest OPS)
Method: mpz< ((OP1 mpz) (OP2 mpz))

Return true if each argument is strictly less than the following argument; otherwise return false. 

(mpz< (mpz 1) (mpz 2))          ⇒ t
(mpz< (mpz 1) (mpz 1))          ⇒ nil
(mpz< (mpz 2) (mpz 1))          ⇒ nil
Generic Function: mpz> (OP &rest OPS)
Method: mpz> ((OP1 mpz) (OP2 mpz))

Return true if each argument is strictly greater than the following argument; otherwise return false. 

(mpz> (mpz 1) (mpz 2))          ⇒ nil
(mpz> (mpz 1) (mpz 1))          ⇒ nil
(mpz> (mpz 2) (mpz 1))          ⇒ t
Generic Function: mpz<= (OP &rest OPS)
Method: mpz<= ((OP1 mpz) (OP2 mpz))

Return true if each argument is strictly less than, or equal to, the following argument; otherwise return false. 

(mpz<= (mpz 1) (mpz 2))         ⇒ t
(mpz<= (mpz 1) (mpz 1))         ⇒ t
(mpz<= (mpz 2) (mpz 1))         ⇒ nil
Generic Function: mpz>= (OP &rest OPS)
Method: mpz>= ((OP1 mpz) (OP2 mpz))

Return true if each argument is greater than, or equal to, the following argument; otherwise return false. 

(mpz>= (mpz 1) (mpz 2))         ⇒ nil
(mpz>= (mpz 1) (mpz 1))         ⇒ t
(mpz>= (mpz 2) (mpz 1))         ⇒ t
Generic Function: mpz= (OP &rest OPS)
Method: mpz= ((OP1 mpz) (OP2 mpz))

Return true if each argument is equal to the following argument; otherwise return false. 

(mpz= (mpz 1) (mpz 2))          ⇒ nil
(mpz= (mpz 1) (mpz 1))          ⇒ t
(mpz= (mpz 2) (mpz 1))          ⇒ nil
