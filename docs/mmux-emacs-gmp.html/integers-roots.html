3.8 Root extraction functions

Generic Function: mpz-root (ROP OP N)
Method: mpz-root ((ROP mpz) (OP mpz) (N integer))

Set ROP to the truncated integer part of the Nth root of OP. 

(let ((rop    (mpz))
      (op     (mpz 8))
      (N      3))
  (mpz-root rop op N)
  (mpz-get-si rop))
⇒ 2
Generic Function: mpz-rootrem (root rem U N)
Method: mpz-rootrem ((root mpz) (rem mpz) (U mpz) (N integer))

Set ROOT to the truncated integer part of the Nth root of U. Set REM to the remainder, U-ROOT^N. 

;; 9 = 2^3 + 1
(let ((rop    (mpz))
      (rem    (mpz))
      (op     (mpz 9))
      (N      3))
  (mpz-rootrem rop rem op N)
  (mpz-get-si rop)      ⇒ 2
  (mpz-get-si rem))     ⇒ 1
Generic Function: mpz-sqrt (ROP OP)
Method: mpz-sqrt ((ROP mpz) (OP mpz))

Set ROP to the truncated integer part of the square root of OP. 

(let ((rop    (mpz))
      (op     (mpz 16)))
  (mpz-sqrt rop op)
  (mpz-get-si rop))
⇒ 4
Generic Function: mpz-sqrtrem (rop1 rop2 OP)
Method: mpz-sqrtrem ((rop1 mpz) (ROP mpz) (OP mpz))

Set ROP1 to the truncated integer part of the square root of OP. Set ROP2 to the remainder OP-ROP1*ROP1. 

;; 17 = 4^2 + 1
(let ((rop    (mpz))
      (rem    (mpz))
      (op     (mpz 17)))
  (mpz-sqrtrem rop rem op)
  (mpz-get-si rop)      ⇒ 4
  (mpz-get-si rem))     ⇒ 1
Generic Function: mpz-perfect-power-p (OP)
Method: mpz-perfect-power-p ((OP mpz))

Return true if OP is a perfect power; otherwise return false. 

(mpz-perfect-power-p (mpz 8))           ⇒ t
(mpz-perfect-power-p (mpz 7))           ⇒ nil
Generic Function: mpz-perfect-square-p (OP)
Method: mpz-perfect-square-p ((OP mpz))

Return true if OP is a perfect square; otherwise return false. 

(mpz-perfect-square-p (mpz 16))         ⇒ t
(mpz-perfect-square-p (mpz 7))          ⇒ nil
