5.2 Floating–point numbers initialisation functions

Generic Function: mpf-set-default-prec (prec)
Method: mpf-set-default-prec ((prec integer))

Set the default precision of mpf object to at least prec, which must be an exact integer number object.

Generic Function: mpf-get-default-prec
Method: mpf-get-default-prec

Return an exact integer number representing the current default precision of (mpf objects.)

Generic Function: mpf-set-prec (ROP prec)
Method: mpf-set-prec ((ROP mpf) (prec integer))

Set the precision of the mpf object ROP to at least prec, which must be an exact integer number object.

Generic Function: mpf-get-prec (ROP)
Method: mpf-get-prec ((ROP integer))

Return an exact integer number representing the precision of the mpf object ROP.
