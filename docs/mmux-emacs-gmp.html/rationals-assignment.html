4.3 Assigning values to rational number objects

Generic Function: mpq-canonicalize (OP)
Method: mpq-canonicalize ((OP mpq))

Remove any factors that are common to the numerator and denominator of OP, and make the denominator positive.

Generic Function: mpq-set (ROP OP)
Method: mpq-set ((ROP mpq) (OP mpq))

Assign the value of OP to ROP; both the operands must be of type mpq. 

(let ((rop (mpq))
      (op  (mpq)))
  (mpq-set-si rop 3 4)
  (mpq-set    op rop)
  (mpq-get-str 10 op))
⇒ "3/4"
Generic Function: mpq-set-si (ROP numerator-integer denominator-integer)
Method: mpq-set-si ((ROP mpq) (numerator-integer integer) (denominator-integer integer))

Assign the value of the signed exact integer numbers numerator-integer and denominator-integer to the mpq object ROP. 

(let ((rop (mpq)))
  (mpq-set-si rop 3 4)
  (mpq-get-str 10 rop))
⇒ "3/4"
Generic Function: mpq-set-ui (ROP numerator-integer denominator-integer)
Method: mpq-set-ui ((ROP mpq) (numerator-integer integer) (denominator-integer integer))

Assign the value of the unsigned exact integer numbers numerator-integer and denominator-integer to the mpq object ROP. 

(let ((rop (mpq)))
  (mpq-set-ui rop 3 4)
  (mpq-get-str 10 rop))
⇒ "3/4"
Generic Function: mpq-set-d (ROP flo)
Method: mpq-set-d ((ROP mpq) (flo float))

Assign the value of a floating–point object to an mpq object. 

(let ((rop (mpq)))
  (mpq-set-d rop 1.2)
  (mpq-get-str 10 rop))
⇒ "5404319552844595/4503599627370496"
Generic Function: mpq-set-z (ROP OP)
Method: mpq-set-z ((ROP mpq) (OP mpz))

Assign the value of an mpz object to an mpq object. 

(let ((rop (mpq))
      (op  (mpz 123)))
  (mpq-set-z rop op)
  (mpq-get-str 10 rop))
⇒ "123"
Generic Function: mpq-set-f (ROP OP)
Method: mpq-set-f ((ROP mpq) (OP mpf))

Assign the value of an mpf object to an mpq object. 

(let ((rop (mpq))
      (op  (mpf 1.2)))
  (mpq-set-f rop op)
  (mpq-get-str 10 rop))
⇒ "5404319552844595/4503599627370496"
Generic Function: mpq-set-str (ROP STR BASE)
Method: mpq-set-str ((ROP mpq) (STR string) (BASE integer))

Assign the value of a string object to an mpq object. The argument BASE must be an exact integer representing the base of the numeric string representation: 0 or a value between 2 and 62. 

(let ((rop (mpq)))
  (mpq-set-str rop "3/4" 10)
  (mpq-get-str 10 rop))
Generic Function: mpq-swap (OP1 OP2)
Method: mpq-swap ((OP1 mpq) (OP2 mpq))

Swap the values between two mpq objects. 

(let ((op1 (mpq))
      (op2 (mpq)))
  (mpq-set-si op1 3 4)
  (mpq-set-si op2 5 6)
  (mpq-swap op1 op2)
  (mpq-get-str 10 op2)  ⇒ "3/4"
  (mpq-get-str 10 op1)  ⇒ "5/6"
  )
