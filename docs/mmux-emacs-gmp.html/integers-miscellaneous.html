3.13 Miscellaneous functions

Generic Function: mpz-fits-ulong-p (OP)
Method: mpz-fits-ulong-p ((OP mpz))

Return true if the operand fits an unsigned long int; otherwise return false.

Generic Function: mpz-fits-slong-p (OP)
Method: mpz-fits-slong-p ((OP mpz))

Return true if the operand fits a ‘igned long int; otherwise return false.

Generic Function: mpz-fits-uint-p (OP)
Method: mpz-fits-uint-p ((OP mpz))

Return true if the operand fits an unsigned int; otherwise return false.

Generic Function: mpz-fits-sint-p (OP)
Method: mpz-fits-sint-p ((OP mpz))

Return true if the operand fits n ‘igned int; otherwise return false.

Generic Function: mpz-fits-ushort-p (OP)
Method: mpz-fits-ushort-p ((OP mpz))

Return true if the operand fits an unsigned short int; otherwise return false.

Generic Function: mpz-fits-sshort-p (OP)
Method: mpz-fits-sshort-p ((OP mpz))

Return true if the operand fits a signed short int; otherwise return false.

Generic Function: mpz-odd-p (OP)
Method: mpz-odd-p ((OP mpz))

Return true if the operand is odd; otherwise return false.

Generic Function: mpz-even-p (OP)
Method: mpz-even-p ((OP mpz))

Return true if the operand is even; otherwise return false.

Generic Function: mpz-sizeinbase (OP BASE)
Method: mpz-sizeinbase ((OP mpz) (BASE integer))

Return the size of OP measured in number of digits in the given BASE.

The argument BASE can vary from 2 to 62.
