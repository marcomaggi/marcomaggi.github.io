3.3 Assigning values to objects

Generic Function: mpz-set (ROP OP)
Method: mpz-set ((ROP mpz) (OP mpz))

Assign the value of OP to ROP; both the operands must be of type mpz. 

(defconst rop (mpz))
(defconst op  (mpz))

(mpz-set-si rop 123)
(mpz-set    op rop)
(mpz-get-str 10 op)     ⇒ "123"
Generic Function: mpz-set-si (ROP INT)
Method: mpz-set-si ((ROP mpz) (INT integer))

Assign the value of an exact integer object to an mpz object. 

(defconst rop (mpz))

(mpz-set-si rop 123)
(mpz-get-str 10 op)     ⇒ "123"
Generic Function: mpz-set-d (ROP FLO)
Method: mpz-set-d ((ROP mpz) (FLO float))

Assign the value of a floating–point object to an mpz object. 

(defconst rop (mpz))

(mpz-set-d rop 12.3)
(mpz-get-str 10 rop)    ⇒ "12"
Generic Function: mpz-set-q (ROP OP)
Method: mpz-set-q ((ROP mpz) (OP mpq))

Assign the value of an mpq object to an mpz object. 

(defconst rop (mpz))
(defconst op  (mpq))

(mpq-set-si op 10 7)
(mpz-set-q rop op)
(mpz-get-str 10 rop)    ⇒ "1"
Generic Function: mpz-set-f (ROP OP)
Method: mpz-set-f ((ROP mpz) (OP mpf))

Assign the value of an mpf object to an mpz object. 

(defconst rop (mpz))
(defconst op  (mpf))

(mpf-set-d op 12.34)
(mpz-set-f rop op)
(mpz-get-str 10 rop)    ⇒ "12"
Generic Function: mpz-set-str (ROP STR BASE)
Method: mpz-set-str ((ROP mpz) (STR string) (BASE integer))

Assign the value of a string object to an mpz object. The argument BASE must be an exact integer representing the base of the numeric string representation: 0 or a value between 2 and 32. 

(defconst rop (mpz))

(mpz-set-str rop "123" 10)
(mpz-get-str 10 rop)    ⇒ "123"
Generic Function: mpz-swap (OP1 OP2)
Method: mpz-swap ((OP1 mpz) (OP2 mpz))

Swap the values between two mpz objects. 

(defconst op1 (mpz))
(defconst op2 (mpz))

(mpz-set-si op1 123)
(mpz-set-si op2 456)
(mpz-swap op1 op2)
(mpz-get-str 10 op1)  ⇒ "456"
(mpz-get-str 10 op2)  ⇒ "123"
