3.6.1 Rounding towards +infinity (ceil)

For the examples in this section, remember that the template operation is: 

N = Q * D + R

and with ceil rounding we have: 

N = 10
D = 3
Q = ceil(N/D) = ceil(10 / 3) = ceil(3.33333) = 4
R = N - Q * D = 10 - 4 * 3 = -2

for the 2exp functions: 

N = 10
B = 3
D = 2^B = 2^3 = 8
Q = ceil(N/D) = ceil(10 / 8) = ceil(1.25) = 2
R = N - Q * D = 10 - 2 * 8 = 10 - 16 = -6
Generic Function: mpz-cdiv-q (Q N D)
Method: mpz-cdiv-q ((Q mpz) (N mpz) (D mpz))

Divide N by D, forming a quotient Q and/or remainder R. 

(let ((Q      (mpz))
      (N      (mpz 10))
      (D      (mpz 3)))
  (mpz-cdiv-q Q N D)
  (mpz-get-si Q))
⇒ 4
