5.4 Assigning values to floating–point number objects

Generic Function: mpf-set (ROP OP)
Method: mpf-set ((ROP mpf) (OP mpf))

Assign the value of OP to ROP; both the operands must be of type mpf. 

(defconst rop (mpf))
(defconst op  (mpf))
(mpf-set-si rop 123)
(mpf-set    op rop)
(mpf-get-str* 10 8 op)  ⇒ "+0.123e+3"
Generic Function: mpf-set-si (ROP INT)
Method: mpf-set-si ((ROP mpf) (INT integer))

Assign the value of an exact integer object to an mpf object. 

(defconst rop (mpf))
(mpf-set-si rop 123)
(mpf-get-str* 10 8 rop) ⇒ "+0.123e+3"
Generic Function: mpf-set-ui (ROP INT)
Method: mpf-set-ui ((ROP mpf) (INT integer))

Assign the value of an exact integer object to an mpf object. 

(defconst rop (mpf))
(mpf-set-ui rop 123)
(mpf-get-str* 10 8 rop) ⇒ "+0.123e+3"
Generic Function: mpf-set-d (ROP flo)
Method: mpf-set-d ((ROP mpf) (FLO float))

Assign the value of a floating–point object to an mpf object. 

(let ((rop (mpf)))
  (mpf-set-d rop 12.3)
  (mpf-get-str* 10 8 rop))
⇒ "+0.123e+2"
Generic Function: mpf-set-z (ROP OP)
Method: mpf-set-z ((ROP mpf) (OP mpz))

Assign the value of an mpz object to an mpf object. 

(let ((rop    (mpf))
      (op     (mpz 123)))
  (mpf-set-z rop op)
  (mpf-get-str* 10 8 rop))
⇒ "+0.123e+3"
Generic Function: mpf-set-q (ROP OP)
Method: mpf-set-q ((ROP mpf) (OP mpq))

Assign the value of an mpq object to an mpf object. 

(let ((rop    (mpf))
      (op     (mpq 2 3)))
  (mpf-set-q rop op)
  (mpf-get-str* 10 8 rop))
⇒ "+0.66666667e0"
Generic Function: mpf-set-str (ROP STR BASE)
Method: mpf-set-str ((ROP mpf) (STR string) (BASE integer))

Assign the value of a string object to an mpf object. The argument BASE must be an exact integer representing the base of the numeric string representation: a positive value between 2 and 62 or a negative value between -2 and -62. 

(let ((rop (mpf)))
  (mpf-set-str rop "123" 10)
  (mpf-get-str* 10 8 rop))
⇒ "+0.123e+3"
Generic Function: mpf-swap (OP1 OP2)
Method: mpf-swap ((OP1 mpf) (OP2 mpf))

Swap the values between two mpf objects. 

(let ((op1 (mpf))
      (op2 (mpf)))
  (mpf-set-si op1 123)
  (mpf-set-si op2 456)
  (mpf-swap op1 op2)
  (mpf-get-str* 10 8 op2)       ⇒ "+0.123e+3"
  (mpf-get-str* 10 8 op1))      ⇒ "+0.456e+3"
