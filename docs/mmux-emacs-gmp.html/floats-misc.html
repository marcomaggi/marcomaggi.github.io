5.8 Miscellaneous functions

Generic Function: mpf-ceil (ROP OP)
Method: mpf-ceil ((ROP mpf) (OP mpf))

Set ROP to OP rounded to an integer. 

(let ((rop    (mpf))
      (op     (mpf 1.5)))
  (mpf-ceil rop op)
  (mpf-almost-equal-p rop 2.0 1e-5))
⇒ t

Generic Function: mpf-urandomb (ROP STATE NBITS)
Method: mpf-urandomb ((ROP mpf) (STATE gmp-randstate) (NBITS integer))

Generate a uniformly distributed random float in ROP. 

(let ((rop    (mpf))
      (state  (gmp-randstate))
      (nbits  3))
  (gmp-randinit-default state)
  (gmp-randseed-ui state 123)
  (mpf-urandomb rop state nbits)
  (mpf<= 0 rop)         ⇒ t
  (mpf<  rop 1.0))      ⇒ t
Generic Function: mpf-random2 (ROP MAX-SIZE EXP)
Method: mpf-random2 ((ROP mpf) (MAX-SIZE integer) (EXP integer))

Generate a random float of at most MAX-SIZE limbs, with long strings of zeros and ones in the binary representation. 

(let ((rop            (mpf))
      (max-size       1)
      (exp            3))
  (mpf-random2 rop max-size exp)
  (mpf-p rop))
⇒ t
