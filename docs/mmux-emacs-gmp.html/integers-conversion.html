3.4 Converting exact integers to other objects

Generic Function: mpz-get-ui (OP)
Method: mpz-get-ui ((OP mpz))

Convert an object OP of type mpz to an unsigned exact integer number. 

(mpz-get-ui (mpz 123))  ⇒ 123
Generic Function: mpz-get-si (OP)
Method: mpz-get-si ((OP mpz))

Convert an object OP of type mpz to an signed exact integer number. 

(mpz-get-si (mpz 123))  ⇒ 123
Generic Function: mpz-get-d (OP)
Method: mpz-get-d ((OP mpz))

Convert an object OP of type mpz to a floating–point number. 

(mpz-get-d (mpz 123))  ⇒ 123.0
Generic Function: mpz-get-d-2exp (OP)
Method: mpz-get-d-2exp ((OP mpz))

Convert an object OP of type mpz to a floating–point number, yielding the exponent separately. The return value is a pair whose car is a floating–point number and whose cdr is the exponent: 

(defconst rv (mpz-get-d-2exp (mpz 123)))

rv                                      ⇒ (0.9609375 . 7)
(* (car rv) (expt 2 (cdr rv)))          ⇒ 123.0
Generic Function: mpz-get-str (BASE OP)
Method: mpz-get-str ((BASE integer) (OP mpz))

Convert an object OP of type mpz to a string; return the resulting string. BASE must be an exact integer representing the base of the numeric string representation: positive between 2 and 36 or negative between -2 and -36. 

(defconst op (mpz))
(mpz-set-si op 15)
(mpz-get-str  10 op)    ⇒ "15"
(mpz-get-str  16 op)    ⇒ "f"
(mpz-get-str -16 op)    ⇒ "F"
