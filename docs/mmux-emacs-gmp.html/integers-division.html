3.6 Division operations

The division functions divide numerator N by the denominator D, forming a quotient Q and/or remainder R; for the 2exp functions, D=2^B where B is the bit count. The rounding is in three styles, each suiting different applications.

cdiv

Rounds Q up towards +infinity, and R will have the opposite sign to D. The c stands for “ceil”.

fdiv

Rounds Q down towards -infinity, and R will have the same sign as D. The f stands for “floor”.

tdiv

Rounds Q towards zero, and R will have the same sign as N. The t stands for “truncate”.

In all cases Q and R will satisfy N = Q * D + R, and R will satisfy 0 <= abs(R) < abs(D).
