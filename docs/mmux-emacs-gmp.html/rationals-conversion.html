4.4 Converting rational numbers to other objects

Generic Function: mpq-get-d (OP)
Method: mpq-get-d ((OP mpq))

Convert an object OP of type mpq to a floating–point number. 

(mpq-get-d (mp1 12.3))  ⇒ 12.3
Generic Function: mpq-get-str (BASE OP)
Method: mpq-get-str ((BASE integer) (OP mpq))

Convert an object OP of type mpq to a string; return the resulting string. BASE must be an exact integer representing the base of the numeric string representation: positive between 2 and 36 or negative between -2 and -36. 

(defconst op (mpq))
(mpq-set-si op 17 13)

(mpq-get-str +10 op)    ⇒ "17/13"
(mpq-get-str +16 op)    ⇒ "11/d"
(mpq-get-str -16 op)    ⇒ "11/D"
