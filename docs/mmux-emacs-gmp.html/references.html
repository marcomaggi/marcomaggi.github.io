Appendix C Bibliography and references

The documentation of MMUX Emacs GMP is available online:

https://marcomaggi.github.io/docs/mmux-emacs-gmp.html

the latest release of this package can be downloaded from:

https://bitbucket.org/marcomaggi/mmux-emacs-gmp/downloads

development takes place at:

https://github.com/marcomaggi/mmux-emacs-gmp

and as backup at:

https://bitbucket.org/marcomaggi/mmux-emacs-gmp

https://gmplib.org/

https://github.com/emacs-mirror/emacs

https://github.com/emacs-pe/emacs-modules
