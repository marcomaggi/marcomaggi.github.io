3.6.4 Modulo functions

The following functions use floor rounding, Rounding towards -infinity (floor).

Generic Function: mpz-mod (R N D)
Method: mpz-mod ((R mpz) (N mpz) (D mpz))

Set R to N mod D. 

(let ((R      (mpz))
      (N      (mpz 10))
      (D      (mpz 3)))
  (mpz-mod R N D)
  (mpz-get-si R))
⇒ 1
Generic Function: mpz-mod-ui (R N D)
Method: mpz-mod-ui ((R mpz) (N mpz) (D integer))

Set R to N mod D. 

(let ((R      (mpz))
      (N      (mpz 10))
      (D      3))
  (mpz-mod-ui R N D))
⇒ 1
