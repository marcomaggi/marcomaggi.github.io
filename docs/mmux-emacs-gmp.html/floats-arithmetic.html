5.6 Arithmetic operations

Generic Function: mpf-add (ROP OP1 OP2)
Method: mpf-add ((ROP mpf) (OP1 mpf) (OP2 mpf))

Set ROP to OP1 + OP2. 

(let ((rop    (mpf))
      (op1    (mpf 1.2))
      (op2    (mpf 3.4)))
  (mpf-add rop op1 op2)
  (mpf-almost-equal-p rop (+ 1.2 3.4) 1e-5))
⇒ t
Generic Function: mpf-add-ui (ROP OP1 OP2)
Method: mpf-add-ui ((ROP mpf) (OP1 mpf) (OP2 integer))

Set ROP to OP1 + OP2. 

(let ((rop    (mpf))
      (op1    (mpf 1.2))
      (op2    3))
  (mpf-add-ui rop op1 op2)
  (mpf-almost-equal-p rop (+ 1.2 3.0) 1e-5))
⇒ t
