3.5 Arithmetic operations

Generic Function: mpz-add (ROP OP1 OP2)
Method: mpz-add ((ROP mpz) (OP1 mpz) (OP2 mpz))

Add two mpz objects. 

(let ((rop      (mpz))
      (op1      (mpz 10))
      (op2      (mpz 1)))
  (mpz-add rop op1 op2)
  (mpz-get-ui rop))
⇒ 11
Generic Function: mpz-add-ui (ROP OP1 OP2)
Method: mpz-add-ui ((ROP mpz) (OP1 mpz) (OP2 integer))

Add an mpz object to an unsigned exact integer number. 

(let ((rop      (mpz))
      (op1      (mpz 10))
      (op2      1))
  (mpz-add-ui rop op1 op2)
  (mpz-get-ui rop))
⇒ 11

Generic Function: mpz-addmul (ROP OP1 OP2)
Method: mpz-addmul ((ROP mpz) (OP1 mpz) (OP2 mpz))

Multiply two mpz objects, then add the result to another mpz object.

      (op1     (mpz 11))
      (op2     (mpz 2)))
  (mpz-addmul rop op1 op2)
  (mpz-get-ui rop))
