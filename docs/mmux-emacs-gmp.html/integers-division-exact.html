3.6.5 Exact division

The following functions will return correct results only when D is an exact divisor of N.

Generic Function: mpz-divexact (Q N D)
Method: mpz-divexact ((Q mpz) (N mpz) (D mpz))

Set Q to N/D. 

(let ((Q      (mpz))
      (N      (mpz 9))
      (D      (mpz 3)))
  (mpz-divexact Q N D)
  (mpz-get-si Q))
⇒ 3
Generic Function: mpz-divexact-ui (Q N D)
Method: mpz-divexact-ui ((Q mpz) (N mpz) (D integer))

Set Q to N/D. 

(let ((Q      (mpz))
      (N      (mpz 9))
      (D      3))
  (mpz-divexact-ui Q N D)
  (mpz-get-si Q))
⇒ 3

Exact division tests

Generic Function: mpz-divisible-p (N D)
Method: mpz-divisible-p ((N mpz) (D mpz))

Return true if N is exactly divisible by D. 

(let ((N      (mpz 9))
      (D      (mpz 3)))
  (mpz-divisible-p N D))
⇒ t
Generic Function: mpz-divisible-ui-p (N D)
Method: mpz-divisible-ui-p ((N mpz) (D integer))

Return true if N is exactly divisible by D. 

(let ((N      (mpz 9))
      (D      3))
  (mpz-divisible-ui-p N D))
⇒ t
Generic Function: mpz-divisible-2exp-p (N B)
Method: mpz-divisible-2exp-p ((N mpz) (B integer))

Return true if N is exactly divisible by 2^B. 

(let ((N      (mpz 16))
      (B      3))
  (mpz-divisible-2exp-p N B))
⇒ t

Congruency tests

N is congruent to C modulo D if there exists an integer Q satisfying N = C + Q * D.

Generic Function: mpz-congruent-p (N C D)
Method: mpz-congruent-p ((N mpz) (C mpz) (D mpz))

Return non-zero if N is congruent to C modulo D. 

(let ((N      (mpz 7))
      (C      (mpz 1))
      (D      (mpz 3)))
  (mpz-congruent-p N C D))
⇒ t
Generic Function: mpz-congruent-ui-p (N C D)
Method: mpz-congruent-ui-p ((N mpz) (C integer) (D integer))

Return non-zero if N is congruent to C modulo D. 

(let ((N      (mpz 7))
      (C      1)
      (D      3))
  (mpz-congruent-ui-p N C D))
⇒ t
Generic Function: mpz-congruent-2exp-p (N C B)
Method: mpz-congruent-2exp-p ((N mpz) (C mpz) (B integer))

Return non-zero if N is congruent to C modulo 2^B. 

(let ((N      (mpz 17))
      (C      (mpz 1))
      (B      3))
  (mpz-congruent-2exp-p N C B))
⇒ t
