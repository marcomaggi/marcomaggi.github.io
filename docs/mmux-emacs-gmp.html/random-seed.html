6.3 Random number generators seeding

Generic Function: gmp-randseed (STATE SEED)
Method: gmp-randseed ((STATE gmp-randstate) (SEED mpz))

Set an initial seed value into STATE. 

(let ((state  (gmp-randstate)))
  (gmp-randinit-default state)
  (gmp-randseed state (mpz 123))
  (let ((num (gmp-urandomm-ui state 3)))
    (and (<= 0 num) (< num 3))))
⇒ t
Generic Function: gmp-randseed-ui (STATE SEED)
Method: gmp-randseed-ui ((STATE gmp-randstate) (SEED integer))

Set an initial seed value into STATE. 

(let ((state  (gmp-randstate)))
  (gmp-randinit-default state)
  (gmp-randseed-ui state 12)
  (let ((num (gmp-urandomm-ui state 3)))
    (and (<= 0 num) (< num 3))))
⇒ t
