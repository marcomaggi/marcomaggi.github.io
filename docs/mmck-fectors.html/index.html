Fectors for CHICKEN

This document describes version 0.6.0-devel.1 of MMCK Fectors, a CHICKEN language library implementing functional vectors. CHICKEN is a Scheme–to–C compiler supporting the language features as defined in the “Revised^5 Report on Scheme”.

The package supports POSIX platforms. This package should work with CHICKEN version 5+; it has been tested with CHICKEN version 5.1.0. The code depends on the CHICKEN egg coops. To run the tests: this package depends upon the package MMCK Checks.

The latest release can be downloaded from:

https://bitbucket.org/marcomaggi/mmck-fectors/downloads
