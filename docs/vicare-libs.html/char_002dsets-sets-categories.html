21.6.6 Sets corresponding to Unicode general categories

The following character sets are defined and exported by the (vicare containers char-sets categories) library.

Notice that not all the code points accepted by char=? are part of a general category, so the union of the category sets is different from the full character set char-set:full.

Categories are disjoint, so the intersection of the category sets is the empty set.
