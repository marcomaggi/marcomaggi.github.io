1.12.5 Special cases

The following bindings are exported by the library (vicare language-extensions syntaxes).

— Macro: case-fixnums ?expr ((?fixnum0 ?fixnum ...) ?fx-body0 ?fx-body ...) ...
— Macro: case-fixnums ?expr ((?fixnum0 ?fixnum ...) ?fx-body0 ?fx-body ...) ... (else ?else-body0 ?else-body ...)
— Auxiliary Syntax: else

Special implementation of case expecting only fixnums as datums; for maximum speed, it makes use of unsafe operations on fixnums. else must be the binding exported by (rnrs) and (vicare).

?expr must be an expression evaluating to a fixnum, else the behaviour is undefined (and you do not want it).

— Macro: case-integers ?expr ((?int0 ?int ...) ?int-body0 ?int-body ...) ...
— Macro: case-integers ?expr ((?int0 ?int ...) ?int-body0 ?int-body ...) ... (else ?else-body0 ?else-body ...)
— Auxiliary Syntax: else

Special implementation of case expecting only exact integers as datums. else must be the binding exported by (rnrs) and (vicare).

?expr must be an expression evaluating to an exact integer, else the behaviour is undefined (and you do not want it).

— Macro: case-symbols ?expr ((?symbol0 ?symbol ...) ?sym-body0 ?sym-body ...) ...
— Macro: case-symbols ?expr ((?symbol0 ?symbol ...) ?sym-body0 ?sym-body ...) ... (else ?else-body0 ?else-body ...)
— Auxiliary Syntax: else

Special implementation of case expecting only symbols as datums. else must be the binding exported by (rnrs) and (vicare).

?expr must be an expression evaluating to a symbol, else the behaviour is undefined.

— Macro: case-strings ?expr ((?string0 ?string ...) ?sym-body0 ?sym-body ...) ...
— Macro: case-strings ?expr ((?string0 ?string ...) ?sym-body0 ?sym-body ...) ... (else ?else-body0 ?else-body ...)
— Auxiliary Syntax: else

Special implementation of case expecting only strings as datums. else must be the binding exported by (rnrs) and (vicare).

?expr must be an expression evaluating to a string, else the behaviour is undefined.

— Macro: case-chars ?expr ((?char0 ?char ...) ?sym-body0 ?sym-body ...) ...
— Macro: case-chars ?expr ((?char0 ?char ...) ?sym-body0 ?sym-body ...) ... (else ?else-body0 ?else-body ...)
— Auxiliary Syntax: else

Special implementation of case expecting only characters as datums. else must be the binding exported by (rnrs) and (vicare).

?expr must be an expression evaluating to a char, else the behaviour is undefined.

— Macro: case-identifiers ?expr ((?identifier0 ?identifier ...) ?sym-body0 ?sym-body ...) ...
— Macro: case-identifiers ?expr ((?identifier0 ?identifier ...) ?sym-body0 ?sym-body ...) ... (else ?else-body0 ?else-body ...)
— Auxiliary Syntax: else

Special implementation of case expecting only identifiers as datums; identifiers are compared with free-identifier=?. else must be the binding exported by (rnrs) and (vicare).

?expr must be an expression evaluating to an identifier, else the behaviour is undefined.
