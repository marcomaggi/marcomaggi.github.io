28 Bytevector compounds

A bytevector compound is a sequence of octets split into a sequence of bytevectors. Bytevector compounds have a special API to handle the sequence of octets as a First–in First–out queue. Bytevector compounds are defined by the library (vicare containers bytevector-compounds).
