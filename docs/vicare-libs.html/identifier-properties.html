1.23 Expand–time identifier properties

The library (vicare language-extensions identifier-properties) allows the association of objects to identifiers at expand time, using other identifiers as keywords; this library is meant to be imported for the expand phase. Meaningless usage examples: 

     (import (vicare)
       (for (vicare language-extensions identifier-properties)
            expand))
     
     (define-syntax (type-of stx)
       (syntax-case stx ()
         ((_ ?id)
          (identifier-property-ref #'type-of #'?id #f))))
     
     (eval-for-expand
       (register-identifier-property #'type-of))
     
     (let ((a "ciao"))
       (eval-for-expand
         (identifier-property-set! #'type-of #'a #'(quote string)))
       (type-of a) ==> (quote string)
       (type-of a))
     ⇒ string

The following bindings are by the library (vicare language-extensions identifier-properties).

— Function: register-identifier-property subject

Allocate a new identifier properties table associated to the identifier subject. If a property associated to subject already exists: raise an exception.

— Function: forget-identifier-property subject

Release all the resources of the properties table associated to the identifier subject. If a property associated to subject does not exist: raise an exception.

— Function: identifier-property-exists? subject

Return #t if subject is an identifier associated to a properties table; otherwise return #f. Raise an exception if subject is not an identifier.

— Function: identifier-property-set! subject key value

Associate the identifier key to value in subject's table. Raise an assertion violation if subject is not a previously defined object property. If key already has an identifier property for subject: silently overwrite the old value.

— Function: identifier-property-ref subject key default

Return the value associated to the identifier key in subject's table. Raise an assertion violation if subject is not a previously defined object property. If key does not have an identifier property for subject: return default.
