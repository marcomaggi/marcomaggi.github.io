2.38 SRFI-114 comparators

The libraries (srfi :114) and (srfi :114 comparators) are derived from the reference implementation of SRFI-114 by John Cowan; see:

http://srfi.schemers.org/srfi-114/srfi-114.html

for more details. (srfi :114) and (srfi :114 comparators) export the same bindings.
