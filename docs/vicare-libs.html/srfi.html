2 Scheme Requests For Implementation

A port to R6RS of the SRFI libraries is included in the distribution of Vicare. The libraries’ names conform to SRFI 97: SRFI Libraries. There is a registry of available SRFIs and other platform features which works alongside SRFI 0: Feature–Based Conditional Expansion.

The SRFI libraries are optionally installed by the package; the availability of the SRFI libraries can be tested at run–time with the function vicare-built-with-srfi-enabled from (vicare) ((vicare-scheme)vicare-built-with-srfi-enabled), and at expand–time with the clause srfi-enabled of cond-expand from SRFI-0 (see linux-enabled).
