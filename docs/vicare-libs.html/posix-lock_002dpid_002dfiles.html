4.29 Creating lock PID files

The library (vicare posix lock-pid-files) implements facilities for the creation of lock PID files, built on top of the library (vicare posix).

  • posix lock-pid-files intro: Introduction to lock PID files.
  • posix lock-pid-files api: Programming interface to lock PID files.
