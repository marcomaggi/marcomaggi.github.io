4.27 POSIX functions not interfaced

Not all the functions defined by POSIX are exposed by (vicare posix); this section lists some of them along with reasons for not interface them. This is not the last word: if good reasons arise to interface some of the functions, they will.

popen()
pclose()

There seems to be no real advantage (in a Scheme API) to use these rather than explicitly call pipe, fork and exec. pipe for details.
