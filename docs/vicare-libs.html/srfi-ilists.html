2.40 SRFI-116 immutable lists

The library (srfi :116) is by John Cowan as reference implementation for SRFI-116; see:

http://srfi.schemers.org/srfi-116/srfi-116.html

for more details. The comparators API is exported by the library (srfi :116 comparators).
