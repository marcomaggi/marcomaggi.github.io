1.4 Running coroutines

The library (vicare language-extensions coroutines) implements coroutines on top of Scheme continuations. The implementation is a simple queue of escape functions: whenever coroutine A yields control to the “next” coroutine, it enqueues an escape function to its current continuation and causes the next escape function to be invoked.

For discussions about other implementations of coroutines and continuations see references.
