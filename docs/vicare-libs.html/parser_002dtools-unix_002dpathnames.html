54.1 Parsing Unix file system pathnames

The library (vicare parser-tools unix-pathnames) implements facilities for parsing and manipulating Unix file system pathnames. When importing it, it is suggested to prefix the bindings with ‘uxptn.’ as in: 

(import (vicare)
  (prefix (vicare parser-tools unix-pathnames) uxptn.))
