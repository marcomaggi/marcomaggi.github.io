4.17.3 Data structure struct-tms

— Structure Type: struct-tms

Scheme level representation of struct tms, struct tms. It has the following fields:

tms_utime
Exact integer.
tms_stime
Exact integer.
tms_cutime
Exact integer.
tms_cstime
Exact integer.
