1.12.3 Sequencing execution

The following bindings are exported by the library (vicare language-extensions syntaxes).

— Macro: begin0-let ((?var0 ?init0) (?var ?init) ...) ?form0 ?form ...

Expand to: 

          (let ((?var0 ?init0)
                (?var  ?init)
                ...)
            ?form0 ?form ...
            ?var0)
