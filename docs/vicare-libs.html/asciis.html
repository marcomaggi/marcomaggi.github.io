1.10 Utilities for ASCII characters

The library (vicare language-extensions ascii-chars) provides functions and macros dealing with fixnums representing characters in ASCII encoding range.

• asciis generic:  Generic utilities.
• asciis conversion:  Conversion utilities.
• asciis predicates:  Character predicates.
• asciis uri:  Character predicates related to RFC 3986, Uniform Resource Identifiers.
