20.2.4 The <binary-input/output-channel> type

The following syntactic bindings are exported by the library (vicare net channels).

Record Type: <binary-input/output-channel>
Parent Type: <channel>

Expose the methods for all the binary input/output channels. These channels implement all the methods of <binary-input-only-channel> and all the methods of <binary-output-only-channel>.
