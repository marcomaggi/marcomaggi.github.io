17.5 Predicates and arguments validation

The following bindings are exported by the library (vicare net channels).

— Function: receiving-channel? chan

Return #t if chan is in the course of receiving a message, else return #f. It is an error if chan is not an instance of channel.

— Function: sending-channel? chan

Return #t if chan is in the course of sending a message, else return #f. It is an error if chan is not an instance of channel.

— Function: inactive-channel? chan

Return #t if chan is neither in the course of sending nor receiving a message, else return #f. It is an error if chan is not an instance of channel.

— Function: input-channel? chan

Return #t if chan is an input or input/output channel, else return #f. It is an error if chan is not an instance of channel.

— Function: output-channel? chan

Return #t if chan is an output or input/output channel, else return #f. It is an error if chan is not an instance of channel.

— Function: input/output-channel? chan

Return #t if chan is an input/output channel, else return #f. It is an error if chan is not an instance of channel.

The following bindings are validation clauses to be used with the facilities of the library (vicare arguments validation).

— Validation Clause: receiving-channel obj

Succeed if obj is an instance of channel and it is in the course of receiving a message.

— Validation Clause: not-receiving-channel obj

Succeed if obj is an instance of channel and it is not in the course of receiving a message.

— Validation Clause: sending-channel obj

Succeed if obj is an instance of channel and it is in the course of sending a message.

— Validation Clause: not-sending-channel obj

Succeed if obj is an instance of channel and it is not in the course of sending a message.

— Validation Clause: inactive-channel obj

Succeed if obj is an instance of channel and it is neither in the course of sending nor receiving a message.

— Validation Clause: not-inactive-channel obj

Succeed if obj is an instance of channel and it is either in the course of sending or receiving a message.
