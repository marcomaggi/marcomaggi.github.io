Lightweight testing for CHICKEN

This document describes version 0.3.1-devel.0 of MMCK Checks, a CHICKEN language library implementing features to organise test suites and it is derived from SRFI-78 “Lightweight testing”. CHICKEN is a Scheme-to-C compiler supporting the language features as defined in the “Revised^5 Report on Scheme”.

The library supports POSIX platforms. This package should work with CHICKEN version 5+; it has been tested with CHICKEN version 5.1.0.
