12 Exceptional–condition object–types

Function: make-pfds-assertion-violation

Build and return a new exceptional–condition object with kind pfds-assertion-violation.

Function: pfds-assertion-violation? OBJ

Return #t if OBJ is an exceptional–condition object with kind pfds-assertion-violation.

Function: pfds-assertion-violation WHO MESSAGE IRRITANT …

Raise a non–continuable exception having exceptional–condition object with kind pfds-assertion-violation and kind exn with location, message and arguments properties.
