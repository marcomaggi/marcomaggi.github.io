1 Overview of the package

This document describes version 0.1b3 of Nausicaa/Gcrypt, a distribution of libraries for R6RS Scheme implementations; it provides a binding to the Libgcrypt C language library. It is built upon Nausicaa/Scheme, making use of its foreign–functions interface.

Libgcrypt is a general purpose library of cryptographic building blocks. It is originally based on code used by GnuPG. It does not provide any implementation of OpenPGP or other protocols. Thorough understanding of applied cryptography is required to use Libgcrypt.

The libraries support Ikarus, Mosh and Ypsilon Scheme; the package supports GNU+Linux systems.

The installed libraries are:

(foreign crypto gcrypt shared-object)
Exports the single binding gcrypt-shared-object which references the foreign shared library.
(foreign crypto gcrypt sizeof)
Export bindings to constant values and data structures accessors.
(foreign crypto gcrypt enumerations)
Defines enumeration types used to handle Gcrypt options and flags.
(foreign crypto gcrypt platform)
Exports one binding for each foreign function; the Scheme functions have the same name of the C functions, so gcry_check_version() is available as gcry_check_version. This library makes the original Libgcrypt documentation relevant for this package. The Libgcrypt library.
(foreign crypto gcrypt primitives)
Exports one binding for each callout function in the platform library, implementing marshaling of arguments and return values. The Scheme functions have the same name of the C functions, but with underscores converted to dashes; so gcry_check_version() is available as gcry-check-version.

This documentation file describes the API exported by this library; only functions implementing marshaling of values are described: if a function is not documented here it is exported “as is” from the platform library.

(foreign crypto gcrypt)
Compound library exporting the high–level API; it reexports bindings from the ‘primitives’ library and the constant values from the ‘sizeof’ library. Additionally bindings from (foreign crypto gpg-error conditions) are rexported.
(foreign crypto gcrypt compensated)
Exports bindings to constructor functions making use of the (compensations) library from Nausicaa/Scheme. Compensation stacks.

6.1 Introduction

Here we see some examples of MPI usage for basic operations; the examples give us some hint about how to manage resources allocation. Every MPI number must be allocated with gcry-mpi-new or gcry-mpi-snew and released with gcry-mpi-release.

The following program will add two numbers, 100 and 50: 

     (import (rnrs)
       (foreign crypto gcrypt)
       (foreign crypto gcrypt compensated))
     
     (with-compensations
       (let ((a (gcry-mpi-new/c 50))
             (b (gcry-mpi-new/c 50))
             (c (gcry-mpi-new/c 50)))
         (gcry-mpi-set-ui a 100)
         (gcry-mpi-set-ui b 50)
         (gcry-mpi-add c a b)))

notice that there is no way to convert an MPI number to a string in human readable form; the only way to make a number readable is to use gcry-mpi->uint, which will return an exact integer in the range allowed by the unsigned int C language type of the underlying platform.

7.1 Introduction

Gcrypt's sexps format is based on Ron Rivest's work; basically, we can think of a sexps as a string of characters with nested, matched, parentheses and strings prefixed with their own length: 

     (4:ciao)

4 is the number of characters in ciao and the : character acts as separator. In the canonical format white spaces are not used as separator; the following example is a sexp holding two elements, the strings ciao and hello: 

     (4:ciao5:hello)

MPI numbers are embedded in sexps in their serialised formats.

8.1 Introduction

Here is an example which does: key pair generation, MPI number encryption and decryption, MPI number signature and verification. The MPI number is 456 and its hexadecimal representation is 1c8. 

     (import (rnrs)
       (compensations)
       (foreign crypto gcrypt)
       (foreign crypto gcrypt compensated))
     
     (with-compensations
       (define data
         `(data
           (flags raw)
           (value ,(let ((n (gcry-mpi-new/c 50)))
                     (gcry-mpi-set-ui n 456)
                     n))))
     
       (define key-pair
         (gcry-pk-genkey/c '(genkey
                             (rsa (nbits 1024)))))
     
       (define pub-key
         (gcry-sexp-find-token/str/c key-pair 'public-key))
     
       (define pri-key
         (gcry-sexp-find-token/str/c key-pair 'private-key))
     
       (define signature
         (gcry-pk-sign/c data pri-key))
     
       (define encrypted-data
         (gcry-pk-encrypt/c data pub-key))
     
       (define decrypted-data
         (gcry-pk-decrypt/c encrypted-data pri-key))
     
       (gcry-sexp->string decrypted-data
                          (gcry-sexp-format canon))
