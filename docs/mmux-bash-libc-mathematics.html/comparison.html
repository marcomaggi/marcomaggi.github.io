6 Comparison builtins ¶

Builtin: isgreater REAL1 REAL2 ¶

Return true if REAL1 is greater than REAL2; otherwise return false. 

if isgreater 2 1
then echo F yeah
fi
Builtin: isgreaterequal REAL1 REAL2 ¶

Return true if REAL1 is greater than, or equal to, REAL2; otherwise return false.

Builtin: isless REAL1 REAL2 ¶

Return true if REAL1 is less than REAL2; otherwise return false.

Builtin: islessequal REAL1 REAL2 ¶

Return true if REAL1 is less than, or equal to, REAL2; otherwise return false.

Builtin: islessgreater REAL1 REAL2 ¶

Return true if REAL1 is less than, or greater than, REAL2; otherwise return false.

Builtin: isunordered REAL1 REAL2 ¶

Return true if REAL1 is unordered with respect to REAL2; otherwise return false. 

isunordered 2 1                 ⇒ 1
isunordered 1 2                 ⇒ 1
isunordered 1 1                 ⇒ 1
isunordered 'NaN' 1             ⇒ 0
isunordered 1 'NaN'             ⇒ 0
isunordered 'NaN' 'NaN'         ⇒ 0
