9 Exponentiation and logarithms ¶

Builtin: exp REAL ¶
Builtin: cexp COMPLEX ¶

Compute e raised to the power of a number, print the result on stdout.

Builtin: exp2 REAL ¶

Compute 2 raised to the power of a real number, print the result on stdout.

Builtin: exp10 REAL ¶

Compute 10 raised to the power of a real number, print the result on stdout.

Builtin: expm1 REAL ¶

Compute exp(REAL) - 1, print the result on stdout.

Builtin: log REAL ¶
Builtin: clog COMPLEX ¶

Compute the base e logarithm of a number, print the result on stdout.

Builtin: log2 REAL ¶

Compute the base 2 logarithm of a real number, print the result on stdout.

Builtin: log10 REAL ¶
Builtin: clog10 REAL ¶

Compute the base 10 logarithm of a number, print the result on stdout.

Builtin: logb REAL ¶

Extract the exponent of a real number, print the result on stdout. 

logb 12.3               -| 3
Builtin: log1p REAL ¶

Compute log(1 + REAL), print the result on stdout.

Builtin: pow REAL_BASE REAL_EXPONENT ¶
Builtin: cpow COMPLEX_BASE COMPLEX_EXPONENT ¶

Compute BASE raised to the power of EXPONENT, print the result on stdout.

Builtin: sqrt REAL ¶
Builtin: csqrt REAL ¶

Compute square root of a number, print the result on stdout. 

sqrt 64                 -| 8
Builtin: cbrt REAL ¶

Compute cube root of a real number, print the result on stdout. 

cbrt 8                  -| 3
Builtin: hypot REAL_X REAL_Y ¶

Compute sqrt(X^2 + Y^2), print the result on stdout. 

hypot 2 3               -| 3.6056
