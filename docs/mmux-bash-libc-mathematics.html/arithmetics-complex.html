8.2 Arithmetic operations between complex numbers ¶

Builtin: cadd COMPLEX0 COMPLEX1 … ¶

Print a string which is the result of the operation: 

COMPLEX0 + COMPLEX1 + ...

If only one operand is given: print that operand.

Builtin: csub COMPLEX0 COMPLEX1 … ¶

Print a string which is the result of the operation: 

COMPLEX0 - (COMPLEX1 + ...)

If only one operand is given: print that operand.

Builtin: cmul COMPLEX0 COMPLEX1 … ¶

Print a string which is the result of the operation: 

COMPLEX0 * COMPLEX1 * ...

If only one operand is given: print that operand.

Builtin: cdiv COMPLEX0 COMPLEX1 … ¶

Print a string which is the result of the operation: 

COMPLEX0 / (COMPLEX1 * ...)

If only one operand is given: print that operand.

Builtin: cneg COMPLEX ¶

Print a string which is the result of the operation: 

- COMPLEX

just invert the sign of the operand.
