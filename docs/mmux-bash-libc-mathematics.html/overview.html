1 Overview of the package ¶

This document describes version 0.1.0-devel.0 of MMUX Bash Libc Mathematics, a package implementing additional builtin commands for the GNU Bash shell; to do so it installs a C language library and a shell script which is meant to be sourced in a Bash session or script.

MMUX Bash Libc Mathematics installs a shared library under $libdir; for example:

/usr/local/lib64/libmmux-bash-libc-mathematics.so

it also installs a shell script, a “shell library”, under $datavar/bash; for example:

/usr/local/share/bash/libmmux-bash-libc-mathematics.bash

to use the builtins, we have to load the shell library using the builtin source.
