11 Hyperbolic builtins ¶

Builtin: sinh REAL ¶
Builtin: cosh REAL ¶
Builtin: tanh REAL ¶

Print a string representing the result of applying the hyperbolic sine, cosine and tangent function to REAL.

Builtin: asinh REAL ¶
Builtin: acosh REAL ¶
Builtin: atanh REAL ¶

Print a string representing the result of applying the hyperbolic arc sine, arc cosine and arc tangent function to REAL.

Builtin: csinh COMPLEX ¶
Builtin: ccosh COMPLEX ¶
Builtin: ctanh COMPLEX ¶

Print a string representing the result of applying the complex hyperbolic sine, cosine and tangent function to COMPLEX.

Builtin: casinh COMPLEX ¶
Builtin: cacosh COMPLEX ¶
Builtin: catanh COMPLEX ¶

Print a string representing the result of applying the complex hyperbolic arc sine, arc cosine and arc tangent function to COMPLEX.
