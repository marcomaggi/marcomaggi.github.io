8.1 Arithmetic operations between real numbers ¶

Builtin: add REAL0 REAL1 … ¶

Print a string which is the result of the operation: 

REAL0 + REAL1 + ...

If only one operand is given: print that operand. 

add 1.3 2.5             -| 3.80000
Builtin: sub REAL0 REAL1 … ¶

Print a string which is the result of the operation: 

REAL0 - (REAL1 + ...)

If only one operand is given: print that operand. 

sub 3 1                 -| 2.00000
sub 10 1 3              -| 6.00000
Builtin: mul REAL0 REAL1 … ¶

Print a string which is the result of the operation: 

REAL0 * REAL1 * ...

If only one operand is given: print that operand. 

mul 2 3 5               -| 30.00000
Builtin: div REAL0 REAL1 … ¶

Print a string which is the result of the operation: 

REAL0 / (REAL1 * ...)

If only one operand is given: print that operand. 

div 12 3                -| 4.00000
div 24 2 3              -| 4.00000
Builtin: mod REAL_NUMERATOR0 REAL_DENOMINATOR1 ¶

Print a string representing the mod of the arguments.

Builtin: remainder REAL_NUMERATOR0 REAL_DENOMINATOR1 ¶

Print a string representing the remainder of the arguments.

Builtin: neg REAL ¶

Print a string which is the result of the operation: 

- REAL

just invert the sign of the operand. 

neg 12                  -| -12.00000
