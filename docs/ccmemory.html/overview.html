1 Overview of the package

This document describes version 0.2.1-devel.4 of CCMemory, a C11 language library implementing an interface to custom memory allocators.

This package supports POSIX platforms. This package depends on the package CCExceptions, at least version 0.8.0. To run the test suite: this package depends upon the package CCTests, at least version 0.3.0.

CCMemory installs the single header file ccmemory.h. All the function names in the API are prefixed with ccmem_; all the preprocessor symbol names are prefixed with CCMEM_; all the type names are prefixed with ccmem_ and suffixed with _t.
