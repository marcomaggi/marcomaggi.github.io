5.1.3 Guarded realloc() with generic allocators

Usage example: 

cce_location_t        L[1];
ccmem_clean_handler_t P_H[1];

if (cce_location(L)) {
  cce_run_catch_handlers_raise(L, upper_L);
} else {
  void *      P = ccmem_malloc_guarded(L, P_H,
                    ccmem_standard_allocator, 4096);

  memset(P, 0, 4096);
  {
    void *    Q = ccmem_realloc_guarded(L, P_H,
                    ccmem_standard_allocator, P, 2 * 4096);

    memset(Q, 0, 2 * 4096);
  }
  cce_run_body_handlers(L);
}
Function: void * ccmem_realloc_guarded_clean (cce_destination_t L, ccmem_clean_handler_t * P_H, ccmem_allocator_t const * A, void * P, size_t newsize)

Wrapper for ccmem_realloc() that registers the allocated memory to be released by the handler referenced by P_H. Such handler must be the same one used in a previous call to ccmem_malloc_guarded_clean(), ccmem_calloc_guarded_clean() or ccmem_realloc_guarded_clean().

Function: void * ccmem_realloc_guarded_error (cce_destination_t L, ccmem_error_handler_t * P_H, ccmem_allocator_t const * A, void * P, size_t newsize)

Wrapper for ccmem_realloc() that registers the allocated memory to be released by the handler referenced by P_H. Such handler must be the same one used in a previous call to ccmem_malloc_guarded_clean(), ccmem_calloc_guarded_clean() or ccmem_realloc_guarded_clean().

Preprocessor Macro: void * ccmem_realloc_guarded (cce_destination_t L, P_H, ccmem_allocator_t const * A, void * P, size_t newsize)

If the argument P_H is a pointer of type:

ccmem_clean_handler_t

The macro use expands into a call to ccmem_realloc_guarded_clean().

ccmem_error_handler_t

The macro use expands into a call to ccmem_realloc_guarded_error().
