7.1.4 Guarded allocation: calloc()

Function: void * cce_sys_calloc_guarded_clean (cce_destination_t L, cce_clean_handler_t * P_H, size_t count, size_t eltsize)
Function: void * cce_sys_calloc_guarded_error (cce_destination_t L, cce_error_handler_t * P_H, size_t count, size_t eltsize)

Like cce_sys_calloc(), but register the resulting pointer in the handler referenced by P_H.

Preprocessor Macro: void * cce_sys_calloc_guarded (cce_destination_t L, cce_clean_handler_t * P_H, size_t count, size_t eltsize)
Preprocessor Macro: void * cce_sys_calloc_guarded (cce_destination_t L, cce_error_handler_t * P_H, size_t count, size_t eltsize)

This preprocessor macro uses _Generic to dispatch the call according to the type of the pointer P_H; if the type is:

cce_clean_handler_t

The macro expands into a call to cce_sys_calloc_guarded_clean().

cce_error_handler_t

The macro expands into a call to cce_sys_calloc_guarded_error().
