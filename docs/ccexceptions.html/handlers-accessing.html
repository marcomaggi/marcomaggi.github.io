5.3 Accessing location handlers

Function: cce_handler_t * cce_clean_handler_handler (cce_clean_handler_t * H)
Function: cce_handler_t * cce_error_handler_handler (cce_error_handler_t * H)

Given a pointer to clean or error handler, return a pointer to the embedded handler.

Macro: cce_handler_handler (cce_clean_handler_t * H)
Macro: cce_handler_handler (cce_error_handler_t * H)

Generic macro which using _Generic to dispatch its expansion depending on the type of the argument H. If the type of structure referenced by H is:

cce_clean_handler_t

The macro expands into a call to cce_clean_handler_handler().

cce_error_handler_t

The macro expands into a call to cce_error_handler_handler().
