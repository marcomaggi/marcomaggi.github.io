5.6.2 Example of asynchronous resource handling using a clean handler

Let’s say we need to allocate memory with malloc(), and to release it with free() whenever the dynamic extent of the local function terminates; we can use built–in functions for this, system memory for details. To show the mechanism, in the following example: we implement a handler explicitly. 

void
P_handler (cce_condition_t const * C, cce_handler_t * H)
{
  free(H->pointer);
}

void
func (void)
{
  cce_location_t       L[1];
  cce_clean_handler_t  P_H;

  if (cce_location(L)) {
    cce_run_catch_handlers_final(L);
  } else {
    void *  P;

    errno = 0;
    P = malloc(4096);
    if (P) {
      P_H.handler.function = P_handler;
      P_H.handler.pointer  = P;
      cce_register_clean_handler(L, &P_H);
    } else {
      cce_raise(L, cce_condition_new_errno_clear(errno));
    }
    do_something(L, P);
    cce_run_body_handlers(L);
  }
}

whether do_something() raises an exception or not: the clean handler is invoked and the memory is released.
