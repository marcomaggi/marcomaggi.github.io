5 The location handlers API

Location handlers allow us to execute callback functions whenever we exit a location context, either by normal execution or by raising an exception. Normal–execution handlers are run by calling the function cce_run_body_handlers(); exception–reaction handlers are run by calling the function cce_run_catch_handlers(), but we should use the more useful functions cce_run_catch_handlers_final() and cce_run_catch_handlers_raise().

The basic code template is the following: 

cce_location_t  L[1];

if (cce_location(L)) {
  /* location handler */
  cce_run_catch_handlers_final(L);
} else {
  /* location body */
  cce_run_body_handlers(L);
}

CCExceptions defines clean handlers, which are run as both normal–execution and exception-reaction handlers; it also defines error handlers, which are run only as exception-reaction handlers.
