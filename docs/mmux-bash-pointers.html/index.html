MMUX Bash Pointers ¶

This document describes version 0.1.0-devel.0 of MMUX Bash Pointers, a package implementing additional builtin commands for the GNU Bash shell; to do so it installs a C language library and a shell script which is meant to be sourced in a Bash session or script.

This package makes raw memory pointers available in shell scripts: 

declare PTR VALUE

libc_malloc PTR 1024
{
  pointer-set-sint $PTR 0 123
  pointer-ref-sint VALUE $PTR 0
}
libc_free $PTR
$VALUE  ⇒ 123

The package targets POSIX platforms. Only for testing: this package depends on the external package MBFL (at least version ‘v3.0.0-devel.8’).
