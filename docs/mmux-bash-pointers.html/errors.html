8 Handling errors ¶

Builtin: libc_strerror MSGVAR ERRNUM ¶

Store in MSGVAR the error message corresponding to ERRNUM, which must be the string representation of an exact integer representing a valid value for errno; ERRNUM must fit the C language type int. 

declare MSG

if test -v libc_EINVAL
then libc_strerror MSG $libc_EINVAL
else MSG='EINVAL undefined'
fi

"$MSG"  ⇒ Invalid argument
Function: mmux-bash-pointers-errno-to-string SYMVAR ERRNUM ¶

Store in SYMVAR a string representation of the errno code ERRNUM. When successful return true, otherwise return false. 

declare SYM

mmux-bash-pointers-errno-to-string SYM $libc_ENOMEM
"$SYM"  ⇒ ENOMEM
