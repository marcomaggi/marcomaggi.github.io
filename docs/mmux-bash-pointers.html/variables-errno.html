3.2 Value of errno ¶

If a C language constant referenced by one of the following variables is undefined: the corresponding Bash variable is undefined; we can test this condition with test -v; to test if the underlying platform implements the constant ENOMEM, we can do: 

if test -v libc_ENOMEM
then # implemented
else # not implemented
fi
