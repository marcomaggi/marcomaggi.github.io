3.1 C language type sizes ¶

If a C language type referenced by one of the following variables is undefined: the corresponding Bash variable is undefined; we can test this condition with test -v; to test if the underlying platform implements the type long long, we can do: 

if test -v libc_SIZEOF_SLLONG
then # implemented
else # not implemented
fi
Variable: libc_SIZEOF_POINTER ¶

Global variable holding the size, measured in bytes, of the C language type void *.
