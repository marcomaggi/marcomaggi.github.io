7.2.2 Constructors for standalone instances from ASCIIZ blocks

Function: ccptn_t const * ccname_new(ccptn_t, asciiz) (cce_destination_t L, ccmem_allocator_t const * A, ccmem_asciiz_t input_rep)

Allocate a new instance of ccptn_t, using the memory allocator A, and the input representation input_rep. When successful: just return. If an error occurs: raise an exception by performing a non–local exit to L.

The argument input_rep must reference an ASCIIZ string, which is referenced by the instance of ccptn_t; (ccmemory)Zero terminated strings. 

cce_location_t             L[1];
ccptn_clean_handler_t      H[1];
ccmem_allocator_t const *  A = ccmem_standard_allocator;

if (cce_location(L)) {
  cce_run_catch_handlers_raise(L, upper_L);
} else {
  ccmem_asciiz_t  input = \
    ccmem_new_asciiz_from_str("/path/to/file.ext");
  ccptn_t const * P;

  P = ccname_new(ccptn_t, asciiz)(L, A, input);
  ccptn_init_and_register_handler(L, H, P);

  cce_run_body_handlers(L);
}
