8 Pathname predicates

Function: bool ccptn_is_absolute (ccptn_t const * P)

Return true if P represents an absolute pathname; otherwise return false. A pathname is absolute if its first octet is the ASCII representation of ‘/’.

Function: bool ccptn_is_relative (ccptn_t const * P)

Return true if P represents a relative pathname; otherwise return false. A pathname is relative if its first octet is not the ASCII representation of ‘/’.

Function: bool ccptn_is_realpath (ccptn_t const * P)

Return true if P is the result of a call to the system function realpath(); otherwise return false. (libc)realpath.

Function: bool ccptn_is_normalised (ccptn_t const * P)

Return true if P is the result of either a call to the system function realpath() or to one of the normalisation functions; otherwise return false.
