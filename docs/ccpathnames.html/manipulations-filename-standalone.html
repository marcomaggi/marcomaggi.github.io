11.5.2 Obtaining a filename standalone pathname

A standalone instance of ccptn_t is allocated on the heap. The constructors of such instances do have to allocate memory for the instance itself.

Function: ccptn_t const * ccname_new(ccptn_t, filename) (cce_destination_t L, ccmem_allocator_t const * A, ccptn_t const * src)

Build a new ccptn_t instance representing the result of extracting the filename from the pathname represented by src; use the allocator A. When successful: return a pointer to the new instance. If an error occurs: raise an exception by performing a non–local exit to to L. 

cce_location_t             L[1];
ccptn_clean_handler_t      P_H[1], Q_H[1];
ccmem_allocator_t const *  A = ccmem_standard_allocator;

if (cce_location(L)) {
  cce_run_catch_handlers_final(L);
} else {
  char const     *input = "/path/to/file.ext";
  ccptn_t const  *P, *Q;

  P = ccname_new(ccptn_t, pointer, clean)(L, A, P_H, input);
  Q = ccname_new(ccptn_t, filename)(L, A, P);
  ccptn_init_handler(L, Q_H, Q);

  cce_run_body_handlers(L);
}
Function: ccptn_t const * ccname_new(ccptn_t, filename, clean) (cce_destination_t L, ccmem_allocator_t const * A, ccptn_clean_handler_t * H, ccptn_t const * src)
Function: ccptn_t const * ccname_new(ccptn_t, filename, error) (cce_destination_t L, ccmem_allocator_t const * A, ccptn_error_handler_t * H, ccptn_t const * src)

Like ccname_new(ccptn_t, filename)() but register the pathname instance to be destroyed using the handler H in the context of the location L. 

cce_location_t           L[1];
ccptn_clean_handler_t    P_H[1], Q_H[1];
ccmem_allocator_t const  *A = ccmem_standard_allocator;

if (cce_location(L)) {
  cce_run_catch_handlers_final(L);
} else {
  char const     *input = "/path/to/file.ext";
  ccptn_t const  *P, *R;

  P = ccname_init(ccptn_t, pointer,  clean)(L, A, P_H, input);
  Q = ccname_init(ccptn_t, filename, clean)(L, A, R_H, P);

  cce_run_body_handlers(L);
}
