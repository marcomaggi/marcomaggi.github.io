9 Pathname accessors

Function: char const * ccptn_ptr (ccptn_t const * P)

Return a pointer to an ASCIIZ representation of the pathname.

Function: size_t ccptn_len (ccptn_t const * P)

Return the number of octets in the pathname representation, not including the terminating zero.

Function: ccmem_ascii_t ccptn_ascii (ccptn_t const * P)

Return an ASCII block referencing the pathname representation; (ccmemory)Non–zero terminated strings.

Function: ccmem_asciiz_t ccptn_asciiz (ccptn_t const * P)

Return an ASCIIZ block referencing the pathname representation; (ccmemory)Zero terminated strings.

Function: ccmem_block_t ccptn_block (ccptn_t const * P)

Return a block referencing the pathname representation; (ccmemory)Blocks of memory. The pointer of the block references an ASCIIZ representation of the pathname; the length of the block represents the number of octets in the pathname representation, including the terminating zero.

Function: ccmem_allocator_t const * ccptn_allocator (ccptn_t const * P)

Return the memory allocator; (ccmemory)Type definitions.
