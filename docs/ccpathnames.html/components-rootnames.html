11.2 Extracting pathname rootnames

The rootname of an input pathname is a pathname representing the input with the extension of the last segment stripped. Examples: 

"/path/to/file.ext"     ⇒ "/path/to/file"
"/path/to/file"         ⇒ "/path/to/file"
"/path/to/dir/"         ⇒ "/path/to/dir"
"file.ext"              ⇒ "file.ext"
".fvwmrc"               ⇒ ".fvwmrc"
"."                     error→ no rootname
".."                    error→ no rootname
"/"                     error→ no rootname

It is an error to attempt rootname extraction from an input pathname that has no rootname; in this case the functions below will raise an exception of type ccptn_condition_no_rootname_t.

Rootname extraction is possible only for an input pathname that has been previously rootnamed; if the input is not rootnamed: the functions below will raise an exception of type ccptn_condition_rootnamed_pathname_t.
