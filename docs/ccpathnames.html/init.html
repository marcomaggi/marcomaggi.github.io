3 Library initialisation

Function: void ccptn_library_init (void)

Initialise the internal structures of the library. It is fine to call this function multiple times.

NOTE This function is defined with the GCC attribute constructor, so, when the library is built with GCC: it will be called automatically by the library initialisation code.
