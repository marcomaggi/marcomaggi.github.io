13.5 The deserialiser trait

The deserialiser trait offers a common API to deserialise a struct from a memory block for storage retrieval purposes; (ccstructs)The deserialiser trait.

Function: ccstructs_deserialiser_T ccname_trait_new(ccstructs_deserialiser_T, ccptn_t) (ccptn_t * P)

Build and return an instance of deserialiser trait which we can use to deserialise an instance of ccptn_t into the struct referenced by P.
