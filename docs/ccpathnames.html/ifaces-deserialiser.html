13.5 The deserialiser interface

The deserialiser interface offers a common API to deserialise a struct from a memory block for storage retrieval purposes; (ccstructs)The deserialiser interface.

Function: ccstructs_deserialiser_I ccname_iface_new(ccstructs_deserialiser_I, ccptn_t) (ccptn_t * P)

Build and return an instance of deserialiser interface which we can use to deserialise the instance of ccptn_t.
