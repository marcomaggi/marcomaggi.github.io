Pregexp for Cyclone

This document describes version 0.1.0-devel.1 of MMUX Cyclone Pregexp, a Cyclone language library implementing regular expressions. Cyclone is a Scheme-to-C compiler supporting the language features as defined in the “Revised^7 Report on Scheme”.

The package supports POSIX platforms. This package should work with Cyclone version 0.17. To run the test suite this package depends upon the external package: MMUX Cyclone Checks, at least version 0.1.0.
