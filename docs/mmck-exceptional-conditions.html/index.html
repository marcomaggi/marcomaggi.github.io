Exceptional Conditions for CHICKEN

This document describes version 0.1.0-devel.0 of MMCK Exceptional Conditions, a CHICKEN language library implementing facilities to handle exceptional–conditions. CHICKEN is a Scheme–to–C compiler supporting the language features as defined in the “Revised^5 Report on Scheme”.

The library supports POSIX platforms. This package has been tested with CHICKEN version 5.1.0.
