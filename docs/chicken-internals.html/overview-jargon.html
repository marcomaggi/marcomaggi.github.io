1.1 Jargon used in this document

Throughout this document some syntactic bindings are described as System Operations; we can think of a system operation as a Scheme macro that, in the end, directly expands into C language code integrated at the macro–use site. As usual with macros: we must take care of not causing multiple evaluation of expressions used as arguments to such operations.
