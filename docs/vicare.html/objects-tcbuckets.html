12.22 Components for hashtables

A tcbucket is a fixed–length memory block referenced by machine words tagged as vectors. 

     |------------------------|-------------| reference to tcbucket
           heap pointer         vector tag

A tcbucket is not a Scheme object like all the others; it is used only to store values in a hashtable. The memory layout of a tcbucket is as follows: 

     |-----|-----|-----|-----| tcbucket memory block
      tconc  key   val  next

For details on such objects, see the paper:

Ghuloum, Dybvig. “Generation–Friendly Eq Hash Tables”. Proceedings of the 2007 Workshop on Scheme and Functional Programming.
