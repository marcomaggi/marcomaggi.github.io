9.1.2 Raw memory mutators

Bit–sized values

In all the following definitions: pointer must be a pointer object, offset and value must be an exact integers.

— Function: ikptr ikrt_set_uint8 (ikptr pointer, ikptr offset, ikptr value)

Store the 8-bit unsigned integer value at offset from pointer.

— Function: ikptr ikrt_set_sint8 (ikptr pointer, ikptr offset, ikptr value)

Store the 8-bit signed integer value at offset from pointer.

— Function: ikptr ikrt_set_uint16 (ikptr pointer, ikptr offset, ikptr value)

Store the 16-bit unsigned integer value at offset from pointer.

— Function: ikptr ikrt_set_sint16 (ikptr pointer, ikptr offset, ikptr value)

Store the 16-bit signed integer value at offset from pointer.

— Function: ikptr ikrt_set_uint32 (ikptr pointer, ikptr offset, ikptr value)

Store the 32-bit unsigned integer value at offset from pointer.

— Function: ikptr ikrt_set_sint32 (ikptr pointer, ikptr offset, ikptr value)

Store the 32-bit signed integer value at offset from pointer.

— Function: ikptr ikrt_set_uint64 (ikptr pointer, ikptr offset, ikptr value)

Store the 64-bit unsigned integer value at offset from pointer.

— Function: ikptr ikrt_set_sint64 (ikptr pointer, ikptr offset, ikptr value)

Store the 64-bit signed integer value at offset from pointer.

C language integer type–sized values

In all the following definitions: pointer must be a pointer object, offset and value must be an exact integers.

— Function: ikptr ikrt_set_char (ikptr pointer, ikptr offset, ikptr value)

Store the exact integer value representing a C language signed char value at offset from pointer.

— Function: ikptr ikrt_set_uchar (ikptr pointer, ikptr offset, ikptr value)

Store the exact integer value representing a C language unsigned char value at offset from pointer.

— Function: ikptr ikrt_set_short (ikptr pointer, ikptr offset, ikptr value)

Store the exact integer value representing a C language signed short int value at offset from pointer.

— Function: ikptr ikrt_set_ushort (ikptr pointer, ikptr offset, ikptr value)

Store the exact integer value representing a C language unsigned short int value at offset from pointer.

— Function: ikptr ikrt_set_int (ikptr pointer, ikptr offset, ikptr value)

Store the exact integer value representing a C language signed int value at offset from pointer.

— Function: ikptr ikrt_set_uint (ikptr pointer, ikptr offset, ikptr value)

Store the exact integer value representing a C language unsigned int value at offset from pointer.

— Function: ikptr ikrt_set_long (ikptr pointer, ikptr offset, ikptr value)

Store the exact integer value representing a C language signed long value at offset from pointer.

— Function: ikptr ikrt_set_ulong (ikptr pointer, ikptr offset, ikptr value)

Store the exact integer value representing a C language unsigned long value at offset from pointer.

— Function: ikptr ikrt_set_longlong (ikptr pointer, ikptr offset, ikptr value)

Store the exact integer value representing a C language signed long long value at offset from pointer.

— Function: ikptr ikrt_set_ulonglong (ikptr pointer, ikptr offset, ikptr value)

Store the exact integer value representing a C language unsigned long long value at offset from pointer.

Other C language type–sized values

In all the following definitions: pointer must be a pointer object, offset must be an exact integer.

— Function: ikptr ikrt_set_float (ikptr pointer, ikptr offset, ikptr value)

Store the flonum value representing a C language float value at offset from pointer.

— Function: ikptr ikrt_set_double (ikptr pointer, ikptr offset, ikptr value)

Store the flonum value representing a C language double value at offset from pointer.

— Function: ikptr ikrt_set_pointer (ikptr pointer, ikptr offset, ikptr value)

Store the pointer value representing a C language void * value at offset from pointer.
