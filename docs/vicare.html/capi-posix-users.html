9.2.15 Users and groups

— Function: ikptr ikrt_posix_getuid (void)

Interface to the C function getuid(), getuid. Return a fixnum representing the real user ID of the process.

— Function: ikptr ikrt_posix_getgid (void)

Interface to the C function getgid(), getgid. Return a fixnum representing the real group ID of the process.

— Function: ikptr ikrt_posix_geteuid (void)

Interface to the C function geteuid(), geteuid. Return a fixnum representing the effective user ID of the process.

— Function: ikptr ikrt_posix_getegid (void)

Interface to the C function getegid(), getegid. Return a fixnum representing the effective group ID of the process.

— Function: ikptr ikrt_posix_getgroups (ikpcb * pcb)

Interface to the C function getgroups(), getgroups. Return a list of fixnums representing the supplementary group IDs of the process; if an error occurs return an encoded errno value.

— Function: ikptr ikrt_posix_seteuid (ikptr new_uid)

Interface to the C function seteuid(), seteuid. Set the effective user ID to new_uid, which must be a fixnum. If successful return the fixnum zero, else return an encoded errno value.

— Function: ikptr ikrt_posix_setuid (ikptr new_uid)

Interface to the C function setuid(), setuid. Set both the real and effective user IDs to new_uid, which must be a fixnum. If successful return the fixnum zero, else return an encoded errno value.

— Function: ikptr ikrt_posix_setreuid (ikptr real_uid, ikptr effective_uid)

Interface to the C function setreuid(), setreuid. Set the real user ID to real_uid and the effective user ID to effective_uid, both the arguments must be fixnums. If successful return the fixnum zero, else return an encoded errno value.

— Function: ikptr ikrt_posix_setegid (ikptr new_gid)

Interface to the C function setegid(), setegid. Set the effective group ID to new_gid, which must be a fixnum. If successful return the fixnum zero, else return an encoded errno value.

— Function: ikptr ikrt_posix_setgid (ikptr new_gid)

Interface to the C function setgid(), setgid. Set both the real and effective group IDs to new_gid, which must be a fixnum. If successful return the fixnum zero, else return an encoded errno value.

— Function: ikptr ikrt_posix_setregid (ikptr real_gid, ikptr effective_gid)

Interface to the C function setregid(), setregid. Set the real group ID to real_gid and the effective group ID to effective_gid, both the arguments must be fixnums. If successful return the fixnum zero, else return an encoded errno value.

— Function: ikptr ikrt_posix_getlogin (ikpcb * pcb)

Interface to the C function getlogin(), getlogin. Return a bytevector holding the ASCII coding of the name of the user logged in on the controlling terminal of the process; if the name cannot be determined: return false.

— Function: ikptr ikrt_posix_getpwuid (ikptr rtd, ikptr uid, ikpcb * pcb)

Interface to the C function getpwuid(), getpwuid. Retrieve informations on the user with identifier uid, which must be a fixnum. If successful return an instance of struct-passwd initialised with rtd, which must be its type descriptor; if a user with such an ID does not exists: return false.

— Function: ikptr ikrt_posix_getpwnam (ikptr rtd, ikptr name, ikpcb * pcb)

Interface to the C function getpwnam(), getpwnam. Retrieve informations on the user with name represented in ASCII coding by the bytevector name. If successful return an instance of struct-passwd initialised with rtd, which must be its type descriptor; if a user with such an ID does not exists: return false.

— Function: ikptr ikrt_posix_user_entries (ikptr rtd, ikpcb * pcb)

Interface to the C functions setpwent(), getpwent() and endpwent(), setpwent. Iterate the database of users and return a list of struct-passwd representing the entries; the structures are initialised with rtd which must be the type descriptor of struct-passwd.

— Function: ikptr ikrt_posix_getgrgid (ikptr rtd, ikptr gid, ikpcb * pcb)

Interface to the C function getgrgid(), getgrgid. Retrieve informations on the group with identifier gid, which must be a fixnum. If successful return an instance of struct-group initialised with rtd, which must be its type descriptor; if a group with such an ID does not exists: return false.

— Function: ikptr ikrt_posix_getgrnam (ikptr rtd, ikptr name, ikpcb * pcb)

Interface to the C function getgrnam(), getgrnam. Retrieve informations on the group with name represented in ASCII coding by the bytevector name. If successful return an instance of struct-group initialised with rtd, which must be its type descriptor; if a group with such an ID does not exists: return false.

— Function: ikptr ikrt_posix_group_entries (ikptr rtd, ikpcb * pcb)

Interface to the C functions setgrent(), getgrent() and endgrent(), setgrent. Iterate the database of groups and return a list of struct-group representing the entries; the structures are initialised with rtd which must be the type descriptor of struct-group.
