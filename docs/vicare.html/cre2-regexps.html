H.2 Precompiled regular expressions construction

The following bindings are exported by the (vicare cre2) library.

— Function: make-regexp pattern
— Function: make-regexp pattern opt

Build and return a precompiled regular expression object. The returned object must be finalised with delete-regexp; this operation is automatically performed when the object is garbage collected.

pattern must be a string or UTF-8 bytevector holding the regular expression pattern. When used, opt must be a configuration options object returned by make-options.

— Function: delete-regexp rex

Finalise a precompiled regular expression object releasing all the associated resources. This function can be safely applied multiple times to the same rex object: the first time the object is finalised, the subsequent times nothing happens.

— Function: regexp? obj

Return #t if obj is a precompiled regular expression object, return #f otherwise.
