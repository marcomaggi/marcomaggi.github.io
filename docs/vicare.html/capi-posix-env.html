9.2.3 Accessing environment variables

— Function: ikptr ikrt_posix_getenv (ikptr variable, ikpcb * pcb)

Interface to the C function getenv(), getenv. Retrieve the value of environment variables. variable must reference a bytevector object representing the name of the environment variable. If the environment variable is set: return a bytevector representing its value; else return false.

— Function: ikptr ikrt_posix_setenv (ikptr variable, ikptr value, ikptr overwrite)

Interface to the C function setenv(), setenv. Set a new value for an environment variable.

variable must reference a bytevector object representing the name of the environment variable; value must reference a bytevector object representing the new value. If overwrite is IK_FALSE_OBJECT and the environment variable already exists: the environment variable is left untouched; else the new value is set, either creating a new environment variable or replacing the old value.

If successful return IK_TRUE_OBJECT; if an error occurs in setenv() return IK_FALSE_OBJECT.

— Function: ikptr ikrt_posix_unsetenv (ikptr variable)

Interface to the C function unsetenv(), unsetenv. Unset an environment variable. variable must reference a bytevector object representing the name of the environment variable. Return true if variable has the correct format, else return false; there is no way to know if a variable was actually unset.

— Function: ikptr ikrt_posix_environ (ikpcb * pcb)

Interface to the global C variable environ, unsetenv. Retrieve the full environment. Return a list of bytevectors representing the contents of the environ array; if the environment is empty (no environment variables set) return nil.
