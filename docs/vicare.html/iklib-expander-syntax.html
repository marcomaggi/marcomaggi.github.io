3.4.1 Inspecting syntax objects

Under Vicare, a syntax object is the instance of a data structure. The following bindings are exported by the library (vicare).

— Function: syntax-object? stx

Return #t if stx is a syntax object, else return #f.

— Function: syntax-object-expression stx
— Function: syntax-object-marks stx
— Function: syntax-object-substs stx
— Function: syntax-object-source-objects stx

Field accessors for the syntax object structure type, to be used for debugging purposes.
