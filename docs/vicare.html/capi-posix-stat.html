9.2.9 File system inspection

— Function: ikptr ikrt_posix_stat (ikptr pathname, ikptr stat, ikpcb * pcb)
— Function: ikptr ikrt_posix_lstat (ikptr pathname, ikptr stat, ikpcb * pcb)

Interfaces to the C functions stat() and lstat(), stat. Inspect the file system entry selected by pathname, which must be a bytevector, and fill the struct-stat instance referenced by stat with the results. If successful return zero, else return an encoded errno value.

— Function: ikptr ikrt_posix_fstat (ikptr fd, ikptr stat, ikpcb * pcb)

Interface to the C function fstat(), fstat. Inspect the file system entry associated to the file descriptor fd, which must be a fixnum, and fill the struct-stat instance referenced by stat with the results. If successful return zero, else return an encoded errno value.

— Function: ikptr ikrt_posix_file_is_directory (ikptr pathname, ikptr follow)
— Function: ikptr ikrt_posix_file_is_char_device (ikptr pathname, ikptr follow)
— Function: ikptr ikrt_posix_file_is_block_device (ikptr pathname, ikptr follow)
— Function: ikptr ikrt_posix_file_is_regular_file (ikptr pathname, ikptr follow)
— Function: ikptr ikrt_posix_file_is_symbolic_link (ikptr pathname, ikptr follow)
— Function: ikptr ikrt_posix_file_is_socket (ikptr pathname, ikptr follow)
— Function: ikptr ikrt_posix_file_is_fifo (ikptr pathname, ikptr follow)
— Function: ikptr ikrt_posix_file_is_message_queue (ikptr pathname, ikptr follow)
— Function: ikptr ikrt_posix_file_is_semaphore (ikptr pathname, ikptr follow)
— Function: ikptr ikrt_posix_file_is_shared_memory (ikptr pathname, ikptr follow)

Return #t or #f whether the file system entry selected by pathname, which must be a bytevector, is of the specified type; if an error occurs return an encoded errno value. When follow is true: stat() is used to inspect the entry, else lstat() is used.

— Function: ikptr ikrt_posix_access (ikptr pathname, ikptr how)

Interface to the C function access(), access. Test the access mode selected by how, which must be a fixnum, for the file system entry selected by pathname, which must be a bytevector. Return #t or #f or an encoded errno value.

— Function: ikptr ikrt_posix_file_size (ikptr filename, ikpcb * pcb)

Determine the size of the file selected by filename, which must be a bytevector, relying on a call to stat(). If successful: return an exact integer representing the size, else return an encoded errno value.

— Function: ikptr ikrt_posix_file_atime (ikptr pathname, ikptr vector, ikpcb * pcb)
— Function: ikptr ikrt_posix_file_mtime (ikptr pathname, ikptr vector, ikpcb * pcb)
— Function: ikptr ikrt_posix_file_ctime (ikptr pathname, ikptr vector, ikpcb * pcb)

Compute an exact integer representing the access, modification and creation times for pathname, which must be a bytevector. If successful return zero, else return an encoded errno value.

The vector argument must be a vector which is mutated as follows: the first item holds an exact integer representing the number of seconds, the second item holds an exact integer representing the number of microseconds.
