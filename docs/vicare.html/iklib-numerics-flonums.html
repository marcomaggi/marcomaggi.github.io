3.13.2 Additional flonum functions

The following bindings are exported by the (ikarus) and (vicare) libraries.

Exponents and logarithms
— Function: fllog1p fl

Return a flonum equivalent to log(1 + fl).

— Function: flexpm1 fl

Return a flonum equivalent to exp(fl) - 1.
