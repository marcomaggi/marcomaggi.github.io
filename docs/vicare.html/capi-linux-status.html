9.3.1 Process termination status

— Function: ikptr ikrt_linux_waitid (ikptr idtype, ikptr id, ikptr info, ikptr options)

Interface to the Linux specific C function waitid(), see the waitid() manual page for details. Wait for the termination of one or more child processes with finer control of waitpid(). The arguments idtype, id and options must be fixnums. info must be a reference to a structure of type struct-siginfo_t, linux status for details. If successful: return info itself, else raise an exception.

The following functions accept as argument a fixnum representing the process termination status, for example the return value of system() or the status acquired by waitpid().

— Function: ikptr ikrt_linux_WIFCONTINUED (ikptr status)

Interface to the Linux specific C macro WIFCONTINUED(). Return true if the process was resumed by delivery of SIGCONT.
