9.2.12 File system directories

— Function: ikptr ikrt_posix_mkdir (ikptr pathname, ikptr mode)

Interface to the C function mkdir(), mkdir. Create a new directory file system entry with name pathname, which must be a bytevector; mode must be a fixnum specifying access permissions. If successful return the fixnum zero, else return an encoded errno value.

— Function: ikptr ikrt_posix_rmdir (ikptr pathname)

Interface to the C function rmdir(), rmdir. Remove the directory file system entry selected by pathname, which must be a bytevector. If successful return the fixnum zero, else return an encoded errno value.

— Function: ikptr ikrt_posix_getcwd (ikpcb * pcb)

Interface to the C function getcwd(), getcwd. Acquire the pathname of the current working directory. If successful return a bytevector holding the pathname, else return an encoded errno value.

— Function: ikptr ikrt_posix_chdir (ikptr pathname)

Interface to the C function chdir(), chdir. Change the current working directory to the file system entry selected by pathname, which must be a bytevector. If successful return the fixnum zero, else return an encoded errno value.

— Function: ikptr ikrt_posix_fchdir (ikptr fd)

Interface to the C function fchdir(), fchdir. Change the current working directory to the file system entry associated to fd, which must be a fixnum representing a file descriptor. If successful return the fixnum zero, else return an encoded errno value.

— Function: ikptr ikrt_posix_opendir (ikptr pathname, ikpcb * pcb)

Interface to the C function opendir(), opendir. Open a directory stream for the inspection of the file system entry pathname, which must be a bytevector. If successful return a pointer object, else return an encoded errno value.

— Function: ikptr ikrt_posix_fdopendir (ikptr fd, ikpcb * pcb)

Interface to the C function fdopendir(), fdopendir. Open a directory stream for the inspection of the file system entry associated to fd, which must be a fixnum representing a file descriptor. If successful return a pointer object, else return an encoded errno value.

— Function: ikptr ikrt_posix_readdir (ikptr pointer, ikpcb * pcb)

Interface to the C function readdir(), readdir. Acquire the next entry from a directory stream referenced by pointer, which must be a pointer object. If successful and an entry is available: return a pointer object; if successful and no more entries are available: return the false object; else return an encoded errno value.

When no more entries are available or an error occurs: the directory stream is closed.

— Function: ikptr ikrt_posix_closedir (ikptr pointer, ikpcb * pcb)

Interface to the C function closedir(), closedir. Close the directory stream referenced by pointer, which must be a pointer object. If successful return the fixnum zero, else return an encoded errno value.

— Function: ikptr ikrt_posix_rewinddir (ikptr pointer)

Interface to the C function rewinddir(), rewinddir. Rewind to the beginning the directory stream referenced by pointer, which must be a pointer object. Return the void object.

— Function: ikptr ikrt_posix_telldir (ikptr pointer)

Interface to the C function telldir(), telldir. Return an exact integer representing the current position in the directory stream referenced by pointer, which must be a pointer object.

— Function: ikptr ikrt_posix_seekdir (ikptr pointer, ikptr pos)

Interface to the C function seekdir(), seekdir. Set to pos the position of the directory stream referenced by pointer, which must be a pointer object; pos must be the return value of a previous call to telldir(). Return the void object.
