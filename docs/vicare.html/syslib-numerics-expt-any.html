9.8.19.1 Any number as first argument
— Unsafe Operation: $expt-number-fixnum num fx

Compute num raised to the power fx; return an unspecified number object.

— Unsafe Operation: $expt-number-bignum num bn

Compute num raised to the power bn; return an unspecified number object.

— Unsafe Operation: $expt-number-flonum num fl

Compute num raised to the power fl; return an unspecified inexact number object.

— Unsafe Operation: $expt-number-ratnum num rt

Compute num raised to the power rt; return an unspecified number object.

— Unsafe Operation: $expt-number-compnum num cn

Compute num raised to the power cn; return an unspecified number object, but most likely a complex number.

— Unsafe Operation: $expt-number-cflonum num cfl

Compute num raised to the power cfl; return an unspecified inexact complex number object.
