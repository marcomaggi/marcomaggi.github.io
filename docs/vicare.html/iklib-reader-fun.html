6.5.5 Additional reader functions

The following bindings are exported by the library (vicare).

— Function: get-annotated-datum port

Like get-datum, but rather than returning a datum return a hierarchy of annotation objects with the same hierarchy of the datum and embedding the datum itself.

— Function: annotation? obj

Return #t if obj is an annotation object, else return #f.

— Function: annotation-expression ann

Accessor for the expression field of an annotation object. Return a list, vector, identifier, what–have–you that may contain further annotations.

— Function: annotation-stripped ann

Accessor for the stripped field of an annotation object. The same symbolic expression of the expression field with no annotations.

— Function: annotation-textual-position ann

Accessor for the textual-position field of an annotation object. A condition object of type &source-position representing the position of the expression in the source code.

— Function: annotation-source ann

A pair whose car is the port identifier and whose cdr is the offset of the character.
