16.17.1 Data type definitions

The following bindings are exported by the library (vicare net channels).

Base channel type
— R6RS Record Type: channel

Record type representing an interprocess connection channel. Whenever this documentation refers to arguments being instances of this type: it uses the identifier chan.

— Function: channel? obj

Return #t if obj is an instance of channel, else return #f.

— Validation Clause: channel obj
— Validation Clause: false-or-channel obj

Validation clause to be used with the facilities of the library (vicare arguments validation). The first form succeeds if obj is an instance of channel; the second also accepts #f.
