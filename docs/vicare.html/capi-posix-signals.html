9.2.8 Delivering interprocess signals

— Function: ikptr ikrt_posix_raise (ikptr signum)

Interface to the C function raise(), raise. Send the selected signal to the calling process. If successful return zero, else return an encoded errno code.

— Function: ikptr ikrt_posix_kill (ikptr pid, ikptr signum)

Interface to the C function kill(), kill. Send the selected signal to selected process. If successful return zero, else return an encoded errno code.

— Function: ikptr ikrt_posix_pause (void)

Interface to the C function pause(), pause. Suspend the process until a signal is received. Return the void object.
