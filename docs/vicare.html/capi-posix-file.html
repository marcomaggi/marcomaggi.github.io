9.2.10 File system interface

— Function: ikptr ikrt_posix_chown (ikptr pathname, ikptr owner, ikptr group)

Interface to the C function chown(), chown. Changes the owner and group of the file system entry selected by pathname, which must be a bytevector. owner and group must be fixnums. If successful return zero, else return an encoded errno value.

— Function: ikptr ikrt_posix_fchown (ikptr fd, ikptr owner, ikptr group)

Interface to the C function fchown(), fchown. Changes the owner and group of the file system entry selected by fd, which must be a fixnum representing a platform file descriptor. owner and group must be fixnums. If successful return zero, else return an encoded errno value.

— Function: ikptr ikrt_posix_chmod (ikptr pathname, ikptr mode)

Interface to the C function chmod(), chmod. Change to mode, which must be a fixnum, the access permissions for the file system entry selected by pathname, which must be a bytevector. If successful return zero, else return an encoded errno value.

— Function: ikptr ikrt_posix_fchmod (ikptr fd, ikptr mode)

Interface to the C function fchmod(), fchmod. Change to mode, which must be a fixnum, the access permissions for the file system entry associated to fd, which must be a fixnum representing a platform file descriptor. If successful return zero, else return an encoded errno value.

— Function: ikptr ikrt_posix_umask (ikptr mask)

Interface to the C function umask(), umask. Set the current file creation mask to mask which must be a fixnum and return the previous mask as a fixnum.

— Function: ikptr ikrt_posix_getumask (void)

Interface to the C function getumask(), getumask. Return the current file creation mask as a fixnum.

— Function: ikptr ikrt_posix_utime (ikptr pathname, ikptr atime_sec, ikptr mtime_sec)

Interface to the C function utime(), utime. Set the access and modification times of the file system entry selected by pathname, which must be a bytevector, to the specified counts of seconds; both atime_sec and mtime_sec must be fixnums. If successful return zero, else return an encoded errno value.

— Function: ikptr ikrt_posix_utimes (ikptr pathname, ikptr atime_sec, ikptr atime_usec, ikptr mtime_sec, ikptr mtime_usec)
— Function: ikptr ikrt_posix_lutimes (ikptr pathname, ikptr atime_sec, ikptr atime_usec, ikptr mtime_sec, ikptr mtime_usec)

Interface to the C functions utimes() and lutimes(), utimes. Set the access and modification times of the file system entry selected by pathname, which must be a bytevector, to the specified counts of seconds and microseconds; all the numeric arguments must be fixnums. If successful return zero, else return an encoded errno value.

— Function: ikptr ikrt_posix_futimes (ikptr fd, ikptr atime_sec, ikptr atime_usec, ikptr mtime_sec, ikptr mtime_usec)

Interface to the C function futimes(), futimes. Set the access and modification times of the file system entry associated to fd, which must be a fixnum representing a file descriptor, to the specified counts of seconds and microseconds; all the numeric arguments must be fixnums. If successful return zero, else return an encoded errno value.
