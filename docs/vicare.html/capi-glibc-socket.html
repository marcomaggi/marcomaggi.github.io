13.4.5 Network sockets

— Function: ikptr ikrt_glibc_if_nametoindex (ikptr name)

Interface to the C function if_nametoindex(), if_nametoindex. Convert a string representing a network interface to the corresponding internal index. name must be a bytevector holding the UTF-8 representation of the string. Return a fixnum representing the index or false if name is invalid.

— Function: ikptr ikrt_glibc_if_indextoname (ikptr index, ikpcb * pcb)

Interface to the C function if_indextoname(), if_indextoname. Convert the index associated to an interface to the corresponding string name. index must be a fixnum representing the index. Return a bytevector holding the name or false if index is invalid.

— Function: ikptr ikrt_glibc_if_nameindex (ikpcb * pcb)

Interface to the C function if_nameindex(), if_nameindex. Build and return an association list having fixnums as keys and bytevectors as values; the fixnums are the interface indexes, the bytevectors are the corresponding interface names.

— Function: ikptr ikrt_glibc_gethostbyname2 (ikptr rtd, ikptr hostname, ikptr addrtype, ikpcb * pcb)

Interface to the C function gethostbyname2(), gethostbyname2. Like ikrt_posix_gethostbyname() but allows the selection of the address type with addrtype which must be AF_INET or AF_INET6.
