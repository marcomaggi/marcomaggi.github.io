17.26.2 Macros extracting void *

— Preprocessor Macro: void * IK_POINTER_FROM_POINTER_OR_FALSE (ikptr obj)
— Preprocessor Macro: void * IK_VOIDP_FROM_POINTER_OR_FALSE (ikptr obj)

Given #f or a pointer object return a void * pointer referencing the data; when obj is #f: the returned value is NULL.
