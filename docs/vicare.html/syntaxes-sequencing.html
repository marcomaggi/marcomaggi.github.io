10.12.3 Sequencing execution

The following bindings are exported by the library (vicare syntactic-extensions).

— Macro: begin0 ?expr0 ?expr ...

Expand to: 

          (call-with-values
               (lambda () ?expr0)
            (lambda args
              ?expr ...
              (apply values args)))

This syntax comes from the R6RS original document, Appendix A “Formal semantics”.

— Macro: begin0-let ((?var0 ?init0) (?var ?init) ...) ?form0 ?form ...

Expand to: 

          (let ((?var0 ?init0)
                (?var  ?init)
                ...)
            ?form0 ?form ...
            ?var0)
