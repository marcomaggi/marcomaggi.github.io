9.4.3 Temporary files and directories

— Function: ikptr ikrt_glibc_mkstemp (ikptr template, ikpcb * pcb)

Interface to the C function mkstemp(), mkstemp. Create a temporary file, open it and open a file descriptor for reading and writing. template must be a mutable bytevector holding a template pathname on entering and the actual pathname on returning. If successful return a file descriptor, else return an encoded errno value.

— Function: ikptr ikrt_glibc_mkdtemp (ikptr template, ikpcb * pcb)

Interface to the C function mkdtemp(), mkdtemp. Create a temporary directory for reading and writing. template must be a mutable bytevector holding a template pathname on entering and the actual pathname on returning. If successful return template itself, else return an encoded errno value.
