14.26.3 Macros extracting char *

— Preprocessor Macro: char * IK_POINTER_FROM_POINTER_OR_FALSE (ikptr obj)
— Preprocessor Macro: char * IK_CHARP_FROM_POINTER_OR_FALSE (ikptr obj)

Given #f or a pointer object return a char * pointer referencing the data; when obj is #f: the returned value is NULL.
