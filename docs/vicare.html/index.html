Vicare Scheme

This document describes version 0.2d9 of Vicare Scheme, an R6RS compliant native compiler for the Scheme language producing single threaded programs running on Intel x86 32-bit processors, with experimental support for 64-bit machines. Vicare is pronounced the etruscan way.
