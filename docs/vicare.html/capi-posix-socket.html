9.2.14 Network sockets

Address structures
— Function: ikptr ikrt_posix_make_sockaddr_un (ikptr pathname, ikpcb * pcb)

Given a bytevector pathname representing the pathname of a local socket, build and return a bytevector holding the struct sockaddr_un representing it.

— Function: ikptr ikrt_posix_sockaddr_un_pathname (ikptr socket_address, ikpcb * pcb)

Given a bytevector socket_address, which must be the return value of a previous call to ikrt_posix_make_sockaddr_un(), holding a struct sockaddr_un build and return a bytevector holding the corresponding socket pathname. If the data in socket_address is invalid: return false.

— Function: ikptr ikrt_posix_make_sockaddr_in (ikptr host_address, ikptr port, ikpcb * pcb)

Given a bytevector host_address holding a struct in_addr and a fixnum port representing a network port number, build and return a bytevector holding a struct sockaddr_in initialised with this data.

— Function: ikptr ikrt_posix_sockaddr_in_in_addr (ikptr socket_address, ikpcb * pcb)
— Function: ikptr ikrt_posix_sockaddr_in_in_port (ikptr socket_address)

Given a bytevector socket_address holding a struct sockaddr_in, extract the in_addr and in_port fields and return them as bytevector holding the corresponding struct in_addr and as fixnum. If the data in socket_address is invalid: return false.

— Function: ikptr ikrt_posix_make_sockaddr_in6 (ikptr host_address, ikptr port, ikpcb * pcb)

Given a bytevector host_address holding a struct in6_addr and a fixnum port representing a network port number, build and return a bytevector holding a struct sockaddr_in6 initialised with this data.

— Function: ikptr ikrt_posix_sockaddr_in6_in6_addr (ikptr socket_address, ikpcb * pcb)
— Function: ikptr ikrt_posix_sockaddr_in6_in6_port (ikptr socket_address)

Given a bytevector socket_address holding a struct sockaddr_in6, extract the in6_addr and in6_port fields and return them as bytevector holding the corresponding struct in6_addr and as fixnum. If the data in socket_address is invalid: return false.

— Function: ikptr ikrt_posix_in6addr_loopback (ikpcb * pcb)
— Function: ikptr ikrt_posix_in6addr_any (ikpcb * pcb)

Build and return a bytevector holding the constant structures of type struct in6_addr defined by in6addr_loopback and in6addr_any.

— Function: ikptr ikrt_inet_aton (ikptr dotted_quad, ikpcb * pcb)

Given the bytevector dotted_quad holding the ASCII dotted quad representation of a host address, build and return a bytevector holding the corresponding struct in_addr. If dotted_quad is invalid: return false.

— Function: ikptr ikrt_inet_ntoa (ikptr host_address, ikpcb * pcb)

Given the bytevector host_address holding a struct in_addr, build and return a bytevector holding the corresponding ASCII dotted quad representation.

— Function: ikptr ikrt_posix_inet_pton (ikptr af, ikptr presentation, ikpcb * pcb)

Given the fixnum af, being AF_INET or AF_INET6, and the bytevector presentation holding the ASCII presentation of an Internet address, build and return a bytevector holding the corresponding struct in_addr or struct in6_addr. If either af or presentation is invalid: return false.

— Function: ikptr ikrt_posix_inet_ntop (ikptr af, ikptr host_address, ikpcb * pcb)

Given the fixnum af, being AF_INET or AF_INET6, and the bytevector host_address holding a struct in_addr or struct in6_addr, build and return a bytevector holding the corresponding ASCII presentation of the Internet address . If the arguments are invalid: return false.

Host address resolution
— Function: ikptr ikrt_posix_gethostbyname (ikptr rtd, ikptr hostname, ikpcb * pcb)

Interface to the C function gethostbyname(), gethostbyname. Given the bytevector hostname holding the ASCII representation of a host name, build and return an instance of struct-hostent using rtd as its type descriptor. If an error occurs, return an encoded h_errno value.

— Function: ikptr ikrt_posix_gethostbyaddr (ikptr rtd, ikptr addr, ikpcb * pcb)

Interface to the C function gethostbyaddr(), gethostbyaddr. Given the bytevector addr holding a struct in_addr or a struct in6_addr, build and return an instance of struct-hostent using rtd as its type descriptor; the type of address is automatically inferred from the length of the bytevector. If an error occurs, return an encoded h_errno value.

— Function: ikptr ikrt_posix_host_entries (ikptr rtd, ikpcb * pcb)

Interface to the C functions sethostent(), gethostent() and endhostent(), gethostbyaddr. Build and return a list of struct-hostent representing the entries in the hosts database, using rtd as type descriptor.

— Function: ikptr ikrt_posix_getaddrinfo (ikptr rtd, ikptr node, ikptr service, ikptr hints, ikpcb * pcb)

Interface to the C function getaddrinfo(), see the manual page getaddrinfo(3). Given the bytevectors node and service identifying an Internet host and a service, build a list of struct-addrinfo instances representing addresses to which sockets can be bound or connected.

Both node and service can be false, in which case the corresponding arguments handed to the C function are NULL.

hints must be an instance of struct-addrinfo used to select matching hosts. rtd must be the type descriptor of struct-addrinfo.

If successful: return the list of data structures, else return a fixnum representing an EAI_ error code.

— Function: ikptr ikrt_posix_gai_strerror (ikptr error_code, ikpcb * pcb)

Interface to the C function gai_strerror(), see the manual page gai_strerror(3). Convert a EAI_ error code into the corresponding error message, return a bytevector holding such message in ASCII coding.

Network protocols
— Function: ikptr ikrt_posix_getprotobyname (ikptr rtd, ikptr name, ikpcb * pcb)

Interface to the C function getprotobyname(), getprotobyname. Given a bytevector name holding the ASCII coding of a network protocol name, query the network protocols database and build an instance of struct-protoent mirroring the resulting struct protoent entry; rtd must be the type descriptor of struct-protoent. If successful return the data structure, else return false.

— Function: ikptr ikrt_posix_getprotobynumber (ikptr rtd, ikptr proto_num, ikpcb * pcb)

Interface to the C function getprotobynumber(), getprotobynumber. Given a fixnum proto_num being the number of a network protocol, query the network protocols database and build an instance of struct-protoent mirroring the resulting struct protoent entry; rtd must be the type descriptor of struct-protoent. If successful return the data structure, else return false.

— Function: ikptr ikrt_posix_protocol_entries (ikptr rtd, ikpcb * pcb)

Interface to the C functions setprotoent(), getprotoent() and endprotoent(), setprotoent. Scan the network protocol database and build a list of struct-protoent mirroring the resulting struct protoent entries; rtd must be the type descriptor of struct-protoent. Return the list of structures.

Network services
— Function: ikptr ikrt_posix_getservbyname (ikptr rtd, ikptr name, ikptr protocol, ikpcb * pcb)

Interface to the C function getservbyname(), getservbyname. Given the bytevectors name and protocol holding the ASCII coding of a network service name and protocol, query the network services database and build an instance of struct-servent mirroring the resulting struct servent entry. rtd must be the type descriptor of struct-servent. If successful return the data structure, else return false.

— Function: ikptr ikrt_posix_getservbynumber (ikptr rtd, ikptr port, ikptr protocol, ikpcb * pcb)

Interface to the C function getservbynumber(), getservbynumber. Given a fixnum port and a string or bytevector protocol being the port number and protocol name of a network service, query the network services database and build an instance of struct-servent mirroring the resulting struct servent entry. rtd must be the type descriptor of struct-servent. If successful return the data structure, else return false.

— Function: ikptr ikrt_posix_service_entries (ikptr rtd, ikpcb * pcb)

Interface to the C functions setservent(), getservent() and endservent(), setservent. Scan the network service database and build a list of struct-servent mirroring the resulting struct servent entries. rtd must be the type descriptor of struct-servent. Return the list of structures.

Networks database
— Function: ikptr ikrt_posix_getnetbyname (ikptr rtd, ikptr name, ikpcb * pcb)

Interface to the C function getnetbyname(), getnetbyname. Given the bytevector name holding the ASCII coding of a network name, query the networks database and build an instance of struct-netent mirroring the resulting struct netent entry. rtd must be the type descriptor of struct-netent. If successful return the data structure, else return false.

— Function: ikptr ikrt_posix_getnetbyaddr (ikptr rtd, ikptr net, ikptr type, ikpcb * pcb)

Interface to the C function getnetbyaddr(), getnetbyaddr. Given an exact integer net representing a network number and the fixnum type representing a network type, query the networks database and build an instance of struct-netent mirroring the resulting struct netent entry. rtd must be the type descriptor of struct-netent. If successful return the data structure, else return false.

— Function: ikptr ikrt_posix_network_entries (ikptr rtd, ikpcb * pcb)

Interface to the C functions setnetent(), getnetent() and endnetent(), setnetent. Scan the networks database and build a list of struct-netent mirroring the resulting struct netent entries. rtd must be the type descriptor of struct-netent. Return the list of structures.

Sockets creation and shutdown
— Function: ikptr ikrt_posix_socket (ikptr namespace, ikptr style, ikptr protocol)

Interface to the C function socket(), socket. Create a new socket descriptor and return it as non–negative fixnum; if an error occurs return an encoded errno value. All the arguments must be fixnums.

— Function: ikptr ikrt_posix_shutdown (ikptr sock, ikptr how)

Interface to the C function shutdown(), shutdown. Close the socket sock according to how. If successful return the fixnum zero, else return an encoded errno value.

— Function: ikptr ikrt_posix_socketpair (ikptr namespace, ikptr style, ikptr protocol, ikpcb * pcb)

Interface to the C function socketpair(), socketpair. Create a pairs of connected sockets and return a pair holding the descriptors as non–negative fixnums; if an error occurs return an encoded errno value. All the arguments must be fixnums, but pcb.

Socket operations
— Function: ikptr ikrt_posix_connect (ikptr sock, ikptr addr)

Interface to the C function connect(), connect. Connect the socket sock to the address specified by the bytevector addr which must hold a concrete instance of struct sockaddr. If successful return the fixnum zero, else return an encoded errno value.

— Function: ikptr ikrt_posix_listen (ikptr sock, ikptr pending_conns)

Interface to the C function listen(), listen. Enable the socket sock to accept connections; the fixnum pending_conns specifies the maximum length of the pending connection requests queue. If successful return the fixnum zero, else return an encoded errno value.

— Function: ikptr ikrt_posix_accept (ikptr sock, ikpcb * pcb)

Interface to the C function accept(), accept. Accept an incoming connection to the server socket sock. If successful return a pair whose car is a non–negative fixnum representing the socket and whose cdr is a bytevector representing the client address as struct sockaddr; else return an encoded errno value, including EWOULDBLOCK which means that sock is in non–blocking mode and no pending connections exist.

— Function: ikptr ikrt_posix_bind (ikptr sock, ikptr sockaddr)

Interface to the C function bind(), bind. Bind the socket descriptor sock to the address specified by the struct sockaddr in the bytevector sockaddr. If successful return the fixnum zero, else return an encoded errno value.

— Function: ikptr ikrt_posix_getpeername (ikptr sock, ikpcb * pcb)

Interface to the C function getpeername(), getpeername. Retrieve informations about the address of the socket to which the sock socket is connected to. If successful return a bytevector holding a struct sockaddr, else return an encoded errno value.

— Function: ikptr ikrt_posix_getsockname (ikptr sock, ikpcb * pcb)

Interface to the C function getsockanme(), getsockname. Retrieve informations about the address bound to the socket descriptor sock. If successful return a bytevector holding the struct sockaddr, else return an encoded errno value.

— Function: ikptr ikrt_posix_send (ikptr sock, ikptr buffer, ikptr size, ikptr flags)

Interface to the C function send(), send. Like write() but with the additional argument flags which must be a fixnum: write data from the bytevector buffer to the socket sock; if size is a fixnum, write size bytes; if size is false, write a number of bytes equal to the length of buffer. If successful return a non–negative fixnum representing the number of bytes actually sent, else return an encoded errno value.

— Function: ikptr ikrt_posix_recv (ikptr sock, ikptr buffer, ikptr size, ikptr flags)

Interface to the C function recv(), recv. Like read() but with the additional argument flags which must be a fixnum: read data from sock and store it in the bytevector buffer; if size is a fixnum, read size bytes; if size is false, read a number of bytes equal to the length of buffer. If successful return a non–negative fixnum representing the number of bytes actually received, else return an encoded errno value.

— Function: ikptr ikrt_posix_sendto (ikptr sock, ikptr buffer, ikptr size, ikptr flags, ikptr addr)

Interface to the C function sendto(), sendto. Like send() but with the additional argument addr which must be a bytevector holding a struct sockaddr: write data from the bytevector buffer to the socket sock to the destination specified by addr.

If size is a fixnum, write size bytes; if size is false, write a number of bytes equal to the length of buffer.

If successful return a non–negative fixnum representing the number of bytes actually sent, else return an encoded errno value.

— Function: ikptr ikrt_posix_recvfrom (ikptr sock, ikptr buffer, ikptr size, ikptr flags, ikpcb * pcb)

Interface to the C function recvfrom(), recvfrom. Like recv() but additionally retrieve informations about the address of the sender: read data from sock and store it in the bytevector buffer.

If size is a fixnum, read size bytes; if size is false, read a number of bytes equal to the length of buffer.

If successful return a pair whose car is a non–negative fixnum representing the number of bytes actually received and whose cdr is a bytevector holding a struct sockaddr representing the address of the sender; else return an encoded errno value.

— Function: ikptr ikrt_posix_getsockopt (ikptr sock, ikptr level, ikptr option, ikptr optval, ikpcb * pcb)

Interface to the C function getsockopt(), getsockopt. Retrieve the value of option of socket sock at level and store it in the bytevector optval. option and level must be fixnums. If successful return the fixnum zero, else return an encoded errno value.

— Function: ikptr ikrt_posix_setsockopt (ikptr sock, ikptr level, ikptr option, ikptr optval, ikpcb * pcb)

Interface to the C function setsockopt(), setsockopt. Set a new value for option of socket sock at level reading it from the bytevector optval. option and level must be fixnums. If successful return the fixnum zero, else return an encoded errno value.
