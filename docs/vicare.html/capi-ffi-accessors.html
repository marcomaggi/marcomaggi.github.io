9.1.1 Raw memory accessors

Bit–sized values

In all the following definitions: pointer must be a pointer object, offset must be an exact integer.

— Function: ikptr ikrt_ref_uint8 (ikptr pointer, ikptr offset)

Extract and return the 8-bit unsigned integer at offset from pointer.

— Function: ikptr ikrt_ref_sint8 (ikptr pointer, ikptr offset)

Extract and return the 8-bit signed integer at offset from pointer.

— Function: ikptr ikrt_ref_uint16 (ikptr pointer, ikptr offset)

Extract and return the 16-bit unsigned integer at offset from pointer.

— Function: ikptr ikrt_ref_sint16 (ikptr pointer, ikptr offset)

Extract and return the 16-bit signed integer at offset from pointer.

— Function: ikptr ikrt_ref_uint32 (ikptr pointer, ikptr offset, ikpcb * pcb)

Extract and return the 32-bit unsigned integer at offset from pointer.

— Function: ikptr ikrt_ref_sint32 (ikptr pointer, ikptr offset, ikpcb * pcb)

Extract and return the 32-bit signed integer at offset from pointer.

— Function: ikptr ikrt_ref_uint64 (ikptr pointer, ikptr offset, ikpcb * pcb)

Extract and return the 64-bit unsigned integer at offset from pointer.

— Function: ikptr ikrt_ref_sint64 (ikptr pointer, ikptr offset, ikpcb * pcb)

Extract and return the 64-bit signed integer at offset from pointer.

C language integer type–sized values

In all the following definitions: pointer must be a pointer object, offset must be an exact integer.

— Function: ikptr ikrt_ref_char (ikptr pointer, ikptr offset)

Extract and return the exact integer representation of the C language signed char value at offset from pointer.

— Function: ikptr ikrt_ref_uchar (ikptr pointer, ikptr offset)

Extract and return the exact integer representation of the C language unsigned char value at offset from pointer.

— Function: ikptr ikrt_ref_short (ikptr pointer, ikptr offset)

Extract and return the exact integer representation of the C language signed short int value at offset from pointer.

— Function: ikptr ikrt_ref_ushort (ikptr pointer, ikptr offset)

Extract and return the exact integer representation of the C language unsigned short int value at offset from pointer.

— Function: ikptr ikrt_ref_long (ikptr pointer, ikptr offset)

Extract and return the exact integer representation of the C language signed long value at offset from pointer.

— Function: ikptr ikrt_ref_ulong (ikptr pointer, ikptr offset)

Extract and return the exact integer representation of the C language unsigned long value at offset from pointer.

— Function: ikptr ikrt_ref_longlong (ikptr pointer, ikptr offset)

Extract and return the exact integer representation of the C language signed long long value at offset from pointer.

— Function: ikptr ikrt_ref_ulonglong (ikptr pointer, ikptr offset)

Extract and return the exact integer representation of the C language unsigned long long value at offset from pointer.

Other C language type–sized values

In all the following definitions: pointer must be a pointer object, offset must be an exact integer.

— Function: ikptr ikrt_ref_float (ikptr pointer, ikptr offset)

Extract and return the flonum representation of the C language float value at offset from pointer.

— Function: ikptr ikrt_ref_double (ikptr pointer, ikptr offset)

Extract and return the flonum representation of the C language double value at offset from pointer.

— Function: ikptr ikrt_ref_pointer (ikptr pointer, ikptr offset)

Extract and return the pointer representation of the C language void * value at offset from pointer.
