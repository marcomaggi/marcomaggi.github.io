9.4.2 File system directories

— Function: ikptr ikrt_glibc_dirfd (ikptr pointer)

Interface to the C function dirfd(), dirfd. Acquire a file descriptor associated to the directory entry referenced by pointer, which must be a pointer object to a directory entry. If successful return a fixnum representing the file descriptor, else return an encoded errno value.
