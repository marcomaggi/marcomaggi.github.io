6.41 Basic time and date functions

The following bindings are exported by the libraries (vicare), (ikarus).

— Function: current-time

Return an object of disjoint type representing the current time as determined by gettimeofday(): this is the elapsed time since the Epoch.

— Function: time-from-now delta

Return a time object representing the current time plus the time object delta.

— Function: make-time seconds nanoseconds

Build and return a new time object with microseconds resolution. If nanoseconds exceeds 1000000 its value is normalised and the excess added to seconds.

— Function: time? obj

Return #t if obj is a time object returned by current-time.

— Function: time-second time
— Function: time-nanosecond time

Accessors, returning exact integers, for the seconds and nanoseconds components of the time objects returned by current-time.

— Function: time-gmt-offset time

Return the GMT offset in seconds of the time object returned by current-time, as determined ty gmtime_r().

— Function: date-string

Return a string representing the current date in the simple format: year, month, day.

Time operations
— Function: time-addition time1 time2

Compute the addition between two times: time1 + time2 and return a time struct representing it.

— Function: time-difference time1 time2

Compute the difference between two times: time1 - time2 and return a time struct representing it.

Time comparison
— Function: time=? time1 time2

Return #t if the time objects are equal, else return #f.

— Function: time<? time1 time2

Return #t if the time1 is less than time2, else return #f.

— Function: time<=? time1 time2

Return #t if the time1 is less than or equal to time2, else return #f.

— Function: time>? time1 time2

Return #t if the time1 is greater than time2, else return #f.

— Function: time>=? time1 time2

Return #t if the time1 is greater than or equal to time2, else return #f.
