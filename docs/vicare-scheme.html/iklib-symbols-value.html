6.21.1 Symbol values

Symbol values have miscellaneous uses: when a symbol is the unique identifier for a data structure, its symbol value field is set to the type descriptor of the data structure; when a symbol is the unique label of a binding in the expander, its symbol value is a reference to the binding. syslib symbols for more details on symbol values.

The following bindings are exported by the (ikarus) and the (ikarus symbols) libraries.

— Function: set-symbol-value! sym value

Store value in the $symbol-value field of the symbol sym.

— Function: symbol-value sym

Return the value in the $symbol-value field of the symbol sym.

— Function: symbol-bound? sym

Return true if sym is a symbol and its $symbol-value field is not set to the special “unbound” machine word value. objects symbols for details.
