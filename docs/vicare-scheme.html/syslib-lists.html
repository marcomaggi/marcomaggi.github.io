8.11 Low level list operations

— Unsafe Operation: $length ell

Assume ell is a proper list, null included: compute and return the length of ell.

— Unsafe Operation: $map1 proc ell

Assume ell is a proper list, null included, and proc is a unary function: execute the same algorithm of map from (rnrs lists (6)), but use unsafe operations.

— Unsafe Operation: $for-each1 proc ell

Assume ell is a proper list, null included, and proc is a unary function: execute the same algorithm of for-each from (rnrs lists (6)), but use unsafe operations.

— Unsafe Operation: $for-all1 proc ell

Assume ell is a proper list, null included, and proc is a unary function: execute the same algorithm of for-all from (rnrs lists (6)), but use unsafe operations.

— Unsafe Operation: $exists1 proc ell

Assume ell is a proper list, null included, and proc is a unary function: execute the same algorithm of exists from (rnrs lists (6)), but use unsafe operations.
