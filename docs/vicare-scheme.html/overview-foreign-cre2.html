1.9.2 Interface to CRE2

CRE2 is a C language wrapper for the RE2 library, which is implemented in C++. RE2 is a fast, safe, thread–friendly alternative to backtracking regular expression engines like those used in PCRE, Perl, and Python; it is a Google project found at:

http://code.google.com/p/re2/

while CRE2 is available at:

http://github.com/marcomaggi/cre2/

Vicare interface to CRE2 can be included at package configuration time by giving the --with-cre2 option to the configure script. A binding to the library CRE2 for the API documentation.
