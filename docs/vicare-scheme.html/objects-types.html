13.1 Some basic type definitions

Typedef: iksword_t

The type of signed integers matching a machine word size. On 32-bit platforms it is an alias for int32_t. On 64-bit platforms it is an alias for int64_t.

Typedef: ikuword_t

The type of unsigned integers matching a machine word size. On 32-bit platforms it is an alias for uint32_t. On 64-bit platforms it is an alias for uint64_t.

Typedef: ik_int

An alias for signed int.

Typedef: ik_long

An alias for signed long.

Typedef: ik_llong

An alias for signed long long.

Typedef: ik_uint

An alias for unsigned int.

Typedef: ik_ulong

An alias for unsigned long.

Typedef: ik_ullong

An alias for unsigned long long.
