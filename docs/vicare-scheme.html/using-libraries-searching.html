2.4.4 Library search algorithms

In the typical scenario: packages compile source libraries in binary format and store them in FASL files; then such FASL files are installed in one of the directory referenced by the parameter library-binary-search-path; finally, when running an application, imported libraries are loaded in binary form from FASL files.

There are other situations. For this reason Vicare implements “library locators”: different procedures to locate a library in the file system and other sources. There are 3 predefined library locators: run–time, compile–time, source; the default is the run–time locator. Library file locators.
