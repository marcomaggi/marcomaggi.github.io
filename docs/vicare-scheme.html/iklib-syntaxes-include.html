6.4.6 Including source files at expand time

— Syntax: include ?filename
— Syntax: include ?filename #t

Include in the current source file the contents of the file selected by the pathname ?filename, which must be a string. Everything happens as if the code was present in place of the include form. When the optional #t argument is given: print a message on the current error port describing the include operation.
