6.12.9 Procedure precondition violations

Condition Type: &procedure-precondition-violation

Base condition object type representing invalid state or argument for a procedure application; it is derived from &assertion.

Function: make-procedure-precondition-violation

Build and return a new condition object of type &procedure-precondition-violation.

Function: procedure-precondition-violation? obj

Return #t if obj is a condition object of type &procedure-precondition-violation; otherwise return #f.
