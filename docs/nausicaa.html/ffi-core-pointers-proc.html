31.1.5.1 Procedural interface

The following bindings are exported by the (nausicaa ffi pointers) library and reexported by (nausicaa ffi) and (nausicaa ffi memory).

— Function: pointer? x

Return #t if the value of x is a value that the Scheme implementation uses to represent memory blocks, return #f otherwise.

— Function: pointer-null? pointer

Return #t if pointer represents a null pointer according to the underlying Scheme implementation, #f otherwise.

— Constant: pointer-null

Holds a null pointer: if pointer->integer is applied to it the result is 0. This binding exists to hold a prebuilt null pointer to be used as return value or function argument, whenever it can make algorithms simpler.

— Function: pointer->integer pointer

Convert a pointer value to an exact integer.

— Function: integer->pointer exact-integer

Convert an exact integer to a pointer value.

— Function: pointer-diff pointer-1 pointer-2

Return the difference between the pointers. It is implemented as: 

          (define (pointer-diff pointer-1 pointer-2)
            (- (pointer->integer pointer-1)
               (pointer->integer pointer-2)))
— Function: pointer-add pointer offset

Add an offset to a pointer and return the resulting pointer. It is implemented as: 

          (define (pointer-add pointer offset)
            (integer->pointer (+ (pointer->integer pointer)
                                 offset)))
— Syntax: pointer-incr! ?pointer-var ?expr

Expands to: 

          (set! ?pointer-var (pointer-add ?pointer-var ?expr))
— Syntax: pointer-decr! ?pointer-var ?expr

Expands to: 

          (set! ?pointer-var
            (pointer-add ?pointer-var (- ?expr)))
— Function: pointer=? pointer0 pointer ...
— Function: pointer<? pointer0 pointer ...
— Function: pointer>? pointer0 pointer ...
— Function: pointer<=? pointer0 pointer ...
— Function: pointer>=? pointer0 pointer ...
— Function: pointer<>? pointer0 pointer ...

All the arguments must be pointer values and are internally converted to integers. Return #t if the integers satisfy the selected predicate (=, <, >, <=, >=), otherwise return #f. pointer<>? returns #t if all the pointers are different.

With the single exception of pointer<>?: All the functions return #t when applied to a single argument.
