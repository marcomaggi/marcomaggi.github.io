12.5 Parsing Uniform resource identifiers

The library (nausicaa parser-tools uri) library implements a parser to handle URIs compliant with RFC 3986:

http://datatracker.ietf.org/doc/rfc3986/

for an introduction to URIs see:

http://en.wikipedia.org/wiki/Uniform_Resource_Identifier

The purpose of (nausicaa parser-tools uri) is to validate and convert a URI representation as string or bytevector into a data structure in which the components are available as bytevectors; it is suggested to use the object types defined by the library (nausicaa net addresses uri).

When importing the library (nausicaa parser-tools uri), it is suggested to prefix the bindings with uri., as in: 

     (import (nausicaa)
       (prefix (nausicaa parser-tools uri) uri.))

We might be interested in the facilities of the library (nausicaa parser-tools uri utilities), Utility URI parser functions.

  • parser-tools uri intro: Introduction to URI elements.
  • parser-tools uri conditions: Condition object types.
  • parser-tools uri percent: Percent encoding of characters.
  • parser-tools uri parser: URI parser functions.
  • parser-tools uri utilities: Utility URI parser functions.
