71.1.3.2 Fixed–semantic–value token–makers

The following bindings are exported by the library (nausicaa r6rs lexer). All the functions make use of the fixed strings from (nausicaa r6rs fixed-strings) as semantic values (see r6rs fixed-strings).

— Function: make-open-paren-token yygetc yyungetc yytext yyline yycolumn yyoffset

Return a <lexical-token> record representing an open parenthesis, ‘(’. The category is set to the symbol OPAREN; the semantic value is set to the character #\(.

— Function: make-close-paren-token yygetc yyungetc yytext yyline yycolumn yyoffset

Return a <lexical-token> record representing a close parenthesis, ‘)’. The category is set to the symbol CPAREN; the semantic value is set to the character #\).

— Function: make-open-bracket-token yygetc yyungetc yytext yyline yycolumn yyoffset

Return a <lexical-token> record representing an open bracket, ‘[’. The category is set to the symbol OBRACKET; the semantic value is set to the character #\[.

— Function: make-close-bracket-token yygetc yyungetc yytext yyline yycolumn yyoffset

Return a <lexical-token> record representing a close bracket, ‘]’. The category is set to the symbol CBRACKET; the semantic value is set to the character #\].

— Function: make-tick-token yygetc yyungetc yytext yyline yycolumn yyoffset

Return a <lexical-token> record representing a tick, ‘'’. The category is set to the symbol TICK; the semantic value is set to the character #\'.

— Function: make-back-tick-token yygetc yyungetc yytext yyline yycolumn yyoffset

Return a <lexical-token> record representing a back tick, ‘`’. The category is set to the symbol BACKTICK; the semantic value is set to the character #\`.

— Function: make-comma-at-token yygetc yyungetc yytext yyline yycolumn yyoffset

Return a <lexical-token> record representing a comma–at, ‘,@’. The category is set to the symbol COMMAAT; the semantic value is set to the string ",@".

— Function: make-comma-token yygetc yyungetc yytext yyline yycolumn yyoffset

Return a <lexical-token> record representing a comma, ‘,’. The category is set to the symbol COMMA; the semantic value is set to the character #\,.

— Function: make-dot-token yygetc yyungetc yytext yyline yycolumn yyoffset

Return a <lexical-token> record representing a dot, ‘.’. The category is set to the symbol DOT; the semantic value is set to the character #\..

— Function: make-double-quote-token yygetc yyungetc yytext yyline yycolumn yyoffset

Return a <lexical-token> record representing a double quote, ‘"’; double quotes open strings, which must be read with a separate lexer table. The category is set to the symbol DOUBLEQUOTE; the semantic value is set to the character #\".

— Function: make-sharp-paren-token yygetc yyungetc yytext yyline yycolumn yyoffset

Return a <lexical-token> record representing a sharp–open parenthesis, ‘#(’. The category is set to the symbol SHARPPAREN; the semantic value is set to the string "#(".

— Function: make-sharp-vu8-paren-token yygetc yyungetc yytext yyline yycolumn yyoffset

Return a <lexical-token> record representing a sharp–vu8–open parenthesis, ‘#vu8(’. The category is set to the symbol SHARPVU8PAREN; the semantic value is set to the string "#vu8(".

— Function: make-sharp-tick-token yygetc yyungetc yytext yyline yycolumn yyoffset

Return a <lexical-token> record representing a sharp–tick, ‘#'’. The category is set to the symbol SHARPTICK; the semantic value is set to the string "#'".

— Function: make-sharp-back-tick-token yygetc yyungetc yytext yyline yycolumn yyoffset

Return a <lexical-token> record representing a sharp–back–tick, ‘#`’. The category is set to the symbol SHARPBACKTICK; the semantic value is set to the string "#`".

— Function: make-sharp-comma-at-token yygetc yyungetc yytext yyline yycolumn yyoffset

Return a <lexical-token> record representing a sharp–commma–at, ‘#,@’. The category is set to the symbol SHARPCOMMAAT; the semantic value is set to the string "#,@".

— Function: make-sharp-comma-token yygetc yyungetc yytext yyline yycolumn yyoffset

Return a <lexical-token> record representing a sharp–comma, ‘#,’. The category is set to the symbol SHARPCOMMA; the semantic value is set to the string "#,".

— Function: make-sharp-semicolon-token yygetc yyungetc yytext yyline yycolumn yyoffset

Return a <lexical-token> record representing a sharp–semicolon, ‘#;’. The category is set to the symbol SHARPSEMICOLON; the semantic value is set to the string "#;".

— Function: make-open-nested-comment-token yygetc yyungetc yytext yyline yycolumn yyoffset

Return a <lexical-token> record representing the opening of a nested comment, ‘#|’. The category is set to the symbol ONESTEDCOMMENT; the semantic value is set to the string "#|".

— Function: make-sharp-bang-r6rs-token yygetc yyungetc yytext yyline yycolumn yyoffset

Return a <lexical-token> record representing a ‘#!r6rs’ token. The category is set to the symbol SHARPBANGR6RS; the semantic value is set to the string "#!r6rs".
