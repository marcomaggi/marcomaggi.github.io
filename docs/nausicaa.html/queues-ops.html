43.3 Basic operations

— Function: queue-enqueue! que obj

Enqueue obj in the rear of the queue.

— Method on <queue>: enqueue! obj

Enqueue obj in the rear of the queue.

— Function: queue-push! obj que

Push obj in the front of the queue.

— Method on <queue>: push! obj

Push obj in the front of the queue.

— Function: queue-dequeue! que
— Function: queue-pop! que

Pop and return a value from the front of the queue.

— Method on <queue>: dequeue!
— Method on <queue>: pop!

Pop and return a value from the front of the queue.

— Function: queue-purge! que

Discard all the elements from the queue.

— Method on <queue>: purge!

Discard all the elements from the queue.
