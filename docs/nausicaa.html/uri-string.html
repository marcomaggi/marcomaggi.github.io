69.3 Plain conversion between strings and bytevectors

The following bindings are exported by the (nausicaa uri low) library; it is suggested to prefix them when importing this library.

— Function: to-bytevector obj

If obj is a bytevector: obj itself is returned. If obj is a string: convert it to a bytevector representation; characters are taken from obj and inserted into the resulting bytevector after decoding them, char to byte, as ASCII.

If the string has a character whose integer representation is not in the range [0, 255]: raise a condition object with components &parser-error, &who, &message, &irritants; the irritants value is a list holding obj.

— Function: to-string obj

If obj is a string: obj itself is returned. If obj is a bytevector: convert it to a string representation; bytes are taken from obj and inserted into the resulting string after encoding them, byte to char, as ASCII.
