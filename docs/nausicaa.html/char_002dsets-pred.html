30.3 Predicates

— Function: char-set? obj

Return true if obj is a valid char-set.

— Function: char-set-empty? cs

Return true if the char-set is empty.

— Function: char-set-contains? cs char

Return true if char is an element of cs.

— Function: char-set=? cs-a cs-b

Return true if the arguments represent the same char-set.

— Function: char-set<? cs-a cs-b

Return true if all the elements of cs-a are strictly less than all the elements of cs-b. Empty sets cannot be ordered, so if an argument is empty the return value is #f.

— Function: char-set-subset? cs0 cs ...
— Function: char-set-subset?/strict cs0 cs ...

Return #t if the arguments, in the given order, are subsets of each other or strict subsets of each other.

— Function: char-set-superset? cs0 cs ...
— Function: char-set-superset?/strict cs0 cs ...

Return #t if the arguments, in the given order, are supersets of each other or strict supersets of each other.
