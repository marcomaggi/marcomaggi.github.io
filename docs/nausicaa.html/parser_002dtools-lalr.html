11.3 A LALR(1) parser generator

The library (nausicaa parser-tools lalr) is a LALR(1) parser tables generator. The libraries (nausicaa parser-tools lalr lr-driver) and (nausicaa parser-tools lalr glr-driver) generate instances of a parser using a supplied table; the two drivers are not compatible, each of them requires a specifically built table.

The algorithm is the same used in GNU Bison and it is described in:
F. DeRemer and T. Pennello. “Efficient Computation of LALR(1) Look–Ahead Set”. TOPLAS, vol. 4, no. 4, october 1982.

as a consequence, it is not written in a fully functional style. In fact, much of the code is a direct translation from C to Scheme of the Bison sources.

The library is a port to R6RS Scheme of Lalr-scm by Dominique Boucher. The original source is available at (URL last verified Oct 21, 2013):

http://code.google.com/p/lalr-scm/

The LALR libraries make use of the <lexical-token> and <source-location> classes defined by the libraries (nausicaa parser-tools lexical-tokens) and (nausicaa parser-tools source-locations).

  • parser-tools lalr intro: Introduction to the LR parser.
  • parser-tools lalr tables: Creating parser tables.
  • parser-tools lalr lexer: Describing tokens.
  • parser-tools lalr parser: Running the parser.
  • parser-tools lalr grammar: Defining the grammar.
  • parser-tools lalr glr: Generalised LR parsing.
