69.2.5 IPv6 address and address prefix class objects

The following bindings are exported by the (nausicaa net ipv6-addresses) library.

— Class: <ipv6-address>

Represents an IPv6 address. Its constructor accepts as single argument the list of numbers returned by the IPv6 address parser, already validated. 

          (let (((o <ipv6-address>)
                (make* <ipv6-address>
                  (ipv6-address-parse "1:2:3:4:5:6:7:8"))))
            (list o.seventh o.sixth o.fifth o.fourth
                  o.third o.second o.first o.zeroth))
          ⇒ (1 2 3 4 5 6 7 8)
— Immutable Field of <ipv6-address>: zeroth
— Immutable Field of <ipv6-address>: first
— Immutable Field of <ipv6-address>: second
— Immutable Field of <ipv6-address>: third
— Immutable Field of <ipv6-address>: fourth
— Immutable Field of <ipv6-address>: fifth
— Immutable Field of <ipv6-address>: sixth
— Immutable Field of <ipv6-address>: seventh

The numeric components of the address, each representing 16 bits. zeroth is the rightmost component, first is the second from the right, etc.

— Immutable Virtual Field of <ipv6-address>: bignum

A single big exact integer of 128 bits representing the whole address number.

— Immutable Virtual Field of <ipv6-address>: string

The string representation of the address.

— Immutable Virtual Field of <ipv6-address>: unspecified?
— Immutable Virtual Field of <ipv6-address>: loopback?
— Immutable Virtual Field of <ipv6-address>: multicast?
— Immutable Virtual Field of <ipv6-address>: link-local-unicast?
— Immutable Virtual Field of <ipv6-address>: global-unicast?

True if the address has type, respectively: unspecified, loopback, multicast, link–local unicast, global unicast. See Section 2.4 “Address Type Identification” of RFC 4291.

— Class: <ipv6-address-prefix>

Represents an IPv6 address prefix; it is derived from <ipv6-address>. Its constructor accepts two arguments: the list of numbers returned by the IPv6 address parser, already validated; a number specifying how many of the leftmost contiguous bits of the address comprise the prefix.

— Immutable Field of <ipv6-address-prefix>: prefix-length

A number specifying how many of the leftmost contiguous bits of the address comprise the prefix.

— Immutable Virtual Field of <ipv6-address-prefix>: string

The string representation of the address prefix.
