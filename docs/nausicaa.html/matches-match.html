52.1 Basic pattern matching

— Syntax: match <sexp> <clause> ...
— Auxiliary Syntax: =>

Match the S–expression <sexp> against the <clause> arguments; each clause is one of: 

          (<pattern> . <body>)
          (<pattern> (=> next) . <body>)

<expr> is evaluated and the result is matched against each <pattern> in turn; the first to succeed causes the corresponding <body> forms to be evaluated as in begin, and the remaining patterns are ignored. However, if the ‘next’ identifier is labeled, it may be used as a continuation to continue matching, allowing for additional runtime tests on the pattern.

The syntax for the patterns is:

‘_’
The literal underscore is the wildcard: it matches anything, it does not bind any variable.
‘#t’
Matches if non-false.
‘#f’
Matches if false.
‘<literal>’
Matches with equal?.
‘(quote <sexp>)’
A literal S–expression; matches with equal?.
‘<variable>’
When the pattern is an unquoted symbol: it matches anything and binds <variable> to it as in: 
               (let ((<variable> <matched subexpression>))
                 ---)

‘(<pattern1> --- <patternN>)’
Proper list of N elements.
‘(<pattern1> --- <patternN> . <patternN+1>)’
Improper list of N or more elements.
‘(<pattern1> --- <patternN> <patternN+1> ...)’
List of N or more elements with terminating ellipsis; each element of remainder must match <patternN+1>.
‘#(<pattern1> --- <patternN>)’
Vector of N elements.
‘#(<pattern1> --- <patternN> <patternN+1> ...)’
Vector of N or more elements with terminating ellipsis; each element of remainder must match <patternN+1>.
‘(:and <pattern> ---)’
If all of <pattern> match.
‘(:or <pattern> ---)’
If any of <pattern> match.
‘(:not <pattern> ---)’
If all <pattern> do not match at all.
‘(:predicate <pred> <pattern> ---)’
If the predicate function <pred> applied to the subexpression is true and all the <pattern> match.
‘(:accessor <proc> <pattern>)’
If <pattern> matches the result of applying <proc> to the subexpression.
‘(:getter <symbol>)’
Match anything and bind to <symbol> a thunk which, when evaluated, returns the matched element itself.
‘(:setter <symbol>)’
Match anything and bind to <symbol> a function which, when applied to a value, replaces the expression with that value. This involves the use of set-car!, set-cdr! and other mutators.
‘(:free-identifier syntax-object)’
Match an identifier (a syntax object) which is equal to syntax-object according to free-identifier=?.
‘(:bound-identifier syntax-object)’
Match an identifier (a syntax object) which is equal to syntax-object according to bound-identifier=?.
