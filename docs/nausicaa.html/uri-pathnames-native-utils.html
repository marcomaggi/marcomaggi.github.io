11.3.4.3 Native pathname utilities

The following bindings are exported by the library (nausicaa uri pathnames).

— Multimethod: pathname (ptn <string>)
— Multimethod: pathname (ptn <bytevector>)

Build and return a new instance of <absolute-native-pathname> or <relative-native-pathname> from ptn, which must be a string or bytevector representing an absolute or relative native pathname.

— Multimethod: pathname=? (ptn1 <absolute-native-pathname>) (ptn2 <absolute-native-pathname>)
— Multimethod: pathname=? (ptn1 <relative-native-pathname>) (ptn2 <relative-native-pathname>)

Implementations for the multimethod pathname=? defined by (nausicaa uri pathnames abstract).
