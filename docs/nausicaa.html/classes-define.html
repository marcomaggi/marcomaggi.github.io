24.1 Defining new class types

A class type, as defined by the (nausicaa language classes) library, is a regular record type as defined by (rnrs records procedural (6)), with the constraint of being nongenerative; new class types are defined by define-class which generates auxiliary syntaxes to allow for virtual fields, methods, dot notation and multiple construction protocols.

A label type, as defined by the (nausicaa language classes) library, is a special interface to the syntaxes which define dot notation for virtual fields and methods: it is applicable to any value, not just records and it is not associated to a record type as defined by (rnrs records procedural (6)). Labels are defined by define-label.
