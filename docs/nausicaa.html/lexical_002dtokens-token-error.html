72.2.3 Lexical error class

The following bindings are exported by the (nausicaa parser-tools lexical-tokens) library; this library is built on top of (nausicaa language classes).

— Class: <lexer-error>

Hold the informations about a lexer error; it is derived from <lexical-token>. The value of the category field is always ‘*lexer-error*’.

— Immutable Field of <lexer-error>: message

A Scheme string describing the error.

— Syntax on <lexer-error>: make* <lexer-error> location value length message

Build and return a new record of type <lexer-error>.

— Syntax on <lexer-error>: make <lexer-error> clause ...
— Auxiliary Syntax on <lexer-error>: location: location
— Auxiliary Syntax on <lexer-error>: value: value
— Auxiliary Syntax on <lexer-error>: length: length
— Auxiliary Syntax on <lexer-error>: error-message: message

Build and return a new record of type <lexer-error>. The error-message: clause is mandatory; location defaults to #f; value defaults to #f; length defaults to zero.

— Syntax on <lexer-error>: is-a? obj <lexer-error>

Return #t if obj is an instance of <lexer-error>.

Traditional records API
— Function: make-<lexer-error> location value length message

Build and return a new record of type <lexer-error>.

— Function: <lexer-error>? obj

Return true if obj is a record of type of <lexer-error>.

— Function: <lexer-error>-message tok

Accessor for the field of <lexer-error> records.
