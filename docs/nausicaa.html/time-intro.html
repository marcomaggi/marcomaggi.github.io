45.1 Introduction to time scales

When using the (nausicaa times-and-dates) library we aim at dealing with Terrestrial Time in which: seconds are the ones of the International System of Units (SI), defined using caesium atomic clocks; 60 seconds make one minute; 60 minutes make one hour; 24 hours make one day; 365 days make one year. Terrestrial Time is a theoretical ideal, which real clocks can only approximate.
