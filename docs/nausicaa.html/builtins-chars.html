5.10 Character type

— Builtin Label: <char>

Conventional type for objects satisfying char?.

Safe accessors
— Immutable virtual field of <char>: <char> upcase

Accessing this field is like applying char-upcase to the char.

— Immutable virtual field of <char>: <char> downcase

Accessing this field is like applying char-downcase to the char.

— Immutable virtual field of <char>: <char> titlecase

Accessing this field is like applying char-titlecase to the char.

— Immutable virtual field of <char>: <char> foldcase

Accessing this field is like applying char-foldcase to the char.

— Immutable virtual field of <char>: <boolean> alphabetic?

Return the result of applying char-alphabetic? to the character.

— Immutable virtual field of <char>: <boolean> numeric?

Return the result of applying char-numeric? to the character.

— Immutable virtual field of <char>: <boolean> whitespace?

Return the result of applying char-whitespace? to the character.

— Immutable virtual field of <char>: <boolean> upper-case?

Return the result of applying char-upper-case? to the character.

— Immutable virtual field of <char>: <boolean> lower-case?

Return the result of applying char-lower-case? to the character.

— Immutable virtual field of <char>: <boolean> title-case?

Return the result of applying char-title-case? to the character.

— Immutable virtual field of <char>: <symbol> general-category

Return the result of applying char-general-category to the character.
