33 Character sets

The (nausicaa char-sets) library handles sets of characters.

A “character” is every value for which the application of char? returns #t. This library does nothing that can be related to a specific character encoding, like ASCII, UCS or Unicode. When needed, it relies on string->list to split a string into its character components. Unicode.
