30.2.3.3 Conversion between blocks and other objects
— Function: memblock->string-hex mb
— Function: string-hex->memblock str malloc

Convert to and from a <memblock> record and a string, encoding each byte in the block with two characters in the string in hexadecimal format. These functions are slow, but they are fine for debugging purposes.

When converting from a string, the new memory block is allocated with malloc, which must be a function like malloc from (nausicaa ffi memory alloc).

When converting to a string, the returned value is in upper case.
