10.2 Inspecting syntax objects

— Function: quoted-syntax-object? stx

Return true if stx is a syntax object holding a pair whose car is one of the identifiers: quote, quasiquote, syntax, quasisyntax.

— Function: syntax=? stx1 stx2

Unwrap the syntax objects stx1 and stx2 and compare the result: return true if they are equal.

— Function: all-identifiers? stx

Given a syntax object: return true if it is null or a list of identifiers; return false otherwise.

— Function: duplicate-identifiers? stx
— Function: duplicate-identifiers? stx identifier=

Search the syntax object stx, which must hold a list of identifiers, for duplicate identifiers; at the first duplicate found, return it; return false if no duplications are found.

identifier= must be the predicate used to compare identifiers; it is typically bound-identifier=? or free-identifier=?. When not given: identifier= defaults to bound-identifier=?.

— Function: delete-duplicate-identifiers unwrapped-ids

Given a list of identifiers: remove the duplicate identifiers and return a proper list of unique identifiers.

— Function: identifier-memq identifier list-of-syntax-objects

Given a list of syntax objects search for one which is free-identifier=? to identifier and return the sublist starting with it; return false if identifier is not present.

— Function: symbol-identifier=? id1 id2

Return true if the identifier arguments have the same name.

— Function: identifier-subst ids-map stx

Substitute occurrences of identifiers in stx, return the result which can be stx itself. stx must be a syntax object, wrapped or unwrapped. ids-map must be a syntax object holding a possibly empty lists of lists, each holding two identifeirs: 

          ((src dst) ...)

each src identifier is substituted with dst.

For every couple of corresponding identifiers: if they satisfy free-identifier=?, then no substitution is performed.
