10.8.3 Iterating over lists

The following bindings are exported by the library (nausicaa containers iterators).

— Class: <spine-iterator>

A forward stride iterator for Scheme lists; it is derived from <iterator>.

— Common constructor on <spine-iterator>: constructor the-list stride

Build a new instance of <spine-iterator> using the-list as list upon which to iterate and stride, which must be an exact positive integer, as stride. The stride is the number of elements in the list to jump forward to when iterating; it must be 1 when we want to visit all the elements.

— Maker on <spine-iterator>: <spine-iterator> (?clauses ...)
— Auxiliary Syntax on <spine-iterator>: subject: ?the-list
— Auxiliary Syntax on <spine-iterator>: stride: ?stride

Build and return a new class instance using the common constructor. The clause subject: is mandatory; the clause stride: is optional, when not used the stride defaults to 1.

Example: 

     #!r6rs
     (import (nausicaa)
       (nausicaa containers iterators))
     
     (let ()
       (<spine>    L '(c i a o))
       (<iterator> I  (<spine-iterator> ((subject: L))))
     
       (I more?)     ⇒ #t
       (I next)      ⇒ c
       (I more?)     ⇒ #t
       (I next)      ⇒ i
       (I more?)     ⇒ #t
       (I next)      ⇒ a
       (I more?)     ⇒ #t
       (I next)      ⇒ o
       (I more?))    ⇒ #f
     
     (let ()
       (<spine>    L '(h e l L o))
       (<iterator> I (<spine-iterator> ((subject: L)
                                        (stride:  2))))
       (I more?)     ⇒ #t
       (I next)      ⇒ h
       (I more?)     ⇒ #t
       (I next)      ⇒ l
       (I more?)     ⇒ #t
       (I next)      ⇒ o
       (I more?))    ⇒ #f
