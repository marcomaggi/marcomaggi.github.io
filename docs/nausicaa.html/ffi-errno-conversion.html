30.4.3 Conversion of error codes

— Function: strerror errno-numeric-value

An interface to the system strerror() function. Return the string message coresponding to errno-numeric-value.

— Function: symbol->errno symbol
— Function: symbol->errno/or-error symbol

Attempt the conversion from an ‘errno’ symbolic value to the corresponding numeric value. symbol->errno returns an integer or #f if the symbol is not defined on the underlying platform. The ‘or-error’ variant raises an assertion violation if symbol is not an ‘errno’ symbolic value.

— Function: errno->symbol number
— Function: errno->symbol/or-error number

Attempt the conversion from an ‘errno’ numeric value to the corresponding symbolic value. errno->symbol returns a symbol or #f if number is not an ‘errno’ numeric value. The ‘or-error’ variant raises an assertion violation if number is not an ‘errno’ numeric value.
