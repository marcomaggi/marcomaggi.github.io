11.2.5.4 URI empty path objects

The following bindings are exported by the (nausicaa uri uri) library.

— Abstract Class: <path-empty>

Base class representing path-empty components of URIs; it is derived from <path>.

— Super constructor on <path-empty>: constructor ()

Build and return an instance of <path-empty>; there is only one instance of <path-empty>: the constructor always returns the same object.
