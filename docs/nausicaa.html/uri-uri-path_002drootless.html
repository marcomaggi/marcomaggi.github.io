11.2.5.7 URI rootless path objects

The following bindings are exported by the (nausicaa uri uri) library.

— Abstract Class: <path-rootless>

Base class representing path-rootless components of URIs; it is derived from <path>.

— Super constructor on <path-rootless>: constructor (obj <list-of-segments>)

Build and return a new instance of <path-rootless>. The argument must be a non–empty, list of objects satisfying the requirements of <list-of-segments>.
