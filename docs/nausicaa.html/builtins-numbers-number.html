5.15.1 Generic number objects

— Builtin Label: <number>

Conventional type for objects satisfying number?.

Safe accessors
— Immutable virtual field of <number>: <boolean> exact?

Accessing this field is like applying to the instance number the function exact? from (rnrs base (6)).

— Immutable virtual field of <number>: <boolean> inexact?

Accessing this field is like applying to the instance number the function inexact? from (rnrs base (6)).

— Immutable virtual field of <number>: <boolean> zero?

Accessing this field is like applying to the instance number the function zero? from (rnrs base (6)).

— Immutable virtual field of <number>: <boolean> finite?

Accessing this field is like applying to the instance number the function finite? from (rnrs base (6)).

— Immutable virtual field of <number>: <boolean> infinite?

Accessing this field is like applying to the instance number the function infinite? from (rnrs base (6)).

— Immutable virtual field of <number>: <boolean> nan?

Accessing this field is like applying to the instance number the function nan? from (rnrs base (6)).

Accessors: conversion
— Immutable virtual field of <number>: <string> string

Accessing this field is like applying to the number the function number->string from (rnrs base (6)) without other arguments.

Methods: arithmetic
— Method on <number>: + num ...

Apply + to the instance and the given optional arguments; return the result.

— Method on <number>: - num ...

Apply - to the instance and the given optional arguments; return the result.

— Method on <number>: * num ...

Apply * to the instance and the given optional arguments; return the result.

— Method on <number>: / num ...

Apply / to the instance and the given optional arguments; return the result.
