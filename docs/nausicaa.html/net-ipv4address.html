68.1 Ipv4 address objects

The library (nausicaa net ipv4addresses) exports utilities to handle IPv4 addresses specifications, according to RFC 5735.

  • net ipv4address lexer: Tokenising an IPv4 address.
  • net ipv4address parser: Parsing an IPv4 address.
  • net ipv4address errors: Condition objects and error handlers.
  • net ipv4address class: IPv4 address and address prefix class objects.
