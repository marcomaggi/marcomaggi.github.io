33.5 Iterating over vectors

The following bindings are exported by the (nausicaa iterators) library.

— Class: <vector-iterator>

An iterator for Scheme vectors; it is derived from <iterator>.

— Common constructor on <vector-iterator>: constructor the-vector start past stride

Build a new instance of <vector-iterator> using the-vector as vector upon which to iterate, start as index of the first character, past as index of the character past the last one to return and stride as stride.

start, past and stride must be exact integers. start must be between 0 and one less than the vector length. past must be between -1 and the vector length. stride cannot be zero and it must have the same sign of the difference past minus start.

The stride is the number of characters in the vector to jump forward to when iterating; it must be 1 when we want to visit all the characters.

— Syntax on <vector-iterator>: make <vector-iterator> . <clauses>
— Auxiliary syntax on <vector-iterator>: subject: <the-vector>
— Auxiliary syntax on <vector-iterator>: start: <start>
— Auxiliary syntax on <vector-iterator>: past: <past>
— Auxiliary syntax on <vector-iterator>: stride: <stride>

Build and return a new class instance using the common constructor.

If the start: clause is not used: the start index defaults to zero. If the past: clause is not used: the past index defaults to the length of the vector. If the stride: clause is not used: the stride defaults to 1.
