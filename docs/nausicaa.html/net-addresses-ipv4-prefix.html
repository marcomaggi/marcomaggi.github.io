12.1.1.2 IPv4 address prefix objects class

The following bindings are exported by the library (nausicaa net addresses ipv4).

— Class: <ipv4-address-prefix>

Represents an IPv4 address prefix; it is derived from <ipv4-address>.

Constructors and makers
— Common constructor on <ipv4-address-prefix>: constructor numbers prefix-length

Build and return a new object of type <ipv4-address-prefix>. numbers must be a list of 4 fixnums in the range [0, 255]; prefix-length must be a non–negative fixnum specifying how many of the leftmost contiguous bits of the address comprise the prefix.

— Maker on <ipv4-address-prefix>: <ipv4-address-prefix> (?numbers ?prefix-length)

Build and return a new object of type <ipv4-address-prefix> using the common constructor. 

          (<ipv4-address-prefix> O
            (receive (numbers len)
                (parse-ipv4-address-prefix "1.2.3.4/8")
              (<ipv4-address-prefix> (numbers len))))
          
          (list (O third) (O second) (O first) (O zeroth))
          ⇒ (1 2 3 4)
          
          (O prefix-length)
          ⇒ 8
Fields
— Immutable Field of <ipv4-address-prefix>: <ipv4-address-prefix-length> prefix-length

A fixnum specifying how many of the leftmost contiguous bits of the address comprise the prefix.

— Immutable Virtual Field of <ipv4-address-prefix>: <string> string

The string representation of the address prefix.
