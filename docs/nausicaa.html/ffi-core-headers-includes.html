23.1.9.2 Including header files
— Autoconf Macro: NAUSICAA_INCLUDES_DEFAULT (headers)

Expand to the specification of C language headers inclusion. Its value is: 

          AC_INCLUDES_DEFAULT
          #ifdef HAVE_ERRNO_H
          #  include <errno.h>
          #endif
          #ifdef HAVE_FCNTL_H
          #  include <fcntl.h>
          #endif
          #ifdef HAVE_INTTYPES_H
          #  include <inttypes.h>
          #endif
          #ifdef HAVE_LIMITS_H
          #  include <limits.h>
          #endif
          #ifdef HAVE_STDINT_H
          #  include <stdint.h>
          #endif
          #ifdef HAVE_UNISTD_H
          #  include <unistd.h>
          #endif

and additionally: the current value of the ‘NAUSICAA_CUSTOM_INCLUDES’ macro is appended along with the optional headers. ‘AC_INCLUDES_DEFAULT’ is defined by Autoconf. Default Includes

For example, it can be used as third argument for ‘AC_COMPUTE_INT’, without arguments: 

          AC_COMPUTE_INT([THE_IMAX],[INT_MAX],
            [NAUSICAA_INCLUDES_DEFAULT])

or with arguments: 

          AC_COMPUTE_INT([SIZEOF_MPFR],[sizeof(mpfr_t)],
            NAUSICAA_INCLUDES_DEFAULT([#include <gmp.h>
          #include <mpfr.h>])
— Autoconf Macro: NAUSICAA_CUSTOM_INCLUDES

Appended to the expansion of NAUSICAA_INCLUDES_DEFAULT; by default defined to the empty string, it is meant to be redefined in the configure.ac template file.

For example an extension for the cURL library can do: 

          m4_define([NAUSICAA_CUSTOM_INCLUDES],[
          #include <curl/curl.h>
          ])

while an extension for the Zlib library can do: 

          m4_define([NAUSICAA_CUSTOM_INCLUDES],[
          #include <zlib.h>
          ])
