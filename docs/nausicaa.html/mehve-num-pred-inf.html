6.3.5 Infinity and not–a–number predicates generic functions

The generic functions finite?, infinite? and nan? exported by (nausicaa mehve numeric-predicates), and reexported by (nausicaa mehve), generalise the corresponding functions exported by (rnrs base (6)) and (nausicaa).

We have to remember that, according to R6RS, the only values satisfying the predicate real? and not the predicate rational? are ‘+inf.0’, ‘-inf.0’ and ‘+nan.0’.

— Generic Function: finite? A

Return #t if the argument is finite and not not–a–number.

— Generic Function: infinite? A

Return #t if the argument is infinite.

— Generic Function: nan? A

Return #t if the argument is not–a–number.

The following are the specialised methods.

— Method: finite? (A <flonum>)

Apply to the argument the function flfinite? exported by the (rnrs arithmetic flonums (6)) library, see flfinite?.

— Method: finite? (A <complex>)

Apply to the argument the function finite? exported by the (nausicaa) library, see finite?.

— Method: infinite? (A <flonum>)

Apply to the argument the function flinfinite? exported by the (rnrs arithmetic flonums (6)) library, see flinfinite?.

— Method: infinite? (A <complex>)

Apply to the argument the function infinite? exported by the (nausicaa) library, see infinite?.

— Method: nan? (A <flonum>)

Apply to the argument the function flnan? exported by the (rnrs arithmetic flonums (6)) library, see flnan?.

— Method: nan? (A <complex>)

Apply to the argument the function nan? exported by the (nausicaa) library, see nan?.
