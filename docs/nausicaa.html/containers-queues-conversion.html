10.6.4 Converting queues to other objects

The following bindings are exported by the library (nausicaa containers queues). The bindings whose name is prefixed with $ are unsafe operations: they do not validate their arguments before accessing them.

— Method on <queue>: list
— Method on <queue>: $list

Return null or a proper list holding the objects from the <queue>, top first.

— Function: list->queue list

Convert a proper list to a <queue> object; objects from the list are pushed on the queue from left to right. 

          (import (nausicaa)
            (nausicaa containers queues))
          
          (<queue> S (list->queue '(1 2 3)))
          
          (S front)       ⇒ 1
— Method on <queue>: vector
— Method on <queue>: $vector

Return a vector holding the objects from the <queue>, top first.

— Function: vector->queue vector

Convert a vector to a <queue> object; objects from the vector are pushed on the queue from left to right.
