73.1.2 Building records

— Syntax on <source-location>: make <source-location> <clause> ...
— Syntax on <source-location>: make* <source-location> <input> <line> <column> <offset>
— Auxiliary Syntax on <source-location>: input: <input-source>
— Auxiliary Syntax on <source-location>: line: <line>
— Auxiliary Syntax on <source-location>: column: <column>
— Auxiliary Syntax on <source-location>: offset: <offset>

Build and return a new instance of <source-location>.

Procedural interface
— Function: make-<source-location> input line colum offset

Build and return a record of type <source-location>.

— Function: make-<source-location>/start input

Build and return a <source-location> record representing the very beginning of an input stream. The line and column counters are set to 1, the offset counter is set to 0; input is stored in the ‘input’ field.
