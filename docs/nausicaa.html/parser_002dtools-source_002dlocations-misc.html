12.1.5 Miscellaneous methods and functions

The procedures described here make use of multimethods; Multimethod dispatching.

— Method on <source-location>: <string> string

Return a string representing the instance object. If the instance object is specified the string has the format ‘?line:?column’, where ?line is the line number and ?column is the column number; otherwise the returned value is the one referenced by the parameter unspecified-source-location-string.

— Method: object->string (O <source-location>)

Return a string representation of O. It makes use of the method string of <source-location>.
