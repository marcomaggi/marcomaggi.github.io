10.6 Simple queues

The library (nausicaa containers queues) implements queues holding arbitrary scheme objects and designed for efficient first–in/first–out operations; it is built on top of (vicare containers queues).

  • containers queues objects: Queue objects.
  • containers queues inspection: Inspecting queue objects.
  • containers queues access: Queue accessors and mutators.
  • containers queues conversion: Converting queues to other objects.
