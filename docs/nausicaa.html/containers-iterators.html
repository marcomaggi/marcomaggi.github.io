10.8 Iterating over sequences

Iterators are objects used to visit values stored in sequences, like lists and vectors, or computed at the time they are requested; iterators are not compliant with the traditional Scheme language style, but many developers are used to them. Vicare Scheme's iterators API is loosely inspired by Python's iterators.

  • containers iterators base: Basic iterators API.
  • containers iterators conditions: Condition object types.
  • containers iterators lists: Iterating over lists.
  • containers iterators strings: Iterating over strings.
  • containers iterators vectors: Iterating over vectors.
  • containers iterators bytevectors: Iterating over bytevectors.
