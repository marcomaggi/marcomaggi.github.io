12.1.4 Unspecified source locations

— Function: unspecified-source-location

Return a <source-location> object representing an unspecified source location. This function always returns the same object.

— Parameter: unspecified-source-location-string

Contains a string representation for <source-location> objects representing an unspecified source location. It is initialised to ‘"no-source"’.
