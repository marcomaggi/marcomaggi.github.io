25.2.8 Allocation from a memory pool

The API described in this section allows us to allocate memory blocks contiguously from a preallocated memory block accessed through a <mempool> record. Requested memory blocks are just marked as used in the status of the pool. When using this allocator there is no need to release individual blocks: Releasing the pool releases all of them.

The following bindings are exported by (nausicaa ffi memory mempool). The library is built on top of (nausicaa language classes).

— Class: <mempool>

Record type descriptor of the memory pool object.

— Immutable Field of <mempool>: pointer

Pointer to the allocated block.

— Immutable Field of <mempool>: size

Number of allocated bytes.

— Mutable Filed of <mempool>: pointer-free

Pointer to the first free byte in the allocated block.

— Immutable Field of <mempool>: free-size

Number of free bytes in the memory pool.

— Function: make-<mempool> pointer number-of-bytes

Build and return a new <mempool> record. Notice that the allocated memory must be released explicitly by accessing the ‘pointer’ field of the record.

— Parameter: memory-pool

Holds #f or the <mempool> record from which memory blocks must be allocated; it is preset to #f.

— Function: primitive-malloc/mempool number-of-bytes

Allocates number-of-bytes in the current memory pool and return a pointer to the first allocated byte, or #f if there is not enough room in the pool.

Notice that this function has the correct interface to be used as value for the primitive-malloc-function parameter. Allocation functions

— Function: malloc/mempool number-of-bytes

A wrapper for primitive-malloc/mempool that raises an &out-of-memory exception if there is not enough room in the pool.

Example: 

     (import (nausicaa)
       (nausicaa ffi memory)
       (nausicaa ffi memory mempool))
     
     (define pool
       (make-<mempool> (malloc 4096) 4096))
     
     (parametrise ((memory-pool pool))
       (let ((p (malloc/mempool 10))
             (q (malloc/mempool 20)))
         ---))
     
     (with-class ((pool <mempool>))
       (primitive-free pool.pointer))

or using compensations: 

     (import (nausicaa)
       (nausicaa ffi memory compensate)
       (nausicaa ffi memory mempool))
     
     (with-compensations
       (parametrise ((memory-pool
                        (make-<mempool> (malloc/c 4096) 4096)))
         (let ((p (malloc/mempool 10))
               (q (malloc/mempool 20)))
           ---)))
