45.4 Seconds and nanoseconds objects

The <seconds-and-nanoseconds> class is meant to represent counts of seconds with neither selected interpretation as points in time or duration, nor selected time scale. The following bindings are exported by the (nausicaa times-and-dates) library.

— Class: <seconds-and-nanoseconds>

Base type representing a count of seconds and anoseconds. The two counts are normalised to be both non–negative or both non–positive.

— Immutable Field of <seconds-and-nanoseconds>: seconds

Exact integer representing seconds count. It can be positive, zero or negative.

— Immutable Field of <seconds-and-nanoseconds>: nanoseconds

Exact integer representing nanoseconds count. It can be positive, zero or negative and it is normalised to the range [-999999999, 999999999].

— Syntax: make* <seconds-and-nanoseconds> <seconds> <nanoseconds>

Expand to an expression evaluating to a new <seconds-and-nanoseconds> instance. <seconds> and <nanoseconds> must expand into expressions evaluating to exact integers representing the counts of seconds and nanoseconds.

— Syntax: make <seconds-and-nanoseconds> <clause> ...

Expand to an expression evaluating to a new <seconds-and-nanoseconds> instance. The seconds and nanoseconds counts are computed from the values given in the following maker clauses:

seconds <seconds>
milliseconds <milliseconds>
microseconds <microseconds>
nanoseconds <nanoseconds>
Select the count in the specified seconds and subseconds units. All the values are normalised to seconds and nanoseconds counts.

— Method on <seconds-and-nanoseconds>: deep-clone
— Method on <seconds-and-nanoseconds>: shallow-clone

Return a copy of the object.

— Immutable Field of <seconds-and-nanoseconds>: full-seconds
— Immutable Field of <seconds-and-nanoseconds>: full-milliseconds
— Immutable Field of <seconds-and-nanoseconds>: full-microseconds
— Immutable Field of <seconds-and-nanoseconds>: full-nanoseconds

Exact numbers representing the time in seconds or the specified subseconds counts.

Traditional record API
— Function: make-<seconds-and-nanoseconds> seconds nanoseconds

Build and return a new instance of <seconds-and-nanoseconds>.

— Function: <seconds-and-nanoseconds>? obj

Return true if obj is an instance of <seconds-and-nanoseconds>.

— Function: <seconds-and-nanoseconds>-seconds sn
— Function: <seconds-and-nanoseconds>-nanoseconds sn
— Function: <seconds-and-nanoseconds>-full-seconds sn
— Function: <seconds-and-nanoseconds>-full-microseconds sn
— Function: <seconds-and-nanoseconds>-full-nanoseconds sn

Accessors for concrete and virtual fields of a <seconds-and-nanoseconds> instance.

— Function: <seconds-and-nanoseconds>-deep-clone sn
— Function: <seconds-and-nanoseconds>-shallow-clone sn

Return a copy of a <seconds-and-nanoseconds> instance.
