64.1.1 Parsed components

The following is an overview of an email address' components.

Comments
Comments are strings enclosed in parentheses and may be present at any point in an address.
Domain
A list of strings separated by dots: 
          alpha.beta.gamma
          ⇒ domain strings = alpha beta gamma

Domain literal
A dotted quad enclosed in square brackets: 
          [101.102.103.104]
          domain literal ⇒ 101.102.103.104

Local part
A list of strings separated by dots: 
          alpha.beta.gamma
          ⇒ domain strings = alpha beta gamma

Address specification
          marco@localhost
          ⇒ local part = marco
          ⇒ domain     = localhost

Route
          @alpha.it,@beta.it:
          ⇒ first domain  = alpha.it
          ⇒ second domain = beta.it

Route address
          <@alpha.it,@beta.it:marco@there.it>
          ⇒ route      = @alpha.it,@beta.it
          ⇒ local part = marco
          ⇒ domain     = there.it

Mailbox
          "Marco Maggi" <@alpha.it,@beta.it:marco@there.it>
          ⇒ phrase (quoted literal) = "Marco Maggi"
          ⇒ route                   = @alpha.it,@beta.it
          ⇒ local part              = marco
          ⇒ domain                  = there.it
          
          my address <marco@there.it>
          ⇒ phrase (atoms)          = my address
          ⇒ route                   =
          ⇒ local part              = marco
          ⇒ domain                  = there.it

Lists of mailboxes are mailboxes separated by ,: 

          <marco@there.it>, <marco@localhost>, <root@localhost>
          ⇒ first mailbox  = <marco@there.it>
          ⇒ second mailbox = <marco@localhost>
          ⇒ third mailbox  = <root@localhost>

Group
List of mailboxes enclosed in ‘: ... ;’ with optional phrase: 
          : marco@localhost, <marco@there.com> ;
          ⇒ phrase         =
          ⇒ first mailbox  = marco@localhost
          ⇒ second mailbox = marco@there.com
          
          :marco@localhost, "Marco Maggi" (a comment) <marco@there.com> ;
          ⇒ phrase         =
          ⇒ first mailbox  = marco@localhost
          ⇒ second mailbox = "Marco Maggi" <marco@there.com>
          
          mailing-list :marco@localhost, ,,, ,, ,
             ,, "Marco Maggi" <marco@there.com> ;
          ⇒ phrase         = mailing-list
          ⇒ first mailbox  = marco@localhost
          ⇒ second mailbox = "Marco Maggi" <marco@there.com>

Address
An address is a list of mailboxes and groups separated by ‘,’.
