30.2.9 Reference counting

Reference counting is implemented by allocating a block of memory with the requested size plus ‘strideof-long’. The block will look like this: 

      strideof-long            data area
     |-------------|--------------------------------------|
                   ^
                   |
     pointer returned by MALLOC/REFCOUNT

the reference counter is handled as an unsigned long and it is located at a negative offset from the address we use to access data.

The following bindings are exported by (nausicaa ffi memory refcount).

— Function: malloc/refcount number-of-bytes
— Function: malloc/refcount number-of-bytes malloc-func
— Function: malloc/rc number-of-bytes
— Function: malloc/rc number-of-bytes malloc-func

Allocate a block of memory wide enough to hold number-of-bytes using reference counting; return a pointer object. Memory is allocated with the procedure malloc-func which defaults to malloc. Return a pointer to the first byte in the data area of the allocated block; the reference counter is initially set to zero.

— Function: pointer-acquire pointer

Increment the reference counter of the memory block referenced by pointer. The block must have been allocated with malloc/rc.

— Function: pointer-release pointer
— Function: pointer-release pointer free-func

Decrement the reference counter of the memory block referenced by pointer; the block must have been allocated with malloc/rc. If the counter reaches zero: The block is freed using the supplied free-func, which defaults to primitive-free.

Usage example with dynamic-wind: 

     (let ((p (malloc/rc 4096)))
       (dynamic-wind
         (lambda () (pointer-acquire p))
         (lambda () (do-something-with p))
         (lambda () (pointer-release p))))

usage example with compensations (see compensations): 

     (with-compensations
       (letrec ((p (compensate
                       (begin0-let ((p (malloc/rc 4096)))
                         (pointer-acquire p))
                     (with
                      (pointer-release p)))))
         (do-something-with p)))))
