6.3.8 Label type for common condition types

The following bindings are exported by the (nausicaa language conditions) library.

— Label: <common-conditions>

Label type for compound condition objects with components: &who, &message, &irritants. It is derived from <condition> exported by (nausicaa language builtins).

— Common constructor on <common-conditions>: <common-conditions> constructor who message irritants

Build and return a new compound condition object with components: &who, &message, &irritants.

Safe virtual fields
— Immutable virtual fields of <common-conditions>: &who who

Accessing this field is like applying to the instance object the function condition-who from (rnrs conditions (6)).

— Immutable virtual fields of <common-conditions>: &message message

Accessing this field is like applying to the instance object the function condition-message from (rnrs conditions (6)).

— Immutable virtual fields of <common-conditions>: &irritants irritants

Accessing this field is like applying to the instance object the function condition-irritants from (rnrs conditions (6)).

OOPP syntaxes
— Maker on <common-conditions>: <common-conditions> (?who ?message ?irritants)

Build and return a new compound condition object with components: &who, &message, &irritants.
