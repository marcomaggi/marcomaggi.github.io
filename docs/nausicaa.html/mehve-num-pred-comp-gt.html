6.3.2.3 Numeric greater than predicate

The binding > exported by (nausicaa mehve) generalises the corresponding function exported by (rnrs base (6)); it is implemented using the following generic functions exported by (nausicaa mehve numeric-predicates).

— Generic Function: greater-than-predicate-0

Used when > is invoked with no arguments.

— Generic Function: greater-than-predicate-1 A

Used when > is applied to one argument.

— Generic Function: greater-than-predicate-2 A B

Used when > is applied to two arguments.

The following are the specialised methods.

— Method: greater-than-predicate-0

Return #f.

— Method: greater-than-predicate-1 (A <top>)

Return #f.

— Method: greater-than-predicate-2 (A <fixnum>) (B <fixnum>)

Apply to the arguments the function fx>? exported by (rnrs arithmetic fixnums (6)), see fx>?.

— Method: greater-than-predicate-2 (A <flonum>) (B <flonum>)

Apply to the arguments the function fl>? exported by (rnrs arithmetic flonums (6)), see fl>?.

— Method: greater-than-predicate-2 (A <complex>) (B <complex>)

Apply to the arguments the function > exported by (nausicaa), see >.
