6.6.2 Exponentiation and logarithm generic functions

The generic functions expt, sqrt and exp exported by (nausicaa mehve transcendental), and reexported by (nausicaa mehve), generalise the corresponding functions from (rnrs base (6)).

The binding log is exported by (nausicaa mehve) and generalises the corresponding function from (rnrs base (6)); its implementation functions log-1 and log-2 are exported by (nausicaa mehve transcendental).

— Generic Function: expt A B

Compute A raised to the power of B.

— Generic Function: sqrt A

Compute square root of A.

— Generic Function: exp A

Compute the base e exponential.

— Generic Function: log-1 A

Compute the base e logarithm.

— Generic Function: log-2 A B

Compute the base B logarithm of A.

The following are the specialised methods.

— Method: expt (A <flonum>) (B <flonum>)

Compute A raised to the power of B using flexpt from (rnrs arithmetic flonums (6)), see flexp.

— Method: expt (A <complex>) (B <complex>)

Compute A raised to the power of B using expt from (rnrs base (6)), see expt.

— Method: sqrt (A <flonum>)

Compute the square root of A using flsqrt from (rnrs arithmetic flonums (6)), see flexp.

— Method: sqrt (A <complex>)

Compute the square root of A using sqrt from (rnrs base (6)), see sqrt.

— Method: exp (A <flonum>)

Compute the base e exponential using flexp from (rnrs arithmetic flonums (6)), see flexp.

— Method: exp (A <complex>)

Compute the base e exponential using exp from (rnrs base (6)), see exp.

— Method: log-1 (A <flonum>)

Compute the base e logarithm using fllog from (rnrs arithmetic flonums (6)), see fllog.

— Method: log-1 (A <complex>)

Compute the base e logarithm using log from (rnrs base (6)), see log.

— Method: log-2 (A <flonum>) (B <flonum>)

Compute the base B logarithm of A using fllog from (rnrs arithmetic flonums (6)), see flexp.

— Method: log-2 (A <complex>) (B <complex>)

Compute the base B logarithm of A using log from (rnrs base (6)), see log.
