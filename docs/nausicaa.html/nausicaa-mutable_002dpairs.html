5.6 Mutable pairs

The library (nausicaa mutable-pairs) defines a label interface for mutable pairs and also reexports all the bindings from (rnrs mutable-pairs (6)).

— Label: <mutable-pair>

Label for handling mutable pairs through the facilities of the (nausicaa language classes) library.

— Mutable Virtual Field of <mutable-pair>: car

Access and mutate the car of a pair.

— Mutable Virtual Field of <mutable-pair>: cdr

Access and mutate the cdr of a pair.

— Syntax on <mutable-pair>: make <mutable-pair> car cdr
— Syntax on <mutable-pair>: make* <mutable-pair> car cdr

Build and return a new pair using cons.

— Syntax on <mutable-pair>: is-a? obj <mutable-pair>

Evaluate to true if obj is a pair.

Examples: 

     (import (nausicaa) (nausicaa mutable-pairs))
     
     (let-make ((p <mutable-pair> 1 2))
       (is-a? p <mutable-pair>))
     ⇒ #t
     
     (let-make ((p <mutable-pair> 1 2))
       p.car ⇒ 1
       p.cdr ⇒ 2
       )
     
     (let-make ((p <mutable-pair> 1 2))
       (set! p.car 3)
       p)
     ⇒ (3 . 2)
     
     (let-make ((p <mutable-pair> 1 2))
       (set! p.cdr 3)
       p)
     ⇒ (1 . 3)
