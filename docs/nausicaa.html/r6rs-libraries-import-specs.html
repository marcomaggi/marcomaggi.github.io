77.4.4.2 Import specifications

The bindings documented in this section are exported by the (nausicaa parser-tools r6rs libraries import) library.

— Class: <import-spec>

Type representing import specifications as specified by R6RS, scheme library form import.

— Immutable field of <import-spec>: <import-set> set

The core of this import specification.

— Immutable field of <import-spec>: <list> levels

A list of exact integers representing the import levels for which this import set was required. It contains at least one element.

— Immutable field of <import-spec>: <sexp> original

The original symbolic expression of this import set. <sexp> is the label from (nausicaa language matches).

— Immutable field of <import-spec>: <library-reference> library-reference

The library reference nested in this import set.

— Common constructor on <import-spec>: constructor sexp

Parse the symbolic expression sexp as an import specification as specified by R6RS; return a new <import-spec> object. Raise an assertion violation if sexp is invalid.

— Syntax on <import-spec>: make <import-spec> sexp
— Syntax on <import-spec>: make* <import-spec> sexp

Build and return a new <import-spec> instance using the common constructor.

— Syntax on <import-spec>: is-a? obj <import-spec>

Return #t if obj is a record of type <import-spec>.

— Method on <import-spec>: apply renamings

Apply the import set specification to the list of renamings, returning the processed list of renamings (which can be null). It applies recursively the specifications of the nested import sets. 

          (let (((o <import-spec>)
                 (make* <import-spec>
                   '(prefix
                       (except
                         (only
                           (rename (alpha beta (1))
                                   (ae aee)
                                   (be bee)
                                   (ce cee))
                           bee cee)
                         bee)
                       px.))))
            (o.apply '((a ae)
                       (b be)
                       (c ce)
                       (d de)
                       (f fe)
                       (g ge))))
          ⇒ ((c px.cee))
