11.2.3 URI host objects

The following bindings are exported by the (nausicaa uri uri) library.

— Label: <host>

Label used to tag bytevectors representing the host component of URIs; it is derived from <ip-address> defined by (nausicaa uri ip).

Methods
— Method on <host>: put-bytevector (port <binary-output-port>)

Write to port a bytevector representing the host component in the format requested by RFC 3986.
