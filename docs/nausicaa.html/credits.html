Language and implementations

Scheme is a statically scoped and properly tail–recursive dialect of the Lisp programming language invented by Guy Lewis Steele Jr. and Gerald Jay Sussman. It was designed to have an exceptionally clear and simple semantics and few different ways to form expressions.

Several libraries depend upon the port to R6RS of the Scheme Requests For Implementation (SRFI). The original SRFI code is the work of many authors, reference of which you can find the at the top of the source code files; the port to R6RS included in this project is the work of Derick Eddington.

Mosh
An R6RS compliant implementation of the Scheme programming language. It is the creation of Taro Minowa (higepon).
Petite Chez Scheme
An also R6RS compliant implementation of the Scheme programming language. It is produced by Cadence Research Systems.
Racket
A distribution of languages and libraries with support for R6RS. It is the work of many people (see the website).
Vicare
An experimental fork of Ikarus. Ikarus is an almost R6RS compliant implementation of the Scheme programming language. It is the creation of Abdulaziz Ghuloum.
Ypsilon
An R6RS compliant implementation of the Scheme programming language. It is the creation of Yoshikatsu Fujita at LittleWing Company.

Nausicaa/Scheme is the creation of Marco Maggi. If this work exists, it is because of the GNU Software tools he uses all the time.
