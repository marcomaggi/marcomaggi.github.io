76.5.3 Converting between 10 and 13 digits representations

The following bindings are exported by the (nausicaa standard-numbers imei) library.

— Function: imei-drop-separators imei-string

Convert imei-string to its minimal representation: strip the string of any valid separators. If the input string is well formed according to IMEI?/WELL-FORMED: the returned string is also well formed. 

          (imei-drop-separators "35686800-004141-20")
          ⇒ "3568680000414120"
          
          (imei-drop-separators "3568680000414120")
          ⇒ "3568680000414120"
— Function: imei->imei16 imei software-version

Convert the Scheme string imei to a 16 digits IMEI; return imei itself if it is already in the correct format, else return a new string. The input string is expected to be well formed according to imei?/well-formed.

This function uses the Scheme string software-version as software version when imei is in 15 digits format. 

          ;; 35-417803-685978-9
          (imei->imei16 "35-417803-685978-9" "87")
          ⇒ "35-417803-685978-87"
          
          (imei->imei16 "354178036859789" "87")
          ⇒ "3541780368597887"
          
          ;; 35686800-004141-20
          (imei->imei16 "35686800-004141-20" "87")
          ⇒ "35686800-004141-20"
          
          (imei->imei16 "3568680000414120" "87")
          ⇒ "3568680000414120"
— Function: imei->imei15 imei

Convert the Scheme string imei to a 15 digits IMEI; return imei itself if it is already in the correct format, else return a new string. The input string is expected to be well formed according to imei?/well-formed. 

          ;; 35-417803-685978-9
          (imei->imei15 "35-417803-685978-9")
          ⇒ "35-417803-685978-9"
          
          (imei->imei15 "354178036859789")
          ⇒ "354178036859789"
          
          ;; 35686800-004141-20
          (imei->imei15 "35686800-004141-20")
          ⇒ "35686800-004141-8"
          
          (imei->imei15 "3568680000414120")
          ⇒ "356868000041418"
