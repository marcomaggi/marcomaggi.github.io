6.6.3 Trigonometric generic functions

The generic functions sin, cos, tan, asin and acos exported by (nausicaa mehve transcendental), and reexported by (nausicaa mehve), generalise the corresponding functions from (rnrs base (6)).

The binding atan is exported by (nausicaa mehve) and generalises the corresponding function from (rnrs base (6)); its implementation functions atan-1 and atan-2 are exported by (nausicaa mehve transcendental).

— Generic Function: sin A
— Generic Function: cos A
— Generic Function: tan A

Compute the sine, cosine and tangent functions.

— Generic Function: asin A
— Generic Function: acos A

Compute the arc sine, arc cosine and arc tangent functions.

— Generic Function: atan-1 A

Compute the arc tangent function when atan is called with one argument.

— Generic Function: atan-2 A B

Compute the arc tangent function when atan is called with two arguments.

The following are the specialised methods.

— Method: sin (A <flonum>)
— Method: cos (A <flonum>)
— Method: tan (A <flonum>)

Compute the sine, cosine and tangent functions using flsin, flcos and fltan from (rnrs arithmetic flonums (6)), see flsin.

— Method: asin (A <flonum>)
— Method: acos (A <flonum>)
— Method: atan-1 (A <flonum>)
— Method: atan-2 (A <flonum>) (B <flonum>)

Compute the arc sine, arc cosine and arc tangent functions using flasin, flacos and flatan from (rnrs arithmetic flonums (6)), see flasin.

— Method: sin (A <complex>)
— Method: cos (A <complex>)
— Method: tan (A <complex>)

Compute the sine, cosine and tangent functions using sin, cos and tan from (rnrs base (6)), see sin.

— Method: asin (A <complex>)
— Method: acos (A <complex>)
— Method: atan-1 (A <complex>)
— Method: atan-2 (A <real>) (B <real>)

Compute the arc sine, arc cosine and arc tangent functions using asin, acos and atan from (rnrs base (6)), see asin.
