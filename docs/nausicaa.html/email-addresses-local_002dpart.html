60.1.2.2 Local part
— Class: <local-part>

Holds a list of local part strings. The local part is a single string, but for parsing reasons it is accumulated as a list of strings.

— Immutable Field of <local-part>: subparts

A list of strings.

— Function: <local-part>-subparts lp

Field accessor.

— Generic Function: object->string (o <local-part>)

Return the string representation of o.
