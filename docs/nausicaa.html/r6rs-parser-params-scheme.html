73.2.3.2 Scheme value datum makers

The following bindings are exported by the (nausicaa r6rs parser) library.

— Parameter: identifier-datum-maker

Set to a function returning a value representing an identifier. The default is make-identifier-datum.

— Parameter: boolean-datum-maker

Set to a function returning a value representing a boolean. The default is make-boolean-datum.

— Parameter: number-datum-maker

Set to a function returning a value representing a number. The default is make-number-datum.

— Parameter: string-datum-maker

Set to a function returning a value representing a string. The default is make-string-datum.

— Parameter: character-datum-maker

Set to a function returning a value representing a character. The default is make-character-datum.

— Parameter: pair-datum-maker

Set to a function returning a value representing a pair. The default is make-pair-datum. This parameter is used as follows: 

          ((pair-datum-maker)  yypushback yycustom car cdr)
— Parameter: list-datum-maker

Set to a function returning a value representing a list. The default is make-list-datum.

The semantic value handed to the referenced function is a proper or improper list of datums (that is: the list of values returned by the functions referenced by the parameters).

— Parameter: vector-datum-maker

Set to a function returning a value representing a vector. The default is make-vector-datum.

The semantic value handed to the referenced function is a proper list of datums (that is: the list of values returned by the functions referenced by the parameters).

— Parameter: bytevector-datum-maker

Set to a function returning a value representing a bytevector. The default is make-bytevector-datum.

The semantic value handed to the referenced function is the list of datums (that is: the list of values returned by the functions referenced by the parameters); it is responsibility of the referenced function to make sure that all the datums in the list are unsigned exact integers in the range [0, 255].
