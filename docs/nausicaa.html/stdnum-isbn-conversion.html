72.4.3 Converting between 10 and 13 digits representations

The following bindings are exported by the (nausicaa standard-numbers isbn) library.

— Function: isbn-drop-separators isbn-string

Convert isbn-string to its minimal representation: strip the string of any valid separators. If the input string is well formed according to ISBN?/WELL-FORMED: the returned string is also well formed. 

          (isbn-drop-separators "1-85798-218-5")
          ⇒ "1857982185"
          
          (isbn-drop-separators "  1-85798-218-5  ")
          ⇒ "1857982185"
          
          (isbn-drop-separators "81 7525 766 0")
          ⇒ "8175257665"
          
          (isbn-drop-separators "8175257660")
          ⇒ "8175257665"
— Function: isbn->isbn13 isbn-string
— Function: isbn->isbn13 isbn-string prefix-string

Convert isbn-string to a 13 digits ISBN; return isbn-string itself if it is already in the correct format, else return a new string. The input string is expected to be well formed according to isbn?/well-formed.

This function uses the 978 prefix by default or prefix-string if one is given; prefix-string should be the string 978 or 979.

— Function: isbn->isbn10 isbn-string

Convert isbn-string to a 10 digits ISBN; return isbn-string itself if it is already in the correct format, else return a new string. The input string is expected to be well formed according to isbn?/well-formed.
