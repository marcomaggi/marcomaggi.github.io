78.6.1 Introduction to GRid identifiers

For an introduction on GRid identifiers see (URL last verified Jun 3, 2011):

http://en.wikipedia.org/wiki/Global_Release_Identifier

for additional informations on GRid identifiers, including a handbook and an online check digit calculator, see (URL last verified Jun 3, 2011):

http://www.ifpi.org/content/section_resources/grid.html

The (nausicaa standard-numbers grid) library handles GRid identifiers as Scheme strings, stripped of any blank character before and after them; GRid identifiers often appear with the string GRid: in front of them, but the library assumes that this tag has already been stripped.

The following are examples of the same GRid identifier: 

     A12425GABC1234002M
     A1-2425G-ABC1234002-M
     GRid:A1-2425G-ABC1234002-M

an identifier has the fixed length of 18 characters, optionally separated by hyphens; all the characters are digits or upper case letters in ASCII encoding.

A GRid has 4 fields of fixed length: the Identifier Scheme element, always ‘A1’ for GRids; the Issuer Code element, always 5 characters; the Release Number element, always 10 characters; the check digit, always a single character.
