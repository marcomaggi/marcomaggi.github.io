Nausicaa for Scheme

This document describes version 0.2d6 of Nausicaa/Scheme, a set of Scheme libraries defining a slightly modified R6RS Scheme language and augmenting the features of the base and standard R6RS libraries.

The project home page of Nausicaa is at:

http://marcomaggi.github.com/nausicaa.html

development of Nausicaa takes place at:

http://github.com/marcomaggi/nausicaa/
