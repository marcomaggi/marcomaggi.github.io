49.6 Points in time

A time point is described by seconds and nanoseconds counts on the TAI scale (no leap seconds, time zone zero, no daylight saving time) using the Unix Epoch as origin. The following bindings are exported by the (nausicaa times-and-dates) library.

— Class: <time>

Type representing a time point; it is derived from <seconds-and-nanoseconds>.

— Immutable Field of <time>: seconds

Exact integer representing seconds count on the TAI scale.

— Immutable Field of <time>: nanoseconds

Exact integer representing nanoseconds count.

— Immutable Virtual Field of <time>: julian-day
— Immutable Virtual Field of <time>: modified-julian-day

Exact numbers representing the Julian Day Number and the Modified Julian Day Number corresponding to the instance's seconds on the TAI scale.

— Syntax on <time>: make* <time> seconds nanoseconds

Expand to an expression evaluating to a new <time> instance. seconds and nanoseconds must expand into expressions evaluating to exact integers representing the counts of seconds on the TAI scale and the count of nanoseconds.

Seconds and nanoseconds are normalised to be both non–negative or both non–positive.

— Syntax on <time>: make <time> clause ...
— Auxiliary Syntax on <time>: seconds seconds
— Auxiliary Syntax on <time>: milliseconds milliseconds
— Auxiliary Syntax on <time>: microseconds microseconds
— Auxiliary Syntax on <time>: nanoseconds nanoseconds

Expand to an expression evaluating to a new <time> instance. The seconds and nanoseconds counts are computed from the values given in the following maker clauses and normalised to be both non–negative or both non–positive:

seconds seconds
milliseconds milliseconds
microseconds microseconds
nanoseconds nanoseconds
Select the count in the specified seconds and subseconds units. The seconds count must be on the TAI scale.

— Method on <time>: deep-clone
— Method on <time>: shallow-clone

Return a copy of the object.

Comparing time points

Comparison aims at estabilishing if a time point comes before or after another time point; comparison functions do consider the sign of seconds and nanoseconds counts.

— Method on <time>: = time
— Method on <time>: < time
— Method on <time>: <= time
— Method on <time>: > time
— Method on <time>: >= time

Compare the instance's time to the time of the <time> object time and return a boolean value.

— Function: time= time ...
— Function: time< time ...
— Function: time<= time ...
— Function: time> time ...
— Function: time>= time ...

Compare the <time> arguments and return a boolean value. When there are no arguments or only one argument: the return value is true.

Arithmetic functions
— Method on <time>: + duration

Add the <duration> object duration to the instance and return a new <time> object representing the result.

— Method on <time>: - (duration <duration>)

Subtract the <duration> object duration from the instance and return a new <time> object representing the result. 

          (let* (((D <time>)      (make* <time> 1 2))
                 ((E <duration>)  (make* <duration> 3 5))
                 ((F <time>)      (D.- E)))
            (list (is-a? F <time>) F.seconds F.nanoseconds))
          ⇒ (#t -2 -3)
— Method on <time>: - (time <time>)

Subtract the <time> object time from the instance and return a new <duration> object representing the result. 

          (let* (((D <time>)      (make* <time> 1 2))
                 ((E <time>)      (make* <time> 3 5))
                 ((F <duration>)  (D.- E)))
            (list (is-a? F <duration>) F.seconds F.nanoseconds))
          ⇒ (#t -2 -3)
Utility functions
— Method on <time>: date
— Method on <time>: date tz-offset

Return a <date> object representing the same time point of the instance. tz-offset is the time zone offset in seconds, it defaults to the local time zone offset.

Traditional record API
— Function: make-<time> seconds nanoseconds

Build and return a new <time> instance.

— Function: <time>? obj

Return true if obj is an instance of <time>.

— Function: <time>-seconds sn
— Function: <time>-nanoseconds sn
— Function: <time>-full-seconds sn
— Function: <time>-full-microseconds sn
— Function: <time>-full-nanoseconds sn

Accessors for concrete and virtual fields of a <time> instance.

— Function: <time>-deep-clone sn
— Function: <time>-shallow-clone sn

Return a copy of a <time> instance.
