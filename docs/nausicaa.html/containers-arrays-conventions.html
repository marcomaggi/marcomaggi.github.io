10.4.1 Arguments conventions

In the documentation of (nausicaa containers arrays) we call:

items
The values collected in Scheme's built–in lists and vectors.
elements
The values collected in the arrays.

The following conventions are used in the documentation of arguments to the functions of (nausicaa containers arrays):

array
A proper array object.
shape
A proper shape object.
position
A proper position object.
position-step
A Scheme built–in vector used to represent the difference between two position values.
dimension
A strictly positive, exact integer representing the extension of a dimension in an array.
index
A non–negative, exact integer representing the coordinate of an element.
start
last
past
When specifying a range of items or elements: start is the left–included bound, last is the right–included bound, past is the right–excluded bound.
obj
Can be any value.
