31.3.5 Converting arrays of cstrings

A null–terminated array of C strings looks like this: 

                       -
     argv pointer ---> | pointer to string -----> C string 0
                       -
                       | pointer to string -----> C string 1
                       -
                       | pointer to string -----> C string 2
                       -
                       | pointer to string -----> C string 3
                       -
                       | null pointer
                       -
— Function: strings->argv args
— Function: strings->argv args malloc-func

Convert the list of Scheme strings and symbols in args into a null–terminated array of strings. Return the pointer to the array.

The array and the strings are allocated with the function malloc-func, or with malloc if malloc-func is not given.

— Function: argv-length argv

Return the number of strings in the null–terminated array of strings referenced by the argv pointer.

— Function: argv->strings argv
— Function: argv->strings argv argc

Without the argc arguments: Convert the NULL–terminated array of strings referenced by the argv pointer to a list of Scheme strings.

With the argc arguments: Convert argc strings in the array of strings referenced by the argv pointer to a list of Scheme strings.
