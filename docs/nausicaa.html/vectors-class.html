36.17 Class interface to vector functions

The (nausicaa vectors xvectors) library exports the single binding <xvector>, which provides an interface to the (nausicaa vectors) functions through the facilities of the (nausicaa language classes) library.

— Label: <xvector>

Label interface to the vector functions. It is derived from the foreign class <vector>, inheriting all its fields and methods.

All the following methods are implemented as syntaxes; in all the descriptions: self is an identifier bound to the subject vector.

Constructors
— Method on <xvector>: concatenate list-of-vectors

Implement the expansions: 

          (self.concatenate list-of-vectors)
          ==> (vector-concatenate (cons self list-of-vectors))
— Method on <xvector>: concatenate-reverse list-of-vectors
— Method on <xvector>: concatenate-reverse list-of-vectors final-vector
— Method on <xvector>: concatenate-reverse list-of-vectors final-vector nvalues

Implement the expansions: 

          (self.concatenate-reverse list-of-vectors)
          ==> (vector-concatenate-reverse
               (cons self list-of-vectors))
          
          (self.concatenate-reverse list-of-vectors
                                    final-vector)
          ==> (vector-concatenate-reverse
               (cons self list-of-vectors)
               final-vector)
          
          (self.concatenate-reverse list-of-vectors
                                    final-vector nvalues)
          ==> (vector-concatenate-reverse
               (cons self list-of-vectors)
               final-vector nvalues)
— Method on <xvector>: append . args

Implement the expansion:
