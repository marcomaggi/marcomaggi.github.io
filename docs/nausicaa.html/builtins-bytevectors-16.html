5.13.3 Bytevector labels: 16-bit words

— Builtin Label: <bytevector-u16l>
— Builtin Label: <bytevector-s16l>
— Builtin Label: <bytevector-u16b>
— Builtin Label: <bytevector-s16b>
— Builtin Label: <bytevector-u16n>
— Builtin Label: <bytevector-s16n>

Conventional type for objects satisfying bytevector? whose contents must be accessed as 16-bit words. The suffix ‘l’ stands for little–endian; the suffix ‘b’ stands for big–endian; the suffix ‘n’ stands for native endianness.

OOPP syntaxes
— Getter on <bytevector-u16l>: instance (?index)
— Getter on <bytevector-u16b>: instance (?index)
— Getter on <bytevector-u16n>: instance (?index)
— Getter on <bytevector-s16l>: instance (?index)
— Getter on <bytevector-s16b>: instance (?index)
— Getter on <bytevector-s16n>: instance (?index)

16-bit word getter. ?index is the zero–based index of the selected word.

— Setter on <bytevector-u16l>: set!/tags (instance (?index)) ?item
— Setter on <bytevector-u16b>: set!/tags (instance (?index)) ?item
— Setter on <bytevector-u16n>: set!/tags (instance (?index)) ?item
— Setter on <bytevector-s16l>: set!/tags (instance (?index)) ?item
— Setter on <bytevector-s16b>: set!/tags (instance (?index)) ?item
— Setter on <bytevector-s16n>: set!/tags (instance (?index)) ?item

16-bit word setter. ?index is the zero–based index of the selected word.
