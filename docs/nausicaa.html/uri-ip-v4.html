11.1.4 Ipv4 address objects

The library (nausicaa uri ip) exports utilities to handle IPv4 addresses specifications, according to RFC 5735. To parse the string representation of IPv4 addresses we can use the library (nausicaa parser-tools ipv4-addresses).

  • uri ip v4 address: IPv4 address objects class.
  • uri ip v4 prefix: IPv4 address prefix objects class.
  • uri ip v4 aux: IPv4 address auxiliary classes.
