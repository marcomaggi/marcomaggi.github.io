3.8 Defining custom finalisers

— Auxiliary Syntax: finaliser ?expression

Optional clause selecting a finaliser function for class instances; Automatic finalisation of records. This clause can be used in the body of define-class and define-mixin.

?expression, evaluated at run–time, must return a function accepting a single argument; such function is used as second argument in a call to record-destructor-set! setting a destructor function for instances of the class type.
