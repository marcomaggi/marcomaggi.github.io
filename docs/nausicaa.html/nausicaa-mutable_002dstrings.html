5.7 Mutable strings

The library (nausicaa mutable-strings) defines a label interface for mutable strings and also reexports all the bindings from (rnrs mutable-strings (6)).

— Label: <mutable-string>

Label for handling mutable strings through the facilities of the (nausicaa language classes) library. Derived from <string>, inherits all its methods and fields.

— Syntax: make <mutable-string> char ...

Build and return a new string using string.

— Syntax: is-a? obj <mutable-string>

Evaluate to true if obj is a string.

— Method on <mutable-string>: set! position char

Store char at position in the string.

— Method on <mutable-string>: fill! char

Store char in every element of the string.

— Syntax: setf (o index) char

Store char at index of the string o. index is an expression.
