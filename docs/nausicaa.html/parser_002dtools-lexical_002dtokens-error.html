11.2.4 Lexical error class

The following bindings are exported by the library (nausicaa parser-tools lexical-tokens).

— Class: <lexer-error>

Hold the informations about a lexer error; it is derived from <lexical-token>. The value of the category field is always ‘*lexer-error*’.

— Common constructor on <lexer-error>: constructor location error-message

Build and return a new instance of <lexer-error> using the given arguments as field values.

— Maker syntax on <lexer-error>: <lexer-error> (?clause ...)
— Auxiliary Syntax on <lexer-error>: location: ?location
— Auxiliary Syntax on <lexer-error>: error-message: ?message

Build and return a new instance of <lexer-error> using the default constructor.

The clause error-message: is mandatory and its value ?message must evaluate to a string.

?location defaults to the return value of unspecified-source-location from the library (nausicaa parser-tools source-locations).

— Immutable field of <lexer-error>: <string> message

A Scheme string describing the error.
