12.2.3 End–of–input objects

The following bindings are exported by the library (nausicaa parser-tools lexical-tokens).

— Label: <end-of-input>

Hold the informations about a lexical token representing the end–of-input; it is derived from <lexical-token>. The value of the category field is always ‘*eoi*’.

— Common constructor on <end-of-input>: constructor location

Build and return a new instance of <end-of-input> using the given arguments as field values.

— Maker syntax on <end-of-input>: <end-of-input> (?clause ...)
— Auxiliary Syntax on <end-of-input>: location: ?location

Build and return a new instance of <end-of-input> using the default constructor.

?location must evaluate to an object of type <source-location>; it defaults to the return value of unspecified-source-location from the library (nausicaa parser-tools source-locations).
