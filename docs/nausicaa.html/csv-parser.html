62.2 High level parsing

— Function: csv->list port separators
— Function: csv->list port separators swirl-field

Read a CSV file from the textual input port and convert it into a list of lists, with each nested list representing a record.

separators must be a list of characters and used as field separators in a line.

swirl-field must be a procedure accepting a single string argument and returning a string argument. It is applied to each field string before adding it to the output list. This procedure is allowed to raise an exception.

— Function: csv->list/comma port
— Function: csv->list/comma port swirl-field

Like csv->list, but use only the comma as field separator.
