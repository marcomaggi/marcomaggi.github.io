11.1.4.2 IPv4 address prefix objects class

The following bindings are exported by the library (nausicaa uri ip).

— Class: <ipv4-address-prefix>

Represents an IPv4 address prefix.

Constructors and makers
— Common constructor on <ipv4-address-prefix>: constructor prefix-length numbers
— Common constructor on <ipv4-address-prefix>: constructor prefix-length third second first zeroth

Build and return a new object of type <ipv4-address-prefix>.

When called with 2 arguments: prefix-length must be a non–negative fixnum specifying how many of the leftmost contiguous bits of the address comprise the prefix; numbers must be a vector of 4 fixnums in the range [0, 255]; .

When called with 5 arguments: prefix-length must be a non–negative fixnum specifying how many of the leftmost contiguous bits of the address comprise the prefix; each of the other arguments must be a fixnum in the range [0, 255].

— Maker on <ipv4-address-prefix>: <ipv4-address-prefix> (?prefix-length ?numbers)
— Maker on <ipv4-address-prefix>: <ipv4-address-prefix> (?prefix-length ?third ?second ?first ?zeroth)

Build and return a new object of type <ipv4-address-prefix> using the common constructor. 

          (<ipv4-address-prefix> O
            (receive (numbers len)
                (parse-ipv4-address-prefix "1.2.3.4/8")
              (<ipv4-address-prefix> (len numbers))))
          
          (list (O third) (O second) (O first) (O zeroth))
          ⇒ (1 2 3 4)
          
          (O prefix-length)
          ⇒ 8
          
          (O string)
          ⇒ "1.2.3.4/8"
Fields
— Immutable field of <ipv4-address-prefix>: <ipv4-address-fixnum> zeroth
— Immutable field of <ipv4-address-prefix>: <ipv4-address-fixnum> first
— Immutable field of <ipv4-address-prefix>: <ipv4-address-fixnum> second
— Immutable field of <ipv4-address-prefix>: <ipv4-address-fixnum> third

The numeric components of the address prefix, each representing 8 bits. zeroth is the rightmost component, first is the second from the right, etc.

— Immutable Field of <ipv4-address-prefix>: <ipv4-address-prefix-length> prefix-length

A fixnum specifying how many of the leftmost contiguous bits of the address comprise the prefix.

— Immutable virtual field of <ipv4-address-prefix>: <string> string

The string representation of the address prefix.

— Immutable virtual field of <ipv4-address-prefix>: <ascii-bytevector> bytevector

The bytevector representation of the address prefix, in ASCII encoding.
