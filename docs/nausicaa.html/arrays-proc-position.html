43.3.1 Specifying the coordinates of elements

The elements' positions in arrays are represented by builtin Scheme vectors holding exact integers.

Constructors
— Function: array-position index0 index ...

Build and return a new Scheme vector holding the index arguments. The indexes are checked to be exact integers. The number of arguments is meant to be equal to the number of dimensions of the array with which the position value will be used.

Predicates and assertions
— Function: array-position? obj

Return #t if obj is a Scheme vector holding exact integers.

— Function: assert-array-position obj func-name
— Function: assert-array-position/or-false obj func-name

Raise an assertion violation exception if obj satisfies array-position?, or it is #f. func-name is used who argument for the exception. Return #t if no exception is raised.

Conversion
— Function: array-position->string position

Return a string representation of position, specialised for array positions.

Port output
— Function: array-position-display position
— Function: array-position-display position port

Write to port a human readable string representation of position. port defaults to the current output port.

— Function: array-position-write position
— Function: array-position-write position port

Write to port a string representation of position such that it can be read back with read. port defaults to the current output port.
