55.5 Conversion

— Function: stack->list stk
— Function: stack->vector stk

Return a list or vector holding the elements in the stack. In the list case, return a copy of the spine of the list in the stack.

— Method on <stack>: list
— Method on <stack>: vector

Return a list or vector holding the elements in the stack. In the list case, return a copy of the spine of the list in the stack.

— Function: list->stack ell
— Function: vector->stack vec

Build and return a new stack holding the elements from the list or vector. In the list case, the stack holds a copy of the spine of the list argument.
