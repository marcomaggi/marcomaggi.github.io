50.3.4 List types

— Class: <pair>

Class for values that satisfy pair?.

— Class: <list>

Class for values that satisfy list?.

— Class: <circular-list>

Class for values that satisfy circular-list?. Notice that a pair and a proper list are not circular lists.

— Class: <dotted-list>

Class for values that satisfy dotted-list?. Notice that a pair is always a dotted list, but a proper list is not a dotted list.
