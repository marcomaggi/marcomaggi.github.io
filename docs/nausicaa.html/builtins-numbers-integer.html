5.15.8 Integer number objects

— Builtin Label: <integer>

Conventional type for objects satisfying integer?. It is derived from <integer-valued>.

Safe accessors
— Immutable virtual field of <integer>: <boolean> odd?

Accessing this field is like applying to the instance number the function odd? from (rnrs base (6)).

— Immutable virtual field of <integer>: <boolean> even?

Accessing this field is like applying to the instance number the function even? from (rnrs base (6)).
