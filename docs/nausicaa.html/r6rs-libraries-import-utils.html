77.4.4.3 Utility functions for import clauses
— Function: apply-import-spec/only renamings list-of-identifiers

Apply the only import specification. renamings must be a list of renamings. list-of-identifiers must be the list of symbols to accept in the renamings, all the others are discarded. 

          (apply-import-spec/only '((a ea) (b eb) (c ec))
                                  '(eb ec))
          ⇒ ((b eb) (c ec))
— Function: apply-import-spec/except renamings list-of-identifiers

Apply the except import specification. renamings must be a list of renamings. list-of-identifiers must be the list of symbols to discard in the renamings, all the others are accepted. 

          (apply-import-spec/except '((a ea) (b eb) (c ec))
                                    '(eb ec))
          ⇒ ((a ea))
— Function: apply-import-spec/prefix renamings prefix-symbol

Apply the prefix import specification. renamings must be a list of renamings. prefix-symbol must be the symbol to prefix to the second symbol in the renamings. 

          (apply-import-spec/prefix '((a ea) (b eb) (c ec))
                                    'ciao:)
          ⇒ ((a ciao:ea) (b ciao:eb) (c ciao:ec))
— Function: apply-import-spec/rename renamings renamings-spec

Apply the rename import specification. renamings must be a list of renamings. renamings-spec must be the list of renamings specifying the rename operation for renamings. 

          (apply-import-spec/rename '((a ea) (b eb) (c ec))
                                    '((eb ebb) (ec ecc)))
          ⇒ ((a ea) (b ebb) (c ecc))
