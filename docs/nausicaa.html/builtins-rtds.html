5.18 R6RS record type descriptor labels

— Builtin Label: <record-type-descriptor>

Conventional type for objects satisfying record-type-descriptor?.

Safe accessors and mutators
— Immutable virtual field of <record-type-descriptor>: <symbol> name

Return the name of the record type. Retrieving this field is like applying record-type-name to the instance.

— Immutable virtual field of <record-type-descriptor>: <record-type-descriptor> parent

Return #f or the record type descriptor of the parent. Retrieving this field is like applying record-type-parent to the instance.

— Immutable virtual field of <record-type-descriptor>: <symbol> uid

Return #f or a symbol representing the unique identifiers of the instance. Retrieving this field is like applying record-type-uid to the instance.

— Immutable virtual field of <record-type-descriptor>: <boolean> generative?

Returna a boolean, #t if the record type is generative. Retrieving this field is like applying record-type-generative? to the instance.

— Immutable virtual field of <record-type-descriptor>: <boolean> sealed?

Returna a boolean, #t if the record type is sealed. Retrieving this field is like applying record-type-sealed? to the instance.

— Immutable virtual field of <record-type-descriptor>: <boolean> opaque?

Returna a boolean, #t if the record type is opaque. Retrieving this field is like applying record-type-opaque? to the instance.

— Immutable virtual field of <record-type-descriptor>: <vector> field-names

Return a vector of symbols representing the field names. Retrieving this field is like applying record-type-field-names to the instance.

Methods
— Method on <record-type-descriptor>: <procedure> predicate

Return the predicate procedure of the instance. Calling this method is like applying record-predicate to the instance.

— Method on <record-type-descriptor>: <procedure> accessor field-idx

Return the accessor procedure for field field-idx. Calling this method is like applying record-accessor to the instance.

— Method on <record-type-descriptor>: <procedure> mutator field-idx

Return the mutator procedure for field field-idx. Calling this method is like applying record-mutator to the instance.

— Method on <record-type-descriptor>: <boolean> field-mutable? field-idx

Return a boolean, #t if field field-idx is mutable. Calling this method is like applying record-field-mutable? to the instance.
