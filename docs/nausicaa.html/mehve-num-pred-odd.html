6.3.4 Even and odd predicates generic functions

The generic functions odd? and even? exported by (nausicaa mehve numeric-predicates), and reexported by (nausicaa mehve), generalise the corresponding functions exported by (rnrs base (6)).

— Generic Function: odd? A

Return #t if the argument is odd.

— Generic Function: even? A

Return #t if the argument is even.

The following are the specialised methods.

— Method: odd? (A <fixnum>)

Apply to the argument the function fxodd? exported by the (rnrs arithmetic fixnums (6)) library, see fxodd?.

— Method: odd? (A <flonum>)

Apply to the argument the function flodd? exported by the (rnrs arithmetic flonums (6)) library, see flodd?.

— Method: odd? (A <real>)

Apply to the argument the function odd? exported by the (rnrs base (6)) library, see odd?.

— Method: even? (A <fixnum>)

Apply to the argument the function fxeven? exported by the (rnrs arithmetic fixnums (6)) library, see fxeven?.

— Method: even? (A <flonum>)

Apply to the argument the function fleven? exported by the (rnrs arithmetic flonums (6)) library, see fleven?.

— Method: even? (A <real>)

Apply to the argument the function even? exported by the (rnrs base (6)) library, see even?.
