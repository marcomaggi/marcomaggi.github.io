11.1.3 Numeric IP address objects

The following bindings are exported by the library (nausicaa uri ip).

— Abstract Class: <ip-numeric-address>

Base class for all the address types representing numeric IP addresses. It is derived from <ip-address>.

Constructors and makers
— Super constructor on <ip-numeric-address>: constructor
— Super constructor on <ip-numeric-address>: constructor option ...

Supertype constructor for types derived from <ip-numeric-address>. When called with some arguments it supports the following keyword values:

#:bignum-rep obj
The argument obj must be #f or an exact integer representation of the address.

all the other values are handed to the constructor of <ip-address>.

Fields
— Immutable virtual field of <ip-numeric-address>: <exact-integer> bignum

Exact integer object representation of the address. The value of this field is set by the constructor or computed by the multimethod ip-address->bignum and it is memoized in the object.
