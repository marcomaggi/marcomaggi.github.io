5.15.11 Flonum number objects

— Builtin Label: <flonum>

Conventional type for objects satisfying flonum?. It is derived from <real>.

— Builtin Label: <integer-flonum>

Conventional type for objects satisfying flonum? and integer?. It is derived from <flonum>.

— Builtin Label: <rational-flonum>

Conventional type for objects satisfying flonum? and rational?. It is derived from <flonum>.

Safe accessors
— Immutable virtual field of <flonum>: <boolean> integer?
— Immutable virtual field of <flonum>: <boolean> $integer?
