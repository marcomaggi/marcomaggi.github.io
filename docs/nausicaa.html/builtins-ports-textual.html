5.17.4 Textual port labels

— Builtin Label: <textual-port>

Conventional type for objects satisfying textual-port?.

— Builtin Label: <textual-input-port>

Conventional type for objects satisfying textual-port? and input-port?. It is derived from <input-port>.

— Builtin Label: <textual-output-port>

Conventional type for objects satisfying textual-port? and output-port?. It is derived from <output-port>.

— Builtin Label: <textual-input/output-port>

Conventional type for objects satisfying textual-port? and input/output-port?. It is derived from <input/output-port>.
