6 The (nausicaa mehve) language

The (nausicaa mehve) library implements a language in the same way the (nausicaa) library does; in facts, it reexports all the bindings of (nausicaa) and it is a replacement for it.

In addition, (nausicaa mehve) redefines some bindings to functions as bindings to generic functions and so they can be customised with specialised methods, see Multimethods dispatching.

  • mehve intro: Generic functions and their customisation.
  • mehve io: Input/output generic functions.
