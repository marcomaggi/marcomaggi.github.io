68.3.4 Utilities for base32 encoders and decoders

— Function: armored-byte-of-base32/upper-case? byte-integer
— Function: armored-byte-of-base32/lower-case? byte-integer
— Function: armored-byte-of-base32/mixed-case? byte-integer

Return true if byte-integer is an exact integer representation, according to char->integer, of an ASCII character in the base32 encoding's alphabet.

— Function: armored-byte-of-base32/hex/upper-case? byte-integer
— Function: armored-byte-of-base32/hex/lower-case? byte-integer
— Function: armored-byte-of-base32/hex/mixed-case? byte-integer

Return true if byte-integer is an exact integer representation, according to char->integer, of an ASCII character in the base32/hex encoding's alphabet.
