5.19 R6RS record labels

— Builtin Label: <record>

Conventional type for objects satisfying record?.

Safe accessors and mutators
— Immutable virtual field of <record>: <record-type-descriptor> rtd

Return the record type descriptor. Retrieving this field is like applying record-rtd to the instance.
