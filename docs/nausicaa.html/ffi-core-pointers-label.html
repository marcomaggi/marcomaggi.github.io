31.1.5.2 Label interface

The following bindings are exported by the (nausicaa ffi pointers) library and reexported by (nausicaa ffi) and (nausicaa ffi memory).

— Label: <pointer>

Label interface to pointer values.

— Immutable Virtual Field of <pointer>: null?

#t if the pointer represents a null pointer according to the underlying Scheme implementation, #f otherwise.

— Immutable Virtual Field of <pointer>: integer

Return an exact integer representing the pointer value.

— Method on <pointer>: diff pointer

Return the difference between the instance and pointer.

— Method on <pointer>: add offset

Add an offset to a pointer and return the resulting pointer.

— Method on <pointer>: incr! expr
— Method on <pointer>: decr! expr

Increment or decrement the pointer by the result of expr.

— Method on <pointer>: =? pointer ...
— Method on <pointer>: <? pointer ...
— Method on <pointer>: >? pointer ...
— Method on <pointer>: <=? pointer ...
— Method on <pointer>: >=? pointer ...
— Method on <pointer>: <>? pointer ...

All the arguments must be pointer values and are internally converted to integers. Return #t if the integers satisfy the selected predicate (=, <, >, <=, >=), otherwise return #f. pointer<>? returns #t if all the pointers are different.

With the single exception of pointer<>?: all the methods return #t when applied to no arguments.
