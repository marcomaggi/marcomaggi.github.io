73.1.6 Miscellaneous functions

The procedures described here make use of the (nausicaa language generics) library; Multimethod dispatching.

— Function: source-location->string loc

Return a string representation of loc, which must be a record of type <source-location> or #f. If loc is #f, the return value is the string ‘<??>’; else the result is of the form: 

          <input>:line:column

for example ‘main.c:33:7’. <input> is a string produced by applying object->string to the value of the ‘input’ field of loc.

Notice that the use of this function is not mandatory at all.

— Method: object->string (o <source-location>)

Return a string representation of o. It is a wrapper for source-location->string.
