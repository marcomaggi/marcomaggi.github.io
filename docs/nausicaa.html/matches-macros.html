28.4 Syntaxes for pattern matching

— Syntax: match-let ((?pattern ?expression) ...) . ?body
— Syntax: match-let ?variable ((?pattern ?expression) ...) . ?body
— Syntax: match-let* ((?pattern ?expression) ...) . ?body
— Syntax: match-letrec ((?pattern ?expression) ...) . ?body

Match each ?expression against each ?pattern and, if all the matches are successful, evaluate ?body in the resulting environment. If a pattern does not match, a mismatch error is raised. These syntaxes are meant to provide destructuring; they are not meant to test if an S–expression matches a pattern.

When the second form is used: a named match-let is defined, working like the named let. 

          (match-let ((x 1))
            x)
          ⇒ 1
          
          (match-let (((x . y) '(1 . 2))
                      (z       3))
            (list x y z))
          ⇒ (1 2 3)

These syntaxes are matching extensions of the corresponding R6RS let syntaxes. Notice that: 

          (match-let ((_ 1))
            _)
          ⇒ 1
          
          (match-let (((_) '(1)))
            'ok)
          ⇒ ok

normal binding of variables takes precedence over pattern matching.

— Syntax: match-lambda ?clause ...
— Syntax: match-lambda* ?clause ...

Define a lambda which matches its arguments against a set of patterns. 

          (match-lambda ?clause ...) ==
            (lambda (expr) (match expr ?clause ...))
          
          (match-lambda* ?clause ...) ==
            (lambda expr (match expr ?clause ...))
— Syntax: match-define ?name ?clause ...
— Syntax: match-define* ?name ?clause ...

Define, and bind to ?name, a lambda which matches its arguments against a set of patterns. 

          (match-define ?name ?clause ...) ==
            (define ?name (match-lambda ?clause ...))
          
          (match-define* ?name ?clause ...) ==
            (define ?name (match-lambda* ?clause ...))

Examples: 

          (match-define one
            ((x y) (+ x y)))
          
          (one '(1 2))
          ⇒ 3
          
          (match-define* two
            ((x y) (+ x y)))
          
          (two 1 2)
          ⇒ 3
