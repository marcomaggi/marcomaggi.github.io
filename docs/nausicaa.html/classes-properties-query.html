24.9.2 Querying the properties values

The following bindings are exported by the (nausicaa language classes properties) library. The functions with prefix struct- can be applied to all the properties values; the functions with prefix class-, label-, mixin- can be applied only to properties values for classes, labels and mixins respectively.

— Function: make-class . args
— Function: make-label . args
— Function: make-mixin . args

Build and return a new properties value. These function are meant to be for internal use only, so their arguments are undocumented.

— Function: struct? obj
— Function: class? obj
— Function: label? obj
— Function: mixin? obj

Return true if obj is a properties value, a class properties value, a label properties value, a mixin properties value respectively.

— Function: struct-list-of-supers prop
— Function: class-list-of-supers prop
— Function: label-list-of-supers prop
— Function: mixin-list-of-supers prop

Query a properties value. Return an unwrapped syntax object being null or a list of identifiers representing the superclasses or superlabels of the class, label or mixin associated to prop.

The returned value is null when prop is associated to: the <top> class; a class having a record as parent (rather than a class).

— Function: struct-field-specs prop
— Function: class-field-specs prop
— Function: mixin-field-specs prop

Query a properties value. Return an unwrapped syntax object being null or a list of lists of concrete field specifications; the format of the list is: 

          (((mutable name accessor mutator)
            (tag ...))
           ...
           ((immutable name accessor)
            (tag ...))
           ...)

where: mutable and immutable are the auxiliary syntaxes exported by (rnrs records syntactic (6)); name is an identifier representing the name of the field; accessor and mutator are identifiers bound to the accessor and mutator functions for the record; the optional tag values are identifiers representing the field type tags.

— Function: struct-virtual-field-specs prop
— Function: class-virtual-field-specs prop
— Function: label-virtual-field-specs prop
— Function: mixin-virtual-field-specs prop

Query a properties value. Return an unwrapped syntax object being null or a list of lists of virtual field specifications; the format of the list is: 

          (((mutable name accessor mutator)
            (tag ...))
           ...
           ((immutable name accessor)
            (tag ...))
           ...)

where: mutable and immutable are the auxiliary syntaxes exported by (rnrs records syntactic (6)); name is an identifier representing the name of the field; accessor and mutator are identifiers bound to the accessor and mutator functions for the field; the optional tag values are identifiers representing the field type tags.

— Function: struct-method-specs prop
— Function: class-method-specs prop
— Function: label-method-specs prop
— Function: mixin-method-specs prop

Query a properties value. Return an unwrapped syntax object being null or list of method specifications; the format of the list is: 

          ((method-name function-or-macro-identifier)
           ...)

where: method-name is an identifier representing the name of the method; function-or-macro-identifier is an identifier bound to the function or macro implementing the method.

— Function: struct-mixin-identifiers prop
— Function: class-mixin-identifiers prop
— Function: label-mixin-identifiers prop
— Function: mixin-mixin-identifiers prop

Query a properties value. Return an unwrapped syntax object being null or a list of identifiers representing the requested mixins.

— Function: struct-list-of-field-tags prop
— Function: class-list-of-field-tags prop
— Function: label-list-of-field-tags prop
— Function: mixin-list-of-field-tags prop

Query a properties value. Return an unwrapped syntax object being null or a list of identifiers representing the full list of field tags; this value is used internally to detect circular tagging.
