49.1 Type definitions and constructors

— Class: <stack>

Class name of stack objects.

— Mutable Field of <stack>: first-pair

The first pair of the internal list of elements.

— Immutable Field of <stack>: empty?

True if the stack is empty.

— Immutable Field of <stack>: length

The number of elements on the stack.

— Function: make-<stack> arg ...

Build and return a new <stack> record. When called with no arguments, return an empty stack; when applied to arguments, return a stack holding those arguments in the given order.

— Function: <stack>? obj

Return #t if obj is a record of type <stack>.

— Function: <stack>-first-pair stack
— Function: <stack>-first-pair-set! stack obj

Accessor and mutator for the concrete field of <stack> records.
