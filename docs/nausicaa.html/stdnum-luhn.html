71.2 Luhn validation algorithm

The (nausicaa standard-numbers luhn) library is a Scheme implementation of the Luhn checksum used to validate identification numbers.

  • stdnum luhn intro: Introduction to the algorithm.
  • stdnum luhn namesp: Algorithm namespaces.
  • stdnum luhn numbers: Processing exact integers.
  • stdnum luhn low: Low level functions.
