3.19.6 OOPP bindings: do–like syntaxes

— Syntax: do/tags ((?tagged-variable ?init ?step) ...) (?test ?expression ...) ?command ...

Like do and defines the OOPP bindings for the arguments. When all the bindings have no tags: this syntax is equivalent to do as defined by R6RS; an untagged binding is equivalent to a binding tagged with <top>. The ?step, ?expression and ?command expressions can use OOPP notation when accessing the bindings.

— Syntax: do*/tags ((?tagged-variable ?init ?step) ...) (?test ?expression ...) ?command ...

Like do and defines the OOPP bindings for the arguments; in addition the bindings are created in a let* style. When all the bindings have no tags: this syntax is equivalent to do as defined by R6RS; an untagged binding is equivalent to a binding tagged with <top>. The ?step, ?expression and ?command expressions can use OOPP notation when accessing the binding.
