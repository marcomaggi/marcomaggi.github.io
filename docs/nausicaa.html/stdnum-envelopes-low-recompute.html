71.1.4.5 Recomputing check digits

The following bindings are exported by the (nausicaa standard-numbers envelopes low) library. For all the functions, the meaning of the common arguments is the following:

sequence->check-digit
A function accepting a thunk as argument and returning the check digit as fixnum; the thunk must return the digits from the sequence one at a time and finally #f when no more input digits are available.
value->digit
A function mapping a value extracted from the a sequence to a fixnum digit in the range required by an algorithm.
— Function: left-flap-list-envelope->left-fold-check-digit ell sequence->check-digit value->digit

Recompute and return the left–fold check digit from the left–flap list envelope ell.

— Function: left-flap-string-envelope->left-fold-check-digit str sequence->check-digit value->digit
— Function: right-flap-string-envelope->left-fold-check-digit str sequence->check-digit value->digit
— Function: left-flap-string-envelope->right-fold-check-digit str sequence->check-digit value->digit
— Function: right-flap-string-envelope->right-fold-check-digit str sequence->check-digit value->digit

Recompute and return the check digit from the string envelope str.

— Function: left-flap-vector-envelope->left-fold-check-digit vec sequence->check-digit value->digit
— Function: right-flap-vector-envelope->left-fold-check-digit vec sequence->check-digit value->digit
— Function: left-flap-vector-envelope->right-fold-check-digit vec sequence->check-digit value->digit
— Function: right-flap-vector-envelope->right-fold-check-digit vec sequence->check-digit value->digit

Recompute and return the check digit from the vector envelope vec.

— Function: left-flap-bytevector-u8-envelope->left-fold-check-digit bv sequence->check-digit value->digit
— Function: right-flap-bytevector-u8-envelope->left-fold-check-digit bv sequence->check-digit value->digit
— Function: left-flap-bytevector-u8-envelope->right-fold-check-digit bv sequence->check-digit value->digit
— Function: right-flap-bytevector-u8-envelope->right-fold-check-digit bv sequence->check-digit value->digit

Recompute and return the check digit from the unsigned bytes bytevector envelope bv.
