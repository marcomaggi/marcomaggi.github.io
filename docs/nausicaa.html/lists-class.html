27.17 Class interface

The (xlists) library exports the single binding <xlist>, which provides an interface to the (lists) functions through the facilities of the (classes) library.

— Foreign Class: <xlist>

Foreign class interface to the list functions. It is derived from <list>.

All the following methods are implemented as syntaxes; in all the following descriptions: <self> is an identifier bound to the subject list.

— Method on <xlist>: list-copy

Implement the expansion: 

          (<self>.list-copy)
          ==> (list-copy <self>)
— Method on <xlist>: tree-copy

Implement the expansion: 

          (<self>.tree-copy)
          ==> (tree-copy <self>)
— Method on <xlist>: circular-list!
