12.1.2.2 IPv6 address prefix objects class

The following bindings are exported by the (nausicaa net addresses ipv6) library.

— Class: <ipv6-address-prefix>

Represents an IPv6 address prefix; it is derived from <ipv6-address>.

Constructors and makers
— Common constructor on <ipv6-address-prefix>: constructor numbers prefix-length

Build and return a new object of type <ipv6-address-prefix>. numbers must be a list of 8 fixnums in the range [0, #xFFFF] representing the address components. prefix-length must be a non–negative fixnum specifying how many of the leftmost contiguous bits of the address comprise the prefix.

— Maker on <ipv6-address-prefix>: <ipv6-address-prefix> (?numbers ?prefix-length)

Build and return a new object of type <ipv6-address-prefix> using the common constructor. 

          (<ipv6-address-prefix> O
            (<> ('(1 2 3 4 5 6 7 8) 10)))
          
          (list (O third) (O second) (O first) (O zeroth))
          ⇒ (5 6 7 8)
          
          (O prefix-length)
          ⇒ 10
Fields
— Immutable field of <ipv6-address-prefix>: <ipv6-address-prefix-length> prefix-length

A number specifying how many of the leftmost contiguous bits of the address comprise the prefix.

— Immutable virtual field of <ipv6-address-prefix>: <string> string

The string representation of the address prefix.
