77.4.2.1 Class interface to library names

The bindings documented in this section are exported by the (nausicaa parser-tools r6rs libraries names) library.

— Class: <library-name>

Class type representing library names as specified by R6RS, scheme library form names.

— Immutable field of <library-name>: <sexp> sexp

The full symbolic expression representing the library name.

— Immutable field of <library-name>: <list> identifiers

List of identifiers in the library reference specification.

— Immutable field of <library-name>: <list> version

The version specification as a, possibly null, list of exact, non–negative, integers.

— Syntax on <library-name>: make <library-name> sexp
— Syntax on <library-name>: make* <library-name> sexp

Parse sexp as a symbolic expression representing a library name as specified by R6RS; build and return a new <library-name> object representing it. If sexp is not valid: raise an assertion violation.

— Syntax on <library-name>: is-a? obj <library-name>

Return #t if obj is an object of type <library-name>.

— Method on <library-name>: <boolean> identifiers=? (name <library-name>)

Return #t if the arguments are library names having equal list of identifiers; else return #f.

— Method on <library-name>: <boolean> = (name <library-name>)
— Method on <library-name>: <boolean> < (name <library-name>)
— Method on <library-name>: <boolean> > (name <library-name>)
— Method on <library-name>: <boolean> <= (name <library-name>)
— Method on <library-name>: <boolean> >= (name <library-name>)

Return #t if the instance and the argument, in this order, have equal list of identifiers and version specifications satisfying the selected predicate; else return #f.

Version numbers are compared using the predicates library-version=?, library-version<? and library-version<=? (see Low level functions handling library names).

Examples: 

     #!r6rs
     (import (nausicaa)
       (nausicaa r6rs parser-tools libraries names))
     
     (let ((a (make <library-name> '(x y (1 2))))
           (b (make <library-name> '(x y (1 3))))
           (c (make <library-name> '(x y (1 4))))
           (d (make <library-name> '(x y (2 1)))))
     
       (list-sort (lambda ((a <library-name>) (b <library-name>))
                    (a.< b))
         (list a d c b)) ⇒ (list a b c d)
       )
