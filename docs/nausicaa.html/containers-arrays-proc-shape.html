10.4.3.2 Specifying the shape of arrays

Values of type shape represent the shape of an array, that is: the number of dimensions, the start index of each dimension, the past index of each dimension. shape values are disjoint from the other values.

Constructors
— Function: array-shape start-list past-list

Build and return a new shape describing an array or array view through the list of start indexes in start-list and the list of past indexes in past-list.

The arguments must have the same length and be composed by non–negative, exact integers. Indexes from the arguments are taken in couples: The first start with the first past, the second start with the second past, and so on. Each couple describes the range of valid indexes for each dimension of an array.

Predicates and assertions
— Function: array-shape? obj

Return #t if obj is a shape value.

— Function: assert-array-shape obj func-name
— Function: assert-array-shape/or-false obj func-name

Raise an assertion violation exception if obj is not a shape value, or #f. func-name is used who argument for the exception. Return #t if no exception is raised.

Comparison and set operations
— Function: array-shape=? shape0 shape ...

Return #t if the arguments have equal number of dimensions and hold the same indexes.

— Function: array-supershape? shape0 shape ...
— Function: array-supershape?/strict shape0 shape ...

Return #t if the arguments, in the given order, are supershapes of each other or strict supershapes of each other. The shape a is supershape of b if:

  1. They have the same number of dimensions.
  2. For each dimension, the range of elements in b is a subset of the range of elements in a.

The following two forms are equivalent: 

          (array-supershape? a b c d)
          (and (array-supershape? a b)
               (array-supershape? b c)
               (array-supershape? c d))
— Function: array-subshape? shape0 shape ...
— Function: array-subshape?/strict shape0 shape ...

Return #t if the arguments, in the given order, are subshapes of each other or strict subshapes of each other. These functions are like the supershape functions with the arguments reversed.

— Function: array-shape-contains? shape position

Return #t if position selects an array element in the range of elements described by shape.

Inspection
— Function: array-shape-number-of-dimensions shape

Return the number of dimensions of the array with which the shape value will be used.

— Function: array-shape-number-of-elements shape

Return the number of elements in the range described by shape.

— Function: array-shape-index-start shape dimension
— Function: array-shape-index-past shape dimension
— Function: array-shape-index-last shape dimension

Return the start, past or last value for position indexes in shape for the selected dimension (zero based). Examples: 

          (define s (array-shape '(-1  2 5)
                                 '( 5 18 7)))
          
          (array-shape-index-start s 0) ⇒ -1
          (array-shape-index-past  s 0) ⇒  5
          (array-shape-index-last  s 0) ⇒  4
          
          (array-shape-index-start s 1) ⇒  2
          (array-shape-index-past  s 1) ⇒ 18
          (array-shape-index-last  s 1) ⇒ 17
          
          (array-shape-index-start s 2) ⇒  5
          (array-shape-index-past  s 2) ⇒  7
          (array-shape-index-last  s 3) ⇒  6
Conversion
— Function: array-shape->string shape

Return a string representation of shape.

Port output
— Function: array-shape-display shape
— Function: array-shape-display shape port

Write to port a human readable string representation of shape. port defaults to the current output port.

— Function: array-shape-write shape
— Function: array-shape-write shape port

Write to port a string representation of shape such that it can be read back with read. port defaults to the current output port.
