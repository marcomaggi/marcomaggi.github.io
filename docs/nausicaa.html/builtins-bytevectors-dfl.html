5.13.7 Bytevector labels: double–precision floating–point numbers

— Builtin Label: <bytevector-doublel>
— Builtin Label: <bytevector-doubleb>
— Builtin Label: <bytevector-doublen>

Conventional type for objects satisfying bytevector? whose contents must be accesses as IEEE double–precision floating–point numbers. The suffix ‘l’ stands for little–endian; the suffix ‘b’ stands for big–endian; the suffix ‘n’ stands for native endianness.

OOPP syntaxes
— Getter on <bytevector-doublel>: instance (?index)
— Getter on <bytevector-doubleb>: instance (?index)
— Getter on <bytevector-doublen>: instance (?index)

Floating–point number getter. ?index is the zero–based index of the selected word.

— Setter on <bytevector-doublel>: set!/tags (instance (?index)) ?item
— Setter on <bytevector-doubleb>: set!/tags (instance (?index)) ?item
— Setter on <bytevector-doublen>: set!/tags (instance (?index)) ?item

Floating–point number setter. ?index is the zero–based index of the selected word.
