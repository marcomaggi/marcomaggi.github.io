30.1.9.9 Full inspection of data types
— Autoconf Macro: NAUSICAA_INSPECT_TYPE (suffix, typedef, type-guess, default, headers)

Perform a full inspection for a C language type definition. This macro is a wrapper for other more basic macros, and it expands to: 

          NAUSICAA_SIZEOF_TEST(suffix,typedef,default,headers)
          NAUSICAA_ALIGNOF_TEST(suffix,typedef,default,headers)
          NAUSICAA_STRIDEOF_TEST(suffix,typedef,default,headers)
          NAUSICAA_BASE_TYPE_TEST(suffix,typedef,type-guess)
— Autoconf Macro: NAUSICAA_INSPECT_STRUCT_TYPE (suffix, typedef, default, headers)

Perform a full inspection for a C language structure type definition. This macro is a wrapper for other more basic macros, and it expands to: 

          NAUSICAA_SIZEOF_TEST(suffix,typedef,default,headers)
          NAUSICAA_ALIGNOF_TEST(suffix,typedef,default,headers)
          NAUSICAA_STRIDEOF_TEST(suffix,typedef,default,headers)
— Autoconf Macro: NAUSICAA_INSPECT_FIELD_TYPE (suffix, struct-typedef, field-name, type-guess, default, headers)

Perform a full inspection for a C language field structure type definition. This macro is a wrapper for other more basic macros, and it expands to: 

          NAUSICAA_OFFSETOF_FIELD_TEST(suffix,struct-typedef,
                                       field-name,headers)
          NAUSICAA_SIZEOF_FIELD_TEST(suffix,struct-typedef,
                                     field-name,default,headers)
          NAUSICAA_BASE_TYPE_TEST(suffix,
                                  struct-typedef.field-name,
                                  type-guess)
— Autoconf Macro: NAUSICAA_INSPECT_FIELD_TYPE_POINTER (suffix, struct-typedef, field-name, headers)

Perform an inspection for a C language field structure type definition, but only to acquire the offset. This macro is required when we need to access a structure field through a pointer to it; an example is the ‘d_name’ field of ‘struct dirent’ from dirent.h. Accessing data structures for a discussion of this case.

Only the offset of the field (output variable ‘OFFSETOF_<SUFFIX>’) is defined.
