66.1 Introduction to JSON

JSON (JavaScript Object Notation) is a lightweight data–interchange format; full informations on the notation are available at:1

http://www.json.org/

the specification is also available in RFC 4627.

The (nausicaa json) library parses JSON text using the (nausicaa silex lexer) and (nausicaa lalr lr-driver) libraries; given the terminals, optionally surrounded by white spaces: 

     BEGIN_ARRAY             [
     END_ARRAY               ]
     BEGIN_OBJECT            {
     END_OBJECT              }
     NAME_SEPARATOR          :
     VALUE_SEPARATOR         ,
     
     FALSE                   "false"
     NULL                    "null"
     TRUE                    "true"

with the addition of ‘NUMBER’ and ‘STRING’, the grammar is: 

     ((json-text     (object)
                     (array))
     
      (object        (BEGIN_OBJECT END_OBJECT)
                     (BEGIN_OBJECT pair END_OBJECT)
                     (BEGIN_OBJECT pair pair-rest END_OBJECT))
     
      (pair          (STRING NAME_SEPARATOR value))
      (pair-rest     (VALUE_SEPARATOR pair)
                     (VALUE_SEPARATOR pair pair-rest))
     
      (array         (BEGIN_ARRAY END_ARRAY)
                     (BEGIN_ARRAY value END_ARRAY)
                     (BEGIN_ARRAY value value-rest END_ARRAY))
     
      (value         (FALSE)
                     (NULL)
                     (TRUE)
                     (NUMBER)
                     (STRING)
                     (object)
                     (array))
      (value-rest    (VALUE_SEPARATOR value)
                     (VALUE_SEPARATOR value value-rest)))

(nausicaa json) turns JSON text into a Scheme S–expression; JSON pairs become Scheme pairs: 

     "Height": 600           ==> ("Height" . 600)
     "Hello" : "Ciao"        ==> ("Hello"  . "Ciao")

objects become lists: 

     { "Height": 600" }
     ==> (("Height" . 600))
     
     { "Height": 600, "Hello" : "Ciao" }
     ==> (("Height" . 600) ("Hello"  . "Ciao"))

arrays become vectors: 

     [ "Ciao", "Hello", "Salut" ]
     ==> #("Ciao" "Hello" "Salut")

the literal ‘true’ becomes #t, the literal ‘false’ becomes #f, the literal ‘null’ becomes ().
