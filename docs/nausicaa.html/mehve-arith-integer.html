6.4.6 Integer operations generic functions

The generic functions div, div0, mod, mod0, div-and-mod and div0-and-mod0 exported by (nausicaa mehve arithmetics), and reexported by (nausicaa mehve), generalise the corresponding functions from (rnrs base (6)) and are meant to perform integer operations as explained in see Integer division.

— Generic Function: div a b
— Generic Function: div0 a b

Perform integer division.

— Generic Function: mod a b
— Generic Function: mod0 a b

Compute the modulus of integer division.

— Generic Function: div-and-mod a b
— Generic Function: div0-and-mod0 a b

Compute the integer division and its modulus.

The following are the specialised methods.

— Method: div (A <fixnum>) (B <fixnum>)
— Method: div0 (A <fixnum>) (B <fixnum>)

Apply to the arguments the function fxdiv or fxdiv0 exported by the (rnrs arithmetic fixnums (6)) library, see fxdiv.

— Method: div (A <integer>) (B <integer>)
— Method: div (A <real>) (B <real>)
— Method: div0 (A <integer>) (B <integer>)
— Method: div0 (A <real>) (B <real>)

Apply to the arguments the function div or div0 exported by the (rnrs base (6)) library, see div.

— Method: mod (A <fixnum>) (B <fixnum>)
— Method: mod0 (A <fixnum>) (B <fixnum>)

Apply to the arguments the function fxmod or fxmod0 exported by the (rnrs arithmetic fixnums (6)) library, see fxmod.

— Method: mod (A <integer>) (B <integer>)
— Method: mod (A <real>) (B <real>)
— Method: mod0 (A <integer>) (B <integer>)
— Method: mod0 (A <real>) (B <real>)

Apply to the arguments the function mod or mod0 exported by the (rnrs base (6)) library, see mod.

— Method: div-and-mod (A <fixnum>) (B <fixnum>)
— Method: div0-and-mod0 (A <fixnum>) (B <fixnum>)

Apply to the arguments the function fxdiv-and-mod or fxdiv0-and-mod0 exported by the (rnrs arithmetic fixnums (6)) library, see fxdiv-and-mod.

— Method: div-and-mod (A <integer>) (B <integer>)
— Method: div-and-mod (A <real>) (B <real>)
— Method: div0-and-mod0 (A <integer>) (B <integer>)
— Method: div0-and-mod0 (A <real>) (B <real>)

Apply to the arguments the function div-and-mod or div0-and-mod0 exported by the (rnrs base (6)) library, see div-and-mod.
