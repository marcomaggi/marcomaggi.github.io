6.3.3 Sign related generic functions

The generic functions zero?, positive?, negative?, non-positive? and non-negative? exported by (nausicaa mehve numeric-predicates), and reexported by (nausicaa mehve), generalise the corresponding functions exported by (rnrs base (6)) and (nausicaa).

— Generic Function: zero? A

Return #t if the argument is zero.

— Generic Function: positive? A

Return #t if the argument is positive.

— Generic Function: negative? A

Return #t if the argument is negative.

— Generic Function: non-positive? A

Return #t if the argument is non–positive.

— Generic Function: non-negative? A

Return #t if the argument is non–negative.

The following are the specialised methods.

— Method: zero? (A <fixnum>)

Apply to the argument the function fxzero? exported by the (rnrs arithmetic fixnums (6)) library, see fxzero?.

— Method: zero? (A <flonum>)

Apply to the argument the function flzero? exported by the (rnrs arithmetic flonums (6)), library see flzero?.

— Method: zero? (A <real>)

Apply to the argument the function zero? exported by the (rnrs base (6)) library, see zero?.

— Method: positive? (A <fixnum>)

Apply to the argument the function fxpositive? exported by the (rnrs arithmetic fixnums (6)) library, see fxpositive?.

— Method: positive? (A <flonum>)

Apply to the argument the function flpositive? exported by the (rnrs arithmetic flonums (6)) library, see flpositive?.

— Method: positive? (A <real>)

Apply to the argument the function positive? exported by the (rnrs base (6)) library, see positive?.

— Method: negative? (A <fixnum>)

Apply to the argument the function fxnegative? exported by the (rnrs arithmetic fixnums (6)) library, see fxnegative?.

— Method: negative? (A <flonum>)

Apply to the argument the function flnegative? exported by the (rnrs arithmetic flonums (6)) library, see flnegative?.

— Method: negative? (A <real>)

Apply to the argument the function negative? exported by the (rnrs base (6)) library, see negative?.

— Method: non-positive? (A <real>)

Apply to the argument the function non-positive? exported by the (nausicaa) library, see non-positive?.

— Method: non-negative? (A <real>)

Apply to the argument the function non-negative? exported by the (nausicaa) library, see non-negative?.
