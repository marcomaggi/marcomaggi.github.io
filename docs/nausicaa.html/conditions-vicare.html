6.3.4 Condition redefinition for Vicare types

The following bindings are exported by (nausicaa language conditions) and redefine the bindings for condition type identifiers exported by (vicare).

— Condition Type Label: &errno
— Condition Type Label: &h_errno
— Condition Type Label: &i/o-eagain
— Condition Type Label: &out-of-memory-error

Label identifiers shadowing Vicare's condition type identifiers.
