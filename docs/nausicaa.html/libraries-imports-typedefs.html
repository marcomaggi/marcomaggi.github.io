68.4.1 Type definitions

Import sets
— Class: <import-set>

Type representing import sets as specified by R6RS.

— Immutable Field of <import-set>: subset

The core of this import set; it can be another <import-set> object or a <library-reference> object.

— Immutable Field of <import-set>: bindings-transformer

A function to apply to a list renamings to obtain the imported bindings.

— Immutable Field of <import-set>: original

The original symbolic expression of this import set.

— Immutable Field of <import-set>: library-reference

The <library-reference> object nested in this import set.

— Function: make-<import-set> sexp

Parse the symbolic expression sexp as an import sets as specified by R6RS; return a new <import-set> object. Raise an assertion violation if sexp is invalid.

— Function: <import-set>? obj

Return #t if obj is a record of type <import-set>.

— Function: <import-set>-subset iset
— Function: <import-set>-bindings-transformer iset
— Function: <import-set>-original iset
— Function: <import-set>-library-reference iset

Accessors for the fields of <import-set> records.

— Method on <import-set>: apply renamings

Apply the import set specification to the list of renamings, returning the processed list of renamings (which can be null). It applies recursively the specifications of the nested import sets.

Import specifications
— Class: <import-spec>

Type representing import specifications as specified by R6RS.

— Immutable Field of <import-spec>: import-set

The core of this import specification; it can be an <import-set> object or a <library-reference> object.

— Immutable Field of <import-spec>: import-levels

A list of exact integers representing the import levels for which this import set was requested. It contains at least one element.

— Immutable Field of <import-spec>: original

The original symbolic expression of this import set.

— Immutable Field of <import-spec>: library-reference

The <library-reference> object nested in this import set.

— Function: make-<import-spec> sexp

Parse the symbolic expression sexp as an import specification as specified by R6RS; return a new <import-spec> object. Raise an assertion violation if sexp is invalid.

— Function: <import-spec>? obj

Return #t if obj is a record of type <import-spec>.

— Function: <import-spec>-import-set iset
— Function: <import-spec>-import-levels iset
— Function: <import-spec>-original iset
— Function: <import-spec>-library-reference iset

Accessors for the fields of <import-spec> records.

— Method on <import-spec>: apply renamings

Apply the import set specification to the list of renamings, returning the processed list of renamings (which can be null). It applies recursively the specifications of the nested import sets.
