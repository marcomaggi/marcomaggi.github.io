77.1.2.3 Fixed–semantic–value token–maker parameters

The following bindings are exported by the library (nausicaa parser-tools r6rs lexer).

— Parameter: open-paren-token-maker

Set to a function returning a <lexical-token> record representing an open parenthesis, ‘(’ . The default is make-open-paren-token.

— Parameter: close-paren-token-maker

Set to a function returning a <lexical-token> record representing a close parenthesis, ‘)’. The default is make-close-paren-token.

— Parameter: open-bracket-token-maker

Set to a function returning a <lexical-token> record representing an open bracket, ‘[’. The default is make-open-bracket-token.

— Parameter: close-bracket-token-maker

Set to a function returning a <lexical-token> record representing a close bracket, ‘]’. The default is make-close-bracket-token.

— Parameter: tick-token-maker

Set to a function returning a <lexical-token> record representing a tick, ‘'’. The default is make-tick-token.

— Parameter: back-tick-token-maker

Set to a function returning a <lexical-token> record representing a back tick, ‘`’. The default is make-back-tick-token.

— Parameter: comma-at-token-maker

Set to a function returning a <lexical-token> record representing a comma–at, ‘,@’. The default is make-comma-at-token.

— Parameter: comma-token-maker

Set to a function returning a <lexical-token> record representing a comma, ‘,’. The default is make-comma-token.

— Parameter: dot-token-maker

Set to a function returning a <lexical-token> record representing a dot, ‘.’. The default is make-dot-token.

— Parameter: double-quote-token-maker

Set to a function returning a <lexical-token> record representing a double quote, ‘"’; double quotes open strings, which must be read with a separate lexer table (see r6rs lexer strings). The default is make-double-quote-token.

— Parameter: sharp-paren-token-maker

Set to a function returning a <lexical-token> record representing a sharp–open parenthesis, ‘#(’. The default is make-sharp-paren-token.

— Parameter: sharp-vu8-paren-token-maker

Set to a function returning a <lexical-token> record representing a sharp–vu8–open parenthesis, ‘#vu8(’. The default is make-sharp-vu8-paren-token.

— Parameter: sharp-tick-token-maker

Set to a function returning a <lexical-token> record representing a sharp–tick, ‘#'’. The default is make-sharp-tick-token.

— Parameter: sharp-back-tick-token-maker

Set to a function returning a <lexical-token> record representing a sharp–back tick, ‘#`’. The default is make-sharp-back-tick-token.

— Parameter: sharp-comma-at-token-maker

Set to a function returning a <lexical-token> record representing a sharp–commma–at, ‘#,@’. The default is make-sharp-comma-at-token.

— Parameter: sharp-comma-token-maker

Set to a function returning a <lexical-token> record representing a sharp–comma, ‘#,’. The default is make-sharp-comma-token.

— Parameter: sharp-semicolon-token-maker

Set to a function returning a <lexical-token> record representing a sharp–semicolon, ‘#;’. The default is make-sharp-semicolon-token.

— Parameter: open-nested-comment-token-maker

Set to a function returning a <lexical-token> record representing the opening of a nested comment, ‘#|’; nested comments must be read with a separate lexer table (see r6rs lexer nested comments). The default is make-open-nested-comment-token.

— Parameter: sharp-bang-r6rs-token-maker

Set to a function returning a <lexical-token> record representing a ‘#!r6rs’ comment. The default is make-sharp-bang-r6rs-token.
