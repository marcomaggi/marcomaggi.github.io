73.1.4 Comparison functions

— Function: source-location=? loc1 loc2

Compare loc1 and loc2 which must be #f or records of type <source-location>. Return true if loc1 and loc2 are true and the line, column and offset counters have the same values.

The following comparison functions establish an ordering in the space of the <source-location> records and #f by comparing the line and column counters. By convention:

  • A <source-location> record is more advanced than #f in the input stream.
  • When the arguments of a comparision function application are both #f, the result is #f.
— Function: source-location-point=? loc1 loc2

Return #t if the line and column counters of loc1 and loc2 have the same values.

— Function: source-location-point>? loc1 loc2

Return #t if loc1 is more advanced in the input stream than loc2.

— Function: source-location-point>=? loc1 loc2

Return #t if loc1 is more advanced or at the same point, in the input stream than loc2.

— Function: source-location-point<? loc1 loc2

Return #t if loc1 is less advanced in the input stream than loc2.

— Function: source-location-point<=? loc1 loc2

Return #t if loc1 is less advanced or at the same point, in the input stream than loc2.
