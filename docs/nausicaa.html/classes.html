20 Record utilities

The (classes) library is an alternative to the (rnrs records syntactic (6)) library; it is built upon (rnrs records procedural (6)) but adds some features like dot notation to access fields.

The library is designed to work with class type definitions imported for the run phase. The facilities of the library work only on records of type defined with define-class.

(classes) makes use of bindings from the (auxiliary-syntaxes) library and reexports the ones it uses as auxiliary syntaxes in the class definition clauses.

  • classes define: Defining new class types.
  • classes make: Record makers.
  • classes pred: Record type predicates.
  • classes inspect: Class and instance inspection.
  • classes dot: Using dot notation.
  • classes builtin: Conventional type descriptors.
