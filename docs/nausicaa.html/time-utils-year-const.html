50.8.3.1 Constants

The following bindings are exported from the (nausicaa times-and-dates years-and-weeks) library.

— Constant: $number-of-seconds-in-half-day

Number of seconds in a half day. It is used in the expressions involving the Julian Day, which starts at noon (rather than at midnight).

— Constant: $tai-epoch-in-jd

Julian day number for the Unix Epoch.

— Constant: $number-of-days-per-month/non-leap-year
— Constant: $number-of-days-per-month/leap-year

Scheme vectors holding the numbers of days for each month. The element at index zero is unused; the element at index 1 is for January, the element at index 2 for February, etc.

— Constant: $number-of-days-the-first-day-of-each-month/non-leap-year
— Constant: $number-of-days-the-first-day-of-each-month/leap-year

Number of days from the beginning of the year, at the first day of a month.
