66.4.2 Error handlers for JSON parsers

— Function: make-json-error-handler who

Return a closure to be used as error handler for JSON parsers. who must be the value for the &who condition.

When the handler is invoked it raises an exception of types: &json-parser-error, &who, &message, &irritants.

The value in the &irritants object is a list with a single element: the offending string from the input.
