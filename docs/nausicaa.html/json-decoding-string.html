67.2.4 Decoding JSON strings

The following bindings are exported by the (nausicaa json) library.

— Function: json-decode-string input-string

Decode a JSON string returning a Scheme string. input-string must not be enclosed in double quotes. Examples: 

          (json-decode-string "")         ⇒ ""
          (json-decode-string "a")        ⇒ "a"
          (json-decode-string "ciao")     ⇒ "ciao"
          (json-decode-string "\\\"")     ⇒ "\""
          (json-decode-string "\\/")      ⇒ "/"
          (json-decode-string "\\b")      ⇒ "\b"
          (json-decode-string "\\f")      ⇒ "\f"
          (json-decode-string "\\n")      ⇒ "\n"
          (json-decode-string "\\r")      ⇒ "\r"
          (json-decode-string "\\t")      ⇒ "\t"
          (json-decode-string "\\u005C")  ⇒ "\\"
          
          (json-decode-string "inizio\\\"\\/\\b\\f\\n\\r\\tfine")
          ⇒ "inizio\"/\b\f\n\r\tfine"
          
          (json-decode-string "\\u0063\\u0069\\u0061\\u006f")
          ⇒ "ciao"
          
          (json-decode-string "ciao \\\"hello\\\" salut")
          ⇒ "ciao \"hello\" salut"
