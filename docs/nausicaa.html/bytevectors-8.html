Up: bytevectors

40.1 Bytevectors of signed and unsigned bytes

The (nausicaa bytevectors s8), (nausicaa bytevectors u8), (nausicaa bytevectors s8low) and (nausicaa bytevectors u8low) libraries extend the bytevector operations available in the base library; Bytevectors. While (nausicaa bytevectors s8) and (nausicaa bytevectors u8) are high level interfaces, (nausicaa bytevectors s8low) and (nausicaa bytevectors u8low) are low level interfaces with the same functionalities; there is special support for bytevectors representing strings in ASCII encoding.
