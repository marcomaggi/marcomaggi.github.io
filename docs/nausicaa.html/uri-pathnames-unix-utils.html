11.3.3.4 Unix pathname utilities

The following bindings are exported by the library (nausicaa uri pathnames unix).

— Multimethod: pathname (ptn <string>)
— Multimethod: pathname (ptn <bytevector>)

Build and return a new instance of <absolute-unix-pathname> or <relative-unix-pathname> from ptn, which must be a string or bytevector representing an absolute or relative Unix pathname according to the predicates pathname?, absolute? and relative? defined by (vicare parser-tools unix-pathnames).

— Multimethod: pathname=? (ptn1 <absolute-unix-pathname>) (ptn2 <absolute-unix-pathname>)
— Multimethod: pathname=? (ptn1 <relative-unix-pathname>) (ptn2 <relative-unix-pathname>)

Implementations for the multimethod pathname=? defined by (nausicaa uri pathnames abstract).
