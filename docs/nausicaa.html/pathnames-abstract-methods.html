41.2.2 Extending class methods

The following bindings are exported by the (nausicaa pathnames methods) library; it is meant to be used only when extending the class interface with new types.

— Generic Function: string self

The implementation of the virtual field string of <pathname> from (nausicaa pathnames abstract). It is meant to return a string representation of the pathname in the bytevector field.

— Generic Function: absolute self absolute-base

The implementation of the absolute method of <pathname> from (nausicaa pathnames abstract). It is meant to return a new pathname record representing the absolute version of self normalised with respect to absolute-base.

If self is already absolute: the return value can be self.

— Generic Function: prepend suffix prefix

Return a record holding a relative pathname representing the pathname of prefix with the pathname suffix appended; both prefix and suffix must be relative pathname records.

— Generic Function: append prefix suffix

Return a record holding a pathname (absolute or relative depending on the arguments) representing the pathname of prefix with the pathname suffix appended; prefix can be absolute or relative, suffix must be relative.

— Generic Function: tail absolute-self

Return a new relative pathname object representing the last segment of absolute-self. If the instance represents the root directory: the returned value has a single segment representing the single–dot entry.

— Generic Function: dirname absolute-self

Return a new absolute pathname object representing all the segments in absolute-self but the last. If the instance represents the root directory: the returned value represents the root directory, too.

— Generic Function: rootname absolute-self

Return a new absolute pathname object representing the same pathname of absolute-self, but having the last segment trimmed of the extension.

— Generic Function: extension absolute-self

Return a new bytevector representing the extension of the last segment in absolute-self. If the last segment has no extension: the return value is false.

— Generic Function: name absolute-self

Return a new bytevector representing the name of the last segment in absolute-self. If the instance represents the root directory: the returned value represents a single dot in ASCII coding.

— Generic Function: replace-extension self source

Replace the extension of the self pathname with the one from source, which can be a bytevector, string or pathname instance.
