61.1.5 Updating the location

— Function: source-location-update location char/token-length

If location is #f, the return value is #f. If location is a <source-location> record, build and return a new <source-location> record representing the position of the next char to be parsed in the input stream, starting from location and updated with char/token-length.

  • If char/token-length is an exact integer, it is interpreted as token length. The column and offset counters from location are incremented by that amount.
  • If char/token-length is the ‘#\return’ character the action depends upon the value of the ‘source-location-honor-return’ parameter: if it is true, the column counter is set to 1 and the offset counter is incremented by 1; if it is #f, the column and offset counters are incremented by 1.
  • If char/token-length is the ‘#\newline’ character: The line counter is incremented by 1, the column counter is set to 1 and the offset counter is incremented by 1.
  • If char/token-length is the ‘#\tab’ character: The offset counter is incremented by 1 and the column counter is set to the result of: 
                   ((source-location-tab-function)
                   (<source-location>-column location))

    that is, the current procedure in the ‘source-location-tab-function’ parameter is applied to the column counter to produce the next column counter's value.

— Parameter: source-location-tab-function

Hold a function used to produce the next column counter's value given the current column counter. By default it is set to source-location-tab-function/8chars.

— Function: source-location-tab-function/8chars column

Return the smallest multiple of 8 which is greater than column.

— Function: source-location-tab-function/tab-table column

Produce the next column counter's value drawing it from the list of horizontal tabulation offsets in the ‘source-location-tab-table’ parameter. The returned value is the smallest offset which is greater than column. If the list is null or all the offsets are smaller than column, source-location-tab-function/8chars is used to compute the return value.

— Parameter: source-location-tab-table

Hold a list of horizontal tabulation offsets, sorted in increasing order. The default value is null.

— Parameter: source-location-honor-return

Hold a boolean value which causes source-location-update to honor a ‘#\return’ character or not.
