6.3 Numeric predicates generic functions

The auxiliary library (nausicaa mehve numeric-predicates) defines the generic functions behind the numeric predicates; we need to import it only if we desire to add specialised methods to the numeric predicates.

  • mehve num pred intro: Introduction to numeric predicates generic functions.
  • mehve num pred comp: Numeric comparison generic functions.
  • mehve num pred sign: Sign related generic functions.
  • mehve num pred odd: Even and odd predicates generic functions.
  • mehve num pred inf: Infinity and not--a--number predicates generic functions.
