5.6 Pointer label

— Builtin Label: <pointer>

Conventional type for objects satisfying pointer? from (vicare).

— Common constructor on <pointer>: <pointer> constructor address

Build and return a new pointer object referencing address, which must be a non–negative exact integer.

Safe accessors
— Immutable virtual field of <pointer>: <boolean> null?

Accessing this field is like applying to the instance the function pointer-null? from (vicare).

— Immutable virtual field of <pointer>: <integer> integer

Accessing this field is like applying to the instance the function pointer->integer from (vicare).

Methods: comparison functions
— Method on <pointer>: = pointer2

Calling this method is like applying to the instance, and the argument, the function pointer=? from (vicare).

— Method on <pointer>: <> pointer2

Calling this method is like applying to the instance, and the argument, the function pointer<>? from (vicare).

— Method on <pointer>: < pointer2

Calling this method is like applying to the instance, and the argument, the function pointer<? from (vicare).

— Method on <pointer>: > pointer2

Calling this method is like applying to the instance, and the argument, the function pointer>? from (vicare).

— Method on <pointer>: <= pointer2

Calling this method is like applying to the instance, and the argument, the function pointer<=? from (vicare).

— Method on <pointer>: >= pointer2

Calling this method is like applying to the instance, and the argument, the function pointer>=? from (vicare).

Methods: pointer arithmetic
— Method on <pointer>: add offset

Calling this method is like applying to the instance, and the argument, the function pointer-add from (vicare).

— Method on <pointer>: diff pointer2

Calling this method is like applying to the instance, and the argument, the function pointer-diff from (vicare).

Other methods
— Method on <pointer>: <pointer> clone

Calling this method is like applying to the instance the function pointer-clone from (vicare).

— Method on <pointer>: set-null!

Calling this method is like applying to the instance the function set-pointer-null! from (vicare).

OOPP syntaxes
— Maker on <pointer>: <pointer> (?address)

Build and return a new pointer object referencing ?address, which must be an expression evaluating to a non–negative exact integer.
