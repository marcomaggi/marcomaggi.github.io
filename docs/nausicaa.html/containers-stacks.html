10.5 Simple stacks

The library (nausicaa containers stacks) implements stacks holding arbitrary scheme objects and designed for efficient last–in/first–out operations; it is built on top of (vicare containers stacks).

  • containers stacks objects: Stack objects.
  • containers stacks inspection: Inspecting stack objects.
  • containers stacks access: Stack accessors and mutators.
  • containers stacks conversion: Converting stacks to other objects.
