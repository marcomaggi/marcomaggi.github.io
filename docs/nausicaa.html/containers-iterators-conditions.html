10.8.2 Condition object types

The following bindings are exported by the library (nausicaa containers iterators).

— Condition type: &stop-iteration

Condition type used to signal that the iteration over a subject is over; it is derived from &serious.

This type is defined by define-condition-type from (nausicaa language conditions), Defining new condition types.

— Immutable field of &stop-iteration: <iterator> iterator

The <iterator> instance which raised the condition object.

— Maker on &stop-iteration: &stop-iteration (?iterator)

Build and return a new instance of &stop-iteration using ?iterator as value for the iterator field.
