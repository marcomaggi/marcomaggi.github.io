72.3 A collection of common strings

The library (nausicaa r6rs fixed-strings) exports bindings to common strings which are meant to be comparable using eq?.

Scheme source strings
— Constant: comma-at

Bound to the string ‘",@"’.

— Constant: sharp-paren

Bound to the string ‘"#("’.

— Constant: sharp-vu8-paren

Bound to the string ‘"#vu8("’.

— Constant: sharp-tick

Bound to the string ‘"#'"’.

— Constant: sharp-back-tick

Bound to the string ‘"#`"’.

— Constant: sharp-comma-at

Bound to the string ‘"#,@"’.

— Constant: sharp-comma-

Bound to the string ‘"#,"’.

— Constant: sharp-semicolon

Bound to the string ‘"#;"’.

— Constant: sharp-bang-r6rs

Bound to the string ‘"#!r6rs"’.

— Constant: sharp-bang

Bound to the string ‘"#!"’.

— Constant: open-nested-comment

Bound to the string ‘"#|"’.

— Constant: true-small

Bound to the string ‘"#t"’.

— Constant: true-capital

Bound to the string ‘"#T"’.

— Constant: false-small

Bound to the string ‘"#f"’.

— Constant: false-capital

Bound to the string ‘"#F"’.
