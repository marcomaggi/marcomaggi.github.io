5.13.1 Bytevector labels: 8-bit words

— Builtin Label: <bytevector>

Conventional type for objects satisfying bytevector?.

— Builtin Label: <nonempty-bytevector>

Conventional type for objects satisfying bytevector? and having non–zero length.

Accessors

Virtual field values whose name is prefixed by $ are built using unsafe operations.

— Immutable virtual field of <bytevector>: <fixnum> length
— Immutable virtual field of <bytevector>: <fixnum> $length

Access the length of the bytevector.

— Immutable virtual field of <bytevector>: <boolean> ascii-encoded?
— Immutable virtual field of <bytevector>: <boolean> $ascii-encoded?

Accessing this field is like applying ascii-encoded-bytevector? to the bytevector, Converting bytevectors.

— Immutable virtual field of <bytevector>: <boolean> latin1-encoded?
— Immutable virtual field of <bytevector>: <boolean> latin1-encoded?

Accessing this field is like applying latin1-encoded-bytevector? to the bytevector, Converting bytevectors.

— Immutable virtual field of <bytevector>: <boolean> percent-encoded?
— Immutable virtual field of <bytevector>: <boolean> $percent-encoded?

— Immutable virtual field of <bytevector>: <percent-encoded-bytevector> percent-encoded
— Immutable virtual field of <bytevector>: <percent-encoded-bytevector> $percent-encoded

— Immutable virtual field of <bytevector>: <bytevector> percent-decoded
