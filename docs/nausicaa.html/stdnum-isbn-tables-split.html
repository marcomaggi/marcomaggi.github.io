72.4.4.4 Splitting ISBN identifiers

The following bindings are exported by the (nausicaa standard-numbers isbn) library.

— Function: isbn10-split/no-separators isbn

Given a 10 digits ISBN string with no separators and satisfying isbn10?/well-formed, split it and return a validated list holding: a string representing the language group number, a string representing the publisher number, a string representing the book number, a string representing the check digit. If isbn is invalid: the return value is #f.

For the purpose of searching the ISBN tables: this function assumes that all the 10 digits ISBNs have an implicit prefix of 978. 

          (isbn10-split/no-separators "316148410X")
          ⇒ ("3" "16" "148410" "X")
— Function: isbn13-split/no-separators isbn

Given a 13 digits ISBN string with no separators and satisfying isbn13?/well-formed, split it and return a validated list holding: a string representing the prefix, a string representing the language group number, a string representing the publisher number, a string representing the book number, a string representing the check digit. If isbn is invalid: the return value is #f. 

          (isbn13-split/no-separators "9783161484100")
          ⇒ ("978" "3" "16" "148410" "0")
— Function: isbn-split/no-separators isbn

Wrapper for isbn10-split/no-separators and isbn13-split/no-separators selecting the appropriate function according to the length of isbn. 

          (isbn-split/no-separators "316148410X")
          ⇒ ("3" "16" "148410" "X")
          
          (isbn-split/no-separators "9783161484100")
          ⇒ ("978" "3" "16" "148410" "0")
— Function: isbn10-split isbn

Given a 10 digits ISBN string (with or without separators) satisfying isbn10?/well-formed, split it and return a validated list holding: a string representing the language group number, a string representing the publisher number, a string representing the book number, a string representing the check digit. If isbn is invalid: the return value is #f.

For the purpose of searching the ISBN tables: this function assumes that all the 10 digits ISBNs have an implicit prefix of 978. 

          (isbn10-split "316148410X")
          ⇒ ("3" "16" "148410" "X")
          
          (isbn10-split "3-16-148410-X")
          ⇒ ("3" "16" "148410" "X")
          
          (isbn10-split "3 16 148410 X")
          ⇒ ("3" "16" "148410" "X")
— Function: isbn13-split isbn

Given a 13 digits ISBN string (with or without separators) satisfying isbn13?/well-formed, split it and return a validated list holding: a string representing the prefix, a string representing the language group number, a string representing the publisher number, a string representing the book number, a string representing the check digit. If isbn is invalid: the return value is #f. 

          (isbn13-split "9783161484100")
          ⇒ ("978" "3" "16" "148410" "0")
          
          (isbn13-split "978-3-16-148410-0")
          ⇒ ("978" "3" "16" "148410" "0")
          
          (isbn13-split "978 3 16 148410 0")
          ⇒ ("978" "3" "16" "148410" "0")
— Function: isbn-split isbn

Wrapper for isbn10-split and isbn13-split selecting the appropriate function according to the length of isbn.

— Function: isbn-tokenise isbn

Split an ISBN string with hyphen or space separators in its components and return them as a list of strings. If isbn has no separators: the return value is a list holding isbn itself. No validation is performed on the input string. 

          (isbn-tokenise "978 3 16 148410 0")
          ⇒ ("978" "3" "16" "148410" "0")
          
          (isbn-tokenise "978-3-16-148410-0")
          ⇒ ("978" "3" "16" "148410" "0")
