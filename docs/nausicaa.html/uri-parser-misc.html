70.5.6 Miscellaneous parser functions

The functions can be used to parse the elements of an URI; all the functions accept a port argument: it must be a binary input port drawing characters from a bytevector.

The following bindings are exported by the (nausicaa uri low) library; it is suggested to prefix them when importing this library.

— Function: valid-component? port

Scan bytes from port until EOF or an invalid byte is found; return two values, when success: true and the port position of the last byte read; when failure: false and the port position of the invalid byte. In any case: the port position is reverted to the state it had before the call to this function.

Ensure that:

  • A byte representing the percent character in ASCII encoding is followed by two bytes representing hexadecimal digits in ASCII encoding.
  • All the non percent–encoded bytes are in the unreserved set defined by RFC 3986.
