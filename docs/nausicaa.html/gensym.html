30 Generating unique symbols

The library (nausicaa language gensym) implements an API to generate a sequence of “unique” symbols. Usage examples: 

     #!r6rs
     (import (nausicaa))
     
     (parametrise ((gensym-counter 0))
       (let ((a (gensym))
             (b (gensym))
             (c (gensym)))
         (list a b c (gensym))))
     ⇒ (nausicaa.gensym.0
         nausicaa.gensym.1
         nausicaa.gensym.2
         nausicaa.gensym.3)
     
     (parametrise ((gensym-counter 0)
                   (gensym-prefix "ciao"))
       (let ((a (gensym))
             (b (gensym))
             (c (gensym)))
         (list a b c (gensym))))
     ⇒ (ciao0 ciao1 ciao2 ciao3)
     
     (parametrise ((gensym-counter 0)
                   (gensym-prefix "G")
                   (gensym-seed ".1."))
       (let ((a (gensym))
             (b (gensym))
             (c (gensym)))
         (list a b c (gensym))))
     ⇒ (G.1.0 G.1.1 G.1.2 G.1.3)
— Parameter: gensym-counter

Hold an exact integer used as suffix for generated symbol names. It is initialised to zero.

— Parameter: gensym-prefix

Hold a Scheme string used as prefix for generated symbol names. It is initialised to ‘"nausicaa.gensym."’.

— Parameter: gensym-seed

Hold a Scheme string used as middle content for generated symbol names. It is initialised to the empty string.

— Function: gensym

Return a new symbol built using the parameters.
