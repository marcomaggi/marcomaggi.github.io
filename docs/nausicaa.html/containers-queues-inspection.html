10.6.2 Inspecting queue objects

The following bindings are exported by the library (nausicaa containers queues). The bindings whose name is prefixed with $ are unsafe operations: they do not validate their arguments before accessing them.

— Immutable virtual field of <queue>: <boolean> empty?
— Immutable virtual field of <queue>: <boolean> $empty?

A boolean, #t if the instance is empty; otherwise return #f.

— Immutable virtual field of <queue>: <boolean> not-empty?
— Immutable virtual field of <queue>: <boolean> $not-empty?

A boolean, #t if the instance is not empty; otherwise return #f.

— Immutable virtual field of <queue>: <integer> size
— Immutable virtual field of <queue>: <integer> $size

An exact integer representing the number of objects in the instance.
