5.5 Keyword label

— Builtin Label: <keyword>

Conventional type for objects satisfying keyword?.

— Common constructor on <keyword>: <keyword> constructor name

Build and return a new keyword object whose name is name, which must be a symbol or string.

Safe accessors
— Immutable virtual field of <keyword>: <symbol> symbol

Accessing this field is like applying keyword->symbol to the instance.

— Immutable virtual field of <keyword>: <string> string

Return a the string representing the name of the symbol embedded in the keyword object. Mutating the returned string leads to undefined behaviour.

OOPP syntaxes
— Maker on <keyword>: <keyword> (?name)

Build and return a new keyword object whose name is the symbol resulting from the evaluation of the expression ?name, which must return a symbol or string.
