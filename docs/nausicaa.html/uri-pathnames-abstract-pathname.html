11.3.2.1 Base pathname abstract class

The following bindings are exported by the library (nausicaa uri pathnames abstract).

— Abstract Class: <pathname>

Base type for file system pathname objects.

Constructors and predicates
— Super constructor on <pathname>: constructor

Constructor to be used by the derived classes' constructors.

Public fields
— Immutable virtual field of <pathname>: <bytevector> bytevector

Bytevector representation of the pathname; the value itself must not be mutated. The value of this field is produced by the multimethod pathname-bytevector, pathname-bytevector.

— Immutable virtual field of <pathname>: <string> string

String representation of the pathname; the value itself must not be mutated. The value of this field is produced by the multimethod pathname-string, pathname-string.

— Immutable virtual field of <pathname>: <bytevector> uri

Bytevector representation of the pathname in the format specified by RFC 3986; the value itself must not be mutated. The value of this field is produced by the multimethod pathname-uri-representation, pathname-uri-representation.

Methods
— Method on <pathname>: extension

Return a bytevector representing the extension of the last segment of the instance pathname, dot separator not included. This method is implemented by the multimethod pathname-extension, pathname-extension.

— Method on <pathname>: dirname

Return a pathname object representing the directory part of the instance pathname. This method is implemented by the multimethod pathname-dirname, pathname-dirname.

— Method on <pathname>: tailname

Return a pathname object representing the last segment of the instance pathname. This method is implemented by the multimethod pathname-tailname, pathname-tailname.

— Method on <pathname>: rootname

Return a pathname object representing the instance pathname with the extension of the last segment stripped, dot separator not included. This method is implemented by the multimethod pathname-rootname, pathname-rootname.

— Method on <pathname>: split

Return 2 values: a boolean, #t if the instance pathname is absolute; a list of bytevectors each representing a segment of the instance pathname. This method is implemented by the multimethod pathname-split, pathname-split.

— Method on <pathname>: prefix? (ptn <pathname>)

Return #t if the instance pathname is a prefix of ptn, otherwise return #f. This method is implemented by the multimethod pathname-prefix?, pathname-prefix?.

— Method on <pathname>: suffix? (ptn <pathname>)

Return #t if the instance pathname is a suffix of ptn, otherwise return #f. This method is implemented by the multimethod pathname-suffix?, pathname-suffix?.

— Method on <pathname>: prepend (ptn <pathname>)

Return a new pathname object representing the instance pathname prepended to ptn. This method is implemented by the multimethod pathname-prepend, pathname-prepend.

— Method on <pathname>: replace-extension ext

Return a new pathname object representing the instance pathname with the extension of the last segment replaced by ext. This method is implemented by the multimethod pathname-replace-extension, pathname-replace-extension.
