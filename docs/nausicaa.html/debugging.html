49 Debugging utilities

The (nausicaa debugging) library provide simple utilities to help debugging. All the following bindings are re–exported by the (nausicaa checks) library.

— Parameter: debugging

When set to true debug will print its message. It is initialised to #f.

— Function: debug string
— Function: debug format-string arg ...

If the debugging parameter is set to true, print string to the current error port, or hand format-string and the optional arg values to format and print the result to the current error port.

— Function: debug-print-condition message exc

A wrapper for debug that prints message and does its best to print exc, which must be a Scheme condition value.
