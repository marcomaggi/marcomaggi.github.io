3.18 Tags and objects inspection

— Syntax: tag-unique-identifiers ?tag

Return the list of UIDs in the class hierarchy of ?tag. 

          (tag-unique-identifiers <top>)
          ⇒ (nausicaa:builtin:<top>)
          
          (tag-unique-identifiers <pair>)
          ⇒ (nausicaa:builtin:<pair>
              nausicaa:builtin:<top>)
          
          (tag-unique-identifiers <textual-port>)
          ⇒ (nausicaa:builtin:<textual-port>
              nausicaa:builtin:<port>
              nausicaa:builtin:<top>)
