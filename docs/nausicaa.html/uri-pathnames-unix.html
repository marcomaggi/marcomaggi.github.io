11.3.3 Manipulating Unix pathnames

The library (nausicaa uri pathnames unix) specialises the pathnames API defined by (nausicaa uri pathnames abstract) to handle Unix–style pathnames. It implements all the methods required by the abstract base classes.

The following bindings from (nausicaa uri pathnames abstract) are reexported by (nausicaa uri pathnames unix): 

     <pathname>
     <relative-pathname>
     <absolute-pathname>
     pathname=?

The string and bytevector representations of Unix–pathnames used by (nausicaa uri pathnames unix) are the same as the ones defined by (vicare parser-tools unix-pathnames), Parsing Unix file system pathnames.

  • uri pathnames unix intro: Introduction to Unix pathnames.
  • uri pathnames unix absolute: Absolute Unix pathname objects.
  • uri pathnames unix relative: Relative Unix pathname objects.
  • uri pathnames unix utils: Unix pathname utilities.
  • uri pathnames unix errors: Condition object types.
