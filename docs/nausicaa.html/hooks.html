45 Running lists of functions

The (nausicaa hooks) library exports an API defining hooks: lists of functions which can be applied to arguments at a specified point in a program. Nausicaa/Scheme hooks are somewhat similar to GNU Emacs and GNU Guile hooks. 

     #!r6rs
     (import (nausicaa) (nausicaa hooks))
     
     (let (((H <hook>) (make <hook>))
           (result     0))
       (H.register! (lambda ()
                      (incr! result 2)))
       (H.register! (lambda ()
                      (incr! result 4)))
       (H.run)
       result)
     ⇒ 6
     
     (let (((H <hook>) (make <hook>
                         (number-of-arguments 1)))
           (result     0))
       (H.register! (lambda (x)
                      (incr! result x)))
       (H.register! (lambda (x)
                      (incr! result x)))
       (H.apply 2)
       result)
     ⇒ 4
— Class: <hook>

Type of hook objects.

— Syntax on <hook>: make <hook> ?clause ...
— Auxiliary Syntax on <hook>: number-of-arguments num

Build and return a new instance of <hook>.

The optional number-of-arguments clause sets the number of arguments to num; when not used, the number of arguments defaults to zero.

— Immutable Field of <hook>: number-of-arguments

The fixed number of arguments of all the registered hook functions.

— Method on <hook>: register! func
— Method on <hook>: register! func last?

Register func as hook function; return a unique identifier associated to func. If last? is false or not used: the new function is registered as first, else it is registered as last.

— Method on <hook>: forget! id

Remove a previously registered hook function, the one associated to id which must be the return value of a previous call to register!.

— Method on <hook>: reset!

Remove all the previously registered hooks.

— Method on <hook>: empty?

Return true if the hook holds no registered functions.

— Method on <hook>: apply arg ...

Apply all the hook functions to the selected arg which must be in number equal to the number-of-arguments field; the order of evaluation is from the last registered to the first. Hooks can be run any number of times.

— Method on <hook>: run

Run all the hook functions without arguments; to be used only when the number-of-arguments field is set to zero. Hooks can be run any number of times.
