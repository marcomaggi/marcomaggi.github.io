5.17.1 Base port label

— Builtin Label: <port>

Conventional type for objects satisfying port?.

Safe accessors and mutators
— Immutable virtual field of <port>: <transcoder> transcoder

Accessing this field is like applying to the port the function port-transcoder from (rnrs io ports (6)).

— Immutable virtual field of <port>: <boolean> textual?

Accessing this field is like applying to the port the function textual-port? from (rnrs io ports (6)).

— Immutable virtual field of <port>: <boolean> binary?

Accessing this field is like applying to the port the function binary-port? from (rnrs io ports (6)).

— Immutable virtual field of <port>: <boolean> input?

Accessing this field is like applying to the port the function input-port? from (rnrs io ports (6)).

— Immutable virtual field of <port>: <boolean> output?

Accessing this field is like applying to the port the function output-port? from (rnrs io ports (6)).

— Immutable virtual field of <port>: <boolean> has-port-position?

Accessing this field is like applying to the port the function port-has-port-position? from (rnrs io ports (6)).

— Immutable virtual field of <port>: <boolean> has-set-port-position?

Accessing this field is like applying to the port the function port-has-set-port-position? from (rnrs io ports (6)).

— Mutable virtual field of <port>: <integer> port-position

Accessing and mutating this field is like applying to the port, respectively, the functions port-position and set-port-position! from (rnrs io ports (6)).

— Immutable virtual field of <port>: <boolean> closed?

Accessing this field is like applying to the port the function port-closed? from (vicare).

— Immutable virtual field of <port>: fd

Accessing this field is like applying to the port the function port-fd from (vicare).

— Immutable virtual field of <port>: <string> id

Accessing this field is like applying to the port the function port-id from (vicare).

— Immutable virtual field of <port>: <symbol> uid

Accessing this field is like applying to the port the function port-uid from (vicare).

— Immutable virtual field of <port>: <integer> hash

Accessing this field is like applying to the port the function port-hash from (vicare).

— Immutable virtual field of <port>: <boolean> non-blocking-mode?

Accessing this field is like applying to the port the function port-in-non-blocking-mode? from (vicare). Mutating this field is like applying to the port the function port-set-non-blocking-mode! and port-unset-non-blocking-mode! from (vicare).
