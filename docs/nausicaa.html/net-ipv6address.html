68.2 IPv6 address objects

The library (nausicaa net ipv6addresses) exports utilities to handle IPv6 addresses specifications, according to RFC 4291.

  • net ipv6address lexer: Tokenising an IPv6 address.
  • net ipv6address parser: Parsing an IPv6 address.
  • net ipv6address utilities: IPv6 address parsing utilities.
  • net ipv6address errors: Condition objects and error handlers.
  • net ipv6address class: IPv6 address and address prefix class objects.
