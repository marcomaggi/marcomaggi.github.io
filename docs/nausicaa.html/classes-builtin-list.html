22.8.3 Pairs and lists types

— Foreign Class: <pair>

Conventional type of objects satisfying pair?.

— Immutable Virtual Field of <pair>: car

Access the car of the pair.

— Immutable Virtual Field of <pair>: cdr

Access the cdr of the pair.

— Foreign Class: <list>

Conventional type of objects satisfying list?. It is derived from <pair>.

— Immutable Virtual Field of <list>: length

Access the length of the list.

— Immutable Virtual Field of <list>: null?

Return true if the list is empty.

— Immutable Virtual Field of <list>: caar
— Immutable Virtual Field of <list>: cadr
— Immutable Virtual Field of <list>: cdar
— Immutable Virtual Field of <list>: cddr
— Immutable Virtual Field of <list>: caaar
— Immutable Virtual Field of <list>: caadr
— Immutable Virtual Field of <list>: cadar
— Immutable Virtual Field of <list>: caddr
— Immutable Virtual Field of <list>: cdaar
— Immutable Virtual Field of <list>: cdadr
— Immutable Virtual Field of <list>: cddar
— Immutable Virtual Field of <list>: cdddr

Apply the corresponding function to the list. Notice that (as a special exception to goot style) referencing these virtual fields may raise an error if the list structure is not suitable.

— Method on <list>: find proc

Apply find to the list.

— Method on <list>: reverse proc

Apply reverse to the list.

— Method on <list>: for-all proc
— Method on <lits>: exists proc

Apply for-all and exists to the list.

— Method on <list>: filter proc
— Method on <list>: partition proc

Apply filter and partition to the list.

— Method on <list>: fold-left nil combine

Apply fold-left to the list.

— Method on <list>: fold-right nil combine

Apply fold-right to the list.

— Method on <list>: remp proc
— Method on <list>: remove obj
— Method on <list>: remv obj
— Method on <list>: remq obj

Apply remp, remove, remv, remq to the list.

— Method on <list>: memp proc list
— Method on <list>: member obj list
— Method on <list>: memv obj list
— Method on <list>: memq obj list

Apply memp, member, memv, memq to the list.

— Method on <list>: assp proc
— Method on <list>: assoc obj
— Method on <list>: assv obj
— Method on <list>: assq obj

Apply assp, assoc, assv, assq to the list.
