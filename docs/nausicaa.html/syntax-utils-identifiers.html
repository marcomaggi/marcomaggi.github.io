10.3 Constructing identifiers

— Macro: identifier->string id

Given an identifier syntax object return its string name.

— Macro: string->identifier context string

Convert string to a symbol and then into an identifier in the same context as the identifier context.

— Function: identifier-prefix prefix identifier

Return a new identifier, in the same context of identifier, whose name is built by prepending prefix to the name of identifier. prefix must be a string, symbol or identifier.

— Function: identifier-suffix identifier suffix

Return a new identifier, in the same context of identifier, whose name is built by appending suffix to the name of identifier. suffix must be a string, symbol or identifier.

— Function: syntax-maker-identifier identifier

Return a new identifier, in the same context of identifier, whose name is built by prepending make- to the name of identifier.

— Function: syntax-predicate-identifier

Return a new identifier, in the same context of identifier, whose name is built by appending ? to the name of identifier.

— Function: syntax-accessor-identifier type-identifier field-identifier
— Function: syntax-mutator-identifier type-identifier field-identifier

Return a new identifier, in the same context of type-identifier, whose name is built by joining the arguments with a dash -; additionally syntax-mutator-identifier appends -set!.

— Function: syntax-method-identifier type-identifier field-identifier

Return a new identifier, in the same context of type-identifier, whose name is built by joining the arguments with a dash -. This function is meant to be used to build class's method function names.

— Function: syntax-dot-notation-identifier variable-identifier field-identifier

Return a new identifier, in the same context of type-identifier, whose name is built by joining the arguments with a dot.

— Function: identifier-general-append arg0 arg ...

Build and return a new string by appending the given arguments. The arguments can be identifiers, strings and symbols.
