29.1.7.4 Utility macros

The following bindings are exported by the (nausicaa ffi) library.

In the following macro descriptions: the type symbols in <ret-type> and <arg-type> are normalised by applying to each a function named clang-maybe-foreign-type->clang-external-type, which must be present in the lexical context of the symbol; such function must exist and convert any external type to a type in the enum-clang-external-types enumeration; when <ret-type> or <arg-type> is already a symbol in enum-clang-external-types, the function should just return its argument.

— Syntax: make-c-callout* <ret-type> <address> (<arg-type0> <arg-type> ...)
— Syntax: make-c-callout/with-errno* <ret-type> <address> (<arg-type0> <arg-type> ...)

Wrappers for the make-c-callout functions which allow us not to quote the arguments and to perform types normalisation.

— Syntax: make-c-function* <shared-object> <ret-type> <funcname> (<arg-type0> <arg-type> ...)
— Syntax: make-c-function/with-errno* <shared-object> <ret-type> <funcname> (<arg-type0> <arg-type> ...)

Wrappers for the make-c-function functions which determine the address of the function by looking for the <funcname> symbol in the <shared-object>.

As an example, the main deflate functions from Zlib can be defined with: 

          (define-shared-object zlib-shared-object 'libz.so)
          
          (define deflateInit
            (make-c-function* zlib-shared-object
                              int deflateInit_ (pointer int)))
          
          (define deflate
            (make-c-function* zlib-shared-object
                              int deflate (pointer int)))
          
          (define deflateEnd
            (make-c-function* zlib-shared-object
                              int deflateEnd (pointer)))

but we can also take advantage of these macro features to define them like this: 

          (define-shared-object zlib-shared-object 'libz.so)
          
          (define (clang-maybe-foreign-type->clang-external-type type)
            (case type
              ((z_streamp) 'pointer)
              (else type)))
          
          (define deflateInit
            (make-c-function* zlib-shared-object
                              int deflateInit_ (z_streamp int)))
          
          (define deflate
            (make-c-function* zlib-shared-object
                              int deflate (z_streamp int)))
          
          (define deflateEnd
            (make-c-function* zlib-shared-object
                              int deflateEnd (z_streamp)))

Variadic functions, like fprintf(), are not supported, but we can interface with them by nailing the arguments; for example: 

          (define-shared-object uuid-shared-object 'libuuid.so)
          
          (define uuid_make
            (let ((arg-2  (make-c-function* uuid-shared-object
                            int uuid_make (void* unsigned-int)))
                  (arg-4  (make-c-function* uuid-shared-object
                            int uuid_make
                            (void* unsigned-int void* char*))))
              (case-lambda
               ((*id mode)
                (arg-2 *id mode))
               ((*id mode uuid name)
                (arg-4 *id mode uuid name)))))
— Syntax: make-c-callback* <ret-type> <scheme-proc> (<arg-type0> <arg-type> ...)

Wrapper for the make-c-callback function which allows us not to quote the arguments and to normalise the types.

— Syntax: define-c-functions <lib-spec> (<pub-name> (<ret-type> <funcname> (<arg-type0> <arg-type> ...))) ...
— Syntax: define-c-functions/with-errno <lib-spec> (<pub-name> (<ret-type> <funcname> (<arg-type0> <arg-type> ...))) ...

Interfaces to the make-c-function* and make-c-function/with-errno* macros. Example: 

          (define-shared-object zlib-shared-object 'libz.so)
          
          (define (clang-maybe-foreign-type->clang-external-type type)
            (case type
              ((z_streamp) 'pointer)
              (else type)))
          
          (define-c-functions zlib-shared-object
            (deflateInit  (int deflateInit_ (z_streamp int)))
            (deflate      (int deflate      (z_streamp int)))
            (deflateEnd   (int deflateEnd   (z_streamp))))
— Syntax: define-c-callouts (<pub-name> (<ret-type> <address> (<arg-type0> <arg-type> ...))) ...
— Syntax: define-c-callouts/with-errno (<pub-name> (<ret-type> <address> (<arg-type0> <arg-type> ...))) ...

Interfaces to the make-c-callout* and make-c-callout/with-errno* macros. Example: 

          (define-shared-object zlib-shared-object 'libz.so)
          
          (define (clang-maybe-foreign-type->clang-external-type type)
            (case type
              ((z_streamp) 'pointer)
              (else type)))
          
          (define-c-callouts
            (deflateInit
             (int (lookup-shared-object zlib-shared-object 'deflateInit_)
                  (z_streamp int)))
          
            (deflate
             (int (lookup-shared-object zlib-shared-object 'deflate)
                  (z_streamp int)))
          
            (deflateEnd
             (int (lookup-shared-object zlib-shared-object 'deflateEnd)
                  (z_streamp))))
