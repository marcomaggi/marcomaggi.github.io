35.20 Rabin–Karp string search

The (nausicaa strings rabin-karp) library implements the Rabin–Karp string search algorithm.

— Class: <rabin-karp>

Hold the state for repeated string searches using the same pattern over different texts.

— Default constructor on <rabin-karp>: constructor pattern base prime

Build an instance. pattern must be a Scheme string representing the pattern to search for. base must be a fixnum used in the rolling hash algorithm (when in doubt use 256). prime must be a “large” prime fixnum (when in doubt use 472882049).

— Syntax: make <rabin-karp> pattern <clause> ...
— Auxiliary Syntax: base <base>
— Auxiliary Syntax: fixnum-prime <prime>

Build an instance using the default constructor. The clauses are optional, <base> defaults to 256 and <prime> defaults to 472882049.

— Method on <rabin-karp>: search text

Perform the search of pattern over text, which must be a Scheme string. If successful: return the offset of the first character of the first occurrence pattern in text; else return #f.

Examples: 

     #!r6rs
     (import (nausicaa)
       (nausicaa strings rabin-karp))
     
     (let (((o <rabin-karp>) (make <rabin-karp> "ciao")))
       (o.search "012345ciao")       ⇒ 6
       (o.search "hello world"))     ⇒ #f
