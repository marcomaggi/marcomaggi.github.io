16 Overall configuration

The (nausicaa configuration) library defines a set of parameters (in the sense defined by the (nausicaa language parameters) library) which can be used to configure the behaviour of Nausicaa. It is suggested to import the library with the config. prefix, as in: 

     (import (nausicaa)
       (prefix (nausicaa configuration) config.))
— Parameter: enable-assertions?

A boolean value, true if the code should include assertions to validate values. If set to false: the assertions are excluded at code expansion time.

The default value is true. When the library is instantiated, the system environment variable NAUSICAA_ENABLE_ASSERTIONS is accessed and used to configure this parameter; recognised values are: true, false, 1, 0.

— Parameter: enable-contracts?

A boolean value, true if the code should include validation of contracts as defined by the (nausicaa contracts) library. If set to false: the contracts are excluded at code expansion time.

The default value is true. When the library is instantiated, the system environment variable NAUSICAA_ENABLE_CONTRACTS is accessed and used to configure this parameter; recognised values are: true, false, 1, 0.

— Parameter: enable-function-arguments-validation?

A boolean value, true if the functions should include assertions to validate their arguments. If set to false: the assertions are excluded at code expansion time.

The default value is true. When the library is instantiated, the system environment variable NAUSICAA_ENABLE_ARGUMENTS_VALIDATION is accessed and used to configure this parameter; recognised values are: true, false, 1, 0.
