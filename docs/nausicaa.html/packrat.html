75 Packrat parser

The (nausicaa parser-tools packrat) library implements parsing combinators using the Packrat parsing algorithm: A memoizing, backtracking, recursive descent parsing technique that runs in time and space linear in the size of the input text. There is no single Scheme object which can be named “Packrat parser”, rather a parser is a collection of combinator functions calling each other.

The library makes use of the <lexical-token> and <source-location> record types, and follows their protocol for end–of–input and lexer errors handling. Describing lexical tokens.

When first experimenting with it, we sould import the library with the following directive: 

     (import (rnrs)
       (prefix (nausicaa parser-tools packrat) packrat.))

to avoid imported bindings name conflicts.

  • packrat intro: Introduction to the algorithm.
  • packrat calc: Calculator example.
  • packrat types: Parser state and results.
  • packrat comb: Combinators.
