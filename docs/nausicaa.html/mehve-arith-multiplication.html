6.4.4 Multiplication generic function

The generic function * exported by (nausicaa mehve) is implemented using the following bindings exported by (nausicaa mehve arithmetics). Notice that * is used also in the implementation of /.

— Generic Function: multiplication-0

It has a single method used when * is called with no arguments.

— Generic Function: multiplication-1 A

Used when * is called with one argument.

— Generic Function: multiplication-2 A B

Used when * is called with two arguments.

The following are the specialised methods.

— Method: multiplication-0

Return the exact integer 1.

— Method: multiplication-1 (A <number>)

Return A itself.

— Method: multiplication-2 (A <fixnum>) (B <fixnum>)

Apply to the arguments the function fx* exported by the (rnrs arithmetic flonums (6)) library, see fx*.

— Method: multiplication-2 (A <flonum>) (B <flonum>)

Apply to the arguments the function fl* exported by the (rnrs arithmetic flonums (6)) library, see fl*.

— Method: multiplication-2 (A <number>) (B <number>)

Apply to the arguments the function * exported by the (rnrs base (6)) library, see *.
