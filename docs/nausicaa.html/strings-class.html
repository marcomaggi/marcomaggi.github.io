36.19 Label string interface

The (nausicaa strings xstrings) library exports the single binding <xstring>, which provides an interface to the (nausicaa strings) functions through the facilities of the (nausicaa language classes) library.

— Label: <xstring>

Label interface to the string functions. It is derived from the foreign class <string>, inheriting all its fields and methods.

All the following methods are implemented as syntaxes; in all the following descriptions: self is an identifier bound to the subject string.

Constructors
— Method on <xstring>: concatenate list-of-strings

Implement the expansion: 

          (self.concatenate list-of-strings)
          ==> (string-concatenate (cons self list-of-strings))
— Method on <xstring>: concatenate-reverse list-of-strings
— Method on <xstring>: concatenate-reverse list-of-strings final-string
— Method on <xstring>: concatenate-reverse list-of-strings final-string nvalues

Implement the expansions: 

          (self.concatenate-reverse list-of-strings)
          ==> (string-concatenate-reverse
               (cons self list-of-strings))
          
          (self.concatenate-reverse list-of-strings
                                    final-string)
          ==> (string-concatenate-reverse
               (cons self list-of-strings)
               final-string)
          
          (self.concatenate-reverse list-of-strings
                                    final-string nvalues)
          ==> (string-concatenate-reverse
               (cons self list-of-strings)
               final-string nvalues)
