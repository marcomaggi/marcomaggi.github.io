6.3.6 General mismatch error

The following bindings are exported by the (nausicaa language conditions) library.

— Condition Type Label: &mismatch

Signal a mismatch between two values. It is derived from &assertion.

— Function: make-mismatch-condition

Build and return a new &mismatch condition object.

— Function: mismatch-condition? obj

Return #t if obj is a condition of object of type &mismatch.
