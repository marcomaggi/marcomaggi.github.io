12.5.4.4 Path component parser functions

The following functions can be used to parse the path-abempty, path-absolute, path-noscheme, path-rootless and path-empty components; the relevant portion of the grammar is: 

     path-abempty  = *( "/" segment )
     path-absolute = "/" [ segment-nz *( "/" segment ) ]
     path-noscheme = segment-nz-nc *( "/" segment )
     path-rootless = segment-nz *( "/" segment )
     path-empty    = 0<pchar>

notice that all the path-* components appear in the grammar as last components of a hier-part or relative-part component, which in turn can be followed only by EOF, a query (starting with a ‘?’ byte) or a fragment (starting with a ‘#’ byte).

The following bindings are exported by the library (nausicaa parser-tools uri). All the functions accept a port argument: it must be a binary input port supporting the port position.

— Function: parse-path-empty port

Parse a path-empty component. Lookahead one byte from port: if it is EOF or a question mark or number–sign in ASCII encoding: return null; else return #f.

In any case leave the port position where it was before the function call. 

          (import (vicare)
            (prefix (nausicaa parser-tools uri) uri.))
          
          (define (mkport S)
            (open-bytevector-input-port (string->ascii S)))
          
          (uri.parse-path-empty (mkport ""))
          ⇒ ()
          
          (let ((P (mkport "ciao")))
            (uri.parse-path-empty P)      ⇒ #f
            (ascii->string
              (get-bytevector-some P)))   ⇒ "ciao"
          
          (let ((P (mkport "/ciao")))
            (uri.parse-path-empty P)      ⇒ #f
            (ascii->string
              (get-bytevector-some P)))   ⇒ "/ciao"
          
          (let ((P (mkport "?query")))
            (uri.parse-path-empty P)      ⇒ ()
            (ascii->string
              (get-bytevector-some P)))   ⇒ "?query"
          
          (let ((P (mkport "#fragment")))
            (uri.parse-path-empty P)      ⇒ ()
            (ascii->string
              (get-bytevector-some P)))   ⇒ "#fragment"
— Function: parse-path-abempty port

Parse from port a, possibly empty, sequence of sequences: byte representing the slash character in ASCII encoding, segment component. Return a possibly empty list holding bytevectors representing the segments.

If successful: leave the port position to the byte after the last read byte; if an error occurs: rewind the port position to the one before this function call.

If an invalid percent–encoded sequence is read, an exception is raised with type &parser-error; the port position is rewind to the one before this function call. 

          (import (vicare)
            (prefix (nausicaa parser-tools uri) uri.))
          
          (define (mkport S)
            (open-bytevector-input-port (string->ascii S)))
          
          (uri.parse-path-abempty (mkport ""))   ⇒ ()
          
          (let ((P (mkport "?query")))
            (uri.parse-path-abempty P)                ⇒ ()
            (ascii->string (get-bytevector-some P)))  ⇒ "?query"
          
          (let ((P (mkport "#fragment")))
            (uri.parse-path-abempty P)                ⇒ ()
            (ascii->string (get-bytevector-some P)))  ⇒ "#fragment"
          
          (let ((P (mkport "/ciao")))
            (map ascii->string
              (uri.parse-path-abempty P))             ⇒ ("ciao")
            (eof-object? (lookahead-u8 P)))           ⇒ #t
          
          (let ((P (mkport "/ciao?query")))
            (map ascii->string
              (uri.parse-path-abempty P))             ⇒ ("ciao")
            (ascii->string (get-bytevector-some P)))  ⇒ "?query"
          
          (let ((P (mkport "/ciao#fragment")))
            (map ascii->string
              (uri.parse-path-abempty P))             ⇒ ("ciao")
            (ascii->string (get-bytevector-some P)))  ⇒ "#fragment"
          
          (let ((P (mkport "/ciao/hello")))
            (map ascii->string
              (uri.parse-path-abempty P))    ⇒ ("ciao" "hello")
            (eof-object? (lookahead-u8 P)))  ⇒ #t
          
          (let ((P (mkport "/ciao/")))
            (map ascii->string
              (uri.parse-path-abempty P))    ⇒ ("ciao" "")
            (eof-object? (lookahead-u8 P)))  ⇒ #t
          
          (map ascii->string
            (uri.parse-path-abempty (mkport "///")))
          ⇒ ("" "" "")
— Function: parse-path-absolute port

Parse from port a path-absolute component; it is like parse-path-abempty, but expects a slash character as first byte and a non–slash as second byte. Return a possibly empty list holding bytevectors representing the segments, or #f if invalid bytes are read from the port.

If successful: leave the port position to the byte after the last read byte; if an error occurs: rewind the port position to the one before this function call.

Notice that a path-absolute can be just a slash character, with no segments attached. 

          (import (vicare)
            (prefix (nausicaa parser-tools uri) uri.))
          
          (define (mkport S)
            (open-bytevector-input-port (string->ascii S)))
          
          (uri.parse-path-absolute (mkport ""))
          ⇒ #f
          
          (let ((P (mkport "ciao")))
            (uri.parse-path-absolute P)               ⇒ #f
            (ascii->string (get-bytevector-some P)))  ⇒ "ciao"
          
          (let ((P (mkport "/")))
            (map ascii->string
              (uri.parse-path-absolute P))            ⇒ ("")
              (eof-object? (lookahead-u8 P)))         ⇒ #t
          
          (let ((P (mkport "//")))
            (uri.parse-path-absolute P)               ⇒ #f
            (ascii->string (get-bytevector-some P)))  ⇒ "//"
          
          (let ((P (mkport "/ciao")))
            (map ascii->string
              (uri.parse-path-absolute P))            ⇒ ("ciao")
            (eof-object? (lookahead-u8 P)))           ⇒ #t
          
          (let ((P (mkport "/ciao/salut")))
            (map ascii->string
              (uri.parse-path-absolute P))   ⇒ ("ciao" "salut")
            (eof-object? (lookahead-u8 P)))  ⇒ #t
          
          (let ((P (mkport "/ciao/")))
            (map ascii->string
              (uri.parse-path-absolute P))   ⇒ ("ciao" "")
            (eof-object? (lookahead-u8 P)))  ⇒ #t
          
          (let ((P (mkport "/?query")))
            (map ascii->string
              (uri.parse-path-absolute P))            ⇒ ("")
            (ascii->string (get-bytevector-some P)))  ⇒ "?query"
          
          (let ((P (mkport "/ciao?query")))
            (map ascii->string
              (uri.parse-path-absolute P))            ⇒ ("ciao")
            (ascii->string (get-bytevector-some P)))  ⇒ "?query"
          
          (let ((P (mkport "/#fragment")))
            (map ascii->string
              (uri.parse-path-absolute P))            ⇒ ("")
            (ascii->string (get-bytevector-some P)))  ⇒ "#fragment"
          
          (let ((P (mkport "/ciao#fragment")))
            (map ascii->string
              (uri.parse-path-absolute P))            ⇒ ("ciao")
            (ascii->string (get-bytevector-some P)))  ⇒ "#fragment"
— Function: parse-path-noscheme port

Parse from port a path-noscheme component. Return a non–empty list holding bytevectors representing the segments, or #f if invalid bytes are read from the port.

If successful: leave the port position to the byte after the last read byte; if an error occurs: rewind the port position to the one before this function call.

Notice that a path-noscheme must not start with a slash character, and then it must have at least one non–empty segment component; the first segment must not contain a colon caracter. 

          (import (vicare)
            (prefix (nausicaa parser-tools uri) uri.))
          
          (define (mkport S)
            (open-bytevector-input-port (string->ascii S)))
          
          (uri.parse-path-noscheme (mkport "")) ⇒ #f
          
          (let ((P (mkport "ciao")))
            (map ascii->string
              (uri.parse-path-noscheme P))   ⇒ ("ciao")
            (eof-object? (lookahead-u8 P)))  ⇒ #t
          
          (let ((P (mkport "/")))
            (uri.parse-path-noscheme P)               ⇒ #f
            (ascii->string (get-bytevector-some P)))  ⇒ "/"
          
          (let ((P (mkport "/ciao")))
            (uri.parse-path-noscheme P)               ⇒ #f
            (ascii->string (get-bytevector-some P)))  ⇒ "/"
          
          (let ((P (mkport "ciao/hello")))
            (map ascii->string
              (uri.parse-path-noscheme P))   ⇒ ("ciao" "hello")
            (eof-object? (lookahead-u8 P)))  ⇒ #t
          
          (let ((P (mkport "ciao/")))
            (map ascii->string
              (uri.parse-path-noscheme P))  ⇒ ("ciao" "")
            (eof-object? (lookahead-u8 P))) ⇒ #t
          
          (let ((P (mkport "ciao/he:llo")))
            (map ascii->string
             (uri.parse-path-noscheme P))   ⇒ ("ciao" "he:llo")
            (eof-object? (lookahead-u8 P))) ⇒ #t
          
          (let ((P (mkport "ci:ao/hello")))
            (uri.parse-path-noscheme P)     ⇒ #f
            (ascii->string
              (get-bytevector-some P)))     ⇒ "ci:ao/hello"
          
          (let ((P (mkport "?query")))
            (uri.parse-path-noscheme P)     ⇒ #f
            (ascii->string
              (get-bytevector-some P)))     ⇒ "?query"
          
          (let ((P (mkport "hello?ciao")))
            (map ascii->string
             (uri.parse-path-noscheme P))   ⇒ ("hello")
            (ascii->string
              (get-bytevector-some P)))     ⇒ "?ciao"
          
          (let ((P (mkport "#fragment")))
            (uri.parse-path-noscheme P)     ⇒ #f
            (ascii->string
              (get-bytevector-some P)))     ⇒ "#fragment"
          
          (let ((P (mkport "hello#fragment")))
            (map ascii->string
              (uri.parse-path-noscheme P))  ⇒ ("hello")
            (ascii->string
              (get-bytevector-some P)))     ⇒ "#fragment"
— Function: parse-path-rootless port

Parse from port a path-rootless component; it is like parse-path-abempty, but expects a non–slash character as first byte. Return a non–empty list holding bytevectors representing the segments, or #f if invalid bytes are read from the port.

If successful: leave the port position to the byte after the last read byte; if an error occurs: rewind the port position to the one before this function call.

Notice that a path-rootless must not start with a slash character, and then it must have at least one non–empty segment component. 

          (import (vicare)
            (prefix (nausicaa parser-tools uri) uri.))
          
          (define (mkport S)
            (open-bytevector-input-port (string->ascii S)))
          
          (uri.parse-path-rootless (mkport ""))  ⇒ #f

            (map ascii->string
              (uri.parse-path-rootless P))   ⇒ ("ciao")

          (let ((P (mkport "/")))

          (let ((P (mkport "/ciao")))

            (ascii->string
              (get-bytevector-some P))       ⇒ "#fragment"
