29.1.9.11 Common groups of tests
— Autoconf Macro: NAUSICAA_SIZEOF

Perform tests to determine the size (the number of bytes) of the common C language types. The following output variables are defined:

Variable Size of C type
SIZEOF_INT8 int8_t
SIZEOF_INT16 int16_t
SIZEOF_INT32 int32_t
SIZEOF_INT64 int64_t
SIZEOF_CHAR char
SIZEOF_SHORT_INT short int
SIZEOF_SHORT_UINT unsigned short int
SIZEOF_INT int
SIZEOF_UINT unsigned int
SIZEOF_LONG long
SIZEOF_ULONG unsigned long
SIZEOF_LLONG long long
SIZEOF_ULLONG unsigned long long
SIZEOF_FLOAT float
SIZEOF_DOUBLE double
SIZEOF_POINTER void *

Variable Alignment of C type
ALIGNOF_INT8 int8_t
ALIGNOF_INT16 int16_t
ALIGNOF_INT32 int32_t
ALIGNOF_INT64 int64_t
ALIGNOF_CHAR char
ALIGNOF_SHORT_INT short int
ALIGNOF_SHORT_UINT unsigned short int
ALIGNOF_INT int
ALIGNOF_UINT unsigned int
ALIGNOF_LONG long
ALIGNOF_ULONG unsigned long
ALIGNOF_LLONG long long
ALIGNOF_ULLONG unsigned long long
ALIGNOF_FLOAT float
ALIGNOF_DOUBLE double
ALIGNOF_POINTER void *

Variable Array stride of C type
STRIDEOF_INT8 int8_t
STRIDEOF_INT16 int16_t
STRIDEOF_INT32 int32_t
STRIDEOF_INT64 int64_t
STRIDEOF_CHAR char
STRIDEOF_SHORT_INT short int
STRIDEOF_SHORT_UINT unsigned short int
STRIDEOF_INT int
STRIDEOF_UINT unsigned int
STRIDEOF_LONG long
STRIDEOF_ULONG unsigned long
STRIDEOF_LLONG long long
STRIDEOF_ULLONG unsigned long long
STRIDEOF_FLOAT float
STRIDEOF_DOUBLE double
STRIDEOF_POINTER void *

Additionally ‘TYPEOF_POINTER_INTEGER’ is set to the type specification of the integer whose size equals the size of a pointer. It is one among ‘unsigned-int’ and ‘unsigned-long’

— Autoconf Macro: NAUSICAA_C_TYPES_LIMITS

Perform tests to determine the value of limits for the common C language types. The following output variables are defined:

Variable Original C expression
VALUEOF_CHAR_MAX CHAR_MAX
VALUEOF_CHAR_MIN CHAR_MIN
VALUEOF_SCHAR_MAX SCHAR_MAX
VALUEOF_SCHAR_MIN SCHAR_MIN
VALUEOF_UCHAR_MAX UCHAR_MAX
VALUEOF_SHRT_MAX SHRT_MAX
VALUEOF_SHRT_MIN SHRT_MIN
VALUEOF_USHRT_MAX USHRT_MAX
VALUEOF_INT_MAX INT_MAX
VALUEOF_INT_MIN INT_MIN
VALUEOF_UINT_MAX UINT_MAX
VALUEOF_LONG_MAX LONG_MAX
VALUEOF_LONG_MIN LONG_MIN
VALUEOF_ULONG_MAX ULONG_MAX
VALUEOF_LLONG_MAX LLONG_MAX
VALUEOF_LLONG_MIN LLONG_MIN
VALUEOF_ULLONG_MAX ULLONG_MAX
VALUEOF_WCHAR_MAX WCHAR_MAX
VALUEOF_SSIZE_MAX SSIZE_MAX

Additionally the output variable ‘WORDS_BIGENDIAN’ is set to #t if the underlying platform is big endian, to #f otherwise.
