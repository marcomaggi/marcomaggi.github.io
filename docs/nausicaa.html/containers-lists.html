10.1 Extended lists processing

The label <xlist> provides an interface to the functions (vicare containers lists) functions through the facilities of the language (nausicaa).

— Label: <xlist>

Label interface to the list functions. Note that <xlist> is not derived from <list> and it has no predicate.

All the following methods are implemented as syntaxes; in all the following descriptions: self is an identifier bound to the subject list.

— Method on <xlist>: list-copy

Implement the expansion: 

          (self list-copy)
          ==> (list-copy self)
— Method on <xlist>: tree-copy

Implement the expansion: 

          (self tree-copy)
          ==> (tree-copy self)
— Method on <xlist>: circular-list!
