11.3.2.3 Relative pathname abstract class

The following bindings are exported by the library (nausicaa uri pathnames abstract).

— Abstract Class: <relative-pathname>

Base type for relative pathnames; it is derived from <pathname>.

Constructors and predicates
— Super constructor on <relative-pathname>: constructor

Constructor to be used by the derived classes' constructors.

Methods
— Method on <pathname>: append (ptn <pathname>)

Return a new pathname object representing the instance pathname appended to ptn. This method is implemented by the multimethod pathname-append, pathname-append.
