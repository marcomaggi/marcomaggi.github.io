5.4 Symbol label

— Builtin Label: <symbol>

Conventional type for objects satisfying symbol?.

— Common constructor on <symbol>: <symbol> constructor string

Return the result of applying string->symbol to string.

Safe accessors
— Immutable virtual field of <symbol>: <string> string

Accessing this field is like applying symbol->string to the instance.

Methods
— Method on <symbol>: hash

Return an integer usable as hash value of the instance.

OOPP syntaxes
— Maker on <symbol>: <symbol> (?string)

Build and return a new symbol whose name is the string resulting from the evaluation of the expression ?string.
