33.3 Iterating over lists

The following bindings are exported by the (nausicaa iterators) library.

— Class: <list-iterator>

A forward stride iterator for Scheme lists; it is derived from <iterator>.

— Common constructor on <list-iterator>: constructor the-list stride

Build a new instance of <list-iterator> using the-list as list upon which to iterate and stride, which must be an exact positive integer, as stride. The stride is the number of elements in the list to jump forward to when iterating; it must be 1 when we want to visit all the elements.

— Syntax on <list-iterator>: make <list-iterator> . <clauses>
— Auxiliary Syntax on <list-iterator>: subject: <the-list>
— Auxiliary Syntax on <list-iterator>: stride: <stride>

Build and return a new class instance using the common constructor.

If the stride: clause is not used: the stride defaults to 1. 

     #!r6rs
     (import (nausicaa)
       (nausicaa iterators))
     
     (let* (((L <list>)          '(c i a o))
            ((I <list-iterator>) (make <list-iterator>
                                   (subject: L))))
     
       (I.more?)     ⇒ #t
       (I.next)      ⇒ c
       (I.more?)     ⇒ #t
       (I.next)      ⇒ i
       (I.more?)     ⇒ #t
       (I.next)      ⇒ a
       (I.more?)     ⇒ #t
       (I.next)      ⇒ o
       (I.more?))    ⇒ #f
     
     (let* (((L <list>)          '(h e l L o))
            ((I <list-iterator>) (make <list-iterator>
                                   (subject: L)
                                   (stride: 2))))
     
       (I.more?)     ⇒ #t
       (I.next)      ⇒ h
       (I.more?)     ⇒ #t
       (I.next)      ⇒ l
       (I.more?)     ⇒ #t
       (I.next)      ⇒ o
       (I.more?))    ⇒ #f
