76.4.3.1 Class interface to library references

The following bindings are exported by the (nausicaa parser-tools r6rs libraries references) library.

— Class: <library-reference>

Class type representing library references as specified by R6RS, scheme library form import.

— Immutable field of <library-reference>: <sexp> sexp

The full library reference specification.

— Immutable field of <library-reference>: <list> identifiers

List of identifiers in the library reference specification.

— Immutable field of <library-reference>: <sexp> version-reference

The version reference as a possibly null symbolic expression representing constraints on the version numbers.

— Common constructor on <library-reference>: constructor sexp

Build and return a new <library-reference> object representing the result of parsing sexp as a symbolic expression representing a library reference as specified by R6RS.

If sexp is not valid: raise an assertion violation.

— Syntax on <library-reference>: make <library-reference> sexp
— Syntax on <library-reference>: make* <library-reference> sexp

Build and return a new <library-reference> object using the common constructor.

— Syntax on <library-reference>: is-a? obj <library-reference>

Return #t if obj is an object of type <library-reference>, else return #f.

— Method on <library-reference>: <boolean> conforming? (name <library-name>)

Return #t if the library name conforms to the reference specified by this instance; else return #f.
