77.1.2.4 Variable–semantic–value token–maker parameters

The following bindings are exported by the library (nausicaa parser-tools r6rs lexer).

— Parameter: line-comment-token-maker

Set to a function returning a <lexical-token> record representing a line comment with line–ending included, ‘; ---’. The default is make-line-comment-token.

— Parameter: line-comment-noend-token-maker

Set to a function returning a <lexical-token> record representing a line comment without line–ending; only end–of–input should be allowed after this token. The default is make-line-comment-noend-token.

— Parameter: white-space-token-maker

Set to a function returning a <lexical-token> record representing a white space. The default is make-white-space-token.

— Parameter: line-ending-token-maker

Set to a function returning a <lexical-token> record representing a line ending. The default is make-line-ending-token.

— Parameter: sharp-bang-token-maker

Set to a function returning a <lexical-token> record representing a sharp–bang comment, ‘#!---’. The default is make-sharp-bang-token.
