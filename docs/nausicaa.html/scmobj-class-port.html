50.3.7 Port types

The following classes for input/output ports are predefined.

— Class: <port>

Class for values that satisfy port?.

— Class: <input-port>

Class for values that satisfy input-port?.

— Class: <output-port>

Class for values that satisfy output-port?.

— Class: <textual-port>

Class for values that satisfy textual-port?.

— Class: <binary-port>

Class for values that satisfy binary-port?.

The binary/textual attributes are not mututally exclusive with the input/output attributes, so if we need it, we can use multiple inheritance: 

     (define-class <binary-input-port> (<binary-port> <input-port>))
