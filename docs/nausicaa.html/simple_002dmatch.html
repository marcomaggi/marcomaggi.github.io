8 Simple destructuring match syntax

The library (nausicaa language simple-match) provides a simple implementation of the destructuring–bind match syntax; it is not a fully featured match syntax. The library has API similar to (vicare language-extensions simple-match), but it recognises the identifiers exported by the libraries in the hierarchy (nausicaa language ---) as auxiliary syntaxes.

  • simple-match syntax: Pattern matcher generator.
