77.4.4.1 Import sets

The bindings documented in this section are exported by the (nausicaa parser-tools r6rs libraries import) library.

— Class: <import-set>

Type representing an import set as specified by R6RS, scheme library form import.

— Immutable field of <import-set>: subset

The core of this import set, it can be another <import-set> object or a <library-reference> object.

— Immutable field of <import-set>: bindings-transformer

A function to be applied to a list renamings to obtain the imported bindings. It is meant to be applied to the full list of renamings representing the export set of the referenced library.

— Immutable field of <import-set>: <sexp> original

The original symbolic expression of this import set. <sexp> is the label from (nausicaa language matches).

— Immutable field of <import-set>: <library-reference> library-reference

The <library-reference> object nested in this import set.

— Common constructor on <import-set>: constructor sexp

Parse the symbolic expression sexp as an import sets as specified by R6RS; return a new <import-set> object. Raise an assertion violation if sexp is invalid.

— Syntax on <import-set>: make <import-set> sexp
— Syntax on <import-set>: make* <import-set> sexp

Build and return a new <import-set> object using the common constructor.

— Syntax on <import-set>: is-a? obj <import-set>

Return #t if obj is a record of type <import-set>.

— Method on <import-set>: apply renamings

Apply the import set specification to the list of renamings, returning the processed list of renamings (which can be null). It applies recursively the specifications of the nested import sets.
