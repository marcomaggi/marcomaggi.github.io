77.2.4.4 Interlexeme datum makers

The following bindings are exported by the (nausicaa parser-tools r6rs parser) library.

— Function: make-interlexeme-space-datum yypushback yycustom value

Default function for the interlexeme-space-datum-maker parameter. Accept as value a list of values representing the interlexeme spaces; return an instance of <interlexeme-space> (see <interlexeme-space>).

— Function: make-whitespace-datum yypushback yycustom value

Default function for the whitespace-datum-maker parameter. Accept as value the string representing the white space; return the string itself.

— Function: make-line-comment-datum yypushback yycustom value

Default function for the line-comment-datum-maker parameter. Accept as value the string representing the line comment; return the string itself.

— Function: make-nested-comment-datum yypushback yycustom value

Default function for the nested-comment-datum-maker parameter. Accept as value the string representing the nested comment; return the string itself.

— Function: make-sharp-semicolon-datum yypushback yycustom interlexeme-space datum

Default function for the sharp-semicolon-datum-maker parameter. Accept as interlexeme-space argument #f or a value representing the optional interlexeme space, as datum the commented datum (r6rs parser inter).

— Function: make-sharp-bang-datum yypushback yycustom value

Default function for the sharp-bang-datum-maker parameter. Accept as value the string representing the sharp–bang comment; return the string itself.

— Function: make-sharp-bang-r6rs-datum yypushback yycustom value

Default function for the sharp-bang-r6rs-datum-maker parameter. Accept as value the string representing the #!r6rs comment; return the string itself.
