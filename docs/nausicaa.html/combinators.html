57 Combinators

The following combinators are exported by (nausicaa combinators).

— Syntax: K form . forms

Evaluate all the forms like the standard begin, but return the value of the first form. 

          (define-syntax K
            (syntax-rules ()
              ((_ ?expr0 ?expr ...)
               (call-with-values
                   (lambda () ?expr0)
                 (lambda x
                   ?expr ...
                   (apply values x))))))

This is the same as the begin0 syntax.

— Syntax: Y func

Build a recursive function with no name. 

          (define-syntax Y
            (syntax-rules ()
              ((_ ?func)
               (let ((F ?func))
                 ((lambda (f)
                    (F (lambda (. args)
                         (apply (f f) args))))
                  (lambda (f)
                    (F (lambda (. args)
                         (apply (f f) args)))))))))

The combinator has to called like this: 

          ((Y (lambda (f)
                (lambda (. args)
                  (do-something-with-args args)
                  (if (condition) (f args))))
              arg1 arg2 ...))
— Syntax: S x y z

The S combinator: 

          (define-syntax S
            (syntax-rules ()
              ((_ ?x ?y ?z)
               (let ((Z ?z))
                 (?x Z (?y Z))))))
