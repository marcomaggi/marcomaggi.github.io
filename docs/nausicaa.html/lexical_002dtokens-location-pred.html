73.1.3 Predicates

— Syntax on <source-location>: is-a? ?obj <source-location>

Return #t if ?obj is an instance of <source-location>.

Procedural interface
— Function: <source-location>? obj
— Function: <source-location>?/or-false obj

Return true if obj is a record of type <source-location>. The second variant accepts #f, too.

— Function: <source-location>?/start obj
— Function: <source-location>?/start/or-false obj

Return true if obj is a record of type <source-location> and it represents the first possible location: Column and line counters set to 1. The offset counter is ignored. The second variant accepts #f, too.
