50.4 Instances

— Macro: make class-object . init-slots

Build and return a new instance object of class class-object. The optional arguments are symbol/value couples used to initialise the slots: The symbol must be the name of a slot, the value its value.

It is not mandatory to initialise all the slots in a class instance. Uninitialised slots will be initialised to the symbol :uninitialised.

Examples

Instantiate a class with 3 slots: 

     (let* ((<one> (make-class () :a :b :c))
            (o     (make <one>
                      :a 1 :b 2 :c 3)))
       ---)

Instantiate a class with superclasses, initialising all the slots: 

     (let* ((<one>   (make-class () :a :b :c))
            (<two>   (make-class (<one>) :d :e :f))
            (<three> (make-class (<two>) :g :h :i))
            (o       (make <three>
                        :a 1 :b 2 :c 3
                        :d 4 :e 5 :f 6
                        :g 7 :h 8 :i 9)))
       ---)
