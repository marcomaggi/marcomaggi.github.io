28.3 Programming interface

— Syntax: defer-exceptions form ...

Evaluate all the forms as in the body of a begin.

If an exception is raised: its value is added to the list of exceptions in the dynamic environment of the current deferred exception handler.

If defer-exception appears outside of a with-deferred-exception-handler form: the exceptions are silently discarded.

— Syntax: with-deferred-exceptions-handler handler thunk

Setup a deferred exception handler dynamic environment, in which handler is the exceptions handler, for the evaluation of thunk. If a deferred exception is raised run-deferred-exceptions-handler is invoked.

— Function: run-deferred-exceptions-handler

Apply the current deferred exceptions handler to each deferred exception value in the current list. If the handler raises an exception: it is silently ignored.
