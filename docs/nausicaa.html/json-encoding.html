67.3 Encoding JSON text

— Function: json-make-pair name value
— Function: json-make-pair name value encode-value?

Build and return a string representing a JSON pair.

name must be a string and it is always encoded with json-encode-string.

value can be a string, a number, #t, #f or (). When value is a string and encode-value? is true: value is encoded with json-encode-string; encode-value? defaults to true.

— Function: json-make-pair* name value
— Function: json-make-pair* name value encode-value?

Like json-make-pair but with the following extensions:

  • If value is positive infinity, it is encoded as the literal ‘Infinity’.
  • If value is positive infinity, it is encoded as the literal ‘-Infinity’.
  • If value is not–a–number, it is encoded as the literal ‘NaN’.

— Function: json-make-object pair ...

Given a list of strings representing JSON pairs return a string representing a JSON object.

— Function: json-make-array list-or-vector

Given a list or vector of strings representing JSON values return a string representing a JSON array. 

          (json-make-array '("12" "34" "56"))
          ⇒ "[ 12, 34, 56 ]"
          
          (json-make-array '#("12" "34" "56"))
          ⇒ "[ 12, 34, 56 ]"
— Function: json-encode-string input-string

Encode a Scheme string to its representation as JSON string; return a string. All the characters whose integer representation is outside the range [32, 126] are represented with the ‘\uXXXX’ sequence, where ‘X’ is a hexadecimal digit. Examples: 

          (json-encode-string "")         ⇒ ""
          (json-encode-string "ciao")     ⇒ "ciao"
          (json-encode-string "\"")       ⇒ "\\\""
          (json-encode-string "/")        ⇒ "\\/"
          (json-encode-string "\b")       ⇒ "\\b"
          (json-encode-string "\f")       ⇒ "\\f"
          (json-encode-string "\n")       ⇒ "\\n"
          (json-encode-string "\r")       ⇒ "\\r"
          (json-encode-string "\t")       ⇒ "\\t"
          (json-encode-string " ")        ⇒ " "
          (json-encode-string "\x127;")   ⇒ "\\u0127"
          (json-encode-string "\x1234;")  ⇒ "\\u1234"
