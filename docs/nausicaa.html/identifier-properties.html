23 Expand–time identifier properties

The library (nausicaa language identifier-properties) allows the association of objects to identifiers at expand time, using other identifiers as keywords. To avoid name collisions, it is suggested to import the library with a prefix; for example: 

     (prefix (nausicaa language identifier-properties) ip.)

Meaningless usage examples: 

     (import (rnrs)
       (prefix (nausicaa language identifier-properties) ip.)
       (nausicaa language syntax-utilities))
     
     (define-auxiliary-syntax type)
     
     (define a "ciao")
     (ip.define-identifier-property a type 'string)
     
     (define b 123)
     (ip.define-identifier-property b type 'number)
     
     (define-syntax get-type
       (lambda (stx)
         (syntax-case stx ()
           ((_ ?id)
            (ip.ref #'?id #'type)))))
     
     (get-type a) ⇒ string
     (get-type b) ⇒ int

Property keywords used in Nausicaa/Scheme should be exported by the library (nausicaa language property-keywords).

— Syntax: define-identifier-property subject key value

Expand to a dummy definition; associate value (the syntax object) to the identifier subject using the identifier key as key.

The functions below are exported for expand by (nausicaa language identifier-properties).

— Function: define subject key value

Function to be invoked at expand–time (typically in the body of a macro transformer); associate key to value in subject's table; raise an assertion violation if the key is already defined.

— Function: set! subject key value

Associate key to value in subject's table; overwrite the old value, if any.

— Function: ref subject key
— Function: ref subject key default

Function to be invoked at expand–time (typically in the body of a macro transformer); retrieve the value associated to the identifier subject using the identifier key as key. If no property is found, return default which defaults to false.

— Function: table subject key

Return the property table associated to the arguments. This is meant to be used only for debugging purposes.
