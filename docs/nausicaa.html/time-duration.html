50.5 Time duration objects

A time duration is the algebraic difference between two points in time on the TAI scale; durations can be: defined directly, added to points in time or computed as difference between points in time. Durations are represented by a seconds count and a nanoseconds count, which are normalised to be both non–negative or both non–positive. The following bindings are exported by the (nausicaa times-and-dates) library.

— Class: <duration>

Type representing a time duration on the TAI scale; it is derived from <seconds-and-nanoseconds>.

— Immutable Field of <duration>: seconds

Exact integer representing seconds count on the TAI scale.

— Immutable Field of <duration>: nanoseconds

Exact integer representing nanoseconds count.

— Syntax on <duration>: make* <duration> seconds nanoseconds

Expand to an expression evaluating to a new <duration> instance. seconds and nanoseconds must expand into expressions evaluating to exact integers representing the counts of seconds on the TAI scale and the count of nanoseconds.

Seconds and nanoseconds are normalised to be both non–negative or both non–positive.

— Syntax on <duration>: make <duration> clause ...
— Auxiliary Syntax on <duration>: seconds seconds
— Auxiliary Syntax on <duration>: milliseconds milliseconds
— Auxiliary Syntax on <duration>: microseconds microseconds
— Auxiliary Syntax on <duration>: nanoseconds nanoseconds

Expand to an expression evaluating to a new <duration> instance. The seconds and nanoseconds counts are computed from the values given in the following maker clauses and normalised to be both non–negative or both non–positive:

seconds seconds
milliseconds milliseconds
microseconds microseconds
nanoseconds nanoseconds
Select the count in the specified seconds and subseconds units. The seconds count must be on the TAI scale.

— Method on <duration>: deep-clone
— Method on <duration>: shallow-clone

Return a copy of the object.

Comparing time durations

Comparison aims at estabilishing the longer or shorter duration between a couple of durations; comparison functions do not consider the sign of seconds and nanoseconds counts: they compare the absolute values.

— Method on <duration>: = duration
— Method on <duration>: < duration
— Method on <duration>: <= duration
— Method on <duration>: > duration
— Method on <duration>: >= duration

Compare the instance's duration to the duration of the <duration> object duration and return a boolean value.

— Function: duration= duration ...
— Function: duration< duration ...
— Function: duration<= duration ...
— Function: duration> duration ...
— Function: duration>= duration ...

Compare the <duration> arguments and return a boolean value. When there are no arguments or only one argument: the return value is true.

Arithmetic functions
— Method on <duration>: + duration

Add the <duration> object duration to the instance and return a new <duration> object representing the result.

— Method on <duration>: - duration

Subtract the <duration> object duration from the instance and return a new <duration> object representing the result.

— Method on <duration>: + lambda

Multiply the instance by the real number lambda and return a new <duration> object representing the result. The results of the product are turned into exact numbers.

— Method on <duration>: / lambda

Divide the instance by the real number lambda and return a new <duration> object representing the result. The results of the division are turned into exact numbers.

— Function: duration+ duration ...

Add all the <duration> arguments and return a new <duration> object representing the sum. When called with no arguments: return a <duration> of zero seconds and zero nanoseconds.

— Function: duration- duration ...

Subtract the <duration> arguments beyond the first from the first and return a new <duration> object representing the difference. When called with no arguments: return a <duration> of zero seconds and zero nanoseconds.

Traditional record API
— Function: make-<duration> seconds nanoseconds

Build and return a new <duration> instance.

— Function: <duration>? obj

Return true if obj is an instance of <duration>.

— Function: <duration>-seconds sn
— Function: <duration>-nanoseconds sn
— Function: <duration>-full-seconds sn
— Function: <duration>-full-microseconds sn
— Function: <duration>-full-nanoseconds sn

Accessors for concrete and virtual fields of a <duration> instance.

— Function: <duration>-deep-clone sn
— Function: <duration>-shallow-clone sn

Return a copy of a <duration> instance.
