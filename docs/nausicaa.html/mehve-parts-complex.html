6.5.4 Complex numbers generic functions

The generic functions real-part, imag-part, magnitude, angle, make-rectangular and make-polar exported by (nausicaa mehve parts), and reexported by (nausicaa mehve), generalise the corresponding functions from (rnrs base (6)).

— Generic Funcion: real-part A

Return the real part of A.

— Generic Funcion: imag-part A

Return the imaginary part of A.

— Generic Funcion: magnitude A

Return the magnitude of A.

— Generic Funcion: angle A

Return the angle of A.

— Generic Funcion: make-rectangular R I

Return a complex having real part R and imaginary part I.

— Generic Funcion: make-polar M A

Return a complex having magnitude M and angle A.

The following are the specialised methods.

— Generic Funcion: real-part (A <real>)

Return A itself.

— Generic Funcion: real-part (A <complex>)

Apply to the argument the function real-part exported by the (rnrs base (6)) library, see real-part.

— Generic Funcion: imag-part (A <complex>)

Apply to the argument the function imag-part exported by the (rnrs base (6)) library, see imag-part.

— Generic Funcion: magnitude (A <real>)

Apply to the argument the function abs exported by the (nausicaa mehve arithmetics) library, see abs.

— Generic Funcion: magnitude (A <complex>)

Apply to the argument the function magnitude exported by the (rnrs base (6)) library, see magnitude.

— Generic Funcion: angle (A <complex>)

Apply to the argument the function angle exported by the (rnrs base (6)) library, see angle.

— Generic Funcion: make-rectangular (R <real>) (I <real>)

Apply to the arguments the function make-rectangular exported by the (rnrs base (6)) library, see make-rectangular.

— Generic Funcion: make-polar (M <real>) (A <real>)

Apply to the arguments the function make-polar exported by the (rnrs base (6)) library, see make-polar.
