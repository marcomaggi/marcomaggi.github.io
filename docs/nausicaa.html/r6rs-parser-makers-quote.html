76.2.4.3 Quoted and syntax datum makers

The following bindings are exported by the (nausicaa r6rs parser) library.

— Function: make-quoted-datum yypushback yycustom value

Default function for the quoted-datum-maker parameter. Accept as value the quoted datum; return a list of two elements whose car is the symbol quote and whose cadr is the datum itself.

— Function: make-quasiquoted-datum yypushback yycustom value

Default function for the quasiquoted-datum-maker parameter. Accept as value the quoted datum; return a list of two elements whose car is the symbol quasiquote and whose cadr is the datum itself.

— Function: make-unquoted-datum yypushback yycustom value

Default function for the unquoted-datum-maker parameter. Accept as value the quoted datum; return a list of two elements whose car is the symbol unquote and whose cadr is the datum itself.

— Function: make-unquoted-splicing-datum yypushback yycustom value

— Function: make-syntax-datum yypushback yycustom value
