28.3.12 Getting values out of the expression

A getter expression matcher looks like this: 

     (match 2
       ((:getter two)
        (two)))
     ⇒ 2

we can think of it as expanding to: 

     (let ((expr 2))
       (let ((two (lambda () expr)))
         (two)))

so the ‘(:getter two)’ pattern instructs match to define a thunk that can be used to retrieve the expression itself. We understand that the single argument ?getter in the abstract pattern ‘(:getter ?getter)’ must be an indentifier. We can evaluate the thunk any number of times.

Let's look at another example: 

     (match '(1 2 3)
       ((_ (:getter two) _)
        (two)))
     ⇒ 2

we can think of this as expanding to: 

     (let ((expr '(1 2 3)))
       (let ((expr1 (cadr expr)))
         (let ((two (lambda () expr1)))
           (two))))
