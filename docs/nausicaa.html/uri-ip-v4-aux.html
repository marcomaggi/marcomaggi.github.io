11.1.4.3 IPv4 address auxiliary classes

The following bindings are exported by the library (nausicaa uri ip).

— Label: <ipv4-address-fixnum>

Label type whose predicate tests that the object is a non–negative fixnum in the range [0, 255]. It is derived from <nonnegative-fixnum> as defined by (nausicaa language builtins), Fixnum number objects.

— Label: <vector-of-ipv4-address-fixnums>

Label type whose predicate tests that the object is a vector of 4 items all being of type <ipv4-address-fixnum>. It is derived from <vector> as defined by (nausicaa language builtins), Vector type.

— Label: <ipv4-address-prefix-length>

Label type whose predicate tests that the object is a non–negative fixnum in the range [0, 32]. It is derived from <nonnegative-fixnum> as defined by (nausicaa language builtins), Fixnum number objects.
