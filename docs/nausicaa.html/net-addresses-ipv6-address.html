12.1.2.1 IPv6 address objects class

The following bindings are exported by the (nausicaa net addresses ipv6) library.

— Class: <ipv6-address>

Represents an IPv6 address.

Constructors and makers
— Common constructor on <ipv6-address>: constructor numbers
— Common constructor on <ipv6-address>: constructor seventh sixth fifth fourth third second first zeroth

Build and return a new object of type <ipv6-address>. When applied to a single argument: numbers must be a list of 8 fixnums in the range [0, #xFFFF] representing the address components. When applied to 8 arguments: each must be a fixnum in the range [0, #xFFFF] representing an address component.

— Maker on <ipv6-address>: <ipv6-address> (?numbers)
— Maker on <ipv6-address>: <ipv6-address> (?seventh ?sixth ?fifth ?fourth ?third ?second ?first ?zeroth)

Build and return a new object of type <ipv6-address> using the common constructor. 

          (<ipv6-address> O
            (<> (1 2 3 4 5 6 7 8)))
          
          (list (O third) (O second) (O first) (O zeroth))
          ⇒ (5 6 7 8)
Fields
— Immutable field of <ipv6-address>: <ipv6-address-fixnum> zeroth
— Immutable field of <ipv6-address>: <ipv6-address-fixnum> first
— Immutable field of <ipv6-address>: <ipv6-address-fixnum> second
— Immutable field of <ipv6-address>: <ipv6-address-fixnum> third
— Immutable field of <ipv6-address>: <ipv6-address-fixnum> fourth
— Immutable field of <ipv6-address>: <ipv6-address-fixnum> fifth
— Immutable field of <ipv6-address>: <ipv6-address-fixnum> sixth
— Immutable field of <ipv6-address>: <ipv6-address-fixnum> seventh

The numeric components of the address, each representing 16 bits. zeroth is the rightmost component, first is the second from the right, etc.

— Immutable virtual field of <ipv6-address>: <exact-integer> bignum

A single big exact integer of 128 bits representing the whole address number.

— Immutable virtual field of <ipv6-address>: <string> string

The string representation of the address.

— Immutable virtual field of <ipv6-address>: <boolean> unspecified?
— Immutable virtual field of <ipv6-address>: <boolean> loopback?
— Immutable virtual field of <ipv6-address>: <boolean> multicast?
— Immutable virtual field of <ipv6-address>: <boolean> link-local-unicast?
— Immutable virtual field of <ipv6-address>: <boolean> global-unicast?

True if the address has type, respectively: unspecified, loopback, multicast, link–local unicast, global unicast. See Section 2.4 “Address Type Identification” of RFC 4291.
