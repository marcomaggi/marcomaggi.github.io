22.8.4 Non–list compound types

Vectors
— Foreign Class: <vector>

Conventional type for objects satisfying vector?.

— Immutable Virtual Field of <vector>: length

Access the length of the vector.

— Syntax: setf (o index) value

Store value at index of the vector o. index is an expression.

— Syntax: getf (o index)

Return the value at index of the vector o. index is an expression.

Bytevectors
— Foreign Class: <bytevector>

Conventional type for objects satisfying bytevector?.

— Immutable Virtual Field of <bytevector>: length

Access the length of the bytevector.

— Syntax: setf (o index) value
— Syntax: setf (o index type) value
— Syntax: setf (o index type endian) value

Implemented as a macro: store value at index index of the bytevector o. The mutator is selected according to the following rules:

  • When no type nor endian is used: the mutator is bytevector-u8-set!.
  • When both type and endian are used: the mutator is selected accordingly; type can be one among: 
                   u8  s8    u16 s16
               u32 s32   u64 s64
               uint   sint
               single double

    while endian can be one among ‘big’ and ‘little’.

  • When only type is used: the mutator is selected accordingly among the native ones; type can be one among: 
                   u16n s16n
               u32n s32n   u64n s64n
               singlen doublen

— Syntax: getf (o index)
— Syntax: getf (o index type)
— Syntax: getf (o index type endian)

Implemented as a macro: return the value at index index of the bytevector o. The accessor is selected according to the following rules:

  • When no type nor endian is used: the mutator is bytevector-u8-ref.
  • When both type and endian are used: the accessor is selected accordingly; type can be one among: 
                   u8  s8    u16 s16
               u32 s32   u64 s64
               uint   sint
               single double

    while endian can be one among ‘big’ and ‘little’.

  • When only type is used: the accessor is selected accordingly among the native ones; type can be one among: 
                   u16n s16n
               u32n s32n   u64n s64n
               singlen doublen

Hashtables
— Foreign Class: <hashtable>

Conventional type for objects satisfying hashtable?.

— Immutable Virtual Field of <hashtable>: size

Access the size of the table.

— Immutable Virtual Field of <hashtable>: keys

Access the keys. Retrieving this field is like applying hashtable-keys to the hashtable.

— Immutable Virtual Field of <hashtable>: entries

Access the keys and values. Retrieving this field is like applying hashtable-entries to the hashtable.

— Syntax: setf (o key) value

Associate value to key in the hashtable o. The key is an expression.

— Syntax: getf (o key)
— Syntax: getf (o key default)

Return the value associated to key in the hashtable o. When not found: return default or #f if default was not given. key and default are expressions.
