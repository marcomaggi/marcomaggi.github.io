77.2.3 Parser semantic actions parameters

Some of the bindings exported by the (nausicaa parser-tools r6rs parser) library reference parameter functions as defined by (nausicaa language parameters). The semantic action forms in the R6RS parser table invoke the functions referenced by the parameters to produce datums, or equivalent values.

Unless otherwise specified, all the parameters whose name has the -datum-maker suffix are used as follows in the semantic action forms: 

     ((identifier-datum-maker) yypushback yycustom value)

and they must be set to functions with the following (or equivalent) behaviour: 

     (lambda (yypushback yycustom value)
       (make-datum value))

where: yypushback and yycustom are the bindings handed by (nausicaa parser-tools lalr lr-driver) to the semantic action forms (see lalr grammar clauses); value is the semantic value from the lexical token.

  • r6rs parser params general: General datum makers.
  • r6rs parser params scheme: Scheme value datum makers.
  • r6rs parser params quote: Quoted and syntax datum makers.
  • r6rs parser params inter: Interlexeme datum makers.
