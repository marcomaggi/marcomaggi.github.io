3.20.3 OOPP bindings: begin–like syntaxes

— Syntax: begin/tags ?body0 ?body ...
— Syntax: begin/tags (<- ?tag ...) ?body0 ?body ...
— Auxiliary Syntax: <-

When the auxiliary syntax <- is not used: begin/tags behaves like begin from (rnrs (6)). When the auxiliary syntax <- is used: it specifies one tag for each value returned by the last ?body form; such tag is used to validate the corresponding return value with the tag's predicate.
