50.7.4 Invoking the next method

— Function: call-next-method

To be invoked from a :primary or :around method. Call the next most specific method.

— Function: next-method?

Return true if a next method is available. Available next methods are :primary and :around methods. After the first :primary method has been called, only :primary methods are considered.

It can be invoked by any kind of method.
