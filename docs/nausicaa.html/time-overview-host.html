45.2.2 Retrieving the current time

Unix Time is the count of seconds since the Epoch, ignoring leap seconds; by definition: when UTC was at January 1 1972 00:00:00, TAI was at January 1 1972 00:00:10. We can express UTC and TAI times, too, as count of seconds since the Epoch. After noticing that neither 1970 nor 1971 are leap years, we assume that the Unix Time scale and the TAI time scale are equal in those years, so the count of seconds since the Unix Epoch of January 1 1972 00:00:00 is: 

     (1972 - 1970) * 365 * 24 * 60 * 60 = 63072000

we establish that: 

     63072000(Unix) = 63072000(UTC) = 63072010(TAI)

So, let's say that “now” is 2000 January 1 00:00:00 UTC and the host clock is synchronised with this time: we retrieve the current time from a Unix Time clock which returns 946681200, taking into account leap years but not leap seconds; we can verify this using the GNU date command: 

     $ date -d '2000/01/01 00:00:00' +%s
