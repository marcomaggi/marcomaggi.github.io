72.7.1 ISO 7064:1983 mod 37, 36

The (nausicaa standard-numbers iso-7064-1983 mod-37-36) library implements the ISO 7064:1983 mod 37, 36 checksum used to validate alphanumeric identifiers. Informations on the algorithm were derived from the Global Release Identifier (GRid) Standard document version 2.1 (URL last verified Jun 3, 2011):

http://www.ifpi.org/content/section_resources/grid.html
  • stdnum iso 7064 1983 37 intro: Introduction to the algorithm.
  • stdnum iso 7064 1983 37 namesp: Algorithm namespaces.
  • stdnum iso 7064 1983 37 low: Low level functions.
