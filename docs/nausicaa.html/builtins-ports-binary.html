5.17.3 Binary port labels

— Builtin Label: <binary-port>

Conventional type for objects satisfying binary-port?.

— Builtin Label: <binary-input-port>

Conventional type for objects satisfying binary-port? and input-port?. It is derived from <input-port>.

— Builtin Label: <binary-output-port>

Conventional type for objects satisfying binary-port? and output-port?. It is derived from <output-port>.

— Builtin Label: <binary-input/output-port>

Conventional type for objects satisfying binary-port? and input/output-port?. It is derived from <input/input-port>.

Input port methods
— Method on <binary-input-port>: get-single
— Method on <binary-input/output-port>: get-single

Calling this method il like applying to the port the function get-u8 from (rnrs io ports (6)).

— Method on <binary-input-port>: lookahead-single
— Method on <binary-input/output-port>: lookahead-single
