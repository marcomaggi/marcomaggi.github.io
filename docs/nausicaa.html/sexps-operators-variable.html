51.2.2 Variable matching

To extract a value from a sexp form we use variables matching operators. It works like this: 

     (define ?alpha (make-sexp-variable '?alpha 1))
     (define ?beta  (make-sexp-variable '?beta  2))
     
     (sexp-match `(,(sexp-var ?alpha) ,(sexp-var ?beta))
                 '(91 92))
     ⇒ ((#<record sexp-variable ?alpha> . 91)
         (#<record sexp-variable ?beta>  . 92))

the resulting alist of bindings can be inspected with assq using the ‘sexp-variable’ records as keys. A variable matching operator matches any value but null.

— Function: sexp-var variable

Used to extract a value from a form, return a matching operator. variable must be a record of type ‘sexp-variable’.

— Function: sexp-var-rest variable

Used to extract a list of values from a form, return a matching operator. variable must be a record of type ‘sexp-variable’. This operator matches all its input form and binds it to the variable; the input form can be null.

— Class: sexp-variable

Record type used to assign a name to values from sexp forms.

— Immutable Field of <sexp-variable>: name

The public name of the variable, to be used for debugging purposes.

— Immutable Field of <sexp-variable>: default

A default value for the variable in case the sexp form does no bind it. When using a transformer the sentinel value has a special meaning. The sentinel value

— Function: make-sexp-variable name default

Build and return a new ‘sexp-variable’ record.

— Function: sexp-variable? obj

Return #t if obj is a record of type ‘sexp-variable’.

— Function: sexp-variable-name obj
— Function: sexp-variable-default obj

Accessors for the fields of a ‘sexp-variable’ record.

— Syntax: let-sexp-variables ((<var> <default>) ...) <body>

Similar to let, build a set of ‘sexp-variable’ records and bind them to <var> identifiers. The body is evaluated in the current environment extended with the variables bindings. 

          (let-sexp-variables ((?alpha 1)
                               (?beta  2))
            (sexp-match `(,(sexp-var ?alpha) ,(sexp-var ?beta))
                        '(91 92)))
          ⇒ ((#<record sexp-variable ?alpha> . 91)
              (#<record sexp-variable ?beta>  . 92))
