6.3.4 Expecting long integers

Preprocessor Macro: void cctests_assert_equal_long (cce_destination_t L, signed long expected, signed long result)
Preprocessor Macro: void cctests_assert_equal_long (cce_destination_t L, signed long expected, signed long result, char const * message, …)

Assert that expected equals result. If the values are equal: just return. Otherwise raise an exception of type cctests_condition_assertion_expected_long_t by performing a non–local exit to L.

When the optional argument message and the following arguments are used: a dynamically allocated string is formatted using message as template and printf() format specifiers; this string is used as description message for the assertion violation.
