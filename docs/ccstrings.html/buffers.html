5 Collecting output in buffers

Programmatically generated textual output is gathered in buffers: dynamically allocated blocks of memory that can be resized and to which we write with functions similar to the standard vsnprintf(). The definitions of the buffers API are accessible in the header file ccstrings.h.

Struct Typedef: ccstr_buffer_t

Type of data structures representing output buffers. Internally, this structure holds a pointer to a block of memory holding the actual data; such memory is allocated with malloc().
