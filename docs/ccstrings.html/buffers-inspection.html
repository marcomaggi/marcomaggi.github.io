5.4 Inspecting buffers

Function: bool ccstr_buffer_is_full (ccstr_buffer_t * B)

Return true if the buffer is full; otherwise return false.

Function: ccmem_block_t ccstr_buffer_output_block (ccstr_buffer_t * B)
Function: ccmem_ascii_t ccstr_buffer_output_ascii (ccstr_buffer_t * B)

Return a block representing the data in the buffer. If B references an initialised buffer: the pointer in the block is never NULL.

Function: ccmem_block_t ccstr_buffer_target_block (ccstr_buffer_t * B)
Function: ccmem_ascii_t ccstr_buffer_target_ascii (ccstr_buffer_t * B)

Return a block representing the free room at the end of the buffer. If B references an initialised buffer: the pointer in the block is never NULL. The return value of this function is meaningful only if a call to ccstr_buffer_is_full() applied to the same buffer returns false.
