5.3 Writing output to buffers

Function: void ccstr_buffer_fwrite (cce_destination_t L, ccstr_buffer_t * B, FILE * stream)

Write the buffer contents to stream, using the system call fwrite(). If an error occurs: raise an exception by performing a non–local exit to L.

Function: void ccstr_buffer_write (cce_destination_t L, ccstr_buffer_t * B, int filedes)

Write the buffer contents to filedes, using the system call write().

  • If the system call completes successfully and all the data is written: return.
  • If the system call completes successfully but not all the data is written: raise an exception by performing a non–local exit to L.
  • If an error occurs and errno is set to EINTR: the function loops internally trying again to write data.
  • If an error occurs: raise an exception by performing a non–local exit to L.
