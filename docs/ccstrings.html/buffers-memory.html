5.5 Buffers memory handlers

Function: void ccstr_buffer_enlarge (cce_destination_t L, ccstr_buffer_t * B, size_t required_len)

Reallocate the buffer enlarging it so that it can hold more than required_len characters. If an error occurs reallocating: perform a non–local exit by jumping to L; in this case the buffer is left untouched and its contents are still valid.
