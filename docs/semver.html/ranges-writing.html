5.3 Writing to a string

All the following symbols are defined in the header file semver.h.

Function: int semver_range_pwrite (const semver_range_t * rangep, char * output_buf, size_t output_len)

Serialise the range rangep as ASCII string in the memory block referenced by output_buf, which has at least output_len characters available. If the buffer has not enough room to hold the full comparator serialisation: the string is truncated to output_len characters.

Like snprintf(): return the number of characters needed to hold the full serialisation, excluding the trailing null; if the return value is greater than or equal to output_len, the string has been truncated.

Preprocessor Macro: int semver_range_write (semver_range_t rangep, char * output_buf, size_t output_len)

Like semver_range_pwrite(), but accept the range structure as argument, rather than a pointer to it.

Example: serialising a range

The following example serialises a range: 

static const char     input_str[] = \
   "1.2.7 || >=1.2.9 <2.0.0";
size_t                buffer_len = 32;
char                  buffer_ptr[buffer_len];
semver_range_t        range;
size_t                offset = 0;
char                  rv;

memset(buffer_ptr, 0, buffer_len);

rv = semver_range_read(&range, \
        input_str, strlen(input_str), &offset);
if (0 == rv) {
  size_t      needed_count;
  size_t      actual_count;
  needed_count = (size_t)semver_range_write(range, \
     buffer_ptr, buffer_len);
  if (0 < needed_count) {
    actual_count = (needed_count < buffer_len)? \
       needed_count : buffer_len;
    printf("len=%lu, actual_count=%lu, range=",
           strlen(input_str), actual_count);
    fwrite(buffer_ptr, sizeof(char), actual_count, stdout);
    printf("\n");
  }
}
semver_range_dtor(&range);

the code prints: 

len=23, actual_count=23, range=1.2.7 || >=1.2.9 <2.0.0
