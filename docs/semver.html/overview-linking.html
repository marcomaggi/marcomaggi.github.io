1.1 Linking code with the library

Using the GNU Autotools

This package installs a data file for pkg-config, so when searching for the installed library with the GNU Autotools, we can add the following macro use to configure.ac: 

PKG_CHECK_MODULES([LIBSV],[libsv >= 3.1])

which will set the variables LIBSV_LIBS and LIBSV_CFLAGS.

Alternatively we can use the raw GNU Autoconf macros: 

AC_CHECK_LIB([libsv],[semver_ctor],,
  [AC_MSG_FAILURE([test for Libsv library failed])])
AC_CHECK_HEADERS([semver.h],,
  [AC_MSG_FAILURE([test for Libsv header failed])])

One–shot compilation, Unix command line

For the one–shot compilation of a file named demo.c, we can do: 

gcc -std=c99 -Wall \
   `pkg-config libsv --cflags` \
   `pkg-config libsv --libs` \
   -o demo demo.c
