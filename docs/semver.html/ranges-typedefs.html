5.1 Type definitions

All the following symbols are defined in the header file semver.h.

Struct Typedef: semver_range_t

Type of data structures representing parsed ranges; a range is a set of comparators. Every instance of this structure is a node in a simply–linked list. Instances of this structure are initialised with semver_range_read(). Instances of this structure are finalised with semver_range_dtor().

It has the following public fields:

semver_range_t * next

Pointer to the next structure in the linked list; NULL if this instance is the last in the list.

semver_comp_t comp

A version comparator.

Function: void semver_range_dtor (semver_range_t * rangep)

Finalise to empty a data structure representing a parsed range; calls to this function must match a previous call to semver_range_read().

All the resources associated to rangep are released, but the structure itself. The following code is correct: 

semver_range_t  range;

semver_range_read(&range, ...);
{
  /* Do something with "range" here. */
}
semver_range_dtor(&range);

the following code is also correct: 

semver_range_t * rangep;

rangep = malloc(sizeof(semver_range_t));
assert(NULL != rangep);
{
  semver_range_read(&range, ...);
  {
    /* Do something with "rangep" here. */
  }
  semver_range_dtor(rangep);
}
free(rangep);
