7.1 Type definitions

All the following symbols are defined in the header file semver.h.

Struct Typedef: semver_id_t

Type of data structures representing a single component in a parsed identifier; every instance of this structure is a node in a simply–linked list. Instances of this structure are initialised with semver_id_ctor() or semver_id_read(). Instances of this structure are finalised with semver_id_dtor().

The structure has the following public fields:

bool numeric

A boolean value: true if the this component is numeric; false if this component is alphabetic.

int num

If this component is numeric: this field holds its numeric representation. Otherwise this field holds a meaningless value.

size_t len

The number of characters in the input string that where parsed to form this component.

const char * raw

Pointer to the character in the input string representing the first character of this component. This pointer is valid as long as the input string handed to semver_id_read() is not mutated.

semver_id_t * next

Pointer to the next node in the linked list; NULL if this structure is the last node.

Function: void semver_id_ctor (semver_id_t * idp)

Initialise to empty an already allocated data structure representing a parsed identifier. Every call to this function must be matched by a call to semver_id_dtor(): 

semver_id_t     id;

semver_id_ctor(&id);
{
  /* Do something with "id" here. */
}
semver_id_dtor(&id);
Function: void semver_id_dtor(semver_id_t * idp)

Finalise to empty a data structure representing a parsed identifier; calls to this function must match a previous call to semver_id_ctor() or semver_id_read().

All the resources associated to idp are released, but the structure itself. The following code is correct: 

semver_id_t     id;

semver_id_ctor(&id);
{
  /* Do something with "id" here. */
}
semver_id_dtor(&id);

the following code is also correct: 

semver_id_t *   idp;

idp = malloc(sizeof(semver_id_t));
assert(NULL != idp);
{
  semver_id_ctor(idp);
  {
    /* Do something with "idp" here. */
  }
  semver_id_dtor(idp);
}
free(idp);
