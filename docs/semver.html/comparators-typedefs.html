4.2 Type definitions

All the following symbols are defined in the header file semver.h.

Struct Typedef: semver_comp_t

Type of data structures representing parsed comparators. Every instance of this structure is a node in a simply–linked list. Instances of this structure are initialised with semver_comp_ctor() or semver_comp_read(). Instances of this structure are finalised with semver_comp_dtor().

It has the following public fields:

semver_comp_t * next

Pointer to the next comparator in the linked list; NULL if this instance is the last in the list.

enum semver_op op

The operator specification for this comparator.

semver_t version

The version specification for this comparator.

Function: void semver_comp_ctor (semver_comp_t * compp)

Initialise to empty an already allocated data structure representing a parsed comparator. Every call to this function must be matched by a call to semver_comp_dtor(): 

semver_comp_t   comp;

semver_comp_ctor(&comp);
{
  /* Do something with "comp" here. */
}
semver_comp_dtor(&comp);
Function: void semver_comp_dtor (semver_comp_t * compp)

Finalise to empty a data structure representing a parsed comparator; calls to this function must match a previous call to semver_comp_ctor() or semver_comp_read().

All the resources associated to compp are released, but the structure itself. The following code is correct: 

semver_comp_t   comp;

semver_comp_ctor(&comp);
{
  /* Do something with "comp" here. */
}
semver_comp_dtor(&comp);

the following code is also correct: 

semver_comp_t * compp;

compp = malloc(sizeof(semver_comp_t));
assert(NULL != compp);
{
  semver_comp_ctor(compp);
  {
    /* Do something with "compp" here. */
  }
  semver_comp_dtor(compp);
}
free(compp);
