4.1 Constants definitions

All the following symbols are defined in the header file semver.h.

Enumeration: semver_op

Type of enumeration representing the comparison criteria for comparators.

Enumerated Constant: enum semver_op SEMVER_OP_EQ

Represent the “equality” operator.

Enumerated Constant: enum semver_op SEMVER_OP_LT

Represent the “less than” operator.

Enumerated Constant: enum semver_op SEMVER_OP_LE

Represent the “less than or equal to” operator.

Enumerated Constant: enum semver_op SEMVER_OP_GT

Represent the “greater than” operator.

Enumerated Constant: enum semver_op SEMVER_OP_GE

Represent the “greater than or equal to” operator.
