4.5 Matching with versions

All the following symbols are defined in the header file semver.h.

Function: char semver_pmatch (const semver_t * versionp, const semver_comp_t * compp)

Match a version referenced by versionp against all the comparators in the linked list referenced by compp. The return value is 1 if the version matches; it is 0 otherwise.

Preprocessor Macro: int semver_match (semver_t version, semver_comp_t comp)

Like semver_pmatch(), but accept the version and comparator structures as arguments, rather than pointers to them.

Example: matching a version

The following example matches a version against a standalone comparator: 

static const char     version_str[] = "1.2.3";
static const char     compar_str[]  = ">=2.0.0";
semver_t      version;
semver_comp_t comp;
size_t        version_off = 0, comp_off = 0;
char          rv;

rv = semver_read(&version, \
        version_str, strlen(version_str), &version_off);
if (0 == rv) {
  rv = semver_comp_read(&comp, \
          compar_str, strlen(compar_str), &comp_off);
  if (0 == rv) {
    int       result = (int)semver_match(version, comp);
    printf("matches?=%s\n", (result)? "yes" : "no");
  }
  semver_comp_dtor(&comp);
}
semver_dtor(&version);

the code prints: 

matches?=no
