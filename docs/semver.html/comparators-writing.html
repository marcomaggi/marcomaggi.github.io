4.4 Writing to a string

All the following symbols are defined in the header file semver.h.

Function: int semver_comp_pwrite (const semver_comp_t * compp, char * output_buf, size_t output_len)

Serialise the comparator compp as ASCII string in the memory block referenced by output_buf, which has at least output_len characters available. If the buffer has not enough room to hold the full comparator serialisation: the string is truncated to output_len characters.

Like snprintf(): return the number of characters needed to hold the full serialisation, excluding the trailing null; if the return value is greater than or equal to output_len, the string has been truncated.

Preprocessor Macro: int semver_comp_write (semver_comp_t compp, char * output_buf, size_t output_len)

Like semver_comp_pwrite(), but accept the comparator structure as argument, rather than a pointer to it.

Example: serialising a comparator

The following example serialises a comparator: 

static const char     input_str[] = "<=1.2.3";
size_t        buffer_len = 32;
char          buffer_ptr[buffer_len];
semver_comp_t comp;
size_t        offset = 0;
char          rv;

memset(buffer_ptr, 0, buffer_len);

rv = semver_comp_read(&comp, \
        input_str, strlen(input_str), &offset);
if (0 == rv) {
  size_t      needed_count;
  size_t      actual_count;
  needed_count = (size_t)semver_comp_write(comp, \
                    buffer_ptr, buffer_len);
  if (0 < needed_count) {
    actual_count = (needed_count < buffer_len)? \
                       needed_count : buffer_len;
    printf("len=%lu, actual_count=%lu, comp=",
           strlen(input_str), actual_count);
    fwrite(buffer_ptr, sizeof(char), actual_count, stdout);
    printf("\n");
  }
}
semver_comp_dtor(&comp);

the code prints: 

len=7, actual_count=7, comp=<=1.2.3
