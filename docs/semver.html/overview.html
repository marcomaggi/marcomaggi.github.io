1 Overview of the package

This document describes version 0.0.1-forked.1 of Libsv, a C language library implementing semantic versioning for the C language.

Libsv installs the single header file semver.h. All the function names in the API are prefixed with semver_; all the constant names are prefixed with SEMVER_; all the type names are prefixed with semver_ and suffixed with _t.
