6 Parsing numeric components

In the context of Libsv’s parsing functions, numeric components are ASCII strings holding: numeric characters (according to the standard function isdigit()); the standalone character ‘x’; the standalone character ‘X’; the standalone character ‘*’.

Usually we do not need to use the numeric components parser directly.

All the following symbols are defined in the header file semver.h.

Function: char semver_num_read (int * nump, const char * input_buf, size_t input_len, size_t * input_offp)

Parse an input string in ASCII format which is meant to hold an integer number. When successful: the return value is 0. If an error occurs: the return value is 1.

The results of parsing are stored in the variable referenced by nump.

The input string is referenced by input_buf; the length of the input string is input_len characters. The argument input_offp is both an input and output parameter:

  • Upon entering the function: the offset of the first character to parse in input_buf is acquired from the variable referenced by input_offp. If the value in the referenced variable is greater than input_len: the function returns unsuccessfully.
  • When the function returns successfully: the offset of the first non–parsed character is stored in the variable referenced by input_offp; further parsing operations can use such value as input.
  • When the function returns unsuccessfully: the value stored in the variable is invalid.

If the string is empty (input_len is zero): the function returns successfully and stores zero in the variable referenced by nump.

If the input string contains ‘x’, ‘X’ or ‘*’: the function returns successfully and stores the value SEMVER_NUM_X in the variable referenced by nump.

The following example code parses a numeric component: 

static const char     input_buf[] = "123";
int           num;
size_t        input_off = 0;
char          rv;

rv = semver_num_read(&num, \
       input_buf, strlen(input_buf), &input_off);
if (0 == rv) {
  printf("num=%d\n", num);
}
Preprocessor Symbol: int SEMVER_NUM_X

Special value used to represent a numeric component specified as ‘x’, ‘X’ or ‘*’.

Function: char semver_num_comp (int A, int B)

Compare the given integers. Return -1 if A is less than B; return 0 if A is equal to B; return +1 if A is greater than B.

The return values of this function are invalid if one of the arguments is SEMVER_NUM_X.
