5.4 Matching ranges and versions

All the following symbols are defined in the header file semver.h.

Function: char semver_prmatch (const semver_t * versionp, const semver_range_t * rangep)

Match a version referenced by versionp against the comparators in the range referenced by rangep. The return value is 1 if the version matches; it is 0 otherwise.

Preprocessor Macro: int semver_rmatch (semver_t version, semver_range_t range)

Like semver_prmatch(), but accept the version and range structures as arguments, rather than pointers to them.

Example: matching a version

The following example matches a version against a range: 

static const char     version_str[] = "1.4.6";
static const char     range_str[]  = "1.2.7 || >=1.2.9 <2.0.0";
semver_t              version;
semver_range_t        range;
size_t                version_off = 0, range_off = 0;
char                  rv;

rv = semver_read(&version, \
        version_str, strlen(version_str), &version_off);
if (0 == rv) {
  rv = semver_range_read(&range, \
          range_str, strlen(range_str), &range_off);
  if (0 == rv) {
    int       result = (int)semver_rmatch(version, range);
    printf("matches?=%s\n", (result)? "yes" : "no");
  }
  semver_range_dtor(&range);
}
semver_dtor(&version);

the code prints: 

matches?=yes
