7 Parsing identifiers

In the context of Libsv’s parsing functions, identifiers are ASCII strings holding: alphanumeric characters (according to the standard function isalnum()); dash characters; dot characters. Dot characters are interpreted as separators between components.

The identifiers API does not duplicate the input strings.

Usually we do not need to use the identifiers parser directly.

• identifiers typedefs:  Type definitions.
• identifiers reading:  Parsing from a string.
• identifiers writing:  Writing to a string.
• identifiers comparing:  Identifiers comparison.
