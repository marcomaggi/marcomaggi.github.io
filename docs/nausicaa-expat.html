1 Overview of the package

Nausicaa/Expat is a distribution of libraries for R6RS Scheme implementations; it provides bindings for the Expat C language library, an XML parser library. It is built upon Nausicaa/Scheme, making use of its foreign–functions interface and it supports the Scheme implementations Ikarus, Mosh and Ypsilon.

Nausicaa/Expat is meant to use Expat in its UTF8 mode, using the characters type XML_Char defined as char.

(foreign xml expat shared-object)
Exports the single binding expat-shared-object which references the foreign shared library.
(foreign xml expat sizeof)
Exports bindings to constant values and data structures accessors.
(foreign xml expat conditions)
Exports condition object type definitions.
(foreign xml expat platform)
Exports one binding for each foreign function; the Scheme functions have the same name of the C functions, so XML_ParserCreate() is available as XML_ParserCreate. This library makes the original Expat documentation relevant for this package. Expat.
(foreign xml expat primitives)
Exports one binding for each foreign function, implementing marshaling of arguments and return values. The Scheme functions have the same name of the C functions, but with: lower case, underscores converted to dashes, camel case words converted to dash–separated words; so XML_ParserCreate() is available as xml-parser-create.

This document describes the primitives API; only the functions implementing marshaling of values are documented: if a function is not documented, it means that it is reexported “as is” from the platform library.

(foreign xml expat)
Compound library exporting the high–level API; it reexports bindings from the primitives, conditions and sizeof libraries and described in this document.
(foreign xml expat compensated)
Exports bindings to constructor functions making use of the (compensations) library from Nausicaa/Scheme. Compensation stacks.

3.1 Introduction to simple parsing

Basically to parse a XML document contained in a Scheme string we can do: 

     (import (rnrs)
       (compensations)
       (foreign xml expat)
       (foreign xml expat compensated))
     
     (define document ...)
     
     (with-compensations
       (let ((parser (xml-parser-create/c pointer-null))
             (stack '()))
     
         (define (start-callback element attributes)
           (set! stack (cons (cons element attributes) stack)))
     
         (define (end-callback element)
           (set! stack (cons element stack)))
     
         (define (data-callback data)
           (set! stack (cons data stack)))
     
         (xml-set-element-handler parser
           (make-xml-start-callback start-callback)
           (make-xml-end-callback   end-callback))
         (xml-set-character-data-handler parser
           (make-xml-data-callback data-callback))
     
         (xml-parse/string parser document #t)
         (reverse stack)))

which, for example, will convert the following string: 

     <stuff>
      <thing>
       <alpha a1="1" a2="2">one</alpha>
       <beta>two</beta>
      </thing>
      <thing>
       <alpha>123</alpha>
       <beta>456</beta>
      </thing>
     </stuff>

to the following Scheme list (bound to stack in the example): 

     ((stuff)
        (thing)
          (alpha (a1 . "1") (a2 . "2")) "one" alpha
          (beta) "two" beta
        thing
        (thing)
          (alpha) "123" alpha
          (beta) "456" beta
        thing
      stuff)

            (make-c-callback*

Build and return a Scheme callback object of type XML_ExternalEntityRefHandler. It is implemented as follows: 

          (define (make-xml-external-entity-ref-callback scheme-function)
            (make-c-callback*
             int
             (lambda (parser context base system-id public-id)
               (let-syntax ((or-false (syntax-rules ()
                                        ((_ ?arg)
                                         (if (pointer-null? ?arg)
                                             #f
                                           (cstring->string ?arg))))))
                 (if (scheme-function parser (or-false context)
                                      (or-false base) (or-false public-id)
                                      (cstring->string system-id))
                     XML_STATUS_OK
                   XML_STATUS_ERROR)))
             (pointer pointer pointer pointer pointer)))
