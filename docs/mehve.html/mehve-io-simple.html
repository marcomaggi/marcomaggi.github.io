2.3.1 Simple input/output generic functions

The bindings display and write exported by (nausicaa mehve language input-output), and reexported by (nausicaa mehve), generalise the corresponding functions from (rnrs io simple (6)).

— Generic Function: display-1 obj
— Generic Function: display-2 obj port

Invoked when display is applied to one or two arguments.

— Generic Function: write-1 obj
— Generic Function: write-2 obj port

Invoked when write is applied to one or two arguments.

— Method: display-1 (obj <top>)
— Method: display-2 (obj <top>) (port <port>)

Apply to the arguments the function display exported by (rnrs io simple (6)), display.

— Method: write-1 (obj <top>)
— Method: write-2 (obj <top>) (port <port>)

Apply to the arguments the function write exported by (rnrs io simple (6)), write.
