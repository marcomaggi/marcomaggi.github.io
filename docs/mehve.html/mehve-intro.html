2.1 Using the language

The library (nausicaa mehve) exports the language for users; other component libraries in the hierarchy (nausicaa mehve language ---) export multimethod bindings that can be used to extend the generic functions.

Before making use of the facilities of Mehve, we must call the function initialise-mehve.

— Function: initialise-mehve

Initialise the multimethods and the component libraries; we can call this function any number of times, only the first time the library is initialised.

Calling this function is needed because, according to R6RS, a compliant Scheme implementation might instantiate a library only when an exported binding is actually accessed; this behaviour does not play well with multimethods.
