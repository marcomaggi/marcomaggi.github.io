2.4.5 Infix to prefix syntax

The infix syntax exported by (nausicaa mehve language infix), and reexported by (nausicaa mehve), redefines the syntax exported by (nausicaa) to support generic function operators.

All the bindings of infix from (nausicaa) are also supported by infix from (nausicaa mehve); the generic functions used as operators have the same precedence of the corresponding simple functions.
