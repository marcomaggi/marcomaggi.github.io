2.4.3 Parts generic functions

The auxiliary library (nausicaa mehve language numerics parts) defines the generic functions behind the parts operations; we need to import it only if we desire to add specialised methods to parts functions. The parts functions are the following: 

     numerator denominator rationalize sign
     floor ceiling truncate round
     real-part imag-part magnitude angle
     make-rectangular make-polar complex-conjugate

As example we look at how to specialise the functions make-rectangular and make-polar to operate on lists of items: 

     #!r6rs
     (import (nausicaa mehve)
       (prefix (nausicaa mehve language numerics parts)
               mehve.))
     
     (define-method
         (mehve.make-rectangular (R <vector>) (I <vector>))
       (R map make-rectangular I))
     
     (define-method
         (mehve.make-rectangular (R <spine>) (I <spine>))
       (R map make-rectangular I))
     
     (define-method
         (mehve.make-polar (M <vector>) (A <vector>))
       (M map make-polar A))
     
     (define-method
         (mehve.make-polar (M <spine>) (A <spine>))
       (M map make-polar A))
     
     (make-rectangular '(1 2 3) '(4 5 6))
     ⇒ (1+4i 2+5i 3+6i)
     
     (make-polar '(1 2 3) '(4 5 6))
     ⇒ (-0.6536436208636119-0.7568024953079282i
         0.5673243709264525-1.917848549326277i
         2.880510859951098-0.8382464945967776i)
Rational numbers generic functions
— Generic Function: numerator A
— Method: numerator (A <fixnum>)
— Method: numerator (A <bignum>)
— Method: numerator (A <ratnum>)
— Method: numerator (A <flonum>)
— Method: numerator (A <integer>)
— Method: numerator (A <real>)

Return the numerator of the argument. Use the function numerator; when the argument is an integer: return the argument itself.

— Generic Function: denominator A
— Method: denominator (A <fixnum>)
— Method: denominator (A <bignum>)
— Method: denominator (A <ratnum>)
— Method: denominator (A <flonum>)
— Method: denominator (A <integer>)
— Method: denominator (A <real>)

Return the denominator of the argument. Use the function denominator; when the argument is an integer: return the fixum 1.

— Generic Function: rationalize A B
— Method: rationalize (A <real>) (B <real>)

Return a rational number representation of A which differs from it by less than B. Use the function rationalize.

Rounding generic functions
— Generic Function: floor A
— Method: floor (A <fixnum>)
— Method: floor (A <bignum>)
— Method: floor (A <flonum>)
— Method: floor (A <real>)

Return an object representing the largest integer not larger than A. Use the function flfloor or floor as appropriate for the argument; when the argument is an exact integer: return the argument itself.

— Generic Function: ceiling A
— Method: ceiling (A <fixnum>)
— Method: ceiling (A <bignum>)
— Method: ceiling (A <flonum>)
— Method: ceiling (A <real>)

Return an object representing the smaller integer not larger than A. Use the function flceiling or ceiling as appropriate for the argument; when the argument is an exact integer: return the argument itself.

— Generic Function: truncate A
— Method: truncate (A <fixnum>)
— Method: truncate (A <bignum>)
— Method: truncate (A <flonum>)
— Method: truncate (A <real>)

Return the integer object closest to A whose absolute value is not larger than the absolute value of A. Use the function fltruncate or truncate as appropriate for the argument; when the argument is an exact integer: return the argument itself.

— Generic Function: round A
— Method: round (A <fixnum>)
— Method: round (A <bignum>)
— Method: round (A <flonum>)
— Method: round (A <real>)

Return the closest integer object to A. Use the function flround or round as appropriate for the argument; when the argument is an exact integer: return the argument itself.

— Generic Function: sign A
— Method: sign (A <fixnum>)
— Method: sign (A <bignum>)
— Method: sign (A <flonum>)
— Method: sign (A <ratnum>)
— Method: sign (A <real>)

Return an object representing the sign of the argument. Use the function sign or its type–specific variants as appropriate for the argument.
