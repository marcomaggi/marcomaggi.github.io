1 Overview of the package

This document describes Mehve, a collection of libraries for Vicare Scheme version 0.3d7, implementing the language (nausicaa mehve) on top of (nausicaa), The language (nausicaa).

(nausicaa mehve) defines the same language of (nausicaa), but it redefines some bindings to functions as bindings to multimethods and so they can be customised with specialised methods, Multimethods dispatching.

Multimethods are significantly slower than functions, so, while more flexible, Mehve is not for every application.

Before making use of the facilities of Mehve, we must call the function initialise-mehve.
