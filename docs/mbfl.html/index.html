Marco’s Bash Functions Libraries ¶

This document describes version 3.0.0-devel.8 of MBFL, a collection of libraries of functions for the GNU Bash shell. It requires GNU Bash at least version 4.3; it require GNU m4 (other m4 implementations will not do). The package is distributed under the terms of the GNU Lesser General Public License (LGPL).

The latest release can be downloaded from:

https://bitbucket.org/marcomaggi/mbfl/downloads

development takes place at:

https://github.com/marcomaggi/mbfl

and as backup at:

https://bitbucket.org/marcomaggi/mbfl
