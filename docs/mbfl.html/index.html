Marco’s Bash Functions Library

This document describes version 3.0.0-devel.0 of MBFL, a library of functions for the GNU Bash shell. It requires Bash at least version 4.3. The package is distributed under the terms of the GNU Lesser General Public License (LGPL).

The latest release can be downloaded from:

https://bitbucket.org/marcomaggi/mbfl/downloads

development takes place at:

https://github.com/marcomaggi/mbfl
