Function: mbfl_default_classes_are_same_or_superclass_and_subclass MAYBE_PARENT_STDCLS MAYBE_CHILD_STDCLS ¶

Return true if either:

  • MAYBE_PARENT_STDCLS and MAYBE_CHILD_STDCLS are equal;
  • MAYBE_PARENT_STDCLS is a class and MAYBE_CHILD_STDCLS is a subclass of MAYBE_PARENT_STDCLS; otherwise return false.
