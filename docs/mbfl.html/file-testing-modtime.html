20.13.5 Testing modification times ¶

Function: mbfl_file_modification_time_is_newer PATHNAME1 PATHNAME2 ¶

Return true if PATHNAME1 has newer modification time than PATHNAME2.

Function: mbfl_file_modification_time_is_older PATHNAME1 PATHNAME2 ¶

Return true if PATHNAME1 has older modification time than PATHNAME2.
