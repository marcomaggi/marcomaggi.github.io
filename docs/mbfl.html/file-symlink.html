20.9 Creating links and symbolic links ¶

Function: mbfl_file_enable_link ¶
Function: mbfl_file_enable_symlink ¶

Declare the commands required to create links and symbolic links between files. Declaring the intention to use a program

The programs are: ln.

Function: mbfl_file_symlink ORIGINAL_NAME SYMLINK_NAME OPT … ¶

Create a symbolic link from ORIGINAL_NAME to LINK_NAME. Insert in the command line of ln the given optional arguments.

Function: mbfl_file_link ORIGINAL_NAME LINK_NAME OPT … ¶

Create a link from ORIGINAL_NAME to LINK_NAME. Insert in the command line of ln the given optional arguments.

Function: mbfl_exec_ln ARG … ¶

Execute the program ln using the given optional command line arguments.
