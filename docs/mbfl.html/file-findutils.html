20.17 Interfaces to the GNU Findutils ¶

Function: mbfl_exec_find ARG … ¶

Execute the program find using the given optional command line arguments.

The pathname of the executable find is preselected at package–configuration time; mbfl_PROGRAM_FIND for details.

The pathname of the executable locate is preselected at package–configuration time; mbfl_PROGRAM_LOCATE for details.
