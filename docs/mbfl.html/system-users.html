27.3 Gathering users data ¶

At package configuration time we can specify which id and whoami executables to use by setting variables on the command line of the configuration script configure: 

configure ID=/path/to/id WHOAMI=/pash/to/whoami
• system users whoami:  Determining the effective user name.
• system users id:  Determining user’s IDs.
