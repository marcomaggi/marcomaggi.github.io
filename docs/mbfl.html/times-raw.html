25.1 Raw call to date ¶

Function: mbfl_exec_date ARG … ¶
Function: mbfl_exec_date_var RV ARG … ¶

Execute date with the optional arguments appended to the command line. Preselected program pathnames.

Function: mbfl_exec_date_format FORMAT OPT … ¶
Function: mbfl_exec_date_format_var RV FORMAT OPT … ¶

Execute date with the mandatory FORMAT as first argument. Additional, optional, arguments are appended to the command line.
