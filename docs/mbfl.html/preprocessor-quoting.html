2.2 Quotes and quoted macro names ¶

The m4 preprocessor is invoked by mbflpp.bash with the --prefix-builtins option; so all the m4 built–in macros and directives are available prefixed with m4_.

The quoting for the preprocessor is changed with: 

m4_changequote(`[[[', `]]]')
