21.2 Closing file descriptors ¶

Function: mbfl_fd_close FD ¶

Close the file descriptor FD.

If the predefined options --test or --show-program are enabled: print a line on the standard error channel describing the operation. If the predefined option --test is enabled: the file descriptor is still closed. Predefined options.
