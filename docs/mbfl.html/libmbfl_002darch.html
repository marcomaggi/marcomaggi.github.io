Appendix D Archiving and compressing files ¶

This section documents the library libmbfl-arch.bash, an interface to the archiving and compressing facilities implemented by the most common external programs. We can load this library at run–time with: 

mbfl_load_library(/path/to/libmbfl-arch.bash)

for details Libraries loading and embedding. Notice that the preprocessor option --print-libmbfl-arch will print the file pathname of the installed library, Invoking the preprocessor.

• libmbfl-arch tar:  Manipulating tar archives.
• libmbfl-arch compress:  Compressing files.
