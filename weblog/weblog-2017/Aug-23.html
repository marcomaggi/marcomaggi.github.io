More Scheme packages for Chez

Posted on Wed Aug 23, 2017

In the last days I have repackaged some Scheme code for Vicare, to make it available for Chez Scheme; compiling under Chez is so faster than Vicare that it is a pleasure. I have written only a small bit of new Scheme. I miss a lot of features I added to Vicare, though.

I know that it is a totally useless hobby. The thing is: writing Scheme is more fun to me than writing with every other language; writing code is more fun than doing a lot of other things. Damn!
