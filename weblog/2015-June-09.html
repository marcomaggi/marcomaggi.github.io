Industria

Industria is a collection of libraries for R6RS compliant Scheme implementations. I have forked the repository and added a branch vicare-scheme to organise the package under Vicare's infrastructure. The current head of the vicare-scheme branch should work with the current head of Vicare's master branch (after a fight to fix errors in Vicare Scheme).

There is also a Bitbucket mirror for this project.

The original documentation still references a website that appears to not exist anymore (or it is offline for some reason). However the original project is still on Github.
