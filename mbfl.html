Marco’s Bash Functions Library ¶

mbfl is a collection of libraries for the gnu Bash shell. It requires Bash at least version 4.3. It is an attempt to make gnu Bash a viable solution for medium sized scripts. A set of modules implementing common operations and a script template are offered by this package and the author has used them with success in implementing non-small scripts.

Development of this package takes place at:

https://github.com/marcomaggi/bash
