Abstract

This is the home page of Nausicaa/SharpBang, a distribution of libraries for R6RS Scheme implementations built upon the Nausicaa/Scheme package. It provides a simple laboratory to play with client languages by converting their source code to a server language, which is Scheme: either (rnrs (6)) or another derived from it.

The libraries support the Scheme implementations Mosh, Petite Chez, Racket and Vicare. GNU+Linux systems are the only platform supported by the packaging infrastructure.

Introduction

SILex stands for Scheme Implementation of Lex; the library (nausicaa silex) is a port to R6RS Scheme of SILex version 1.0 by Danny Dubé.

Lalr-scm is a LALR(1) parser tables generator by Dominique Boucher; much of the code is a direct translation from C to Scheme of the GNU Bison sources; the library (nausicaa lalr) is a port to R6RS Scheme of Lalr-scm.

The idea of SharpBang is to define a client language, write a lexer for it with the library (nausicaa silex) and a parser for it with the library (nausicaa lalr), then add a conversion tool to a server Scheme language.

SharpBang comes with an example language embedded, Simple, which shows the mechanism and provides a template for new language definitions.

Everybody is writing language conversion tools these days, but this package is especially inspired by Racket (but no code comes from it).
